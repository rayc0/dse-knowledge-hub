# 内地生在港饮食生活指南

对很多内地学生来说，来香港读大学最大的文化冲击不是语言、不是课业，而是**饮食**。粤式菜系、英式下午茶、殖民地遗留的茶餐厅文化、东南亚移民口味、还有香港独有的"快餐厅"——这些都是内地食堂文化完全没有的东西。很多新生到埗第一周瘦了 2 公斤不是因为忙，而是因为"真的不知道吃什么"。

这份指南就是写给即将或刚来港的内地学生：从大学饭堂到茶餐厅入门、从省钱自煮到家乡菜地图，让你在香港吃得饱、吃得省、吃得健康。

## 一、先理解香港饮食文化的几个关键

### 1. 主食偏重米饭和面食
香港饮食以粤菜为基础，米饭是主粮。除了米饭，面食（云吞面、车仔面、米粉）也极为普及。北方面食（馒头、饺子、刀削面、拉面）并非没有，但需要特别寻找。

### 2. 口味偏清淡但油量不低
粤菜讲究"清、鲜、爽、嫩、滑"，口味相对清淡，但烹饪时用油量并不低。**内地北方同学常常觉得"没味道"**，其实是因为没调酱。香港餐桌上常配酱油、辣椒酱、豉油、甜酱、咖喱酱，都可以自己加。

### 3. 餐具和用餐节奏
- 茶餐厅用餐节奏极快，坐下→点餐→上餐→买单通常 20 分钟内完成。
- 要主动"搭台"（与陌生人合桌），这是香港茶餐厅标准做法，不是对你不尊重。
- 筷子不是万能的——吃西式套餐时会配刀叉。

### 4. 份量
- 港式饭餐份量比内地普遍偏小。一份饭大约 250–300 克米饭 + 100–150 克菜肉。**北方男生可能需要点"大碗"或加饭**。
- 饭堂 set 餐一般含一个汤 + 一个主菜 + 米饭 + 茶，是最经济的选择。

### 5. 价格区间（2026 年参考）
| 类型 | 价格（港币） |
|---|---|
| 大学饭堂 set 餐 | $28–45 |
| 茶餐厅普通餐 | $45–75 |
| 连锁快餐（大家乐、大快活、美心）| $45–65 |
| 中档餐厅一人餐 | $80–150 |
| 旅游区/IFC/中环 | $120–250 |
| 自煮一餐（自己买菜） | $15–25 |

## 二、八大院校饭堂实况

### 香港大学（HKU）
- **龙华堂（Lung Wah Hall Canteen）**：最大的本部饭堂，分 3 楼层，中餐、西餐、日韩、印度咖喱、素食皆有。set 餐 $32–42。早餐尤其丰富。
- **学人书店旁饭堂（Starbucks 对面）**：较精致，适合面试日或约朋友。
- **Robert Black College**：教授和研究生常光顾，菜品质量较高。

### 香港中文大学（CUHK）
- **大学饭堂（University Canteen）**：位于本部，中西并重。晚饭时段尤其热闹。
- **Cafe Colabo**：咖啡 + 轻食，适合赶时间的学生。
- **各个书院自己的饭堂**：每个书院（新亚、崇基、联合、逸夫、晨兴、善衡、敬文、伍宜孙、和声）都有自己的饭堂，风格各异。**新亚和崇基的午饭最具中式家常味**，善衡的素食最多样。

### 香港科技大学（HKUST）
- **LG1/LG5 Canteen**：规模大、选择多。中西兼具，日式拉面、韩国菜特别出色。
- **麦当劳**：位于 Lau Ming Wai Hall 附近。
- **Starbucks**：UST 本部有两间。

### 香港理工大学（PolyU）
- **Global Kitchen**：学生饭堂主力，$35–55 一餐。
- **V 饭堂（V Canteen）**：较新，适合聚餐。
- **红磡站 MTR 附近**：出校门即有大量茶餐厅、连锁快餐。

### 香港城市大学（CityU）
- **Phase 2 Chinese Restaurant**：早茶和点心特别受欢迎。
- **City U Sky Canteen**：位置高景观好，气氛舒适。
- **Chan's 潮州**：潮州菜饭堂，特色菜值得尝试。

### 香港浸会大学（HKBU）
- **Kai Tak Campus Canteen**：中西合璧。
- **AAB Canteen**：素食和健康饮食选择多。

### 教育大学（EdUHK）
- **大埔校园饭堂**：规模较小但菜品用心。

### 岭南大学（LingU）
- **屯门校园饭堂**：中式为主，早午晚都开。

**给内地新生的建议**：开学第一周尽量把你所在书院/校园的饭堂全部吃一遍（至少午餐 + 晚餐各一次），你会很快找到自己最喜欢的几家。饭堂比外面便宜 30–50%，长期应该是主力餐点来源。

## 三、茶餐厅入门——每个内地生必学

茶餐厅是香港饮食文化的灵魂。不懂茶餐厅就算不上真正体验过香港生活。

### 茶餐厅基本菜单

**早餐（约 $38–50，早上 7–11 点）：**
- 火腿通粉 + 火腿蛋三文治 + 奶茶/咖啡
- 牛油多士 + 煎双蛋 + 火腿肠 + 咖啡
- 牛肉通粉
- 芝士公仔面

**午餐快餐（约 $55–75，11 点–3 点）：**
- 西炒饭
- 干炒牛河
- 豉椒炒蜆炒饭
- 椒盐猪扒饭
- 咖喱鸡饭
- 焗猪排饭
- 粟米肉粒饭
- 豉油鸡饭

**下午茶（约 $35–50，3 点–5 点）：**
- 奶油多士 + 丝袜奶茶
- 菠萝油 + 鸳鸯
- 蛋挞 + 冻柠茶

**晚饭（约 $60–100）：**
- 小菜 + 饭（可以跟同学一起点 3–4 个小菜分食）
- 煲仔饭（冬天特别受欢迎）
- 海南鸡饭
- 烧味饭（叉烧、烧鸭、烧鹅、白切鸡、油鸡）

### 点餐礼节
- **一坐下就要立刻看菜单**，服务员会很快过来。
- **点餐时直接报菜名 + 饮品**，例如"咖喱牛腩饭、冻柠茶少冰"。
- **一般默认含饮品**（快餐套餐），需要确认是热的还是冻的、是茶、咖啡还是柠水。
- **结账时自己拿单到收银台付**，不是在桌上付。
- **小费是可选的**，茶餐厅一般不留小费。

### 推荐的连锁茶餐厅（内地生友好）
- **翠华餐厅**：价格略高但稳定，适合新手（$75–120 一餐）
- **澳洲牛奶公司（佐敦）**：炒蛋、炖奶、多士都是招牌
- **檀岛咖啡饼店**：最老牌之一，奶茶极佳
- **兰芳园**：丝袜奶茶发源地
- **金华冰厅**：菠萝油代表作

## 四、内地生最常见的饮食困扰与解决方案

### 困扰一：吃不饱

**原因**：港式餐饭份量偏小，且比较清爽，缺乏饱腹感。

**解决**：
- 在饭堂点"加饭"（通常 $5 加一碗）
- 多点蛋白质（蛋、肉、豆腐）
- 煲仔饭、咖喱饭这类"油饭"饱腹感较强
- 下午备点心（饼干、麦片、水果）防饥饿

### 困扰二：口味不适应

**原因**：粤菜清淡、偏甜、偏海鲜，北方同学不习惯。

**解决**：
- **找家乡菜馆**（见第六节）
- **自煮**——买一个电饭煲、一个单头电磁炉，自己煮家乡口味
- **带调料**——老干妈、东北大酱、山西老陈醋、四川花椒，都可以从淘宝/京东国际寄到宿舍
- **拓展口味**——试一试泰国菜、越南菜、日本菜、韩国菜，可能你会找到新爱好

### 困扰三：吃得贵

**原因**：每天外食 = 每月 $3,000–5,000 港币伙食费。

**解决**：
- **饭堂为主力，每周外出不超过 2 次**
- **学会自煮**——每月可以省 $1,500–2,500
- **超市买菜时看黄昏特价**（百佳、惠康、市集在晚上 7 点后常有肉菜 50% off）
- **avoid 旅游区饮食**（尖沙咀、中环、铜锣湾价格高于住宅区 30–50%）

### 困扰四：没时间吃

**原因**：赶课、赶 project、图书馆闭馆。

**解决**：
- **宿舍备速食**：方便面、急冻饺子、烤麦片、即食粥、营养棒
- **午饭主吃**：在饭堂用 30 分钟好好吃午饭，比晚上回到宿舍再叫外卖便宜
- **外卖 app**：Foodpanda、Deliveroo、KeeTa（美团）都已进入香港，部分商家提供学生优惠

### 困扰五：想念家乡菜

**原因**：想家。

**解决**：
- **约同乡朋友**一起去家乡菜馆（见第六节）
- **自己学两三道家乡菜**：锅包肉、糖醋排骨、麻婆豆腐、番茄炒蛋这些都是入门级
- **周末自己炖一锅汤**（鸡汤、排骨汤）：成本 $50，可以吃 3–4 餐

## 五、自煮省钱完全攻略

自煮是在香港长期生活最大的省钱武器。但要注意：

### 宿舍限制
- 大部分大学宿舍**禁止在房间内使用明火**（电磁炉和电饭煲通常允许，煤气炉和小明火锅禁止）
- **公共厨房**：每个宿舍 floor 通常有公共厨房可用，提供冰箱、微波炉、烤箱、洗碗槽
- **check 自己宿舍的具体规则**

### 必备厨具（预算 $500–1,200）
- 电饭煲（Zojirushi、象印、松下入门款 $300–500）
- 单头电磁炉（$200–400）
- 不粘锅 1 个、小汤锅 1 个
- 刀、砧板、筷子、勺
- 保鲜盒 3–5 个

### 食材采购
- **百佳、惠康、Aeon**：全港连锁超市，价格中等
- **华润万家**：比较接近内地超市，价格较低
- **街市（湿市场）**：新鲜肉菜最便宜的地方，但需要早起和一些广东话
  - 西营盘街市、旺角街市、北角街市、大埔街市是学生常去的
- **Market Place by Jasons**：中高端，不推荐学生常购
- **Don Don Donki（ドンキ）**：日式杂货店，零食、即食便当便宜好吃

### 经济菜谱示例
- **番茄炒蛋 + 米饭**：成本 $10，15 分钟
- **青椒肉丝 + 米饭**：成本 $15，20 分钟
- **麻婆豆腐 + 米饭**：成本 $12，15 分钟
- **红烧肉（小锅）**：成本 $35，分 3 餐
- **鸡汤（整鸡）**：成本 $50，分 4 餐

### 省钱小贴士
- **买批发包装**：百佳、Aeon 的 1 kg 米、1 kg 鸡胸肉都比小包装便宜
- **利用冷冻**：急冻肉类、冷冻蔬菜、急冻饺子、急冻汤圆，便宜且不浪费
- **煮一次吃两天**：周末煮一大锅咖喱、红烧肉、番茄汤，分装保鲜盒冰箱保存
- **和舍友分担**：跟 2–3 位舍友合资做饭分食，每人成本更低

## 七、健康饮食建议

内地生来港前 3 个月是饮食习惯调整期，常见问题和应对：

### 常见问题
1. **上火**（因为香港气候潮湿、饮食偏煎炒）
2. **便秘**（纤维摄入下降）
3. **体重变化**（饮食结构改变）
4. **胃部不适**（油腻或辛辣过多）
5. **营养不均衡**（蛋白质、蔬菜、水果、碳水、脂肪比例失衡）

### 应对措施
- **每天吃至少 2 份蔬菜 + 1 份水果**
- **煲汤**：广东人煲汤有其道理，冬瓜汤、苦瓜汤、罗汉果茶都是降火好物
- **少吃油炸**：烧卖、炸云吞、猪扒都是高油，不适合天天吃
- **多走路**：香港是走路城市，每天步行 30 分钟是基本
- **定期自测体重**：前 3 个月每周测一次

### 何时就医
- 连续 3 天腹泻或便秘
- 不明原因体重下降 > 2 kg
- 持续性胃痛
- 过敏反应（皮疹、喉咙肿）

大学医务室通常免费，不要等到严重了才去。

## 八、几条给新生的金句

1. **饭堂是你最好的朋友**——质优、价廉、方便。
2. **茶餐厅是你最好的老师**——学会茶餐厅文化你就学会了一半港式生活。
3. **自煮是你最好的投资**——一个月能省 $1,500–2,500。
4. **家乡菜是你最好的慰藉**——但不要每天吃（贵且不健康）。
5. **尝试新菜系**——泰、越、日、韩、印度，都是宝藏。

香港是一个饮食天堂，在任何角落、任何时段、任何预算都能找到吃的。内地生来港学习的 3–4 年里，饮食不只是"吃饱"的问题，而是认识这个城市、结识朋友、照顾自己的一条重要线索。

吃得聪明，吃得开心，学得安心。祝每一位来港的内地学生都能在这座城市找到自己的味道。
