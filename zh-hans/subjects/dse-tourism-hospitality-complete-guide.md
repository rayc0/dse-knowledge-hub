# DSE旅游与款待科完全指南——内地生视角

DSE旅游与款待科（Tourism and Hospitality Studies，THS）是香港高中应用学习（Applied Learning）方向的选修科之一，聚焦于旅游业与酒店业的理论知识与实践应用。对于内地学生而言，香港作为全球顶级旅游目的地，这门科目既提供了深度了解香港服务业的窗口，也是通往PolyU酒店管理等顶尖课程的重要路径。

## 一、考试结构概览

| 组成部分 | 占分比例 | 时间 | 形式 |
|---|---|---|---|
| Paper 1：资料回应题（Data Response） | 约40% | 1小时30分钟 | 阅读资料后作答短答及分析题 |
| Paper 2：论述题（Essay Questions） | 约60% | 2小时 | 选答论述题，深度分析 |

> 注意：THS没有独立的校本评核（SBA）组成，成绩完全由公开考试决定，对私人候考人相对友好。

## 二、Paper 1：资料回应题详解

### 2.1 题型特点
- 提供旅游/酒店相关的图表、统计数据、新闻摘录等资料
- 考生须解读资料，并结合所学知识作答
- 题目层次从描述（describe）到分析（analyze）到评价（evaluate）递进

### 2.2 常见资料类型
- 访港旅客统计图（来源地、消费模式、住宿类型）
- 酒店入住率数据比较
- 旅游业危机新闻摘录（如疫情、社会事件影响）
- 旅游目的地SWOT分析案例

### 2.3 答题技巧
- 资料引用：必须具体引用资料中的数字/信息，不能只说"资料显示"
- 对比分析：多数题目要求对比两组数据，注意找出差异的原因而非仅描述差异
- 专业术语：旅游乘数效应、可持续旅游、旅客停留日数等术语需准确使用

## 三、Paper 2：论述题详解

### 3.1 题型结构
- 通常设4—5题，考生选答2—3题
- 每题约20—25分，需要800—1200字篇幅
- 要求结构完整：引言→论点→证据→分析→结论

### 3.2 热门论述题主题（历年高频）

| 主题 | 典型题目方向 |
|---|---|
| 香港旅游业挑战 | 讨论影响访港旅客数量的因素及应对策略 |
| 可持续旅游 | 旅游开发与环境保护的矛盾与平衡 |
| 酒店业服务质量 | 如何提升客户满意度及忠诚度 |
| 危机管理 | 旅游业应对突发事件（疫情/自然灾害）的策略 |
| 科技对旅游的影响 | 数字化转型如何改变旅客行为与行业运营 |
| 大湾区旅游合作 | 港澳粤旅游一体化的机遇与挑战 |

## 四、核心课程内容

### 4.1 香港旅游业

| 知识点 | 重点内容 |
|---|---|
| 旅游统计 | 访港旅客来源地构成、消费模式、住宿偏好 |
| 旅游吸引力 | 香港的自然、文化、购物、美食资源 |
| 旅游基础设施 | 机场、港口、酒店分级制度、旅游景点管理 |
| 旅游推广 | 香港旅游发展局（HKTB）的角色与推广策略 |
| 旅游与经济 | 旅游乘数效应、旅游业对GDP和就业的贡献 |
| 旅游挑战 | 过度旅游（Over-tourism）、旅客结构变化、竞争目的地 |

### 4.2 酒店管理

| 知识点 | 重点内容 |
|---|---|
| 酒店分类 | 星级制度、品牌连锁vs独立酒店、青年旅舍 |
| 运营部门 | 前厅部、客房部、餐饮部、销售部的职能 |
| 收益管理 | 动态定价、入住率与平均房价（ADR）、RevPAR |
| 服务质量 | SERVQUAL模型、客人投诉处理、忠诚度计划 |
| 可持续发展 | 绿色酒店认证、能源节约、减废措施 |

### 4.3 旅游规划与目的地管理

| 知识点 | 重点内容 |
|---|---|
| 旅游规划理论 | 旅游生命周期（Butler Model）：探索→发展→巩固→停滞→衰退/复兴 |
| 目的地竞争力 | Porter竞争优势理论在旅游中的应用 |
| 旅游产品开发 | 文化旅游、生态旅游、会展旅游（MICE） |
| 旅游影响 | 经济影响（乘数效应）、社会文化影响、环境影响 |

### 4.4 客户服务

| 知识点 | 重点内容 |
|---|---|
| 服务特性 | 无形性、不可分离性、易逝性、异质性（IIHV） |
| 服务失误与恢复 | 服务补救策略、退款vs升级vs道歉 |
| 跨文化服务 | 不同文化背景旅客的服务需求差异 |
| 科技与服务 | 自助入住、AI聊天机器人、OTA平台（Booking.com等） |

## 五、选修单元

THS设有以下选修单元，考生可在指定选修题中作答：

| 选修单元 | 内容重点 |
|---|---|
| 旅游及款待业的当代议题 | 共享经济（Airbnb）、过度旅游、旅游伦理 |
| 会展旅游（MICE） | 会议、奖励旅游、展览、活动的策划与管理 |
| 生态旅游与可持续旅游 | 生态旅游认证、社区参与、碳足迹管理 |

## 六、与商科/BAFS的比较

| 维度 | THS（旅游与款待） | BAFS（企业、会计及财务概论） |
|---|---|---|
| 行业聚焦 | 旅游/酒店行业专项 | 通用商业知识 |
| 财务内容 | 较少（收益管理基础） | 较多（会计、财务报表） |
| 实践应用 | 强（香港旅游案例）  | 中（商业案例分析） |
| 升学适配 | 主要匹配酒店/旅游相关课程 | 广泛适用于商科课程 |
| 大学认可度 | 中等（专业课程认可） | 较高（商科通用） |
| 内地生熟悉度 | 部分熟悉旅游业，香港本地知识需补充 | 内容与内地商科类似，较易上手 |

## 八、内地生备考特别建议

### 8.1 香港本地旅游知识补充
内地生最大的知识盲区是**香港本地旅游业的详细知识**：
- 了解HKTB（香港旅游发展局）的主要推广计划和最新旅客数据
- 熟悉香港主要旅游区域：尖沙咀、铜锣湾、西九、大屿山、香港迪士尼/海洋公园
- 掌握香港近年旅游业挑战（2019年社会事件、疫情影响、旅客结构转变）

### 8.2 大湾区旅游议题
内地生的优势：对大湾区发展政策和内地旅游市场非常熟悉，在大湾区旅游合作议题上可提供深度分析。

### 8.3 英文答题准备
THS虽允许中文作答，但专业词汇需要双语掌握：

| 英文术语 | 中文翻译 |
|---|---|
| RevPAR | 每间可供出售客房收入 |
| Yield Management | 收益管理 |
| Overtourism | 过度旅游 |
| MICE | 会议、奖励旅游、展览、活动 |
| Butler Model | 旅游区域演化周期模型 |
| Multiplier Effect | 旅游乘数效应 |

## 九、Level 5冲刺策略

### 9.1 分数目标

| 等级 | 备考方向 |
|---|---|
| Level 5 | Paper 1资料引用精准，Paper 2论述结构完整、案例丰富 |
| Level 4 | 基本概念正确，分析稍欠深度 |
| Level 3 | 答案切题，但缺乏具体论据 |

### 9.2 高分策略
1. **案例库建设**：整理10—15个香港旅游业经典案例（疫情影响、过度旅游、大湾区合作等），每题引用2—3个
2. **理论框架**：熟练使用Butler生命周期、SERVQUAL、Porter竞争优势等框架
3. **数据引用**：记忆近3年香港访客统计核心数字（来源地占比、消费总额等）
4. **Paper 2选题策略**：优先选择自己案例最充分的题目，避免在不熟悉主题上浪费时间

## 十、总结

DSE旅游与款待科对内地生而言是一门**可以取得稳定成绩**的选修科，主要挑战在于补强香港本地旅游知识和行业数据。无SBA要求使其对私人候考人更加友好。对于有志于酒店管理（尤其是PolyU HTM）的内地生，THS是最具说服力的选修科选择，也是最直接展示专业兴趣的方式。
