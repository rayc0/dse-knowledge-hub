# DSE音乐科完全指南——内地生视角

DSE音乐科（Music）是香港高中选修科目之一，适合有正式音乐训练背景的考生。对于来自内地、有音乐专业基础或参加过中央音乐学院考级的学生，音乐科提供了一个将特长转化为DSE高分的机会。本指南全面解析音乐科的考试结构、选科标准与备考策略。

## 一、考试结构概览

DSE音乐科由三大部分组成：

| 组成部分 | 占分比例 | 考核形式 |
|---|---|---|
| 聆听卷（Listening Paper） | 约40% | 笔试，听录音作答 |
| 演奏（Performing） | 约30% | 校本评核（SBA）现场演奏 |
| 创作（Creating） | 约30% | 校本评核（SBA）作品提交 |

> 校本评核（SBA）总共占约60%，对整体成绩影响极大，不可忽视。

## 二、聆听卷（Listening Paper）详解

### 2.1 考试格式
- 时间：约1小时30分钟
- 题型：多项选择、简答、填表、分析题
- 考生需在考试期间聆听多段音乐录音，并作答

### 2.2 主要考察内容

| 知识领域 | 具体内容 |
|---|---|
| 乐理基础 | 音符、节拍、调号、和弦、音程 |
| 音乐风格辨识 | 巴洛克、古典、浪漫、现代、爵士、流行 |
| 曲式分析 | 二段式、三段式、奏鸣曲式、变奏曲式 |
| 作曲家与作品 | 西方古典名家（巴赫、莫扎特、贝多芬等）及中国音乐传统 |
| 乐器辨识 | 管弦乐组、中国乐器的音色辨别 |
| 音乐文化 | 非洲、印度、日本等世界音乐文化概述 |

### 2.3 内地生优势与挑战

| 方面 | 内地生优势 | 需补充 |
|---|---|---|
| 中国传统音乐 | 熟悉二胡、琵琶等乐器及五声调式 | 需了解粤剧等香港本地音乐 |
| 西方乐理 | 参加过音乐考级者基础扎实 | 需强化曲式分析（非考级重点） |
| 世界音乐 | 通常了解较少 | 需专项补充非洲、印度音乐知识 |

## 三、演奏部分（Performing SBA）

### 3.1 要求
- 考生须演奏**2—3首**乐曲，总时长约5—10分钟
- 可选择独奏或合奏（合奏不超过总时长50%）
- 由学校内部评核员（通常为音乐科老师）评分，并附送外部审核

### 3.2 乐曲选择建议

| 考虑因素 | 建议 |
|---|---|
| 难度 | 选择能稳定发挥的难度，约ABRSM Grade 7—8水平 |
| 风格多样性 | 涵盖不同时期或风格（如古典+现代） |
| 表达空间 | 选有情感起伏、能展示音乐诠释能力的曲目 |
| 独特性 | 避免过度常见曲目，与老师商讨 |

### 3.3 内地考级背景换算参考

| 内地考级级别 | 大致等同ABRSM水平 | DSE演奏适用性 |
|---|---|---|
| 中央院/中国音协 8级 | 约ABRSM Grade 6—7 | 可参考，需进一步提升 |
| 中央院/中国音协 9—10级 | 约ABRSM Grade 7—8 | 适合DSE演奏要求 |
| 专业附中水平 | 超过ABRSM Grade 8 | 充分符合，建议选高难度曲目 |

> 注意：DSE演奏评核不仅看技术，更重视**音乐表达、乐句处理与舞台表现**，内地考级注重技术的特点需适当调整。

## 四、创作部分（Creating SBA）

### 4.1 要求
- 提交**1—2件**原创音乐作品
- 可以是：乐器作品、电子音乐/MIDI、歌曲创作等多种形式
- 须附上创作说明（解释创作理念、技术手法等）
- 总时长约2—5分钟

### 4.2 评分标准

| 评分维度 | 说明 |
|---|---|
| 音乐材料的使用 | 旋律、和声、节奏是否有效发展 |
| 曲式结构 | 整体布局是否合理、有逻辑性 |
| 创意与个性 | 作品是否有独特的音乐想法 |
| 技术执行 | 乐谱规范、音频录制质量 |
| 创作说明 | 文字说明是否清晰解释创作意图 |

### 4.3 创作建议
- 从自身最熟悉的乐器/风格出发
- 融合中西元素（如在钢琴作品中加入五声调式元素）可获得个性加分
- 提前与音乐科老师沟通，确保方向符合评分标准

## 五、应该选择音乐科吗？

### 5.1 适合选音乐科的学生

| 条件 | 说明 |
|---|---|
| 乐器程度 | ABRSM Grade 5（或相当）以上，能稳定演奏达标曲目 |
| 乐理基础 | 能读谱、理解和声与曲式基础 |
| 创作兴趣 | 对音乐创作有热情或有初步经验 |
| 时间投入 | 愿意持续练习乐器，不因备考减少练习 |
| 目标院校 | 有志于音乐相关专业，或将音乐作为特长科目 |

### 5.2 不适合选音乐科的学生

| 情况 | 原因 |
|---|---|
| 只有初级乐器基础（低于Grade 5） | SBA演奏分数难以达标 |
| 对乐理无基础 | 聆听卷难度较高，短期补底耗时 |
| 期望高分但缺乏系统训练 | 音乐科不像文科靠背诵取分 |
| 备考其他科目时间已很紧张 | 演奏和创作SBA需要持续投入 |

## 六、音乐科工作量估算

| 任务 | 每周时间投入 |
|---|---|
| 乐器练习（维持及提升） | 3—5小时 |
| 乐理/聆听复习 | 2—3小时 |
| 创作作品推进 | 1—2小时 |
| **合计** | **约6—10小时/周** |

> 这是DSE选修科中**工作量较大**的科目，但对于热爱音乐的学生而言，大部分时间投入（乐器练习）与正常兴趣维持重合，实际额外备考负担相对合理。

| HKAPA音乐学院 | DSE仅为参考 | 以专业试音为主要录取标准 |

## 八、内地生备考音乐科特别建议

1. **香港本地音乐了解**：补充粤剧、粤语流行音乐发展史，聆听卷可能涉及
2. **曲式分析专项**：中央音乐学院考级不考曲式分析，需额外学习
3. **世界音乐补足**：非洲鼓乐、印度音乐、日本雅乐——这些是内地学生最薄弱的聆听题领域
4. **乐谱规范**：创作提交需用标准乐谱软件（如Sibelius、MuseScore），提前学习使用
5. **英文音乐术语**：DSE聆听卷以英文为主，建立音乐英语词汇表（tempo, dynamics, texture等）

## 九、总结

DSE音乐科是一门高度依赖**长期音乐训练积累**的科目，无法靠短期突击取得高分。对于拥有ABRSM Grade 5以上乐器程度、具备基础乐理知识的内地学生而言，音乐科是将特长变为成绩优势的绝佳通道。核心备考方向是：**强化聆听卷的曲式分析与世界音乐知识、精心准备SBA演奏曲目、提前规划创作作品**。
