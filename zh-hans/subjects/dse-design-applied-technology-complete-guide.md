# DSE设计与应用科技（DAT）完全攻略：内地学生备考指南

## 科目简介

设计与应用科技（Design and Applied Technology，简称DAT）是DSE的选修科目，融合设计思维、工程原理与实践技能。与其他以笔试为主的科目不同，DAT高度重视实践操作和设计过程，要求学生完成从构思到实物制作的完整设计循环。

对有艺术设计背景、动手能力强、或有意修读设计/工程/建筑类大学专业的内地学生而言，DAT是一个极具吸引力的选择。

## 课程结构与评估组成

DAT的评估分为三大部分：

| 评估部分 | 内容 | 占比 | 说明 |
|---------|------|------|------|
| Paper 1（笔试） | 设计知识与理解（选择题 + 结构题） | 35% | 涵盖材料学、产品分析、人体工程学等 |
| 设计组合（SBA项目一） | 设计过程文件（设计思维记录） | 30% | 全学年持续完成，在校评核 |
| 实践项目（SBA项目二） | 实物制作 + 项目报告 | 35% | 制作一件完整的设计产品 |

**总体特点**：DAT是DSE中SBA（校本评核）比重最高的科目之一，超过60%的成绩来自学校内的持续评核，而非一次性考试。

## Paper 1：书面试卷详解

### 涵盖知识领域

| 主题 | 具体内容 |
|------|---------|
| 设计过程 | 设计循环（Design Cycle）、用户研究、需求分析 |
| 材料与制造 | 木材、金属、塑料、复合材料的性质与加工工艺 |
| 产品分析 | 功能性、美观性、人体工程学、可持续性分析 |
| 绘图技能 | 等角投影（Isometric Drawing）、正投影（Orthographic Drawing） |
| 电子与控制 | 基本电路、传感器、控制系统（部分课程） |
| 人体工程学 | 人体尺寸标准、用户友好设计原则 |
| 可持续设计 | 生命周期分析、环保材料选择、循环经济设计 |

### 题型分布

| 题型 | 说明 |
|------|------|
| 多项选择题 | 测试基础设计知识和材料性质 |
| 结构题 | 分析产品设计图、识别设计问题、提出改进建议 |
| 延伸题 | 以具体情境为背景，综合评价设计方案 |

## 设计组合（SBA项目一）：设计过程文件

设计组合是对学生**设计思维过程**的系统记录，通常以A3画板或数码文件形式呈现。

### 设计组合的评核内容

| 阶段 | 内容 | 评核要点 |
|------|------|---------|
| 1. 情境分析 | 确定设计问题，分析用户需求 | 研究深度、问题定义清晰度 |
| 2. 设计调研 | 参考现有产品、材料研究 | 调研广度与相关性 |
| 3. 设计构思 | 多个设计方案草图 | 创意多样性、可行性 |
| 4. 方案筛选 | 评价并选定最优方案 | 判断力、理由是否有据 |
| 5. 详细设计 | 制作详细工程图 | 绘图准确性、标注完整 |
| 6. 原型制作 | 模型或实物初稿 | 工艺质量 |
| 7. 测试评估 | 测试产品功能，对照设计要求 | 测试方法、批判性评价 |
| 8. 反思改进 | 记录改进建议和学习反思 | 自我评估能力 |

**关键提示**：评核老师重视的是**设计思维的完整性**，而非最终产品是否精美。一个问题识别清晰、调研充分、迭代合理的组合，比一个华丽但过程记录单薄的组合得分更高。

## 实践项目（SBA项目二）：实物制作

实践项目要求学生在学校工作坊中独立完成一件完整的设计产品。

### 实践项目的评核标准

| 评核维度 | 权重 | 说明 |
|---------|------|------|
| 工艺质量 | 高 | 材料切割、接合、表面处理的精细度 |
| 功能实现 | 高 | 产品是否能正常运作 |
| 设计一致性 | 中 | 实物是否与设计组合中的方案一致 |
| 项目报告 | 中 | 制作过程记录、遇到的问题及解决方法 |
| 创意展现 | 中 | 设计是否有独特之处 |

### 常见实践项目类型

- 家具小品（如折叠式书架、灯具设计）
- 生活用品（如环保餐具架、储物装置）
- 玩具或教育产品（如益智拼图、模型）
- 简单电子装置（如感应灯、智能收纳盒）

## 三维设计与数码工具

现代DAT课程越来越重视数码设计工具的运用：

| 工具类别 | 常用软件 |
|---------|---------|
| 2D绘图 | AutoCAD（基础）、Adobe Illustrator |
| 3D建模 | SketchUp、Fusion 360、Tinkercad |
| 渲染与展示 | Keyshot、Blender（进阶） |
| 排版设计 | Adobe InDesign（设计组合排版） |

内地学生若有相关软件使用经验，在制作设计组合时会有明显优势。

## 对接大学专业

DAT成绩被以下类型的大学课程认可：

| 大学专业 | DAT的关联度 | 主要院校 |
|---------|-----------|---------|
| 工业设计 | 极高 | 香港理工大学（PolyU） |
| 产品设计 | 极高 | 香港理工大学、香港知专设计学院 |
| 建筑学 | 高 | 港大、中大、理大建筑系 |
| 室内设计 | 高 | 香港理工大学、香港设计学院 |
| 视觉传达设计 | 中高 | 香港理工大学 |
| 机械工程 | 中 | 各大工程学院 |
| 科技教育 | 高 | 香港教育大学 |

**重要提示**：申请设计类大学课程通常还需提交**设计作品集（Portfolio）**，DAT的设计组合和实践项目作品是构建作品集的核心材料。

## DSE DAT与内地相关课程的比较

内地高中并无直接对应的「设计与应用科技」科目，但通识技术课、信息技术课、劳动课等有部分相似元素：

| 维度 | DSE DAT | 内地相关课程 |
|------|--------|------------|
| 课程定位 | 正式选修科目，计入DSE成绩 | 通常为必修课，不计入高考 |
| 实践深度 | 深，有完整项目制作要求 | 浅，以演示/操作为主 |
| 设计思维训练 | 系统化（设计循环） | 较少系统化训练 |
| 数码工具应用 | 鼓励使用3D软件 | 信息技术课有基础操作 |
| 大学连接 | 直接对口设计/工程专业 | 主要服务于高考文理科选择 |

## 内地学生的备考优势与挑战

### 优势

- **艺术培训背景**：内地有大量学生参加过专业美术培训（素描、色彩、立体构成），这些技能直接转化为DAT优势
- **数学与物理基础**：内地理科训练有助于理解材料力学、电路等工程原理
- **勤奋与执行力**：内地学生对于耗时的项目制作通常有较强的坚持能力
- **3D软件经验**：部分内地学生已自学SketchUp、Fusion 360等软件

### 挑战

- **设计思维转换**：内地培训偏向技法（画得像），DSE更重视「解决问题」的设计思维
- **工作坊操作安全规范**：香港学校工坊有严格的安全规程，内地学生需时间适应
- **英文书面表达**：Paper 1及项目报告须以英文作答
- **时间管理**：设计组合和实践项目占大量课余时间，需要长期规划

## 高分策略

### 设计组合高分技巧

1. **过程重于结果**：每个设计决定都需要有文字解释理由（「我选择A方案因为……」）
2. **用户研究要扎实**：进行真实的用户访谈或问卷，将结果反映在设计中
3. **版面设计整洁专业**：组合的视觉效果直接影响第一印象，但内容才是评核核心
4. **照片记录制作过程**：每个制作步骤拍照，写入项目报告
5. **测试须客观**：测试时记录数据（如强度测试、用户使用测试），而非只写「测试成功」

### Paper 1备考技巧

1. **熟记材料性质对照表**：这是选择题的高频考点（如「哪种材料最适合…」）
2. **工程图练习**：等角投影和正投影图是必考内容，需大量练习
3. **人体工程学数据**：记住常用人体尺寸标准（如桌椅高度、扶手宽度）
4. **产品分析框架**：SCAMPER（替换、组合、调整、改造、其他用途、消除、重排）是常用工具

## 总结

DAT是DSE中少数能让学生「以作品说话」的科目，对有设计天赋和动手能力的学生而言，这是一个高性价比的选修科目。内地学生若具备美术培训背景或理工兴趣，可充分发挥优势；关键在于从「技法展示」思维转换为「设计思维记录」，并重视英文书面表达能力的培养。
