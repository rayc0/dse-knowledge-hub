# JUPAS申请策略完全指南——内地生视角

JUPAS（Joint University Programmes Admissions System，大学联合招生办法）是香港八大资助大学录取本地DSE考生的主渠道。对于以私人候考人身份应考DSE的内地生而言，了解JUPAS机制并善用非本地生名额，是进入港大、港中文、港科大等顶尖院校的关键。

## 一、JUPAS系统概览

### 1.1 参与院校
JUPAS涵盖以下八所教资会资助大学：

| 简称 | 全称 |
|---|---|
| HKU | 香港大学 |
| HKUST | 香港科技大学 |
| CUHK | 香港中文大学 |
| PolyU | 香港理工大学 |
| CityU | 香港城市大学 |
| HKBU | 香港浸会大学 |
| LingU | 香港岭南大学 |
| EdUHK | 香港教育大学 |

### 1.2 时间线

| 阶段 | 时间 | 说明 |
|---|---|---|
| 网上申请 | 每年9月—翌年1月 | 填报最多25个志愿 |
| 修改志愿 | 翌年5月放榜前 | 可按估分调整排序 |
| 成绩放榜 | 7月中旬 | DSE成绩公布 |
| 主轮录取 | 7月下旬 | 按成绩与志愿顺序录取 |
| 补录（Clearing） | 8月 | 余额名额开放 |

## 二、波段制度（Banding System）

### 2.1 波段定义
JUPAS根据考生DSE成绩将其划入不同波段：

| 波段 | 说明 |
|---|---|
| Band A | 成绩高于该课程上年度录取中位数 |
| Band B | 成绩接近上年度录取中位数 |
| Band C | 成绩低于上年度录取中位数 |

- **Band A志愿**：被录取概率最高，建议放置最心仪的课程
- **Band B志愿**：有一定风险，需权衡
- **Band C志愿**：保底志愿，录取概率高但课程可能非首选

### 2.2 志愿排列策略
建议25个志愿按以下比例分布：

| 类型 | 数量 | 说明 |
|---|---|---|
| 冲刺志愿（Band C） | 3—5个 | 目标院校顶尖课程 |
| 主力志愿（Band A/B） | 10—15个 | 核心目标课程 |
| 保底志愿（Band A） | 5—8个 | 确保录取的课程 |

> 重要：JUPAS采用**志愿顺序优先**原则，系统从第1志愿开始逐一检查是否达到录取门槛，达到即录取，不再看后续志愿。因此**第1志愿必须是最心仪的课程**，切勿因求稳把最爱的课程放低。

## 三、DSE成绩如何换算为JUPAS排名

### 3.1 计分方式
各大学通常以**最佳5科**或**最佳6科**（含必修4科+最佳选修）计算入学成绩：

| 等级 | 分值（参考） |
|---|---|
| 5** | 7分 |
| 5* | 6分 |
| 5 | 5分 |
| 4 | 4分 |
| 3 | 3分 |
| 2 | 2分 |
| 1 | 1分 |

> 部分课程（如理工科）对数学、物理等特定科目设**加权**，具体以各大学公布为准。

### 3.2 热门课程参考最低分（近年DSE，最佳5科）

| 课程 | 院校 | 约需总分 | 特殊要求 |
|---|---|---|---|
| 医学（MBBS） | HKU | 34—36/42 | 面试 |
| 医学（MBBS） | CUHK | 33—35/42 | 面试 |
| 法律（LLB） | HKU | 32—34/42 | 面试 |
| 计算机科学 | HKUST | 28—31/42 | 数学5* |
| 工商管理 | CUHK | 27—30/42 | 英文5 |
| 精算学 | CityU | 29—32/42 | 数学5** |
| 教育学 | HKU | 24—27/42 | |

*以上数据为参考，每年有波动，请以各校官方公布为准。*

## 四、非本地生名额

### 4.1 名额比例
根据香港政府政策，资助大学本科课程可招收**最多20%**的非本地学生（2024年起部分课程放宽至25%）。

| 院校 | 非本地生比例（参考） |
|---|---|
| HKU | 约20% |
| HKUST | 约20% |
| CUHK | 约20% |
| PolyU | 约15—20% |

### 4.2 内地生竞争格局
- 非本地生名额中，内地生（中国内地）占绝大多数
- 内地生需与其他非本地生（包括海外华人、外籍生）竞争同一名额池
- 部分热门课程（如医学、法律）非本地生名额**极为有限**（有时仅2—5个）

### 4.3 非本地生申请路径
内地私人候考人可通过以下两种方式申请：

| 路径 | 说明 |
|---|---|
| JUPAS Non-local（非联招） | 通过各大学独立申请入口，提交DSE成绩 |
| 直接向各校申请 | 部分院校开设专门的Non-JUPAS入学计划 |

> 内地生以**Non-JUPAS**方式申请时，可参考JUPAS历年最低分作为竞争基准，但录取标准可能略有不同。

## 五、SEAS早期录取计划

### 5.1 什么是SEAS
SEAS（Scholarship and Early Admission Scheme，奖学金及早期录取计划）允许**优秀考生在DSE考试前**获得有条件录取通知书，无需等待7月放榜。

### 5.2 申请条件
- 通常要求模拟考（Mock DSE）或学校内部成绩达到极优水平
- 各院校自行设定标准，一般针对**预期5**×**×或接近5**×**×**的考生
- 申请窗口：约每年4—5月（DSE考试期间或稍后）

### 5.3 SEAS优缺点

| 优点 | 缺点 |
|---|---|
| 提前锁定学位，减轻放榜压力 | 条件录取，DSE成绩未达标则取消 |
| 可获奖学金激励 | 申请名额有限，竞争激烈 |
| 有助心理稳定，冲刺期更从容 | 若正式成绩超预期，可能错失更好选择 |

## 六、热门课程面试准备

### 6.1 哪些课程需要面试
- **医学**（HKU/CUHK）：几乎所有获提名者须参加面试
- **法律**（HKU LLB）：面试占录取权重较大
- **教育学**（EdUHK/HKU）：部分专项面试
- **社工/护理**：通常有面试

### 6.2 医学面试（HKU MMI格式）
HKU医学院采用**Multiple Mini Interview（MMI）**格式：
- 约8—10个独立面试站，每站5—8分钟
- 考察：伦理情景判断、沟通能力、批判性思维、自我认识
- 准备建议：练习STAR答题法、阅读医疗伦理案例、关注香港医疗政策

### 6.3 法律面试
- HKU法律面试考察：法律推理、时事议题分析、个人动机
- 准备建议：阅读香港法庭判决摘要、时事评论，练习逻辑辩论

### 6.4 内地生面试加分项
- 能展示**双语（普通话+广东话/英语）**沟通能力
- 有内地与香港跨文化经历
- 对香港本地议题有深度了解（非仅内地视角）

## 七、志愿填报实战策略

### 7.1 信息收集清单
在填报前，必须收集以下数据：

- [ ] 目标课程近3年最低收分（各校官网或JUPAS官网）
- [ ] 目标课程是否有特定科目要求（如物理、数学M2）
- [ ] 非本地生历年录取人数
- [ ] SEAS截止日期及申请方式
- [ ] 是否需要面试及面试格式

### 7.2 填报节奏
1. **9—10月**：初步列出30个感兴趣课程，粗筛
2. **11—1月**：缩减至25个，确定大致顺序
3. **5月**（估分后）：根据模拟考成绩调整顺序，特别是前5个志愿
4. **7月放榜后**：无法修改，系统自动处理

### 7.3 常见错误

| 错误 | 后果 | 建议 |
|---|---|---|
| 第1志愿过于保守 | 浪费最优先权重 | 第1志愿放最心仪课程 |
| 全部志愿集中同一院校 | 若该校分高全部落空 | 分散2—3所院校 |
| 忽视课程特定科目要求 | 即使总分达标也被拒 | 细读各课程入学要求 |
| 不了解非本地生名额 | 误以为与本地生竞争相同名额 | 查明各课程非本地生配额 |

## 八、总结

JUPAS申请成功的关键在于：**精准定位自身成绩水平、理解波段机制、合理分布25个志愿、提前了解非本地生名额及面试要求**。内地生应特别注意，非本地生名额有限且竞争激烈，建议在志愿排布时兼顾不同院校，避免把所有赌注押在单一顶尖院校。放榜前充分准备面试，往往是拿到心仪Offer的最后一公里。
