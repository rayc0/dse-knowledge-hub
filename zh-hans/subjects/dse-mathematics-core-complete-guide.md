# DSE数学必修部分完全指南——内地生视角

数学必修部分（Mathematics Compulsory Part，Math Core）是DSE四大必修科之一，也是绝大多数内地生的强项科目。然而，DSE数学的出题风格与高考存在明显差异，若不针对性调整备考策略，高分并非理所当然。本指南从内地生视角出发，系统梳理DSE数学必修的全貌。

## 一、考试结构

| 卷别 | 题型 | 题量 | 时间 | 分值 |
|---|---|---|---|---|
| Paper 1 | 长答题（Long Questions） | 约15题（分A、B两节） | 2小时15分钟 | 105分 |
| Paper 2 | 选择题（Multiple Choice） | 45题 | 1小时15分钟 | 45分 |
| **合计** | | | **3小时30分钟** | **150分** |

### Paper 1详解
- **Section A（Part 1）**：约8题，每题2—4分，考基础计算与概念应用
- **Section A（Part 2）**：约5题，每题4—7分，难度中等，需要推导过程
- **Section B**：约4题，每题10—14分，属于高难度综合题，5**冲刺者必须拿下大部分分数

### Paper 2详解
- 45道选择题，每题1分，无负分
- 考察速度与准确度，需在平均每题约1分40秒内作答
- 常见陷阱：选项设计刻意包含常见错误答案

## 二、核心考点

### 2.1 代数（Algebra）

| 子主题 | 常考内容 |
|---|---|
| 多项式 | 因式分解、余式定理、因式定理 |
| 方程 | 二次方程判别式、韦达定理 |
| 不等式 | 二次不等式、绝对值不等式 |
| 数列 | 等差数列、等比数列、求和公式 |
| 指数与对数 | 指数方程、对数方程、换底公式 |

### 2.2 函数（Functions）

| 子主题 | 常考内容 |
|---|---|
| 线性函数 | 斜率、截距、两点式方程 |
| 二次函数 | 顶点式、对称轴、最大/最小值 |
| 指数/对数函数 | 图像变换、方程求解 |
| 变分 | 正比、反比、联合变分 |

### 2.3 统计与概率（Statistics & Probability）

| 子主题 | 常考内容 |
|---|---|
| 数据处理 | 平均数、中位数、众数、四分位数 |
| 箱型图 | 五数概括、四分位距 |
| 概率 | 古典概型、加法规则、乘法规则 |
| 排列组合 | 排列数P、组合数C的计算与应用 |
| 正态分布（M1相关，Core亦涉及基础） | 标准正态、百分位概念 |

### 2.4 几何（Geometry）

| 子主题 | 常考内容 |
|---|---|
| 直线几何 | 角度计算、平行线截割 |
| 圆 | 圆的方程、切线、圆与直线关系 |
| 三角 | 正弦/余弦定理、三角恒等式 |
| 向量（基础） | 向量加减、数量积 |
| 坐标几何 | 中点、距离、斜率、直线方程 |

### 2.5 数论与其他

| 子主题 | 常考内容 |
|---|---|
| 整数与整除 | 最大公因数、最小公倍数 |
| 比率与比例 | 实际问题建模 |
| 测量 | 面积、体积、误差处理 |

## 三、与高考数学的对比

| 维度 | DSE数学必修 | 高考数学（全国卷） |
|---|---|---|
| 难度 | 中等偏上 | 偏难（尤其是压轴题） |
| 题型 | 选择题+长答题 | 选择题+填空题+大题 |
| 微积分 | **不考**（在M1/M2中） | 必考（导数与积分） |
| 向量 | 基础涉及 | 深度考察（空间向量） |
| 概率统计 | 较重（约20%） | 中等（约15%） |
| 数列 | 中等考察 | 深度考察（含递推） |
| 三角函数 | 中等 | 深度（含辅助角、变换） |
| 解题步骤要求 | 需写完整推导过程（Paper 1） | 同样需要完整过程 |
| 语言 | **英文或中文**（可选） | 中文 |
| 计算器 | **允许使用** | 不允许 |

> 对内地生最重要的差异：**DSE允许使用计算器**，这大幅降低了复杂数字计算的难度，但同时选择题陷阱更多，需注意。

## 四、内地生常见失误

| 失误 | 具体表现 | 解决方法 |
|---|---|---|
| 不善用计算器 | 仍用笔算耗时 | 熟练掌握图形计算器（CASIO fx-9860G或类似型号）常用功能 |
| Paper 2答题超时 | 每题耗时过长 | 练习限时，每题不超过2分钟，不会立即跳过 |
| 步骤不完整（Paper 1） | 跳步写答案，无中间推导 | 参照Marking Scheme，补全每步 |
| 误解英文题目 | 关键词理解偏差 | 整理DSE数学英文术语表 |
| 概率题逻辑错误 | 混淆有放回/无放回抽样 | 专项练习列举法与公式法 |
| 忽视"Show that"题 | 直接写结果而非推导 | 这类题必须写完整证明过程 |
| 圆的方程失误 | 完全平方式转换错误 | 反复练习标准形式与一般形式互转 |

## 五、Level 5**冲刺策略

### 5.1 分数目标

| 等级 | 约需原始分（参考） |
|---|---|
| 5** | ≥约130/150（87%） |
| 5* | ≥约115/150（77%） |
| 5 | ≥约100/150（67%） |
| 4 | ≥约80/150（53%） |

*以上为估算，每年分数线有调整。*

### 5.2 Paper 1 Section B攻略
Section B是5**与5*之间的关键战场：
- 每题约12分，做对3题可额外得36分
- 常见Section B题型：向量综合、函数最值与不等式结合、统计推断、圆的综合
- 策略：先做会的小问（a)(b)，拿稳基础分，（c)(d)高难部分再冲刺

### 5.3 Paper 2选择题满分策略
- 目标：45题全对，不丢分
- 每题检查：计算结果代回验证（利用计算器）
- 时间分配：第一遍30分钟完成，剩余15分钟重检标记题
- 特别注意：坐标几何、概率、数列这三类错误率最高

### 5.4 备考时间线

| 距考试时间 | 任务 |
|---|---|
| 6个月前 | 系统复习全部知识点，完成教材例题 |
| 4—6个月 | 开始练习历年Paper 2，建立速度感 |
| 2—4个月 | 集中攻克Paper 1 Section B，做近5年真题 |
| 最后2个月 | 全套真题计时练习，错题复盘，Marking Scheme对照 |
| 最后2周 | 重温高频考点公式，不引入新题型 |

## 六、Math Core与M1/M2的关系

| 维度 | Math Core（必修） | M1（微积分与统计） | M2（代数与微积分） |
|---|---|---|---|
| 是否必考 | 是 | 选修（第7科） | 选修（第7科） |
| 微积分 | 不含 | 含简单微积分 | 含深度微积分 |
| 适合人群 | 所有考生 | 商科/社科倾向 | 理工科/数学兴趣强 |
| 难度 | 中等 | 中偏难 | 较难 |
| 与Core关系 | 基础 | Core的延伸 | Core的延伸 |

> 大多数顶尖理工科课程（HKUST工程、HKU理学院）建议/要求M2。内地生数学基础强，若目标为理工科，应同时选修M2。

## 八、总结

DSE数学必修是内地生最具竞争力的科目之一。核心优势是数学基础扎实、计算能力强。主要调整方向是：**熟练运用计算器、掌握英文术语、训练Paper 2答题速度、确保Paper 1步骤完整性**。系统操练历年真题，针对性补强概率统计和坐标几何，Level 5**完全在内地生的能力范围之内。
