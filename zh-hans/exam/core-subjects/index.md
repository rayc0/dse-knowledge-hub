# DSE四个必修科目完全解析

香港中学文凭考试（HKDSE）设有四个必修核心科目（Core Subjects），所有考生均须应考。这四科的成绩是香港各大学录取计算的基础，也是非联招（Non-JUPAS）申请时内地生最常以DSE入学的核心指标。本文逐科分析各科特点、等级分布及备考要点。

---

## 一、中国语文（Chinese Language）

### 科目概述

中文科是DSE中公认难度最高的科目之一，也是最常成为内地生短板的科目。考试形式包括：
- **卷一：阅读理解**（占30%）：文言文及白话文篇章，需理解、分析及评价
- **卷二：写作**（占30%）：实用文（书信、报告等）及论说文/记叙文
- **卷三：聆听及综合**（占20%）：聆听录音后进行摘要或综合
- **卷四：说话**（口试，占20%）：小组讨论，用粤语进行（！）

### 重要注意：口试以粤语进行

中文口试须以粤语作答，这是内地学生的主要挑战。普通话母语者若粤语不流利，可能在口试环节失分严重。建议提前数年开始学习粤语，或针对口试格式进行专项训练。

### 等级分布（约5万名考生/年）

| 等级 | 百分比 | 人数（约） |
|------|-------|---------|
| 5** | 2–3% | 1,000–1,500 |
| 5* | 5–6% | 2,500–3,000 |
| 5 | 8–10% | 4,000–5,000 |
| 4 | 15–18% | 7,500–9,000 |
| 3 | 20–22% | 10,000–11,000 |
| 2 | 20–22% | 10,000–11,000 |
| 1及以下 | 约25% | 12,500+ |

### 大学录取最低要求

- 大多数本科课程：**Level 3**
- 中文为主课程（如中文系、教育）：**Level 4–5**
- 部分课程豁免中文要求（如国际学生特别通道）

### 备考建议

- 大量阅读粤语白话文及文言经典，熟悉书面语体系
- 写作须掌握实用文格式（信件抬头、结尾礼仪等固定格式）
- 口试提前练习粤语表达，熟悉小组讨论常见议题（社会时事、价值观探讨）

## 二、英国语文（English Language）

### 科目概述

英文科是影响内地生录取结果最关键的科目，几乎所有香港大学课程均要求英文达**Level 4或以上**。考试分为：
- **卷一：阅读**（25%）：理解各类文体文章
- **卷二：写作**（25%）：功能性写作（邮件、建议书）及延伸写作（议论文、故事）
- **卷三：聆听及综合**（25%）：聆听后做笔记、摘要或回答问题
- **卷四：口语**（25%）：小组讨论及个人发言，全程英语

### 等级分布

| 等级 | 百分比 |
|------|-------|
| 5** | 约2% |
| 5* | 约5% |
| 5 | 约10% |
| 4 | 约18% |
| 3 | 约22% |
| 2 | 约22% |
| 1及以下 | 约21% |

### 大学录取最低要求

- 绝大多数大学课程：**Level 3–4**（Level 3为极低门槛，实际热门课程普遍要求Level 4以上）
- HKU、HKUST理工商科热门课程：通常要求**Level 4–5**
- 英文教学或翻译课程：**Level 5或以上**

### 备考建议

- DSE英文考试强调实用及批判性写作，与内地英语考试风格有所不同，建议多读香港英文报章（South China Morning Post）
- 口试须熟悉讨论流程（提出观点、回应他人、总结），不只是语言流利度
- 写作卷需掌握不同文体格式（商业信函、报告、博客等各有固定结构）

## 三、数学（Mathematics）

### 科目结构

数学科分为两个部分：

**必修部分（Compulsory Part）**——全部考生均须应考
- 代数、数列、函数、统计、概率、几何、三角函数等
- 两份卷，共约3小时
- 难度中等，但拉分效果显著

**延伸单元（Extended Module）**——选修，分两部分：
- **M1（微积分与统计）**：适合商科、生命科学方向学生
- **M2（代数与微积分）**：适合理工科、数学系方向，难度更高

延伸单元成绩独立计分，部分大学（尤其科大、理大工程类）在录取时将M1/M2列为额外要求或加分项。

### 等级分布（必修部分）

| 等级 | 百分比 |
|------|-------|
| 5** | 约3–4% |
| 5* | 约6% |
| 5 | 约10% |
| 4 | 约16% |
| 3 | 约18% |
| 2 | 约20% |
| 1及以下 | 约27% |

数学必修部分的5**率相对较高，是理科生拉开差距的重要科目。

### 大学录取要求

- 文科/社科课程：一般要求**Level 2–3**
- 商科（BBA/会计）：一般要求**Level 3–4**
- 理工科：要求**Level 4**，部分课程要求M2达Level 3以上

### 备考建议

- DSE数学解题步骤须完整呈现，不能只写答案
- M2内容与内地高中数学选修模块有重叠，内地理科生通常较有优势
- 历年试卷（Past Papers）是最有效的备考工具，建议至少完成最近5年全卷

## 四、公民与社会发展科（Citizenship and Social Development，CSD）

### 2021年课程改革背景

这是变化最大的一科。原「通识教育」（Liberal Studies）于2021年改革为「公民与社会发展科」，课程内容及考核形式均有重大调整：

**主要变化：**
- 删除批判性议题辩论，减少争议性内容
- 新增「国家发展」及「当代中国」专题
- 考核方式：不再设公开考试卷，改为**校本评核（SBA）**及**考察活动报告**
- 内地参观（大湾区或其他城市）成为课程核心组成部分

### 科目组成（2021年起）

| 主题 | 内容 |
|------|------|
| 主题一：香港 | 香港历史、身份认同、社会发展 |
| 主题二：国家 | 国家发展成就、「一国两制」、香港与国家关系 |
| 主题三：当代世界 | 全球议题、可持续发展、国际关系 |

### 等级与评核

- CSD采用「达标/不达标（Attained / Not Attained）」评级，**不设数字等级（1–5**）**
- 大学录取时，该科不计入「最佳五科」分数计算，但须达到「Attained」
- 极少数学生会因CSD未达标而影响入学

### 对内地生的影响

对于以Non-JUPAS申请的内地生，若未在香港中学就读，通常**无须提交CSD成绩**，以其他科目成绩申请即可。但在本港就读中学的学生须完成此课程。

## 四科综合策略

| 科目 | 内地生难度 | 策略重点 |
|------|-----------|---------|
| 中文 | ★★★★★（最难） | 提前学粤语、大量练习写作 |
| 英文 | ★★★☆☆（中等） | 多读英文报章、练习口试 |
| 数学 | ★★☆☆☆（内地理科生优势） | 刷历年卷，选修M2增强竞争力 |
| CSD | ★☆☆☆☆（Non-JUPAS通常不计） | 在港就读者按课程要求达标即可 |

四科中，英文成绩往往是决定内地生能否进入顶尖大学的关键。数学科内地理科生通常有优势；中文科是最大挑战，须及早准备。
