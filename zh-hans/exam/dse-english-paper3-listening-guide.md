# HKDSE 英语科 Paper 3 听力及综合卷完整攻略

HKDSE 英语科共有四卷 (Reading、Writing、Listening & Integrated Skills、Speaking),其中 **Paper 3 听力及综合运用卷 (Listening and Integrated Skills Paper)** 占全科总分的 **30%**,考试时长 **约 2 小时**。对于从内地转来香港念书的同学、国际学校背景的学生、或英语为第二语言的考生而言,Paper 3 常常被视为最难的一卷 —— 不是因为单词或语法本身特别难,而是因为它同时测试听力理解、信息筛选、综合写作、时间管理和任务转换能力,考点密集而且紧扣时间。

本攻略写给正在备考 DSE 英语的同学,特别是有内地普通话背景或对香港课程体系还不熟悉的学生及家长。内容涵盖考试结构、听力训练方法、Data File 使用技巧、综合写作的通用框架、时间分配策略、常见错误及 2026 年备考建议。

## 一、Paper 3 考试结构解析

### 考卷构成

Paper 3 分为 **Part A (Easy Section)** 和 **Part B (Difficult Section)** 两部分,考生需全做 Part A,而 Part B 要在 **B1 (较易版本)** 和 **B2 (较难版本)** 中选一个作答。大部分考取大学的考生会选 B2,因为 B2 的最高可达分数较高 (能够拿 5、5*、5**)。

### 时间分配 (2 小时总时长)

- **Part A**: 约 45 分钟 (听力 + 整合写作)
- **Part B**: 约 75 分钟 (听力 + 较长的整合写作)

注意:Part A 和 Part B 并不是完全分开计时的,整卷合计 2 小时。**时间管理极其关键**。

### 考核重点

Paper 3 同时测试:

1. **听力理解 (Listening Comprehension)** — 听音频填空、回答问题、辨认讲者意图
2. **信息整合 (Integrated Skills)** — 从听力材料与 Data File (纸本资料) 中提取信息,重新整理
3. **写作能力 (Writing)** — 以适当的文体 (报告、信、演讲、发言稿、宣传文案等) 呈现
4. **任务意识 (Task Awareness)** — 理解你在写作中扮演的角色、写给谁看、想要达到什么目的

## 二、Data File 是什么?

**Data File** 是 Paper 3 独有的一个配套材料 —— 一份约 20-30 页的资料包,里面包含:

- 新闻报道、海报、宣传单
- 图表、数据表、问卷结果
- 邮件、备忘录、访谈记录
- 文章、评论、广告

考生需要在听力播放期间或之后,从 Data File 中找到相关信息,结合听力内容完成写作任务。Data File 的信息往往比听力材料更多,你要学会 **快速定位所需信息,忽略无关段落**。

**关键提醒**: Data File 会在考试开始时发放,允许你在听力播放前 **先翻阅和标注**。绝对不要等到写作时才第一次看 Data File。

## 三、听力部分的应对策略

### 录音播放规则

- 每段录音通常播放 **两次**
- 第一次播放时:快速捕捉大意与关键信息
- 第二次播放时:验证细节、补充遗漏

### 听力前的准备 (Pre-listening)

在录音开始前的 **预读时间 (通常 2-3 分钟)**,你应该:

1. **快速读题目要求**: 搞清楚每题要填什么 (日期?地点?原因?数字?)
2. **预测答案类型**: 如果题目是"What time?",就准备捕捉时间信息
3. **标注关键词**: 在题目上圈出核心词 (who, what, when, where, why, how)
4. **预览 Data File 的相关页面** (如果时间允许)

### 听力过程中的技巧

- **先听大意,再补细节**: 不要卡在一个词上,否则会错过后面的内容
- **用缩写和符号**: 边听边用 K (thousand)、$ (dollar)、← → (方向) 等符号快速记录
- **注意转折词**: "however"、"but"、"actually"、"in fact" 之后常出现关键信息
- **留意数字和专有名词**: 常是考点
- **不懂的先跳过**: 第二次播放时再补

### 听力中常见的考点类型

- **直接信息题**: 时间、地点、价格、名字 — 抓准即可
- **推断题**: 讲者的态度、情感、意图 — 听语气和关键动词
- **排序题**: 事件先后次序 — 注意时间连接词 (first, then, after that, finally)
- **图表配对题**: 把听到的信息配对到 Data File 的图表或表格
- **句子改写题**: 填空位置需要根据句子的语法结构决定词性

## 四、综合写作 (Integrated Writing) 的通用框架

Part A 与 Part B 都要求你根据听力和 Data File,完成一份写作任务。常见的任务形式包括:

- **写一封信** (给校长、编辑、朋友、市民)
- **写一篇报告** (给公司、学校、政府)
- **写一份演讲稿** (校会、比赛、公开活动)
- **写一篇宣传文案** (海报、网站、社交媒体帖子)
- **写一份对话或访谈记录**

### 写作前的三个关键问题

无论题目如何,在下笔前先问自己:

1. **Who am I?** (我是谁?)— 学生?记者?老板?市议员?
2. **Who am I writing to?** (读者是谁?)— 老师?一般公众?某个具体人?
3. **What is my purpose?** (我写这篇文章的目的是什么?)— 说服?告知?报告?邀请?抱怨?

回答清楚这三个问题,你的语气、用词、结构才能对应 "角色感"。阅卷老师最看重的是 **角色一致性** 和 **任务达成度**,而不仅仅是语法正确。

### 结构示范:正式信函

```
[正式开头]: Dear Principal / Dear Editor / Dear Mr. Chan

[第一段: 自我介绍 + 写信目的]
I am writing as the Chairperson of the Student Association to express our concerns about...

[第二段: 主要论点 1 + 证据 (从听力或 Data File 提取)]
Firstly, based on the recent survey...

[第三段: 主要论点 2 + 证据]
Furthermore, the talk given by Dr. Wong highlighted...

[第四段: 建议或行动呼吁]
In light of this, we suggest that the school should...

[结尾]: Yours faithfully / Yours sincerely + 署名
```

### 结构示范:演讲稿

```
[开场]: Good morning, everyone. My name is... and today I would like to speak about...

[引入背景]: Recently, our school has been facing...

[主要论点段落]: First of all... / Secondly... / Moreover...

[呼吁行动]: Therefore, I urge each of you to...

[结束]: Thank you for your attention.
```

### 结构示范:报告

```
[标题]: Report on [Topic]
[To]: [收件人]
[From]: [你的角色]
[Date]: [日期]

[1. Introduction]: The purpose of this report is to...
[2. Findings]: 从听力和 Data File 中的事实与数据
[3. Analysis]: 解读这些发现的意义
[4. Recommendations]: 基于分析提出的建议
[5. Conclusion]: 总结核心观点
```

## 五、时间分配建议 (2 小时)

| 时间段 | 任务 |
|---|---|
| 0:00 - 0:03 | 读指示页,快速浏览 Data File 目录 |
| 0:03 - 0:08 | 读 Part A 题目,预读 Data File 相关页面 |
| 0:08 - 0:20 | Part A 听力 (录音播放两次) |
| 0:20 - 0:40 | Part A 写作 (约 200 字) |
| 0:40 - 0:45 | 预读 Part B 题目与 Data File |
| 0:45 - 1:05 | Part B 听力 (录音播放两次) |
| 1:05 - 1:50 | Part B 写作 (约 400 字) |
| 1:50 - 2:00 | 检查全卷 (拼写、语法、漏题) |

**提醒**: 这只是参考分配,实际考试中录音播放时间是固定的,你无法自行调整。

## 六、常见扣分陷阱

### 语法与拼写

- **时态混乱**: 报告用过去式,计划用将来式,当前情况用现在式 — 不要乱跳
- **主谓一致**: 特别是 "The student as well as the teachers is/are..." 这类陷阱
- **冠词遗漏**: the/a/an 的运用,是母语英语者与非母语学习者的区别
- **拼写错误**: 即使是简单单词,拼错也会失分 — 常见错误:receive, separate, necessary, definitely
- **标点符号**: 中文思维下容易忘记英文句号

### 内容与任务

- **角色混乱**: 明明是写信给校长却用了朋友聊天的语气
- **信息抄录**: 直接把 Data File 的句子抄下来 — 这叫 "lifting",会严重扣分
- **任务不完整**: 题目要求写三点,你只写了两点
- **读者意识缺失**: 用术语或缩写,读者未必懂
- **结构不清**: 一大段到底,没有段落分明

### 听力方面

- **听漏关键信息**: 没有抓住数字、名字、日期
- **预读时间浪费**: 开始播音时还没读完题目
- **第二遍没有补齐**: 第一次没听懂就放弃,没有用第二次机会

## 七、内地及非本地考生的特别挑战

### 口音适应

DSE 听力录音使用 **英式或香港式英语发音**,与内地或美式英语存在差异。常见适应要点:

- **母音发音**: 英式 "can't" /kɑːnt/,美式 /kænt/
- **辅音 r**: 英式在词尾通常不发 r 音 (car → /kɑː/)
- **语速**: 日常对话较快,新闻播报较慢
- **语调起伏**: 比普通话更有节奏感

**训练建议**: 每天听 BBC News、TED Talks (英式讲者)、香港电台英文节目,或 DSE 历年真题录音。**至少 20 小时** 的针对性英式听力训练是必要的。

### 中式英文思维

内地生常见的问题:

- 直译中文成英文 (例:"Many peoples think that..." 而非 "Many people think that...")
- 滥用 "very" (应该选择更精准的形容词,如 "exceptionally", "particularly")
- 句式过于简单 (整篇文章都是简单句,缺少复合句与从句)
- 论点直白而不含蓄 (英文议论文偏好用数据和例证说服,而非情感呼吁)

### 语域意识

英文写作对 **formal / informal / neutral** 的区分比中文更严格。DSE 阅卷员会扣分的语域错误:

- 给校长写信用 "I wanna" 而不是 "I would like to"
- 在正式报告里写 "stuff" 而不是 "items" 或 "matters"
- 在演讲稿里突然出现 "kinda" 或 "gonna"

## 八、Level 5、5*、5** 的分水岭

DSE 英文 Paper 3 的分数分布大致如下:

- **Level 4**: 基本完成任务,语法正确率约 70-80%,信息提取合理,结构清晰
- **Level 5**: 任务完成度高,语法几乎无错,有一定的词汇多样性,结构完整
- **Level 5***: 用词精准,句式多样,有角色感,能批判性运用信息
- **Level 5****: 语言近乎母语水平,论证深入,有创意,展现个人声音

从 5 到 5**,关键不在多写多少字,而在 **每一句话的质量**。一篇结构清晰、用词精准、角色鲜明的 300 字文章,比一篇语法混乱、堆砌辞藻的 500 字文章得分更高。

## 九、备考 90 天计划

### 第 1-30 天 (基础巩固期)

- **听力训练**: 每天 30 分钟英式英语听力 (BBC 6 Minute English 是入门好材料)
- **词汇积累**: 每天 20 个学术词汇 (报告、信、演讲的常用词)
- **语法复习**: 重点复习时态、被动语态、从句
- **真题研究**: 做 1-2 套 Part A 真题,熟悉题型

### 第 31-60 天 (综合训练期)

- **整套真题**: 每周做 1-2 套完整 Paper 3,严格计时
- **错题本**: 记录每次听力和写作的错误,分类归纳
- **Sample Answer 研究**: 阅读香港考评局发布的 Level 5** 样卷
- **模拟写作**: 每周写 3-4 篇不同体裁的作文,请老师或同学改

### 第 61-90 天 (冲刺调整期)

- **真题集中攻破**: 做近 5 年真题,分析考点趋势
- **时间管理**: 严格按照考试时间分配练习
- **心态调整**: 不要熬夜,保持作息稳定
- **考前一周**: 只做已做过的题,巩固信心,不尝试新题型

## 十一、结语

Paper 3 是 DSE 英语科中最考验综合能力的一卷 —— 它不只测试你懂不懂英语,更测试你能不能在高压时间内从多源信息中快速提取、整合、重组,并用恰当的语言呈现给指定的读者。对内地及非本地考生而言,它需要的是 **系统性训练** 而非仅靠死记硬背。从听力、词汇、语法到写作框架,每一部分都可以通过明确的练习提升。

如果你正在备考 2026 年的 DSE,请记住:Paper 3 的成绩不是考前一个月冲刺出来的,而是通过几个月的每日积累和大量真题训练实现的。**保持稳定节奏、从错误中学习、模拟真实考试环境**,你会看到自己的稳步进步。

祝考试顺利。
