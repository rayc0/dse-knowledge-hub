## 二、申请门槛对比

### 香港八大（大陆高考生走 Non-JUPAS 内地本科生直收）

- **高考成绩**：一般一本线 +120–150 分，顶尖专业（HKU MBBS / CUHK GBUS / HKUST GBUS / CityU Vet Med 等）需一本线 +180 以上。
- **英语**：高考英语 ≥120/150（部分专业 130+），或 IELTS 6.5（小分 5.5+）/ TOEFL 93+。
- **面试**：绝大多数专业必含面试（详见本站面试题库指南）。
- **申请时间线**：提前批 / 独立招生，2026 入学申请窗口普遍 2025-10 至 2026-06。
- **PS / CV / 推荐信**：近年越来越重要（尤其 HKU / CUHK / HKUST）。

### 荷兰 13 所研究型大学（U-NL）

荷兰不像英美那样统一接收"高考成绩"，路径按院校分三类：

1. **直接录取（小部分大学认高考）**
   - Groningen、Tilburg、Erasmus Rotterdam、Maastricht、Twente、Wageningen 等部分专业接受 Gaokao 550+ / 省级一本线（按省份浮动）。
   - 配合 IELTS 6.5（小分 6.0）或 TOEFL 90+。
   - 所有申请通过 Studielink 全国统一平台。

2. **Foundation Year 衔接**
   - UvA、Leiden、Utrecht、TU Delft、TU/e 的部分本科要求完成 Foundation Year（荷兰本土预科）或已在高中读 IB / A-Level / AP。
   - 适合已有国际课程背景或计划 Gap Year 补预科的学生。

3. **大学学院（University Colleges）——纯英文、Liberal Arts**
   - 如 Amsterdam University College (AUC)、Leiden University College The Hague、Erasmus University College、Utrecht University College、Maastricht University College。
   - 录取方式接近美式：需要 PS + motivation letter + 成绩单 + 面试 + 推荐信。
   - 大陆高考生亦可申，但要求更接近"英美式 holistic review"。

**关键变化：Internationalisation in Balance Act**
- 从 2025/26 学年起，荷兰政府要求研究型大学本科**至少 2/3 学分以荷兰语授课**。
- 2026/27 入学起，部分英文本科项目将引入入学名额限制（selectie 或抽签 loting），对非欧盟学生申请难度显著上升（尤其心理学、传播、商科、国际法等"高曝光"英文本科）。
- 例外：工程类短缺专业、边境地区院校、单一校区项目、本质上国际化的项目（如国际关系、IB）。

### 一句话比较

HK 录取可量化（高考分数 + 英语 + 面试），荷兰录取多元（专业 + 学校 + 背景），2026 年对大陆高考生申请英文本科难度上升，建议提早锁定 **Erasmus / Groningen / Maastricht / Twente** 等对高考友好、英文覆盖度仍高的学校。

## 三、学费与 4 年总成本（2026 数据）

### 香港八大（非本地本科生）

- **学费**：HKU / CUHK / HKUST 2025–26 起升至 HK$200K/年（入学后可能分档），PolyU HK$200K、CityU HK$170K、HKBU / Lingnan / EdUHK HK$140–170K。
- **住宿**：校内宿舍 HK$20K–40K/年（竞争激烈，通常仅保障 Y1 或 Y1+Y2），校外 HK$5,000–9,000/月。
- **生活费**：HK$6,000–10,000/月（餐饮 + 交通 + 通讯 + 个人）。
- **保险/医疗**：HK$1,000–3,000/年。
- **杂项（签证、EEP、机票）**：首年 RMB 5,000–15,000。

**4 年总成本**：
- 顶尖校（HKU/CUHK/UST）≈ RMB 140–160 万
- 中档校（PolyU/CityU）≈ RMB 120–140 万
- 三档校（HKBU/Lingnan/EdU）≈ RMB 105–125 万

### 荷兰（非欧学士 Institutional Fee）

- **学费**：
  - 社科/人文：€9,500–€13,000/年（如 Erasmus BA €11,500、Groningen BA €11,500）
  - 商科/经济：€11,000–€16,000/年（如 Tilburg IBA €12,500、UvA Economics €16,500）
  - 理工/CS：€13,000–€19,000/年（如 TU Delft €19,000、TU/e €19,000、UT €14,500）
  - 医学/建筑/特殊专业：€20,000–€24,000/年（仅极少数接受非欧学生）
- **住宿**：合租 €500–€900/月（阿姆斯特丹、乌特勒支偏贵，格罗宁根、马斯特里赫特便宜）。
- **生活费**：€300–€500/月（餐饮 + 交通 + 个人），IND 建议财务证明 €1,130.77/月。
- **健康保险**：学生 Dutch public health insurance €120–€160/月（强制；部分学生保险如 AON Student Insurance €50–€70/月可替代）。
- **杂项（MVV/VVR、机票、初设）**：首年 RMB 15,000–25,000。

**4 年 / 3 年总成本**（按典型本科 3 年制计算；部分研究型大学 liberal arts colleges 为 3 年）：
- TU Delft / TU/e 工程 3 年 ≈ RMB 65–85 万
- UvA / Leiden / Utrecht 社科 3 年 ≈ RMB 55–75 万
- Groningen / Tilburg / EUR 商科 3 年 ≈ RMB 60–80 万
- 若需 Foundation Year + 3 年本科 ≈ 额外 RMB 15–25 万

### 一句话比较

**荷兰本科 3 年总成本约为香港的 40–60%**，工程类（TU Delft / TU/e）毕业生起薪甚至高于 HK 本地本科毕业生，是欧盟内性价比最高的英语本科目的地之一。

## 四、毕业后签证与工作路径

这是两地差异最大的决策变量。

### 香港：IANG（Immigration Arrangements for Non-local Graduates）

- **时长**：毕业后自动获得 24 个月无条件逗留。
- **薪资要求**：无门槛。
- **工作类型**：任何工作、任何薪资、甚至自雇、创业、gap year 均可。
- **延期**：24 个月内找到工作后可按普通工作签证机制逐年延期。
- **PR 路径**：连续居住 7 年 → 香港永居（Right of Abode）。
  - 典型路径：4 年本科 + 3 年 IANG/工作 = 7 年完成 PR。

### 荷兰：Zoekjaar（Orientation Year / Residence Permit for Orientation Year）

- **时长**：12 个月（不可延期、不可续签；一生仅可用一次）。
- **薪资要求**：Zoekjaar 期间无薪资门槛，任何工作均可。
- **工作类型**：任何工作、任何时长（无每周小时限制）、无需雇主 sponsor。
- **申请时间**：毕业（文凭日期）后 3 年内任意时间申请（非必须毕业当月）。
- **申请费**：€254。
- **财务要求**：约 €1,200–€1,500/月自证能力。
- **后续签证**：Zoekjaar 期内找到 HSM（Highly Skilled Migrant）工作 → 转 HSM 工签
  - **2026 HSM Zoekjaar 毕业生薪资门槛**：€3,122/月（不含 8% 假日津贴）——比正常 HSM 门槛（30 岁以下 €3,613/月 / 30 岁以上 €5,331/月）低约 13–40%。
- **PR 路径**：HSM 持有 + 连续居住 5 年 + 荷兰语 A2 → 荷兰永居。

### 核心比较

| 维度 | IANG (HK) | Zoekjaar (NL) |
|---|---|---|
| 时长 | 24 个月 | 12 个月 |
| 薪资门槛 | 无 | 无（Zoekjaar 期间） |
| 转正式工签门槛 | 一般工签，按雇主申请 | HSM €3,122/月（2026 毕业生优惠） |
| PR 年限 | 7 年（含 4 年 BA） | 5 年 HSM + A2 荷兰语 |
| 身份总时长 | 4 + 3 = 7 年 | 3 + 1 + 5 = 9 年（典型） |
| 语言难度 | 英语即可 | 荷兰语 A2（PR 必过） |

**编辑观点**：若以"拿到长期身份"为目标，HK 的 7 年路径在时间和语言门槛上都更宽松；若以"留欧工作 + 欧盟流动性"为目标，荷兰 Zoekjaar + HSM 的灵活度更高（HSM 持有人家属可直接入境工作，且 5 年后即可全欧通行蓝卡路径）。

## 五、专业竞争力对比

### 荷兰强项

- **工程 / 航天 / 水利 / 建筑**：TU Delft（QS Civil #4、Architecture #2–5）、TU/e、UT 集中度高，水利工程世界第一。
- **商科 / 金融**：Tilburg、Erasmus RSM、UvA Business School（RSM 多个项目 FT 欧洲排名前 10）。
- **农业 / 食品 / 生命科学**：Wageningen（QS Agriculture & Forestry 多年世界第一）。
- **国际法 / 国际关系**：Leiden、Utrecht（荷兰海牙国际法传统）。
- **医学研究**：Erasmus、LUMC、AMC（研究导向强，但非欧学生本科临床医学基本无名额，需经 Master 进入研究路径）。
- **经济学**：Tilburg、EUR、UvA（Tinbergen Institute 欧洲顶级经济学培训基地）。

### 香港强项

- **金融 / 商科**：HKU、CUHK、HKUST 商学院亚洲前 3，Big 4 / IBank / PE / HF 的亚洲总部在 HK。
- **法律 / 法学 LLB**：HK 普通法司法辖区 + 粤港澳大湾区法律服务需求。
- **酒店管理 / 旅游**：PolyU SHTM 全球前 15。
- **医学 / 牙医**：HKU、CUHK 本科临床医学（MBBS / MBChB）接受大陆高考生，且毕业即可在 HK 执业；荷兰本科临床医学无此路径。
- **数据科学 / AI**：CityU SDSC、HKUST AI、HKU Data Science 在亚洲持续上升。
- **传媒 / 创意产业**：HKBU 传理学院亚洲老牌。

### 一句话比较

- 工程 / 水利 / 建筑 / 农业 → 荷兰
- 金融 / 法律 / 酒店 / 本科医学 → 香港
- 商科 / 经济 → 两地平分，但荷兰 3 年制 + 性价比更高
- CS / 数据科学 → 两地均强，取决于目标就业地（亚洲 HK / 欧洲 NL）

## 六、大陆学生签证流程对比

### 香港学生签证（2026）

- **签证费**：HK$330（2025-09-08 起）
- **2026 新增**：HK$800 Student Visa Handling Fee（仅适用 deferment / reinstatement / expiry 后重新申请，首次申请不适用）
- **处理时长**：ImmD 6–10 周（视学校 sponsor）
- **方式**：学校 sponsor + 电子签证 e-Visa + 大陆 EEP 加 D 签注
- **财务证明**：建议学校官方 RMB 33–50 万
- 详见本站"香港八大学生签证申请完整流程 2026"

### 荷兰学生签证 MVV + VVR（2026）

- **签证费**：€207（大使馆）+ IND 学生许可费（含在学校打包费用中）
- **财务证明**：€1,130.77/月 × 12 个月 = €13,569/年（约 RMB 10.5 万）
- **处理时长**：IND 60 天法定，实际 4–8 周
- **方式**：学校作为 IND 认可 sponsor → 提交 TEV 合并申请 → 大使馆贴 MVV 入境贴纸 → 抵荷后 BSN 注册 + VVR 领卡
- **特殊要求**：Nuffic Certificate 自 2022 年起对中国学生 MVV 申请**不再强制**，但部分学校在学术录取时仍可能要求 CSSD（教育部学信网）认证报告
- **TB 检测**：中国大陆学生可**免除**TB 测试（长期豁免名单）

### 一句话比较

HK 全流程约 8–10 周 + EEP 签注 2–3 周；NL 全流程约 8–12 周 + 使馆 MVV 贴签 2–4 周。两地都要求学校作为 sponsor，复杂度相当。

## 七、家庭决策场景（5 类）

### 场景 A：985/211 目标 + 7 年身份导向家庭
- **推荐**：HK 八大（HKU / CUHK / HKUST 优先）
- **原因**：4 + 3 = 7 年 PR 路径清晰、全程英语、亚洲职业网络
- **预算**：RMB 130–160 万

### 场景 B：预算紧张 (60–80 万) + 英语授课偏好
- **推荐**：荷兰 Groningen / Tilburg / EUR / Maastricht 3 年本科
- **原因**：3 年制 + 非欧学费低于 €15,000 + 毕业后 Zoekjaar 12 个月机会
- **预算**：RMB 55–80 万
- **风险**：2026/27 起英文本科名额收紧，须早申

### 场景 C：工程 / 理工背景 + 计划留欧
- **推荐**：TU Delft / TU/e / UT（荷兰理工三校）
- **原因**：世界顶级工程教育 + 欧洲短缺专业 HSM 门槛低 + 毕业生起薪强
- **预算**：RMB 65–85 万（3 年）
- **备注**：TU Delft 2026/27 入学英文本科仍有保留（工程类属短缺专业豁免）

### 场景 D：医学 / 法学 / 酒店管理 / 本地执业导向
- **推荐**：HK 八大
- **原因**：荷兰本科临床医学不对非欧开放；法学须普通法体系匹配未来执业
- **预算**：RMB 140–160 万

### 场景 E：计划本科 → 美国研究生衔接
- **推荐**：荷兰 3 年 BA + 美国 MA
- **原因**：荷兰本科便宜 + Liberal Arts Colleges (AUC / UCU / EUC) 类美式 GPA 系统易衔接 + 整体成本仍低于"HK BA + US MA"
- **预算**：本科 RMB 55–75 万 + 美研 RMB 80–150 万 = 总 135–225 万
- **备注**：若目标美国研究生直接就业 → HK BA + US MA 更稳妥（HK 本科美式评估更成熟）

## 九、常见问题 (FAQ)

**Q1：大陆高考生能不能直接申请 TU Delft / UvA 本科？**
A：TU Delft 2026 部分英文本科（如 Aerospace Engineering、Computer Science）接受高考 + IELTS 组合，但要求高考总分约 600+ 且数学 ≥135/150。UvA 多数英文本科仍要求 IB / A-Level / AP，单独高考一般不足，建议考虑 Foundation Year 或 AUC 大学学院路径。

**Q2：荷兰"2/3 学分荷兰语授课"新规是否影响 2026 入学？**
A：2025/26 入学**不受影响**；**2026/27 入学**起，新生部分英文本科可能被撤销或引入 selectie/loting。建议：若目标荷兰英文本科，**2026-01 前尽早通过 Studielink 提交（1 月 15 日法定截止日之前）**，并优先选短缺专业豁免名单内的工程/农业类项目。

**Q3：Zoekjaar 12 个月够不够找工作？**
A：足够（荷兰互联网+工程市场 IT/数据/机械平均招聘周期 6–12 周），但仅限于 HSM 类岗位（Top 500 国际化雇主为主）。对于本土荷兰语岗位（销售、行政、教育），非荷兰语母语的中国毕业生实际获 offer 率较低。

**Q4：3 年制荷兰本科文凭能否做留服认证？**
A：可以。3 年制欧洲本科符合"博洛尼亚进程"（European Higher Education Area）标准，中国教育部留学服务中心（现 CSCSE）接受 3 年制 BSc/BA 为本科学位认证，但要求学位在 Nuffic 数据库（DUO）可查。

**Q5：Gaokao 考不到一本线还能申请荷兰吗？**
A：部分学校接受省级一本线以下但达成全国 550+ 的学生（如 Groningen 部分文科项目），或通过 Foundation Year 进入。但 HK 八大**最低门槛即一本线**，两地要求不同：NL 灵活但专业受限，HK 分数硬门槛但专业开放。

**Q6：荷兰 HSM €3,122/月门槛大陆本科生毕业能达到吗？**
A：TU Delft / TU/e 工程毕业生起薪普遍 €3,500–€4,200/月，达标无难度。但社科 / 人文 / 商科 Graduate Trainee 起薪可能 €2,800–€3,300/月，处于临界线上，需精挑行业（咨询、银行、四大审计对中国毕业生较友好）。

**Q7：留荷 5 年 PR vs 留港 7 年 PR，哪个更快？**
A：表面上 NL 5 年 PR 比 HK 7 年快，但 NL 计算是从"全职 HSM 起算"，即 4 年 BA 不计入 PR 年限；实际 BA 入境到 PR 约 **9 年**（4 本科 + 1 Zoekjaar + 5 HSM）。HK 是 **7 年**（4 本科 + 3 IANG/工作）。**HK 实际上更快拿到 PR**。

**Q8：荷兰本科能转到英国/美国吗？**
A：可以。荷兰本科 3 年学士学位被英美院校广泛承认（博洛尼亚兼容），转学或申请 Master 无障碍。相反 HK 本科也广泛受认可，但 HK 的 4 年 GPA 系统与美国更接近，Master 申请时略占优势。

**Q9：荷兰的 Nuffic Certificate 还需要办吗？**
A：**MVV 签证申请不再需要**（2022 起 IND 取消强制要求）。但部分学校（如 Erasmus、Utrecht 的某些项目）在**学术录取阶段**仍可能要求中国 CSSD / 学信网英文认证报告，费用约 RMB 200–400/份。

**Q10：荷兰生活费真的比香港便宜吗？**
A：住宿层面——阿姆斯特丹/乌特勒支合租 €700–€900/月 ≈ HK$6,000–7,500/月，和 HK 市区合租相当；格罗宁根/马斯特里赫特 €450–€600 明显便宜。餐饮层面——荷兰自炊 €150–€200/月，HK 自炊 HK$2,500 ≈ €280，NL 便宜 30%。整体生活成本 NL 比 HK 低 **15–25%**。

## 十、决策建议（30 秒版）

1. **若家庭预算 ≤ 80 万 RMB 且能接受 3 年制** → 荷兰 Groningen / Tilburg / EUR / Maastricht
2. **若目标工程 / 水利 / 农业 / 国际法** → 荷兰 TU Delft / TU/e / UT / Wageningen / Leiden
3. **若目标金融 / 法律 / 酒店 / 本科医学** → 香港八大
4. **若目标 PR 身份 + 亚洲职业** → 香港（7 年更快）
5. **若目标欧盟流动性 + 工程就业** → 荷兰（Zoekjaar + HSM 组合）
6. **若计划本科 → 美国研究生** → 荷兰 3 年 BA + 美国 MA 组合（总成本更低）

两地均为"性价比梯队"内的优质选择，**不是互斥而是互补**：不少大陆家庭采用"HK 本科 + 荷兰硕士"或"荷兰本科 + HK/新加坡硕士"的双地组合，兼顾亚欧双市场。
