# DSE 放榜日行动指南 2025：成绩公布当天的每小时行动清单、JUPAS 操作与心态准备

DSE 放榜日是香港每年7月中旬最紧张的一天。成绩公布的那一刻，决定了接下来几天的关键决策——接受哪个大学录取？更改哪些 JUPAS 志愿？是否需要启动非联招（Non-JUPAS）申请？是否要考虑重考？

本指南将带你逐小时走过放榜日，从准备文件、查看成绩、分析 JUPAS Band 情况，到应对各种不同场景的具体行动，以及心理调适建议。

---

## 一、放榜前的准备：前一晚该做好什么

放榜日本身已经足够紧张，提前一晚做好准备能让你在成绩公布后更快速、冷静地做出决定。

### 文件清单（提前准备好以下材料）

- [ ] DSE 考生号码（Candidate Number）和身份证
- [ ] JUPAS 登录账号和密码（提前测试能否正常登入）
- [ ] 历年各大学课程收生分数参考（从 JUPAS 官网或升学辅导资料下载）
- [ ] 自己填写的 JUPAS 志愿表复印件（知道自己填了什么课程）
- [ ] 非联招申请课程的截止日期列表（若有提前研究）
- [ ] 联络老师/升学顾问的方式（放榜日老师通常全天可联系）

### 计算你的预期成绩区间

放榜前一晚，根据考后感受和校内模拟试成绩，粗略估计：
- 你的最佳五科加一（Best 5+1）可能落在什么范围
- 各志愿课程的历年收分是多少
- 哪些志愿最可能是 Band A、Band B、Band C

这样，当成绩真正出来时，你不会完全措手不及，而是能立即对照预期进行分析。

### 心理准备

无论结果如何，成绩只是今天的第一步，而不是最后的结论。提醒自己：
- 有 E-application 改志愿机制（放榜当天开始）
- 有非联招并行路线
- 有副学士、IVE 等合理替代方案
- 香港每年有相当比例的同学通过非传统路线进入心仪大学

## 二、成绩公布：放榜当天上午（约 8:00-9:00）

### 成绩公布方式与时间

**2025 年放榜日约为 7 月中旬**（精确日期以 HKEA 官方公告为准）。

成绩通常在早上 **8:00 至 9:00** 之间在以下渠道公布：

1. **HKEA 官方网站**（hkeaa.edu.hk）：输入考生号码和出生日期查询
2. **HKEA 官方手机 App**（DSE 成绩查询）：提前下载并登录
3. **学校现场**：成绩单派发（如学校安排集体放榜）
4. **邮政**：实体成绩单寄出（较慢，当天未必能收到）

**建议**：使用网站和 App 双管齐下，避免服务器拥堵时无法查看。

### 如何读懂成绩单

DSE 成绩以等级表示：

| 等级 | 说明 |
|------|------|
| 5** | 最高级别 |
| 5* | 优秀 |
| 5 | 良好 |
| 4 | 达标（大学基本入学要求通常为 4） |
| 3 | 中等 |
| 2 | 基本 |
| 1 | 不达标 |
| U/Level 1以下 | 不予评级 |

**大学基本入学要求（最低门槛参考）**：通常为中文3、英文3、数学2、通识/公民与社会发展2，加上两科选修，各科要求按院校不同而异。

### 计算最佳五科加一（Best 5+1）

大部分大学按"最佳五科（必包含中英文）+ 一科"计算入学成绩，或各有独特计分方式。计算步骤：

1. 列出所有科目成绩（含 M1/M2 如有修读）
2. 中文和英文必须包含在内
3. 从剩余科目中选出分数最高的科目，组成最佳五科
4. 第六科（+1）通常是选修科中的最高分
5. 若修读 M1/M2，其成绩可计为独立科目

**将你的分数与各志愿课程的历年收分比较**，初步判断自己属于哪个 Band。

## 三、JUPAS 情景分析树：根据 Band 决定行动

### 情景 A：所有主要志愿均为 Band A

**恭喜！这是最理想的情况。**

行动清单：
- [ ] 确认第一志愿是真正最想就读的课程
- [ ] 等待 JUPAS 官方录取通知（通常在放榜后约两周公布）
- [ ] 若对其中一个课程有疑问，可利用 E-application 微调志愿次序
- [ ] 留意各大学迎新活动和入学注册截止日期

**需要注意**：Band A 不等于 100% 录取，只是代表你在历年主要收生范围内。若竞争特别激烈，仍有少数 Band A 考生未获正式录取，但这种情况相对罕见。

---

### 情景 B：第一志愿 Band A，但其他志愿也有 Band B/C

**相对稳健，主要等待第一志愿结果。**

行动清单：
- [ ] 重点关注第一志愿是否能成功录取
- [ ] 利用 E-application 窗口，将 Band B 志愿调整次序（确保保底课程排在后面）
- [ ] 不需要大动作调整，除非第一志愿课程你有所犹豫

---

### 情景 C：主要志愿均为 Band B（边缘线）

**这是最需要主动应对的情况。**

Band B 代表你在历年边缘区域，有机会被录取，但不确定。核心策略：

**立即行动（放榜当天）：**
- [ ] 登入 JUPAS 查看 Band 情况
- [ ] 对照历年收分，计算自己与各课程 Band A 的分数差距
- [ ] 找出分距最小的 Band B 课程，这是最有机会被录取的
- [ ] 打开 E-application，准备重组志愿

**E-application 志愿重组建议：**
1. 删除或降级明显无望的 Band C 志愿
2. 将"分距最小的 Band B"提升至第一志愿位
3. 加入原本没有的保底课程（副学士衔接、IVE 课程）
4. 确保保底课程你真的愿意就读

**平行行动：非联招申请**
- 检查各大学是否有放榜后非联招名额
- PolyU、CityU 等有时在放榜后开放补充申请
- 若有符合资格的国际考试成绩（IB、SAT），可同步申请

---

### 情景 D：主要志愿均为 Band C（希望渺茫）

**需要迅速启动备用计划。**

Band C 代表你的成绩明显低于课程历年收分，被录取概率很低。这种情况需要更大幅度的志愿调整。

**立即行动：**
- [ ] 对 JUPAS 志愿进行全面重组——基本上是重新填写
- [ ] 研究副学士课程（香港专上学院 HKCC、HKUSPACE、CityU SCOPE 等）的收生要求
- [ ] 研究 IVE 课程（职业训练局 VTC 提供，入学要求较低）
- [ ] 联系学校升学辅导老师，当天约见或电话咨询

**当天需要开始行动的平行申请：**

| 方向 | 说明 | 截止日期参考 |
|------|------|------------|
| PolyU HKCC 副学士 | 放榜后开放申请，入学要求较低 | 约放榜后 1-2 周 |
| HKUSPACE 副学士 | 多个课程在放榜后接受申请 | 约放榜后 1-2 周 |
| IVE 高级文憑 | VTC 旗下，工程/IT/设计等实务课程 | 联合申请另有时间表 |
| CityU SCOPE | 部分课程放榜后接受申请 | 视课程而定 |

---

### 情景 E：成绩严重不达标，连副学士也有困难

这种情况下，选项收窄，但仍有出路：

- **重考 DSE**：下一届 DSE 约在来年 4-5 月，需以私校生身份报考
- **海外升学**：部分国家（如澳大利亚、英国）的基础课程（Foundation Programme）接受 DSE 成绩较低的学生
- **工作后再进修**：先在职场累积经验，再通过工作经验申请副学士或学士课程

## 四、非联招（Non-JUPAS）申请：放榜当天的并行操作

许多考生在放榜日只顾 JUPAS，忘记检查非联招渠道。若 JUPAS 结果不理想，当天应同步检查：

### 哪些大学在放榜后接受非联招申请？

各大学每年政策不同，以下为参考（以往年情况为基准）：

- **香港理工大学（PolyU）**：部分课程在放榜后仍接受 Non-JUPAS 申请，尤其是课程名额未满的情况
- **香港城市大学（CityU）**：视年份，有时开放 Clearing 名额
- **香港浸会大学（HKBU）**：部分创意艺术课程有放榜后申请窗口
- **香港中文大学（CUHK）**：通常较严格，放榜后名额有限

### 非联招申请通常需要什么文件？

- DSE 成绩通知书（当天即可获得）
- 身份证复印件
- 个人陈述（Personal Statement）——这是最花时间的部分，建议提前准备草稿
- 推荐信（部分课程要求）——提前联系老师

**提醒**：放榜后非联招申请窗口通常只有 3-7 天，需要迅速行动。

## 五、若考虑重考 DSE：评估框架

重考 DSE 是一个重大决定，不应在放榜当天仓促决定。以下是评估框架：

### 适合重考的情况

- 目标课程明确且需要特定成绩（如医科、法律），差距在 1-2 个等级内
- 有强烈意愿投入一年时间认真备考
- 有家庭支持（经济和心理层面）
- 能分析出上次考试失误的具体原因，并有针对性改善方法

### 不适合重考的情况

- 只是因为没有更好选择，缺乏具体改善计划
- 对大学课程方向仍不明确
- 家庭经济压力较大（重考通常需要补习费用）
- 心理上已精疲力竭，难以再支撑一年高压备考

### 重考 DSE 的技术要点

- 以私校生（School Candidate）身份报考，并非所有科目都可重考（请查阅 HKEA 官方规定）
- 重考成绩与历次成绩并非自动取最高，具体政策需查阅当年 HKEA 规定
- 报名通常在 12 月（为来年 4-5 月的考试）

## 六、国际大学申请作为替代路线

若 DSE 结果不理想，但手持优秀英文成绩，海外大学是值得认真考虑的选项：

### 英国

- UCAS 系统，DSE 5*/5 通常对应 A Level A*/A 等级
- 部分大学（如曼彻斯特、爱丁堡、布里斯托）在放榜后仍有 UCAS Clearing 名额
- 英国大学学位通常 3 年，学费每年约 9,000-38,000 英镑

### 澳大利亚

- 直接以 DSE 成绩申请，换算为 ATAR 分数
- 墨尔本、悉尼、昆士兰等主要大学接受 DSE 申请
- 有 Foundation Programme 选项，适合成绩偏低的学生

### 加拿大

- 多伦多、UBC、麦吉尔等大学接受 DSE 申请
- 申请通常在 12 月至翌年 3 月，放榜后一般已截止，需提前规划

### 台湾

- 港澳学生可通过联合分发委员会申请台湾大学
- 录取参考 DSE 成绩，台湾顶尖大学（台大、成功大学等）具竞争力

## 七、心理支援：放榜日的情绪管理

无论结果如何，放榜日都是极高压力的一天。以下是心理调适建议：

### 结果好于预期

即使成绩超出预期，也需要冷静决策。高兴是应该的，但不要在兴奋状态下仓促接受录取——确认这是你真正想就读的课程。

### 结果低于预期

- **允许自己有情绪**：悲伤、失落、愤怒都是正常的反应，不需要立即假装没事
- **给自己 1-2 小时的情绪缓冲**，然后开始理性分析选项
- **避免将成绩等同于自我价值**：DSE 成绩是特定时刻的特定表现，不代表你的能力上限
- **联系信任的人**：家人、朋友、老师——倾诉比独自承受有效得多

### 若感到难以承受

如果情绪影响到日常功能，或有任何伤害自己的想法，请立即联系：

- **撒玛利亚防止自杀会热线**：2389 2222（24 小时）
- **青少年情绪支援热线 Open Line**：27820088
- **社会福利署热线**：2343 2255

## 八、完整行动清单：放榜日逐小时安排

### 07:00-08:00：最后准备

- [ ] 确认 JUPAS 登录信息
- [ ] 打开 HKEA App 或网站（提前测试可否登入）
- [ ] 准备计算纸和历年收分参考资料

### 08:00-09:00：查看成绩

- [ ] 成绩一出，立即记录各科等级
- [ ] 计算最佳五科加一分数
- [ ] 初步比较各志愿的 Band 情况
- [ ] 告知家人成绩，稳定情绪

### 09:00-11:00：分析与规划

- [ ] 对照历年收分，判断各志愿的实际录取概率
- [ ] 联系升学辅导老师（大多数学校当天安排辅导）
- [ ] 研究 E-application 需要做哪些调整
- [ ] 查阅非联招申请的当天状态

### 11:00-15:00：执行调整

- [ ] 进入 JUPAS E-application 系统，开始更改志愿
- [ ] 若需申请非联招课程，开始准备文件
- [ ] 若考虑副学士，联系相关院校查询

### 15:00-18:00：确认与等待

- [ ] 确认 E-application 修改已成功提交（截止通常在放榜后数天内，但越早越好）
- [ ] 检查非联招申请进度
- [ ] 休息，与家人、朋友聚会，放松身心

### 18:00 之后：充电

- [ ] 今天已做了能做的事，接下来等待 JUPAS 正式录取通知
- [ ] 不需要再做什么——过度焦虑不会改变结果
- [ ] 计划接下来几天的后续行动（JUPAS 录取通知约在两周后）

## 九、放榜后的时间线

| 事项 | 时间（参考） |
|------|------------|
| DSE 放榜 | 7 月中旬 |
| E-application 改志愿 | 放榜当天起至约 4-5 天后 |
| JUPAS 第一轮正式录取通知 | 约 7 月底至 8 月初 |
| 接受录取截止 | 约正式通知后 5-7 天 |
| 各大学迎新及注册 | 8 月 |
| 学期开始 | 9 月 |

---

## 小结

放榜日的核心是：**快速分析、冷静决策、多线并进**。

成绩出来后的几小时内，你需要：读懂成绩、比较历年收分、判断 Band 情况，并在 E-application 窗口期内做出合理的志愿调整。与此同时，若有需要，平行启动非联招或副学士申请。

记住：放榜日是开始，不是终点。无论今天的结果如何，香港的升学体系提供了多条路线——关键是在正确的时间采取正确的行动。

---

*本文资讯以 2025 年为参考基准，具体日期及政策以 JUPAS、HKEA 官方公告为准。如有任何疑问，请直接联系相关大学招生办公室或学校升学辅导老师。*
