# 香港大学工程学科完全指南——内地生视角

工程学（Engineering）是香港高校最受内地学生欢迎的专业之一。香港理工大学、香港科技大学在全球工程院校排名中名列前茅，且毕业生在大湾区科技企业就业优势显著。本指南全面解析香港各大工程学院的课程、入学要求、科研机会及职业发展路径。

## 一、主要院校及工程系别概览

### 1.1 香港科技大学（HKUST）——工学院（School of Engineering）

HKUST工学院是全球工程教育的顶尖学府：

| 学系 | 主要专业 |
|---|---|
| 计算机科学与工程（CSE） | 人工智能、软件工程、网络安全 |
| 电子及计算机工程（ECE） | 芯片设计、嵌入式系统、通信 |
| 化学及生物工程（CBE） | 新材料、生物医学工程、能源化工 |
| 土木及环境工程（CIVL） | 基础设施、智慧城市、环境工程 |
| 工业工程及决策分析（IEDA） | 工程管理、运筹学、数据分析 |
| 机械及航空工程（MAE） | 无人机、机器人、汽车工程 |

**QS工程排名（2025）：** HKUST约全球30—40名，亚洲顶5名

**特色：**
- 研究型大学，本科生有大量接触前沿研究的机会
- 与内地及国际顶尖科技企业有密切产学研合作
- HKUST广州校区的设立进一步强化大湾区影响力

### 1.2 香港大学（HKU）——工程学院（Faculty of Engineering）

| 学系 | 特色 |
|---|---|

| 电机及电子工程（EEE） | 智能电网、光子学、通信 |
| 土木工程（Civil） | 香港大型基建项目合作 |

**特色：**
- 与香港建造业、政府基建项目有紧密联系

## 四、DSE选科要求

### 4.1 通用DSE入学要求

| 科目要求 | 说明 |
|---|---|
| 数学必修 | **Level 4以上**（几乎所有工程课程要求） |
| 数学延伸M2 | **强烈建议/部分课程要求**（微积分是工程基础） |
| 物理（Physics） | **Level 3—4以上**（大多数工程方向要求） |
| 化学（Chemistry） | 化学/材料/生物医学工程要求，其他工程有帮助 |
| 英文 | **Level 3以上**（教学语言以英文为主） |

### 4.2 各工程方向推荐组合

| 工程方向 | 最佳DSE组合 | 特别说明 |
|---|---|---|
| 计算机/软件工程 | 数学+M2+物理（或ICT） | M2是微积分基础，不可或缺 |
| 电气/电子工程 | 数学+M2+物理 | 电磁学和电路学需要数学基础 |
| 机械工程 | 数学+M2+物理 | 流体力学/热力学基础 |
| 土木工程 | 数学+物理（+地理有帮助） | 结构分析需要数学 |
| 化学工程 | 数学+化学+物理 | 化学是必须 |
| 生物医学工程 | 数学+生物+化学（+物理） | 生物背景很重要 |

## 六、4年制 vs 5年制课程

| 维度 | 4年制工程学士 | 5年制（含一年交换/实习） |
|---|---|---|
| 常见于 | HKU、HKUST、CityU | PolyU（部分课程+工业实习年） |
| 优点 | 较快毕业，节省学费 | 获取实习经验，竞争力更强 |
| 缺点 | 无系统实习年 | 多一年学费生活费 |
| 香港工程学会认可 | 均可申请Graduate Membership | 实习年有助更快取得专业资格 |

### 6.1 工程专业资格（香港工程师学会）
- 毕业后可申请**香港工程师学会（HKIE）研究生会员（Graduate Member）**
- 积累4年工作经验（工程相关）后可申请**Corporate Member（企业会员）**，相当于注册工程师资格

## 九、内地生工程学备考建议

1. **数学M2是关键**：所有顶尖工程课程的本科阶段都会用到微积分，M2成绩越高越好
2. **物理必须扎实**：DSE物理Level 4以上是基本门槛，内地生物理基础通常扎实，应充分发挥
3. **英文不可忽视**：工程课程几乎全英文教学，英文读写能力直接影响学习质量
4. **了解ICT（资讯及通讯科技）**：若有志于CS/软件工程，ICT选修有参考价值
5. **提前了解UROP**：HKUST的UROP是本科生加入研究室的主要渠道，了解规则有助于入学后快速参与

## 十、总结

香港工程学科提供了在国际顶尖排名院校接受英文工程教育、同时毗邻大湾区高科技产业集群的独特优势。内地学生在数理基础方面普遍具有竞争力，需要重点强化英文能力和适应英文工程教学环境。HKUST是最具国际声誉的选择，PolyU在土木工程领域全球领先，HKU和CityU则提供更加灵活的跨学科工程教育。
