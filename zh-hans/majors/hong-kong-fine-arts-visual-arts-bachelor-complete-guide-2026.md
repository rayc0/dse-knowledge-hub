# 香港艺术学士完整指南 2026：中大艺术系、浸大视觉艺术、教大艺术教育

在香港修读纯艺术（Fine Arts）或视觉艺术（Visual Arts）本科，并不像建筑、设计或时装那样需要走专门工程或设计学院路线。香港八大里真正提供「以创作为本」本科艺术学士的，主要是三所：**香港中文大学艺术系（Department of Fine Arts）**、**香港浸会大学视觉艺术院（Academy of Visual Arts）** 与 **香港教育大学视觉艺术教育（BA in Visual Arts Education）**。这三所定位完全不同：中大走学术性强、传统书画与现代艺术并重的「学院派」；浸大走当代艺术、跨媒介实验、艺术与社会的「工作室派」；教大走艺术教育结合，培育中小学美术老师与艺术教育工作者。

本文将按 2026 年最新 JUPAS 资讯，深入比较三所课程的收生要求、作品集与面试、课程结构、师资与设施、海外交换与毕业出路，帮助内地与香港同学准确判断哪所适合自己。

## 二、中大艺术文学士（JS4044）

### 1. 学院定位与历史

中大艺术系成立于 1957 年，是香港历史最悠久的大学艺术系，亦是华人地区少数同时教授传统中国书画与西方现代艺术理论的学术单位。中大艺术系采取「**学院派**」教学理念：要求同学既能创作，也要能写论文、做艺术史研究。

### 2. 收生要求与计分

- **最低门槛**：DSE 4 核心 + 2 选修，达大学最低入学要求即可投考
- **必修科**：英文 Level 3、中文 Level 3、数学 Level 2、公民与社会发展达标
- **优先科**：英文 + 中文权重较高；艺术相关选修科（如视觉艺术、文学）会被纳入计算
- **Band A 优先**：把 JS4044 放在 Band A 是进入面试名单的前提
- **历年中位收生分**：约 21–23 分（Best 5），2024–2025 年度有上升趋势

### 3. 作品集要求

中大艺术系作品集规定相对宽松，但内容审查严谨：

- **格式**：PDF，最多 10 页，5MB 以下
- **数量**：建议 8–15 件作品
- **媒介**：不限——可包括素描、油画、水墨、版画、雕塑、摄影、装置、数码艺术
- **关键**：作品需有「**思路演变**」与「**自我反思**」。中大评审看重思考过程而非仅技巧
- **替代方案**：若实在没有创作，可改交 personal statement 详述对艺术的认识与学习计划

### 4. 课程结构（4 年）

- **第一年**：艺术史导论（中国 + 西方）、素描、水墨基础、视觉文化
- **第二年**：进阶绘画、雕塑、版画、艺术理论
- **第三年**：分流——传统中国书画 / 现代媒介 / 艺术史研究
- **第四年**：毕业创作（Graduation Exhibition）+ 毕业论文（Graduation Thesis）

中大艺术系的特色是同时要交「**创作作品**」和「**学术论文**」，这与浸大只交毕业展览不同。

### 5. 设施与师资

- 中大文物馆（Art Museum）馆藏 17,000+ 件中国文物，本科生可在馆员指导下做研究
- 师资以艺术史学者为主，亦有水墨、版画、雕塑创作型教授
- 与中大新亚书院、敦煌研究合作的暑期实习项目

### 6. 出路

中大艺术系毕业生约 30% 升读硕士（M.A. in Fine Arts、海外艺术史 MA）、20% 进入博物馆与画廊业、25% 转入创意产业（出版、文化策展、媒体），其余进入广告、设计或考 PGDE 当美术老师。

## 三、浸大视觉艺术文学士（JS2810）

### 1. 学院定位与建筑

浸大视觉艺术院（Academy of Visual Arts，AVA）成立于 2005 年，过去多年以前皇家空军啟德军营（Kai Tak Campus）为家——这座二级历史建筑本身就是艺术品。2024 年起，浸大已规划新校园发展，但视艺院仍以 **大型工作室空间** 与 **跨媒介设备**（陶瓷窑、版画机、金工、木工、影像录音棚）著称。

### 2. 收生要求与计分

- **最低门槛**：DSE 4C + 2X，达大学最低要求
- **必修科**：英文 3、中文 3、数学 2、公民达标
- **计分公式**：Best 4 或 Best 5，**视觉艺术 / Music / 体育等艺术相关选修加权**
- **Band A 优先**：必须 Band A，否则不会获邀面试
- **历年中位收生分**：约 19–22 分（Best 5）

### 3. 作品集 + 实作测试 + 面试

浸大视觉艺术院是香港八大里**对作品集要求最严格、流程最完整**的：

#### 第一关：网上作品集
- 透过 AVA Online Portfolio System 提交
- 一般要求 10–15 件作品，附短述（artist statement）
- 媒介不限，但建议展示**多元媒介**：手绘、油画、混合媒介、数码、3D、影像
- **关键提示**：浸大重视「**艺术家声明**」，每件作品需短文解释概念

#### 第二关：实作测试 Practical Test
- 入围后到啟德校园参加 3–4 小时的 on-site 创作
- 题目通常是开放命题（如：以「家」为题创作一件作品）
- 评审现场观察学生**创作思维、时间管理、即场应变**

#### 第三关：面试
- 个别面试 15–20 分钟
- 评审 panel 由 3–4 位教授组成
- 问题涵盖：艺术兴趣、欣赏的艺术家、对当代艺术议题的看法、为何选浸大

「**面试表现会被特别考虑**」（Interview performance will be specially considered）是浸大官方招生页明确写明的。

### 4. 课程结构

浸大 AVA 的课程结构是 **「广基础、深专修」** 模式：

- **第一年**：四大媒介基础——绘画、雕塑、版画、影像
- **第二年**：选修工作室——陶瓷、金工、木工、媒体艺术、艺术与生活美学
- **第三年**：海外交流（强制鼓励）+ 工作室深造
- **第四年**：毕业创作 + AVA 毕业展（每年六月於啟德 + 油街实现展出）

### 5. 浸大三大特色

1. **工作室文化浓厚**：学生在大型工作室共用空间，与同侪日夜创作，互相批评
2. **跨媒介鼓励**：很多学生作品横跨 2D + 3D + 影像 + 表演
3. **国际联系**：与英国 Glasgow School of Art、台湾国立台北艺术大学、北京中央美术学院有交流协议

### 6. 出路

浸大 AVA 毕业生职业路径多元：

- **职业艺术家 / 独立创作人**（约 25–30%）：透过 1a Space、JCCAC、油街实现等驻场计划起步
- **艺术教育 + 工作坊导师**（20%）：自由身教学、社区艺术、博物馆教育
- **画廊 / 拍卖行**（15%）：苏富比、佳士得、白石画廊、Tang Contemporary
- **文化机构 / 策展**（15%）：M+、香港艺术馆、大馆、Tai Kwun、香港艺术中心
- **创意行业**（25%）：广告、品牌、UX、出版、电影美术、舞台设计

起薪较中大、教大稍低（约 HK$16,000–22,000/月），但职业弹性极高，第五年后差异收窄。

## 四、教大视觉艺术教育（BA in Visual Arts Education）

### 1. 课程定位

香港教育大学的视觉艺术教育课程，是香港**唯一**结合艺术创作与教师专业培训的本科课程。毕业后**直接取得注册教师资格**（PRT, Permitted to Teach），可任教香港中小学视觉艺术科。

### 2. 收生要求

- **最低门槛**：DSE 4C + 2X
- **必修科**：英文 Level 3+、中文 Level 3+、数学 Level 2、公民达标
- **必修选修**：通常要求一科文学或人文学科
- **历年中位收生分**：约 17–20 分（Best 5）
- **作品集 + 面试**：必须提交作品集；面试一对一，约 20 分钟

面试评核重点不止艺术能力，还会考察**沟通能力、对儿童 / 青少年艺术教育的看法、教学热情**。

### 3. 课程结构

- **艺术创作**（约 40%）：绘画、设计、新媒体、视觉文化
- **教育理论**（约 35%）：教育心理学、课程设计、青少年发展、视艺科教学法
- **通识与语文**（约 15%）：教大要求英文、普通话、IT 三项基本能力达标
- **教学实习**（约 10%）：第三、第四年到中学视艺科实习共 12–16 周

### 4. 出路

- **中小学视艺老师**（约 70%）：起薪点 MPS 15–17（约 HK$36,000+/月），福利稳定
- **艺术教育机构 / 博物馆教育组**（10%）
- **升读教育硕士 / 艺术硕士**（10%）
- **创作 + 自由教学**（10%）

对于希望在香港有稳定职业兼保留艺术创作的同学，教大是最稳的选择。

## 五、三所对比：怎么选？

| 你的特点 | 适合 | 原因 |
|----------|------|------|
| 喜欢艺术史、传统书画、想做研究 | **中大 JS4044** | 学术性最强，艺术馆资源丰富 |
| 想做当代艺术家、爱跨媒介实验 | **浸大 JS2810** | 工作室文化、跨媒介、毕业展曝光大 |
| 想稳定收入兼保留创作 | **教大 BA(VAE)** | 直接注册教师资格 |
| 内地生 / 想出名做策展、入 M+ | **浸大 JS2810** | M+、Tai Kwun 与浸大 AVA 长期合作 |
| 想进苏富比 / 佳士得 / 拍卖行 | **中大 + 浸大** | 中大艺术史背景 + 浸大策展实习皆受认可 |

## 六、作品集准备建议（共通）

无论申请哪一所，以下原则共通：

1. **数量约 10–15 件**：宁精勿滥
2. **媒介多元**：展示绘画、立体、影像、数码至少 3 类
3. **附 artist statement**：每件作品 50–100 字概念说明
4. **创作过程记录**：草图、sketchbook、灵感来源会大加分
5. **避免「漂亮但空洞」**：评审看的是思考，不是技巧
6. **个人风格 + 实验性并存**：既要有 signature，也要敢挑战自己
7. **早 12 个月开始**：DSE 考试当年 1 月已要交，建议中四暑假就开始储备

## 七、内地生注意事项

- **JUPAS 不适用内地生**：内地生需透过各校 Non-JUPAS / Mainland 渠道申请
- **中大**：内地本科招生（提前批），高考分数 + 校测 + 作品集面试
- **浸大**：内地本科生途径，认可高考成绩，但仍需作品集 + 视频面试
- **教大**：内地学额较少，主要透过 Non-JUPAS

内地高考生想透过艺术类校考入读，需特别注意：**香港艺术本科不承认「联考成绩」**，必须看个别学校的作品集与面试表现。

## 八、香港艺术产业生态与长期前景

香港正进入「**文化创意产业 2.0**」阶段：

- **M+ 博物馆**（2021 开馆）已成亚洲当代艺术枢纽
- **大馆 Tai Kwun** 与 **故宫文化博物馆** 提供策展、教育、研究岗位
- **西九文化区**整体扩张，未来五年预计新增 1,500+ 文化艺术岗位
- **Art Basel Hong Kong** 与 **Art Central** 每年三月引入全球画廊
- **NFT、数字艺术、AI 创作**等新领域，香港在亚洲位居前列

艺术本科生的长期前景，不再局限「难搵食」的旧印象。但同学需要清楚：艺术本科**不是「保送」高薪工作**的途径，而是培养创作力与文化资本的起点。第一份工的起薪（HK$15,000–25,000）通常低于商科、工程，但第五至第十年的职业弹性、自营空间、跨界机会远高于一般学科。

## 九、总结

香港艺术本科虽然学额少、竞争激烈、起薪不高，但对**真心热爱视觉艺术、愿意以创作思考人生**的同学，仍是香港教育体系里最完整的路径。中大艺术系适合学术型、爱艺术史与传统的同学；浸大视觉艺术院适合实验型、爱当代艺术与跨媒介的同学；教大视觉艺术教育适合想稳定教书兼创作的同学。

无论选哪一所，**作品集是关键中的关键**——技巧可以训练，思想与好奇心却是评审最难假装的。希望本指南帮你在 DSE 与 JUPAS 路上找到属于你的艺术坐标。
