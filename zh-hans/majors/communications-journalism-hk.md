# 香港大学传播与新闻学科完全指南——内地生视角

传播与新闻学（Communications and Journalism）是香港高校的传统强势学科，培养了大批在香港、内地及国际媒体行业工作的专业人才。香港作为亚洲国际媒体中枢，其新闻传播教育具有**双语（中英文）、国际化、兼顾本地与全球视野**的独特优势。本指南为有志于传播与新闻方向的内地DSE考生提供全面参考。

## 一、主要院校课程概览

### 1.1 香港浸会大学（HKBU）——传播学院（School of Communication）

HKBU是香港新闻与传播教育的旗舰，享誉全港甚至亚太地区：

| 学系/课程 | 主要方向 |
|---|---|
| 新闻系（Journalism） | 新闻写作、调查报道、多媒体新闻 |
| 电影（Cinema and Television） | 影视制作、纪录片、电影研究 |
| 广播（Radio and Television） | 广播新闻、主持与播音 |
| 传播研究（Communication Studies） | 媒体理论、受众研究、健康传播 |
| 数字新媒体（Digital Media and Television） | 融合媒体、数字内容制作 |
| 整合传播管理（COMS） | 公关、广告、危机传播 |

**特色：**
- 业界实习资源丰富，与香港主流媒体（无线电视TVB、香港电台RTHK等）有合作
- 设有亚洲传播研究中心，为学生提供研究机会
- 新闻系排名为香港最高，在亚洲具有重要声誉

### 1.2 香港中文大学（CUHK）——新闻与传播学院（School of Journalism and Communication，SJIC）

| 课程方向 | 特色 |
|---|---|
| 新闻与传播（BAComm+Journ） | 整合新闻采编与传播理论 |
| 广告（Advertising） | 品牌传播、创意策略 |
| 公共关系（PR） | 企业传播、危机管理 |
| 全球传播（Global Communications） | 跨文化传播，英文为主要教学语言 |

**特色：**
- 学术研究实力强，多位教授在传播研究领域有国际影响力
- 两岸三地传播生态研究独树一帜
- 与内地传媒机构（央视、人民日报等）有合作关系

### 1.3 香港大学（HKU）——新闻及媒体研究中心（Journalism and Media Studies Centre，JMSC）

| 课程 | 说明 |
|---|---|
| 新闻学（Journalism，本科专修） | 以英文为主要教学语言的新闻训练 |
| 数字新闻（Digital Journalism） | 数据新闻、多媒体叙事 |

**特色：**
- 以英语新闻报道为主，培养国际新闻记者
- 与全球知名媒体机构（路透社、BBC、彭博社）有实习合作
- 国际化程度最高，适合有志于英文媒体的学生

## 二、核心课程内容

### 2.1 新闻写作与采编

| 课题 | 内容 |
|---|---|
| 新闻价值判断 | 新闻五要素（5W1H）、新闻价值标准 |
| 新闻写作格式 | 倒金字塔结构、标题写作、特写报道 |
| 调查报道 | 信息来源核实、数据新闻、FOIA申请 |
| 多媒体新闻 | 视频新闻、播客制作、新闻摄影 |
| 数据新闻 | 数据可视化（Tableau/Datawrapper）、Excel分析 |

### 2.2 媒体法与伦理

| 课题 | 内容 |
|---|---|
| 香港媒体法律 | 诽谤法、保密来源法、版权法 |
| 新闻伦理 | 准确性、公正性、隐私权与公众利益的平衡 |
| 媒体监管 | 香港通讯事务管理局（ComCom）的角色 |
| 社交媒体与法律 | 平台内容审核、网络发布法律责任 |

### 2.3 数字新闻与新媒体

| 课题 | 内容 |
|---|---|
| 内容策略 | SEO基础、受众分析、内容分发 |
| 社交媒体运营 | Instagram/微信/TikTok等平台特性 |
| 新闻应用开发 | 基础编程（Python数据分析） |
| 互动新闻 | 用户体验设计、互动信息图表 |

### 2.4 公共关系与广告

| 课题 | 内容 |
|---|---|
| 企业传播 | 媒体关系、危机公关（危机传播管理） |
| 广告策略 | 目标受众定位、品牌定位、创意策略 |
| 整合营销传播（IMC） | 多平台协同传播 |
| 公益传播 | NGO传播、社会倡导活动设计 |

## 三、双语训练——香港的独特优势

香港传播教育最显著的特色是**中英双语**训练，这在内地高校难以复制：

| 语言能力 | 培训方式 |
|---|---|
| 英文新闻写作 | 大量英文写作练习，达到国际媒体稿件标准 |
| 广东话播报 | 播音课程训练标准广东话（普通话学生需额外练习） |
| 普通话新闻 | 内地新闻合作项目，为大湾区媒体就业铺路 |
| 跨语言翻译 | 新闻稿中英翻译训练 |

**内地生提示：**广东话是香港媒体工作的重要工具，在港就读期间应主动学习广东话，至少达到能理解和简单沟通的程度。

## 五、DSE选科建议

| 志愿方向 | 强烈建议 | 有帮助 |
|---|---|---|
| 新闻系 | 英文高分（Level 5以上）、中文 | 历史、通识/CS、经济 |
| 广告/公关 | 英文、经济或BAFS | 设计类、数学 |
| 电影/广播 | 视觉艺术（若有）、英文/中文 | 音乐、历史 |
| 传播研究 | 英文、社会科学相关 | 统计学（数学M1） |

**DSE语文要求：**传播与新闻课程普遍对英文成绩要求较高，建议英文达到**Level 4—5**。

## 八、与内地传媒院校对比

| 维度 | 香港（HKBU/CUHK） | 中国传媒大学 | 复旦大学新闻学院 |
|---|---|---|---|
| 全球传媒排名 | 亚洲前20 | 国内顶尖 | 国内顶尖 |
| 英文媒体训练 | 非常强 | 有但比重较低 | 有但比重较低 |

## 九、总结

香港的传播与新闻教育为内地学生提供了在全球视野下学习新闻与传播的理想平台。HKBU传播学院和CUHK新闻与传播学院是首选，HKU适合有志于英文国际媒体的学生。内地生的最大挑战是**提升英文媒体写作能力**和**适应香港媒体生态**，同时可以将内地媒体知识和普通话能力转化为差异化优势。大湾区媒体融合发展的大趋势，使具备双语能力、跨文化传播经验的香港传播毕业生拥有广阔的职业前景。
