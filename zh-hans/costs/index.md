# 赴港读书费用总览

赴港升读大学的总费用通常高于内地，但显著低于英美顶尖大学，且就业市场认可度高。本页提供各院校收费对比、年度生活成本估算与奖学金资讯，帮助内地家庭做准确的财务规划。

---

## 学费对比（2024/25学年）

香港八大非本地学生学费经政府资助后，各院校本科学费基本统一，但部分专业课程有额外附加费：

| 院校 | 标准本科学费（非本地生） | 备注 |
|------|------|------|
| 香港大学（HKU） | HK$145,000/年 | 医学MBBS约HK$261,000 |
| 香港科技大学（HKUST） | HK$145,000/年 | — |
| 香港中文大学（CUHK） | HK$145,000/年 | — |
| 香港理工大学（PolyU） | HK$145,000/年 | 部分专业附加费 |
| 香港城市大学（CityU） | HK$145,000/年 | — |
| 香港浸会大学（HKBU） | HK$145,000/年 | — |
| 岭南大学（LingnanU） | HK$145,000/年 | — |
| 香港教育大学（EdUHK） | HK$145,000/年 | — |

**换算参考（2025年约）：**
- HK$145,000 ≈ 人民币约13.5万元（视汇率波动）

**→ 各校详细学费明细：[大学费用详细指南](/zh-hans/costs/university-fees/)**

## 住宿费用

### 大学宿舍（Hall / Hostel）

宿舍费用因院校、房型与设施不同而有较大差异，通常为内地学生首年首选：

| 院校 | 宿舍费（参考） | 房型 |
|------|------|------|
| HKU | HK$16,000-25,000/年 | 双人房至单人房 |
| HKUST | HK$18,000-28,000/年 | 双人房至单人房 |
| CUHK | HK$14,000-22,000/年 | 书院制，环境佳 |
| PolyU | HK$20,000-30,000/年 | 市区，空间较小 |
| CityU | HK$22,000-30,000/年 | 市区，设施现代 |
| HKBU | HK$16,000-24,000/年 | 部分在九龙塘 |
| LingnanU | HK$16,000-22,000/年 | 保证首年宿位 |
| EdUHK | HK$14,000-20,000/年 | 大埔校园 |

### 校外租房

校外租房在香港属于常见选择，但价格较高：

| 区域 | 月租参考（劏房/套房） | 交通便利度 |
|------|------|------|
| 沙田/大围（CUHK附近） | HK$4,000-7,000 | 良好 |
| 清水湾/将军澳（HKUST附近） | HK$5,000-8,000 | 一般 |
| 港岛/西环（HKU附近） | HK$6,000-10,000 | 极佳 |
| 九龙塘/石硖尾（HKBU/CityU附近） | HK$5,000-8,000 | 良好 |
| 屯门/元朗（EdUHK/LingnanU附近） | HK$3,500-6,000 | 尚可 |

## 年度总预算范围

| 情况 | 年度总费用（学费+生活） |
|------|------|
| 学费 + 宿舍生活（节俭） | HK$21万-24万（≈人民币约19万-22万） |
| 学费 + 宿舍生活（一般） | HK$24万-28万（≈人民币约22万-26万） |
| 学费 + 校外租房（一般） | HK$28万-35万（≈人民币约26万-32万） |
| 学费 + 校外租房（舒适） | HK$35万以上（≈人民币约32万以上） |

*以上为参考范围，实际开支因个人生活方式、选择院校与租房位置而异。*

## 奖学金概览

### 各院校奖学金

大多数香港大学均为内地优秀学生提供入学奖学金：

| 院校 | 主要奖学金 | 金额参考 |
|------|------|------|
| HKU | HKU Scholarship / 许士芬奖学金 | 全额学费至数万港元 |
| HKUST | HKUST Scholarship | 全额至部分学费 |
| CUHK | CUHK Scholarship / 廖伟荣纪念奖学金 | 全额至部分学费 |
| PolyU | PolyU Presidential Scholarship | 全额至部分学费 |
| CityU | CityU Scholarship | 全额至部分学费 |
| HKBU | HKBU Scholarship | 部分学费 |
| LingnanU | President's Scholarship | 部分学费 |

**申请要求：** 大多数奖学金要求在申请时填写额外申请表，或在录取后由院校主动提名，需留意各校具体规定。

**→ 奖学金完整指南：[香港大学奖学金申请指南](/zh-hans/costs/scholarships-guide/)**

### 政府及社会奖学金

- **香港赛马会奖学金** — 全面覆盖学费与生活津贴
- **内地省级政府赴港奖学金** — 部分省份提供，金额不一
- **内地企业赞助奖学金** — 需个别查询

## 学生贷款与融资

内地学生一般不符合香港政府学生资助计划（HKSL）的申请资格，但可留意：

- 部分院校设有紧急助学金，供突发经济困难学生申请
- 部分内地银行提供留学贷款产品，供赴港升学用途

**→ 详细资讯：[学生贷款与融资指南](/zh-hans/costs/student-loans-and-financing/)**

## 财务规划建议

1. **预留缓冲**：港元汇率波动，建议在年度预算上预留10-15%缓冲
2. **宿舍优先**：首年申请宿舍可大幅降低生活成本，同时融入校园生活
3. **奖学金申请**：申请时务必完整填写奖学金申请表，不少学生因疏忽而错失机会
4. **开户准备**：提前了解香港开户要求，详见[银行开户指南](/zh-hans/admission/bank-hkid-octopus-sim-setup-guide-2026/)
5. **保险安排**：确认医疗保险覆盖，各校学生保险有不同保障范围
