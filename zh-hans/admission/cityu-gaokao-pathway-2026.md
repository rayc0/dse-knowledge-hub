## 一、2026年最大变化：退出统招，全面独立招生

香港城市大学（CityU，港城大）自**2026学年**起正式退出"全国普通高等学校统一招生计划（统招）"，改为通过"中国内地高考单独招生"方式独立录取应届高考生。

这是港八大2026年招生政策最重大的结构性变化之一（另一所退出统招的是香港城市大学）。此前，内地学生通过统招志愿填报城大；2026年起，**城大不再占用统招志愿**，学生可在填写统招志愿的同时，单独向城大投递申请，两者互不影响。

### 独立招生的核心要点

| 项目 | 2026年新政策 |
|---|---|
| 招生方式 | 高考单独招生（不参加统招） |
| 是否占统招志愿 | **否**，两者互不冲突 |
| 是否设省市名额 | **否**，全国择优录取 |
| 计划录取规模 | 约300名应届高考生 |
| 选拔依据 | 高考成绩＋申请材料＋面试（如适用） |

## 二、申请资格与分数要求

### 基础资格

- 2026年应届参加普通高考的中国内地学生
- 高考实际成绩（**不含任何加分**）须达到**本科第一批重点大学录取分数线（一本线）**
- 英语科目成绩须达到**110分或以上**（150分制）

### 实际录取参考区间

城大官方仅公布"一本线"为最低门槛，但根据2025年录取数据与各省院校对比，竞争力参考区间如下：

| 专业方向 | 参考分数区间（高出一本线） | 英语要求 |
|---|---|---|
| 商科（BBA、会计、金融） | +90–130分 | 120分以上 |
| 计算机科学 / 数据科学 | +100–140分 | 120分以上 |
| 工程学（电子、土木、机械） | +80–120分 | 115分以上 |
| 理学 / 环境与能源 | +80–120分 | 110分以上 |
| 创意媒体 | +80–110分（含作品集/面试） | 110分以上 |
| 人文及社会科学 | +80–110分 | 110分以上 |
| 生物医学 | +110–140分 | 120分以上 |
| 法律（LLB） | +120–150分 | **135分以上** |
| 兽医医学（BVM） | +130–160分 | **135分以上** |

> **注意**：以上区间为参考竞争力范围，城大不设固定分数线，最终以综合评估为准。法律和兽医学由于专业竞争激烈，英语单科门槛显著更高。

### 复读生政策

城大接受高考复读生申请，复读年份不影响申请资格，以实际高考成绩为准。

## 四、申请材料清单

向城大提交的申请材料通常包括：

1. **身份证明文件**（护照/身份证）
2. **高中成绩单**（高一至高三三年成绩）
3. **高考准考证或含准考证号的官方文件**（证明参考身份）
4. **高考成绩截图**（于6月26日前上传）
5. **英语水平证明**（如高考英语成绩属高考一部分，通常无须额外语言考试）
6. **个人陈述（PS）**（部分专业要求）
7. **奖项/荣誉证明**（如有）

> 城大接受以英语或中文撰写的部分材料，但个人陈述建议使用英文，因为所有课程以英语授课。

## 六、学费与奖学金

### 学费（2026-2027学年）

| 项目 | 非本地学生年学费 |
|---|---|
| 标准本科课程 | **HK$190,000/年** |
| 兽医医学（BVM，6年制） | **HK$392,000/年** |

四年总学费约**HK$760,000（约RMB 68–72万）**，加上住宿和生活开销，四年总费用约**RMB 100–120万**。

### 入门奖学金（Entrance Scholarship）

城大对所有通过高考途径录取的内地学生**自动评审**，无须单独申请：

| 奖学金类别 | 评选标准（省内"一分一段"排名） | 奖励金额 |
|---|---|---|
| **全额奖学金** | 全省前 0.1% + 英语优秀 | 学费全免 ＋ 每年最高 HK$60,000 生活津贴 |
| **学费全免奖学金** | 全省前 0.1%–1% + 英语优秀 | 全额学费免除（约HK$190,000/年） |
| **学费半免奖学金** | 全省前 1%–1.5% + 英语优秀 | 50% 学费免除（约HK$95,000/年） |
| **和谐奖学金** | 符合学费全免/半免条件 + 来自西部省区 | 额外 HK$40,000/年 |

**适用和谐奖学金的西部省区**：内蒙古、西藏、甘肃、广西、贵州、宁夏、云南、青海、新疆。

**续奖条件**：每学年末须维持大学规定的累积绩点（GPA）要求，奖学金自动续发。

**持续年限**：本科4年（双学位5年，兽医6年）。

> **实际意义**：高考"一分一段"全省前0.1%，以广东省为例，约对应理科全省前300名。达到学费全免门槛（前1%）约对应理科全省前3,000名，竞争极激烈。**即使不获奖学金，城大学费HK$190,000/年仍属港八大中等偏低水平**（低于港大/科大的HK$200,000+）。

## 九、内地学生毕业去向

### IANG签证 — 毕业留港

城大本科毕业后可申请**IANG（输入内地人才计划）签证**，无配额限制，2025年批准率约97%，有效期2年，可续期，是港八大毕业生留港工作的标准路径。

### 主要就业方向（城大内地学生）

| 方向 | 代表雇主 / 出路 |
|---|---|
| 金融科技 / 投资银行 | 汇丰、渣打、工银亚洲、摩根大通（香港） |
| 科技行业 | 腾讯、字节跳动（香港/深圳）、阿里云、华为 |
| 管理咨询 / Big4 | 德勤、毕马威、普华永道、埃森哲 |
| 法律行业 | 香港律师事务所（LLB + PCLL + 律师资格） |
| 大湾区 | 深圳、广州科技/金融/地产企业 |
| 继续深造 | 城大约25–30%毕业生升读硕博 |

### 留服认证

城大本科学历可在中国教育部留学服务中心（CDGDC/留服中心）完成认证，认证后等同国内"985/211"同等院校水平处理，适用于国内考研、考公等场合。

## 十、常见问题 (FAQ)

**Q：2026年城大退出统招，意味着我不能通过统招志愿报考城大了吗？**
A：是的。自2026年起，城大不参加统招，你无需（也无法）在统招志愿中填报城大。你需要直接在城大官网独立申请。这意味着你可以**同时填满统招志愿**，再额外申请城大，两者互不影响。

**Q：申请城大会影响我的统招志愿或排名吗？**
A：完全不影响。城大独立招生不占统招志愿，不影响你在国内其他院校的录取机会。

**Q：我的高考英语不是第一外语（如选考日语），还能申请吗？**
A：可以，但须提供有效的英语能力证明（如IELTS 6.5或TOEFL 90+）。城大所有课程以英语授课，英语能力是核心评估维度。

**Q：兽医医学（BVM）对内地学生有多少名额？**
A：城大BVM每年全球录取约60名，其中供内地高考学生的名额约3–5个，竞争极为激烈，须达到一本线+130–160分且英语135+。兽医学费HK$392,000/年，6年总费用约RMB 250–280万，家庭需有充分经济准备。

**Q：法律（LLB）毕业后能否回内地执业？**
A：香港法律学位（LLB）主要针对香港法律体系。如需回内地执业，须参加国内司法考试（法律职业资格考试）。部分跨境业务（尤其大湾区）对同时具备内地与香港法律知识的人才有需求。

**Q：城大的商学院与港大、科大相比如何？**
A：城大商学院获AACSB + EQUIS + AMBA三重认证，是香港唯一集齐三项认证的商学院之一。QS商科排名城大Asia Top 15。相比港大/科大更侧重实践导向，国际化氛围极强（THE全球最国际化#1），适合希望在跨国企业或金融科技领域发展的学生。

**Q：创意媒体专业申请有特殊要求吗？**
A：创意媒体学院（SCM）部分项目可能要求提交作品集（Portfolio）或参加面试，建议提前准备数字媒体、设计、影像等方向的代表作品。

**Q：城大是否提供内地学生专属奖学金以外的其他资助？**
A：是的。城大还设有：①在校表现奖学金（按年度学业成绩评审）；②杰出运动员奖学金（学费全免＋住宿）；③海外交换补贴（部分学院提供）；④紧急经济援助基金（一次性援助）。

**Q：城大本科毕业后申请城大硕士有优势吗？**
A：有一定优先通道，尤其对绩点3.3以上的本科毕业生。城大授课型硕士（如MSc Finance、MSc CS）在香港金融/科技圈认可度高，续读成本相对合理。

**Q：城大哥伦比亚大学2+2双学位的申请门槛是什么？**
A：该项目竞争极为激烈，通常要求入学GPA 3.5以上（前两年城大修读期间），以及良好的数理基础（应用数学/工程方向）。名额极少，建议有意者从入学第一学期起维持高GPA并积极参与相关科研活动。
