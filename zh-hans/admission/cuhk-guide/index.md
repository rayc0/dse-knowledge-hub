# 香港中文大学（中大/CUHK）内地生申请指南

香港中文大学（The Chinese University of Hong Kong，简称CUHK或中大）是香港历史最悠久的综合研究型大学之一，在QS世界大学排名长期位居全球前50位。其书院制传统、深厚的中华文化底蕴，以及地处沙田的优美山水校园，令中大成为许多内地学生的首选目标。

---

## 申请途径：Non-JUPAS

内地学生无法参加联招（JUPAS），需通过**Non-JUPAS渠道**申请中大。主要提交材料包括：
- 香港中学文凭考试（DSE）成绩或高考成绩（见下文）
- 在读/毕业学校成绩单
- 个人陈述（Personal Statement）
- 英语能力证明（如适用）

CUHK Non-JUPAS的统一申请平台为[大学联合招生办法非联招](https://www.jupas.edu.hk/en/non-jupas/)，另外部分学院亦设独立申请系统。申请通常于每年**1月至3月**开放，**3月底前**须完成提交，成绩公布后（约7月）进行录取确认。

---

## 一试两用计划：高考生的重大优势

CUHK是香港首批接受**「一试两用」（HKDSE-Gaokao Mutual Recognition）**的大学之一。这意味着，内地考生可凭**高考成绩**直接申请CUHK部分学士学位课程，无需另行参加DSE考试。

**适用条件（2024/25入学参考）：**
- 高考总分通常须达到所在省份**本科一批录取线以上约80–120分**（视乎学科竞争程度）
- 理工科成绩优秀者更具优势
- 需同时提交高中三年学业成绩

对于已备考或正在准备高考的内地生而言，这一计划大大降低了赴港升学的门槛，无需额外学习DSE课程体系。

## DSE成绩要求

若以DSE成绩申请中大，各院校要求有所不同：

| 学科 | 最低要求 |
|------|---------|
| 中文（Chinese Language） | Level 3 |
| 英文（English Language） | Level 4–5 |
| 数学（Mathematics） | Level 3–4（视学科而定） |
| 通识/公民与社会发展科 | Level 2–3 |
| 选修科目 | Level 4–5（共两科） |

**理学院、工程学院**通常要求数学延伸单元（M1/M2）达Level 3以上；**医学院**要求化学及生物均达Level 5*。整体而言，中大主流热门学科的DSE入学要求较HKU略低，但仍属竞争激烈。

## 医学及法律课程

**医学（MBBS）**：CUHK医学院为香港两所医学院之一，内地生录取名额极少，DSE要求为5科达Level 5以上（含5**）。高考路径录取分数须位于省内顶尖百分之一。

**法律（LLB）**：CUHK法律学院在亚洲享有盛誉。内地生申请须English Language达Level 5以上，整体成绩须极为优异。建议同时准备个人陈述及参考信。

## 奖学金

中大为优秀内地新生提供多项奖学金：

- **中大大学奖学金（CUHK University Scholarships）**：金额最高可达**每年港币128,000元**，涵盖学费及生活费。申请与入学申请同步提交，无需额外报名。
- **理大及书院奖学金**：各书院（如新亚书院、崇基学院）设有独立奖学金，金额10,000–50,000港元不等。
- **香港政府奖学金**：见「政府奖学金」专题文章。

获奖条件通常包括：成绩排名（DSE满分或高考前5%）、课外活动表现及面试表现。建议提早准备奖学金申请材料。

## 沙田校园生活

中大校园占地约137公顷，依山而建，毗邻吐露港，自然风光优美。校内设有：

- **5所书院**（2024年含第9书院），住宿及活动均以书院为单位，归属感强
- **铁路直达**：大学站（东铁线）设于校园内，往来九龙及香港岛约30–50分钟
- **多元餐饮**：饭堂、茶餐厅、国际美食一应俱全，价格相对实惠

---

## 校园语言环境：粤语与普通话

中大历史上以中文（粤语）教学为特色，但近年国际化程度提升：

- **授课语言**：大部分课程以**英文**授课，部分人文学科保留中文（粤语/普通话）教学
- **日常交流**：校园内粤语为主，但普通话使用日益普遍，内地生适应难度不高
- **内地生社群**：中大内地生比例约15–20%，有专属学生会（内地学生联会），新生适应支援完善

对于不懂粤语的内地生，英语沟通已足以应付日常学业及社交需求，粤语可在生活中自然习得。

## 申请时间线（参考）

| 时间 | 事项 |
|------|------|
| 每年10–11月 | 了解课程及奖学金要求，准备申请材料 |
| 1月–3月 | Non-JUPAS网上申请开放，提交所有材料 |
| 3月底 | 申请截止（具体日期以官网为准） |
| 4–6月 | 面试邀请（部分学科） |
| 7月初 | DSE成绩公布，最终录取决定 |
| 7月中下旬 | 确认入学，安排签证及宿舍 |

## 实用建议

1. **尽早准备个人陈述**：中大非常重视申请者的自我表达，建议结合具体经历，避免泛泛而谈。
2. **善用一试两用**：若已参加高考且成绩理想，可直接申请无需额外考DSE。
3. **同时申请多校**：Non-JUPAS可同时提交HKU、HKUST等，不影响中大申请。
4. **关注书院选择**：不同书院文化氛围各异，新亚书院人文氛围浓厚，逸夫书院理工色彩较强。
5. **官网为准**：每年录取要求可能调整，务必参阅[CUHK官网Non-JUPAS页面](https://www.cuhk.edu.hk/adm/non-jupas)。
