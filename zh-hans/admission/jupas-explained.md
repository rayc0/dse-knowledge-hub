# JUPAS 是什么？大陆学生需要了解吗？

> **TL;DR：** JUPAS（大学联合招生办法）是香港本地中学生（持有 DSE 成绩）申请八大公立大学的**集中联招系统**。绝大多数大陆学生**不走 JUPAS**——他们通过 **Non-JUPAS（非联招）** 凭高考成绩直接向各校申请。本文解释两者的根本区别、例外情况（在港就读的大陆生、DSE 自修生、重考生），以及 2026 年 JUPAS 完整时间线。

---

## 一、什么是 JUPAS？

**JUPAS**，全称 **Joint University Programmes Admissions System**，中文官方名称 **大学联合招生办法**（简称「联招」），是香港统一管理公立大学本科入学申请的集中系统。

由**香港大学教育资助委员会（UGC）** 于 1990 年建立，JUPAS 的核心设计目标是：让每位考完 **香港中学文凭考试（HKDSE）** 的本地应届生，只需在一个系统内完成所有大学申请，无需逐校递交。

### JUPAS 参与院校（2026 年）

| 院校 | 简称 |
|---|---|
| 香港大学 | HKU |
| 香港中文大学 | CUHK |
| 香港科技大学 | HKUST |
| 香港理工大学 | PolyU |
| 香港城市大学 | CityU |
| 香港浸会大学 | HKBU |
| 香港教育大学 | EdUHK |
| 岭南大学 | LU |
| 香港都会大学 | HKMU |

以上九所均为 UGC 资助或认可的本地院校，JUPAS 每年处理约 **5 万名** 本地考生的申请。

## 二、JUPAS 怎么运作？

JUPAS 的运作逻辑与内地的「高考志愿填报」非常相似，但有其独特机制：

### 1. 填选课程志愿（Band A / B / C）

申请人在系统中最多可选 **25 个课程**，并将这些课程分组排入三个 Band：

- **Band A**：最希望就读的课程（排名 1-3）
- **Band B**：次选课程（排名 4-12）
- **Band C**：保底课程（排名 13-25）

大学在分配时，**优先满足 Band A** 的志愿。如果学生获得 Band A 某课程录取，系统将不再考虑其 Band B 和 Band C 志愿。

### 2. DSE 成绩是核心

JUPAS 录取几乎完全基于 **HKDSE 成绩**。最低入学门槛为「**3322** 规则」：

- **中文**（Chinese）：3 级或以上
- **英文**（English）：3 级或以上
- **数学**（Mathematics）：2 级或以上
- **通识教育 / 公民与社会发展**（LS/CS）：2 级或以上

但这只是门槛，**热门专业的实际录取分数远高于此**——以 HKU 医学院为例，录取分数可达最佳 5 科 30+（满分 35 分）。

### 3. 「先分配、后确认」机制

DSE 成绩公布（7 月中旬）后，考生有 **48 小时** 窗口修改志愿顺序。系统随后统一进行分配，8 月上旬公布结果。

## 三、大陆学生为什么（几乎）不走 JUPAS？

这是本文的核心问题。答案只有一个字：**身份**。

### 本地生 vs 非本地生

JUPAS 是**专为本地学生设计**的渠道。根据 JUPAS 官方政策，「本地学生」的定义为：

> 持有香港永久居民身份证，或持有相关签注使其依法在港居留的人士

**来自中国大陆、持有大陆户籍、以高考成绩申请的学生，属于「非本地学生」（Non-local students）。**

根据 JUPAS 官网明文规定：

> **所有非本地申请人，无论是否持有 HKDSE 成绩，均须通过 Non-JUPAS 渠道申请，不得通过 JUPAS 报名。**

这意味着：即使一名大陆学生参加并通过了 DSE 考试，只要其不具备「本地学生」身份，仍然**不能**走 JUPAS 通道，而必须走 Non-JUPAS。

### 大陆学生的正确通道：Non-JUPAS

**Non-JUPAS**（非联招），是大陆学生申请香港八大的**标准路径**。学生需：

1. **直接向各大学的招生系统提交申请**（各校系统独立，不统一）
2. 提交**高考成绩**（或其他认可学历证明）
3. 部分学校要求英语证明 + 面试

→ 关于 Non-JUPAS 的完整申请指南，见：[大陆学生申请香港八大完整指南 2026](mainland-bada-guide.md)

## 四、JUPAS vs Non-JUPAS 全面对比

| 对比维度 | JUPAS（联招） | Non-JUPAS（非联招） |
|---|---|---|
| **适用对象** | 香港本地学生（持 HKID） | 大陆学生、海外学生、国际课程学生 |
| **入学考试** | HKDSE（香港中学文凭考试） | 高考、IB、A-Level、SAT+AP 等 |
| **申请方式** | 统一系统（jupas.edu.hk），一次申请多校 | 各校独立系统，逐一申请 |
| **志愿填报** | 最多 25 个志愿，分 Band A/B/C 排序 | 各校分别提交，无统一排序 |
| **申请时间** | 每年 12 月开放，5 月底截止 | 各校 10 月陆续开放，5-6 月截止 |
| **结果公布** | 8 月初统一放榜 | 各校独立通知，6-7 月陆续出 Offer |
| **录取依据** | 几乎 100% DSE 成绩 | 高考成绩 + 面试 + 英语成绩 + 综合评定 |
| **学费（非本地生）** | 港币 14.5 - 17.1 万 / 年 | 港币 14.5 - 17.1 万 / 年（相同） |
| **年申请人数** | 约 5 万人（本地生） | 数万人（来自全球） |
| **竞争性质** | 本地考生之间竞争 | 全球非本地学生竞争 |

**结论：** 两条渠道最终入读的是**同一所学校、同一个课程、同一份学位文凭**，学费和待遇完全相同。唯一的差异是申请路径和所用成绩类型。

## 五、哪些大陆相关学生可能涉及 JUPAS？

虽然绝大多数大陆学生走 Non-JUPAS，但以下几类人群可能会接触到 JUPAS 或需要了解它：

### 情形 1：在港就读国际学校或本地学校，已获香港本地身份

部分大陆家长通过移民（如优秀人才入境计划、投资移民、专才计划）获得香港居留权，其子女在港读书并持有香港身份证。这类学生参加 DSE 后，**可以通过 JUPAS 以本地生身份申请**，享受：
- 本地生学费（港币 4.2 万 / 年，远低于非本地生的 14.5-17.1 万）
- 更多学额（八大大多数学额面向本地生）
- 更高录取率（相同分数，本地生通道竞争者更少）

这是香港移民对子女教育最显著的财务回报之一。

### 情形 2：持依亲签注的未成年人，且入港时不满 18 岁

JUPAS 的「本地学生」定义还包括：**持依亲签注（Dependant Visa/Permit）且首次获得该签注时未满 18 岁**的申请人。

部分通过「内地人才引进」政策来港工作的大陆父母，其子女持依亲签注在港上学。若孩子入港时不满 18 岁，在港就读 DSE 课程后，理论上可以 JUPAS 本地生身份申请。

> ⚠️ 建议在申请前向 JUPAS 秘书处（jupas@jupas.edu.hk）确认个人身份状况，避免因误解资格而浪费时间。

### 情形 3：DSE 自修生，但不具备本地身份

部分大陆学生以「自修生」身份参加 DSE（年满 19 岁或持内地高中学历），取得 DSE 成绩后，**仍然不能走 JUPAS**，而是持 DSE 成绩通过 **Non-JUPAS** 申请各校。

这是很多人误解的地方——**有 DSE 成绩 ≠ 可以走 JUPAS**。身份才是决定性因素。

### 情形 4：在港就读本地中学的 DSE 应届生（内地学籍但有香港居留权）

若大陆籍学生持有香港身份并在港正规中学就读，参加 DSE 应届考试，则按本地学生处理，可走 JUPAS。

## 八、JUPAS 与 Non-JUPAS 录取名额对比

了解 JUPAS 和 Non-JUPAS 的**名额比例**，有助于大陆学生正确评估自己所在的竞争池：

| 名额类别 | 说明 | 占比（约） |
|---|---|---|
| **JUPAS 本地生名额** | 面向持 HKDSE 成绩的本地学生 | 约 60% |
| **Non-JUPAS 非本地生名额** | 面向大陆生、海外生、国际课程学生 | 约 40%（近年政策上调） |

> 背景：香港特区政府自 2024-25 学年起，将非本地生招收上限从本地生名额的 **20%** 大幅提升至 **40%**，以吸引更多优质海外和内地生。这对大陆学生来说是利好消息——Non-JUPAS 通道的名额显著扩大。

换算成实际数字：以 HKU 为例，本科总名额约 2,800 人，40% Non-JUPAS 配额约 **1,120 人**，其中大陆生通常占多数。

## 九、常见问题 FAQ

**Q：我是大陆学生，考了 DSE，能用 JUPAS 申请吗？**
A：不能。JUPAS 仅对「本地学生」开放，判断标准是**香港身份**，而非 DSE 成绩。只要你不具备香港居民身份，即使持有 DSE 成绩，也必须通过 Non-JUPAS 渠道申请。

**Q：JUPAS 和 Non-JUPAS 入读后有区别吗？**
A：几乎没有。最终课程、学位证书、校园设施完全相同。唯一显著差异是**学费**：本地生（JUPAS 通道）每年约港币 **4.2 万**，非本地生（Non-JUPAS 通道）每年约 **14.5-17.1 万**。

**Q：我的孩子在香港读书，持有依亲签注，可以走 JUPAS 吗？**
A：若孩子首次获得依亲签注时未满 18 岁，依 JUPAS 政策可能符合本地生资格。建议向 JUPAS 秘书处确认，或咨询孩子就读学校的升学辅导老师。

**Q：Non-JUPAS 的竞争比 JUPAS 更激烈吗？**
A：竞争性质不同。JUPAS 是约 5 万本地考生争有限名额；Non-JUPAS 是来自全球的非本地生竞争，人数也达数万。两者都很竞争，但评分维度不同：JUPAS 高度依赖 DSE 分数，Non-JUPAS 更综合（高考分数 + 英语 + 面试 + 课外活动）。

**Q：JUPAS 申请费是多少？**
A：2026 年 JUPAS 申请费为港币 **250 元**（普通申请）。逾期申请须额外缴纳逾期附加费。被录取后须额外缴纳 **学额接受费港币 5,000 元**。

**Q：JUPAS 的「Band A」是什么意思？**
A：Band A 是申请人在 JUPAS 系统内标注的「最优先志愿」（1-3 个课程）。大学在分配时优先匹配 Band A 志愿。成功入读 Band A 课程的申请人，其 Band B 和 Band C 志愿自动作废。Band A 的策略性填写至关重要——建议参考各课程历年收生分数，确保 Band A 既有冲劲又不过于脱离实际。

**Q：大陆学生可以同时通过 Non-JUPAS 和 JUPAS 申请同一所大学吗？**
A：不可以。大陆学生不具备 JUPAS 申请资格，只能通过 Non-JUPAS 申请各校。

## 十、给大陆家长的建议：了解 JUPAS，但别花时间在上面

许多大陆家长在研究香港大学时，会搜索到大量关于 JUPAS 的信息——这很正常，因为 JUPAS 是香港本地最主流的升学渠道，相关讨论铺天盖地。

**但请记住：JUPAS 信息对大陆学生几乎没有直接参考价值。** 以下是大陆家长真正需要关注的：

1. **Non-JUPAS 各校申请时间**（每年 10 月陆续开放，12 月早申截止 → **不要拖到 5 月**）
2. **高考分数要求**（各校 + 各省份 + 各专业不同）
3. **英语能力证明**（高考英语 / 雅思 / 托福）
4. **面试准备**（HKU、CUHK、HKUST 基本必须，全英文）
5. **奖学金申请**（大部分自动评定，但部分需主动提交）
