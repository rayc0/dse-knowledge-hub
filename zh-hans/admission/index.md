# 香港大学升学申请总览

香港共有八所受政府资助的大学（俗称"八大"），均接受来自内地的Non-JUPAS（非联招）申请。本页汇整各院校的核心资讯，帮助内地DSE考生与高考生快速比较，制定申请策略。

---

## 各院校详细申请指南

### 香港大学（HKU）

香港排名最高的综合性研究型大学，医学、法律、商科与工程均具国际竞争力。Non-JUPAS申请竞争最为激烈。

**成绩要求（参考）：** DSE Best-5通常需达33分或以上；高考要求视省份而定，通常需达一本线以上较高分段

**特色课程：** 医学（MBBS）、法律（LLB）、经济金融（BEcon&Fin）

**→ 完整申请指南：[港大内地生申请指南](/zh-hans/admission/hku-guide/)**
**→ 高考途径详解：[HKU本科录取指南](/zh-hans/admission/hku-undergraduate-admission-guide-2026/)**

---

### 香港科技大学（HKUST）

以理工科与商科见长，在工程与商业管理领域亚洲顶尖。校园位于西贡清水湾，环境优美。

**成绩要求（参考）：** DSE Best-5通常需达30分或以上；理工类课程要求数学与相关科学科成绩优异

**特色课程：** 计算机科学、工学院各系、商学院（HKUST Business School）、EMBA

**→ 完整申请指南：[科大内地生申请指南](/zh-hans/admission/hkust-guide/)**
**→ 综合指南：[科大本科全面指南](/zh-hans/admission/hkust-undergraduate-guide-2026/)**

---

### 香港中文大学（CUHK）

香港唯一采用书院制的大学，中英双语学习环境，文科与理科均衡发展。"一试两用"政策允许内地高考生同时申请。

**成绩要求（参考）：** DSE Best-5通常需达30分或以上；高考（一试两用）视课程而定

**特色课程：** 工商管理（九成学生有交流机会）、医学、法律、专业会计

**→ 完整申请指南：[中大内地生申请指南](/zh-hans/admission/cuhk-guide/)**
**→ 综合指南：[中大本科全面指南](/zh-hans/admission/cuhk-undergraduate-guide-2026/)**
**→ 高考途径：[CUHK高考入读途径](/zh-hans/admission/cuhk-gaokao-pathway-2026/)** *(如有)*

---

### 香港理工大学（PolyU）

以实用型专业著称，设计、酒店及旅游管理、护理、物理治疗等专业全港顶尖，毕业生就业率高。

**成绩要求（参考）：** DSE Best-5通常需达25-28分；热门课程要求更高

**特色课程：** 设计学院（School of Design）、酒店与旅游管理、护理、物理治疗

**→ 综合指南：[理工大学本科完整指南](/zh-hans/admission/polyu-undergraduate-complete-guide-2026/)**
**→ 高考途径：[PolyU高考入读途径](/zh-hans/admission/polyu-gaokao-pathway-2026/)**

### 香港城市大学（CityU）

位于市区九龙塘，交通便利，以创意媒体、法律、商科与工程见长，商界实习机会丰富。

**特色课程：** 创意媒体、法律、工商管理、电子工程

**→ 综合指南：[城市大学本科完整指南](/zh-hans/admission/cityu-undergraduate-complete-guide-2026/)**

### 香港浸会大学（HKBU）

以人文、传理（传播与新闻）、商科与中医见长，校园文化浓厚，规模较小，师生关系密切。

**特色课程：** 传理学院（新闻、电影、公关）、中医学、商学院

### 岭南大学（LingnanU）

香港唯一以博雅教育（Liberal Arts）为核心的大学，班级规模小，每位学生均有丰富海外交流机会，学生与教授比例全港最低。

**特色课程：** 社会科学、人文学科、工商管理（小班教学）

### 香港教育大学（EdUHK）

专注教育学及相关专业，香港培养中小学教师的主要院校，近年拓展至社会科学与健康学科。

**特色课程：** 幼儿教育、小学教育、中学各科教育、特殊教育

## 选校策略建议

### 按志向选校

| 志向方向 | 推荐院校 |
|----------|----------|
| 金融投资 | HKU、HKUST、CUHK |
| 工程科技 | HKUST、PolyU、CityU |
| 医学 | HKU（MBBS）、CUHK（MBBS） |
| 传媒创意 | HKBU、CityU |
| 设计 | PolyU |
| 酒店旅游 | PolyU |
| 教育 | EdUHK |
| 博雅通识 | LingnanU |

### 按成绩定位

- **DSE Best-5 ≥ 32分**：有竞争力申请HKU、HKUST、CUHK热门课程
- **DSE Best-5 28-31分**：PolyU、CityU热门课程，或HKU/CUHK非热门课程
- **DSE Best-5 24-27分**：HKBU、LingnanU、EdUHK，或PolyU/CityU非热门课程
- **高考生**：各院校分设高考收生途径，不经DSE，参见各院校高考指南

## 申请时间线概览

| 时间 | 事项 |
|------|------|
| 9-10月 | 各大学公布来年收生资讯，研究选校 |
| 10-12月 | 各大学Non-JUPAS网上申请开放 |
| 11月-翌年2月 | 提交申请（各校截止日期不同，需逐一确认） |
| 12月-翌年3月 | 部分院校发出面试邀请 |
| 1-3月 | 参加面试（如适用） |
| 3-5月 | 等待条件录取通知 |
| 4月（第三周） | DSE公开考试 |
| 7月 | DSE放榜，确认录取 |
| 8-9月 | 入学注册、宿舍申请、签证办理 |

**→ 完整时间线详解：[申请时间线完整指南](/zh-hans/admission/application-timeline-2026/)**
