# DSE（香港中学文凭試験）とは：国際学校生徒向けの完全ガイド

香港で中学・高校を卒業する日本人学生にとって、**DSE（Hong Kong Diploma of Secondary Education：香港中学文凭）**は大学進学の重要な鍵となる資格です。このガイドでは、DSEの仕組みから受験スケジュールまで、日本語で詳しく解説します。

---

## 1. DSEの概要

### DSEとは何か

DSEは**香港政府が実施する統一大学入学試験**です。正式名称は「香港中學文憑試験（Hong Kong Diploma of Secondary Education Examination）」で、香港考試及評核局（HKEAA）が主管しています。

| 項目 | 内容 |
|------|------|
| 実施機関 | 香港考試及評核局（HKEAA） |
| 対象者 | 香港の中学6年生（Form 6）修了生 |
| 試験時期 | 毎年3月〜5月（筆記試験） |
| 主な用途 | 香港の大学・専門学院入学（JUPAS経由） |
| 国際的な認知 | 英国・豪州・カナダ・米国の大学で認定 |

### 日本の大学入試との主な違い

DSEは日本の大学入学共通テスト（旧センター試験）と同様の「共通試験」形式ですが、いくつかの重要な違いがあります。

- **複数年の受験結果が使える**：DSEの成績は取得後5年間（一部科目は無期限）有効
- **科目別に合否ではなく5段階評価**：1〜5**（5**が最高）で採点
- **英語が必須コア科目**：すべての受験者が英語（English Language）を受験
- **再受験が可能**：合格後も成績向上を目的として再受験できる

### 国際学校生徒のDSE受験

ESF（英基学校協会）やその他の国際学校に在籍する学生も、**Elective Candidates（課外受験生）**として個人受験が可能です。ただし、国際学校のカリキュラム（IB・A-Levels）とDSEのカリキュラムは異なるため、別途対策が必要です。

## 2. DSEの科目構成

### コア科目（4科目・全員必修）

すべてのDSE受験者は以下の4つのコア科目を受験する必要があります。

| 科目 | 英語名 | 内容 |
|------|--------|------|
| 中国語 | Chinese Language | 読解・作文・口語試験 |
| 英語 | English Language | 読解・作文・口語・聴解 |
| 数学 | Mathematics（Compulsory Part） | 必修パートのみ。数学延伸モジュール（M1/M2）は選択 |
| 市民生活と社会 | Citizenship and Social Development（CSD） | 2021年より「通識教育」に替わって導入 |

**重要**: 中国語は非中国語圏の生徒（NCS学生）向けに代替試験「Applied Learning Chinese（ApL Chinese）」が用意されています。ESFや国際学校の生徒の多くはこちらを選択します。

### 選択科目（2〜3科目）

コア科目に加えて、以下のカテゴリから2〜3科目を選択します。

**人文・社会科学系**
- 中国の歴史（Chinese History）
- 世界の歴史（History）
- 地理（Geography）
- 経済（Economics）
- ビジネス・会計・財務概論（BAFS）
- 倫理・宗教（Ethics and Religious Studies）

**理工系**
- 物理（Physics）
- 化学（Chemistry）
- 生物（Biology）
- 複合科学（Combined Science）
- 情報科学・通信技術（ICT）
- 設計とテクノロジー（Design and Applied Technology）

**芸術・体育系**
- 視覚芸術（Visual Arts）
- 音楽（Music）
- 体育（Physical Education）

**応用学習（ApL）**
- 職業・技術系の資格コース（最大2科目まで「選択科目」として計上可）

### 成績評価

| グレード | 名称 | 目安 |
|----------|------|------|
| 5** | Level 5 Double Star | 最優秀（上位約1%） |
| 5* | Level 5 Star | 優秀（上位約3%) |
| 5 | Level 5 | 非常に良い |
| 4 | Level 4 | 良い |
| 3 | Level 3 | 合格水準以上 |
| 2 | Level 2 | 最低合格水準 |
| 1 | Level 1 | 不合格 |
| U | Unclassified | 採点不能 |

香港の大学入学に必要な最低成績は、一般的に**コア科目で3322（中国語3、英語3、数学2、市民生活と社会2）**です。

## 3. DSEと国際資格の比較表

| 比較項目 | DSE | IB（国際バカロレア） | A-Levels | 日本の共通テスト |
|----------|-----|---------------------|----------|----------------|
| 主な対象地域 | 香港 | 全世界 | 英国・旧英連邦 | 日本 |
| 試験の形式 | 統一試験＋校内評価 | 内部評価＋最終試験 | 科目別試験 | マークシート中心 |
| 科目数（必修） | 4科目（コア） | 6科目＋コア要件 | 通常3〜4科目 | 5〜8科目 |
| 成績評価 | 1〜5**（7段階） | 1〜7点（科目別） | A*〜U（6段階） | 各科目100点満点 |
| 国際認知度 | 高（英国/豪州/カナダ） | 非常に高い（全世界） | 非常に高い（英語圏） | 主に日本国内 |
| 香港大学入学 | 主要ルート（JUPAS） | 非JUPAS経由で可 | 非JUPAS経由で可 | ほぼ使用不可 |
| 取得コスト | 低〜中 | 高（IB認定校が必要） | 中 | 低 |
| 再受験 | 可（5年間有効） | 不可（同年度のみ） | 可（制限あり） | 可（毎年） |

**まとめ**: DSEは香港の大学入学に最も直接的なルートを提供します。国際学校でIBやA-Levelsを学んでいる場合でも、DSEを追加で取得することで、JUPAS経由の香港大学入学に有利になります。

## 4. JUPAS（大学入学制度）でのDSE活用

### JUPASとは

**JUPAS（Joint University Programmes Admissions System：大学联合招生办公室）**は、DSE成績をもとに香港の大学・専門学院に出願するための統一システムです。

JUPASを利用できる主な大学・機関：
- 香港大学（HKU）
- 香港中文大学（CUHK）
- 香港科技大学（HKUST）
- 香港理工大学（PolyU）
- 香港城市大学（CityU）
- 香港浸会大学（HKBU）
- 嶺南大学（LU）
- 香港教育大学（EdUHK）

### JUPAS入学の最低基準（一般的な目安）

ほとんどの学部プログラムは以下を最低要件としています：
- 英語：Level 3以上
- 中国語：Level 3以上（または ApL Chinese「Attained with Distinction」以上）
- 数学：Level 2以上
- 市民生活と社会：Level 2以上
- 選択科目：2科目でLevel 2以上

競争率の高いプログラム（医学部・法学部・工学部など）は、5**や5*が多数必要です。

### 非JUPAS出願との違い

IBやA-Levelsを持つ国際学校卒業生は、JUPAS以外に**非JUPAS（Non-JUPAS）**ルートでも出願できます。ただし、非JUPASは定員が少なく競争が激しいため、DSEを取得してJUPAS経由で出願する方が有利な場合もあります。

## 5. 海外大学でのDSE認定

### 英国

英国のほとんどの大学がDSEを認定しています。

| DSEグレード | 相当するA-Level |
|-------------|----------------|
| 5** | A* |
| 5* | A |
| 5 | B |
| 4 | C |
| 3 | D |

UCASポイントへの換算も行われており、オックスフォード大学・ケンブリッジ大学を含む多くの大学でDSEが入学要件として受け入れられています。

### オーストラリア

オーストラリアの主要大学（メルボルン大学・シドニー大学・ANU等）はDSEを認定しており、各科目のグレードをATAR（オーストラリア大学入学ランク）に換算する基準を設けています。

### カナダ

UBC（ブリティッシュコロンビア大学）・トロント大学・マギル大学など主要校がDSEを認定。英語（Level 4以上）と数学（Level 4以上）が特に重視されます。

### アメリカ

米国の大学によるDSEの認定は英国・豪州・カナダほど標準化されていませんが、HKU・CUHKなど香港名門校の卒業後に米国大学院を目指す場合は問題ありません。一部のリベラルアーツカレッジや州立大学でも個別にDSEを評価します。

### 日本

日本の国立大学への直接入学にはDSEは基本的に使用できませんが、一部の私立大学（早稲田・慶應・立命館アジア太平洋大学等）では国際資格として個別評価される場合があります。

## 6. 日本人学生がDSEを受験するケースと注意事項

### 受験が推奨されるケース

**ケース1：香港の国際学校に在籍・卒業予定**
- ESF（英基学校協会）系列校や他の英語系国際学校在籍者
- 卒業後に香港の大学（HKU・CUHK・HKUST等）への進学を検討している場合
- IBやA-Levelsと並行してDSEの科目（主に英語・数学）を受験するケースもあり

**ケース2：日本帰国後に香港大学を目指す**
- 日本の高校を卒業後、DSEを個人受験（Private Candidate）として受験
- 英語力が高く、香港大学のグローバルプログラムに興味がある場合

**ケース3：英国・豪州・カナダの大学も視野に入れている**
- DSE取得により複数の英語圏大学への出願が可能になる
- 特に英国の大学を目指す場合、DSEはA-Levelsの代替として広く認められている

### 日本人学生特有の注意事項

**中国語科目について**
- 日本人学生の多くは「ApL Chinese」（応用学習中国語）を選択
- 日本語話者は漢字の読み書きに有利な場合もあるが、広東語（口語）の試験準備が必要
- 普通語（北京語）のバックグラウンドがある場合も、DSEの中国語は広東語に基づく点に注意

**英語科目について**
- 国際学校で英語教育を受けている場合は英語科目で高得点を狙いやすい
- 英語Level 3以上が多くの大学入学に必須

**市民生活と社会（CSD）について**
- 2021年に新設された科目で、従来の「通識教育（Liberal Studies）」に替わるもの
- 中国本土・香港・世界の相互関係を学ぶ内容
- 国際学校生徒にとって最も準備が必要な科目の一つ

## 7. DSEの出願・受験スケジュール（11月〜5月）

### 年間スケジュール概要

| 時期 | 内容 |
|------|------|
| 8〜9月 | 出願期間開始（翌年5月試験分） |
| 10〜11月 | 出願締め切り（通常10月末〜11月初旬） |
| 11〜12月 | 受験票・試験会場の通知 |
| 1〜2月 | 口語試験（Speaking exam）実施 |
| 3月下旬 | 筆記試験開始 |
| 4月〜5月 | 筆記試験期間（科目により異なる） |
| 7月中旬 | 成績発表（Result Day） |
| 7月下旬 | JUPAS最終出願・調整（Adjustment）フェーズ |
| 8月 | 大学入学手続き |

### 試験会場について

DSEの試験会場は香港内各所に設置されています。海外在住の学生が受験する場合は、試験期間中に香港に滞在する必要があります。海外試験会場は一部の科目のみに限られるため、事前にHKEAAの公式サイトで確認が必要です。

### 出願費用

2025年度実績（参考）：
- コア科目（4科目）：約HK$630
- 選択科目（1科目あたり）：約HK$245
- 合計（コア4科目＋選択2科目）：約HK$1,120〜HK$1,365

### 公式情報源

- **HKEAA公式サイト**：[www.hkeaa.edu.hk](https://www.hkeaa.edu.hk)（英語・中国語）
- **JUPAS公式サイト**：[www.jupas.edu.hk](https://www.jupas.edu.hk)（英語・中国語）

## よくある質問（FAQ）

**Q: 日本のパスポートを持っていてもDSEを受験できますか？**
A: はい。香港居住者でなくても、HKEAAに登録すれば受験できます（Private Candidate）。

**Q: DSEの成績は日本の大学入試に使えますか？**
A: 一般的には使用できません。一部の私立大学が個別に評価するケースはありますが、公式なルートとして確立されていません。

**Q: IBとDSEを同時に勉強できますか？**
A: 可能ですが、負担が非常に大きくなります。DSEの英語・数学のみを追加で受験するケースが現実的です。

**Q: 成績はいつまで有効ですか？**
A: ほとんどの科目は取得後5年間有効です。ただし、大学によっては独自の有効期限を設けている場合があります。
