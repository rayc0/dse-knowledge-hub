# DSEからカナダ大学進学ガイド

カナダはDSE保有者にとって、費用・安全性・移民後の永住権取得という3つの観点から非常に魅力的な留学先です。UBC（ブリティッシュコロンビア大学）、トロント大学、マギル大学をはじめとする世界トップクラスの大学群がDSEを公式な入学資格として認めており、卒業後は永住権取得への明確なルートも開かれています。

## PGWP→CEC→永住権ルートの解説

カナダは留学後の永住権取得ルートが世界で最も整備された国のひとつです。

### ステップ1: Post-Graduation Work Permit（PGWP）
- **対象**: カナダのDESIG指定校（UBC・トロント・マギル等はすべて該当）で8ヶ月以上の正規課程修了
- **有効期間**: プログラム期間に応じ最長3年
- **働き方**: カナダ国内でオープンワークパーミット（雇用主・職種の制限なし）

### ステップ2: Canadian Experience Class（CEC）
- **条件**: PGWP期間中に1年以上のカナダ国内就労経験（NOC TEER 0/1/2/3）
- **言語**: CLB 5（英語）または NCLC 5（フランス語）
- **申請**: Express Entryシステム経由でオンライン申請

### ステップ3: 永住権（Permanent Resident）取得
- CECで申請後、通常6ヶ月以内に永住権取得
- **所要期間の目安**: 大学4年間 → PGWP 3年 → CEC申請 → 合計約7〜8年で永住権

> このルートはDSE取得後にカナダ大学へ進学する日本人・アジア系学生の間でも注目を集めています。

## バンクーバー・トロントの生活環境

### バンクーバー（UBC・SFU）
- **気候**: 温暖・雨季あり（太平洋性気候）
- **日本語コミュニティ**: 約7万人の日系人が居住、日本語スーパー・レストラン充実
- **アジア系コミュニティ**: 人口の約45%がアジア系、多文化都市
- **就職市場**: IT・資源エネルギー・観光業が活発
- **家賃目安**: 1Rアパート CAD 1,800〜2,500/月

### トロント（Uトロント・マクマスター）
- **気候**: 四季あり（冬は氷点下）
- **日本語コミュニティ**: 日系商工会議所あり、大手日系企業（トヨタ・本田の北米拠点）
- **就職市場**: 金融・テクノロジー・製造業が強い
- **家賃目安**: 1Rアパート CAD 2,000〜2,800/月

## 日本人学生がカナダを選ぶ理由

| 選択理由 | 詳細 |
|--------|------|
| 治安の良さ | 先進国の中でも犯罪率が低く、留学初心者でも安心 |
| 多文化主義 | 政府方針として多文化主義を推進、人種差別が少ない |
| 永住権ルートの明確さ | 留学→就労→PR取得が制度化されている |
| 英語学習環境 | フランス語圏（モントリオール）では英仏二言語環境も |
| コスト | 米国・英国より総費用が抑えられる |
| 日本との時差 | 東部時間は-14時間（PST: -17時間）、国際電話で連絡しやすい |
