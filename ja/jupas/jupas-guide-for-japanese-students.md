# 日本人学生向けJUPASガイド：香港大学出願システムの完全攻略

香港の大学を目指す日本人学生にとって、**JUPAS（ジュパス）**の仕組みを正確に理解することは出願成功の第一歩です。本ガイドでは、JUPASの基本から、日本人学生特有の注意点まで、日本語でわかりやすく解説します。

---

## 1. JUPASの概要

### JUPASとは何か

**JUPAS（Joint University Programmes Admissions System：大學聯合招生辦公室）**は、DSE成績をもとに香港の大学・高等教育機関の学士課程プログラムに出願するための**統一オンラインシステム**です。

毎年約1万9,000〜2万人の受験生がJUPASを通じて出願し、香港8大学のほぼすべての学士課程プログラムがJUPAS経由で入学者を受け入れています。

### JUPASで出願できる主な機関

| 機関名 | 略称 |
|--------|------|
| 香港大学 | HKU |
| 香港中文大学 | CUHK |
| 香港科技大学 | HKUST |
| 香港理工大学 | PolyU |
| 香港城市大学 | CityU |
| 香港浸会大学 | HKBU |
| 嶺南大学 | LU |
| 香港教育大学 | EdUHK |

### JUPASと非JUPAS（Non-JUPAS）の違い

| 項目 | JUPAS | 非JUPAS |
|------|-------|---------|
| 対象者 | DSE受験者（主に香港居住者） | IB・A-Levels・SAT等の国際資格保有者 |
| 定員 | 各プログラムの70〜80%程度 | 20〜30%程度 |
| 競争率 | 人気プログラムは非常に高い | さらに高い場合もある |
| 推薦状 | 一部学部で必要 | 多くの場合必要 |
| 使用できる成績 | DSEのみ | IB/A-Levels/その他 |

**日本人学生へのポイント**: 国際学校でIBを取得した場合でも、DSEを並行して受験していればJUPAS経由で出願できます。定員の多いJUPASルートは非JUPASより有利な場合があります。

## 2. 出願プロセス

### 出願期間・費用

| 項目 | 詳細 |
|------|------|
| 出願開始 | 毎年10月中旬〜11月初旬 |
| 出願締め切り | 翌年1月中旬 |
| 出願費用（基本） | HK$270（最初の5プログラム選択） |
| 追加プログラム費用 | 1プログラムあたりHK$30 |
| 最大選択数 | **25プログラム**（5〜25の範囲で選択可） |

### オンライン出願の流れ

1. **JUPAS公式サイトで登録**（[www.jupas.edu.hk](https://www.jupas.edu.hk)）
2. **DSE候補者番号（Candidate Number）を入力**し、JUPASアカウントと紐付け
3. **志望プログラムを最大25件選択**し、A〜Eリストに分類
4. **出願内容の確認・決済**（クレジットカード・ATM払い等）
5. **出願後も変更可能な期間あり**（詳細は後述）

### 25プログラム選択の戦略

JUPASで最大25プログラムを選択できますが、単に数を埋めるのではなく**戦略的に組み合わせること**が重要です。

**推奨する分布（25プログラムフル活用の場合）：**

| リスト区分 | プログラム数 | 選択基準 |
|-----------|-------------|---------|
| Aリスト（最優先） | 3〜5件 | 第一志望群。自分の成績で合格の見込みがある上位希望 |
| Bリスト | 5〜8件 | 安全圏に近い志望校。妥協せず希望する学部 |
| Cリスト | 8〜12件 | ほぼ確実に合格見込みのバックアップ |
| DリストEリスト | 残り | 安全網。ただし通う意思のあるプログラムのみ |

**注意**: 行く意思のないプログラムを選択することは、他の受験生の機会を奪うことになるため倫理的に問題があります。実際に進学する意思のあるプログラムのみを選んでください。

## 3. Aリスト（First Choice）の重要性

### Aリストとは

JUPASでは選択した25プログラムを5つのリスト（A〜E、各最大5件）に分類します。**Aリストは最も重要な「第一志望群」**で、JUPASシステムはまずAリストのプログラムへの配置を優先的に試みます。

### Aリストの役割と仕組み

JUPASの配置アルゴリズムは以下の順序で機能します：

1. システムがあなたのDSE成績でAリストのプログラムに合格できるか確認
2. Aリストの中で最もランクの高いプログラム（A1）への配置を試みる
3. A1に不合格の場合、A2、A3...と順に確認
4. AリストすべてNGの場合、Bリストへ進む

### Aリスト選択の黄金ルール

**良い例：**
- A1：HKU医学部（成績がほぼ届くが少し背伸びが必要）
- A2：CUHK医学部（同様）
- A3：HKUST理工学部（合格の見込みが高い）

**悪い例：**
- A1〜A5すべてに手が届かないプログラムを入れる（配置ゼロになるリスク）
- A1〜A5すべてに安全すぎるプログラムを入れる（より良い選択肢を逃す）

**戦略の核心**：AリストのA1には「少し背伸びが必要だが不可能ではない」プログラムを置き、A4〜A5には「ほぼ確実に合格できる」プログラムを置くグラデーション配置が理想的です。

## 4. プログラム別DSE最低要件と入学競争

### 主要大学・学部の一般的なDSE要件

以下はJUPAS出願における一般的な成績水準です（年度によって変動します）。

**最難関プログラム（5**・5*レベル多数必要）**

| プログラム | 大学 | 目安となる最低成績 |
|-----------|------|------------------|
| 医学・外科（MBBS） | HKU / CUHK | コア4科目で5**×2以上＋選択2科目5*以上 |
| 法律（LLB） | HKU / CUHK | 全科目ほぼ5レベル以上 |
| コンピュータサイエンス | HKUST | 数学5*以上＋理科選択科目5以上 |

**人気プログラム（4〜5レベル必要）**

| プログラム | 大学 | 目安となる最低成績 |
|-----------|------|------------------|
| ビジネス（BBA） | HKU / HKUST | 英語4＋数学4＋選択2科目4〜5 |
| 工学 | HKUST / PolyU | 数学4以上＋理科4以上 |
| 社会科学 | CUHK | 英語4＋コア3322以上 |

**入学しやすいプログラム（3レベル基準）**

| プログラム | 大学 | 目安となる最低成績 |
|-----------|------|------------------|
| 教育学 | EdUHK | コア3322以上＋選択2科目3以上 |
| 社会福祉 | HKBU | コア3322以上 |
| 人文学 | 各大学 | コア3322以上 |

### 競争率を理解するための指標

JUPASでは各プログラムの**過去3年間のメジアン・スコア（中央値成績）**がJUPASウェブサイトで公開されています。出願前に必ず確認し、自分の成績との差を把握しましょう。

## 5. 面接と入学審査

### 面接が実施される主なプログラム

すべてのプログラムが面接を行うわけではありませんが、以下の学部は特に面接を重視します。

| 学部 | 主な大学 | 面接の形式 |
|------|---------|-----------|
| 医学部 | HKU・CUHK | MMI（Multiple Mini Interview）形式 |
| 法学部 | HKU・CUHK | グループディスカッション・個人面接 |
| 看護学部 | 各大学 | 個人面接 |
| 教育学部 | EdUHK | 英語・中国語口頭試験＋グループ活動 |
| ソーシャルワーク | 各大学 | 個人面接 |

### 面接で評価される要素

1. **英語コミュニケーション能力**（特に英語系プログラム）
2. **中国語（広東語）コミュニケーション能力**（中国語系プログラム）
3. **批判的思考・問題解決能力**
4. **社会貢献・課外活動の経験**
5. **志望動機の明確さ**

### 日本人学生のための面接対策

**英語面接について**：国際学校出身の日本人学生は英語面接に強みを持つ傾向があります。ただし、香港の社会問題や大学のプログラムに関する最新知識が問われることも多いため、事前調査が不可欠です。

**広東語面接について**：一部のプログラムでは広東語での面接が行われます。日常会話レベルの広東語習得が推奨されます。

**推薦状**：一部のプログラム（特にHKUとCUHK）では学校の成績証明書や推薦状の提出を求めます。国際学校の担任教師・進路指導担当者に早めに依頼しましょう。

## 6. 日本人学生に特有のポイント

### 英語力基準の確認

JUPAS経由で英語系プログラムに入学する場合、一般的に以下が求められます。

| 要件 | 内容 |
|------|------|
| DSE英語 | Level 3以上（多くの学部）、Level 4以上（英語重視学部） |
| IELTS（代替可） | 6.5〜7.0以上（非JUPAS経由の場合） |
| TOEFL iBT（代替可） | 80〜100以上（非JUPAS経由の場合） |

**JUPASルート**では原則としてDSE英語の成績が英語力の証明になります。

### 推薦状の準備

推薦状を求めるプログラムに出願する場合：

- **依頼先**：担任教師、学科担当教師、課外活動指導者など
- **依頼時期**：出願開始（10〜11月）の少なくとも2〜3ヶ月前
- **形式**：JUPASのオンラインシステム経由で提出（教師に専用リンクを送付）
- **内容**：学業成績だけでなく、人物・リーダーシップ・課外活動も含める

### 非DSE科目・資格の扱い

JUPAS出願において、DSE以外の資格は以下のように扱われます：

| 資格 | JUPASでの扱い |
|------|--------------|
| IB Diploma | 非JUPAS経由での出願資格として使用可（JUPASでは直接使用不可） |
| A-Levels | 同上 |
| SAT/ACT | 一部の大学で補足資料として参照可能 |
| 日本語能力試験（JLPT） | 日本語関連プログラム出願時に有利 |
| 漢検 | 補足資料として提出可能な場合あり |

### 学費・奨学金情報

日本人学生を含む非本地学生（Non-local Students）は、一般的に本地学生（Local Students）と異なる学費設定となります：

| 大学 | 年間学費目安（非本地学生） |
|------|--------------------------|
| HKU | HK$140,000〜145,000 |
| CUHK | HK$135,000〜145,000 |
| HKUST | HK$140,000〜145,000 |
| PolyU | HK$120,000〜140,000 |

奨学金制度：
- **HKU Global Scholarships**：非本地学生向け成績優秀奨学金
- **HKUST RedBird Award**：学業・課外活動優秀者向け
- **香港政府奨学基金**：一部の非本地学生も対象

## 7. JUPAS調整フェーズ（Clearing & Adjustment）の活用

### 調整フェーズとは

DSE成績発表（7月中旬）後、JUPASの配置結果が発表されます。その後、**調整フェーズ（Adjustment Phase）**が開始され、以下の機会が生まれます。

### 調整フェーズの2つのシナリオ

**シナリオA：成績が予想より良かった場合（Upgrade）**
- 当初のJUPAS配置プログラムより上位のプログラムへのアップグレードを申請可能
- 条件：現在の配置プログラムへの入学権を保持しつつ、上位プログラムの空き枠を狙う
- 期間：成績発表後7〜10日程度

**シナリオB：配置漏れした場合（Clearing）**
- JUPASで配置が決まらなかった学生が空き枠のあるプログラムに直接出願
- 大学側が残余定員を公開し、学生が個別に連絡・応募
- 競争は激しいが、最後のチャンスとなる

### 調整フェーズの実践的な対策

**事前準備（成績発表前）：**
1. 各大学の調整フェーズ用の連絡先・Webページをブックマーク
2. 成績発表当日に動けるよう、追加書類（成績証明書・推薦状）を準備
3. 「本命より上」と「安全網のさらに下」の両方のプログラムリストを作成

**成績発表当日：**
1. JUPASで成績確認後、即座に配置結果を確認
2. アップグレード希望の場合：JUPAS Adjustment申請ページへアクセス
3. 配置漏れの場合：各大学の追加募集ページを即座に確認・応募

**注意事項**：
- 調整フェーズは通常**2〜3日間のみ**と非常に短い
- 日本にいる場合、時差（香港はJSTより1時間遅れ）を考慮して成績発表時刻を確認
- 香港の友人・家族のサポートを事前に依頼しておくことを推奨

## JUPAS出願チェックリスト（日本人学生向け）

### 出願前（9〜10月）
- [ ] JUPASウェブサイトのアカウント作成
- [ ] 希望プログラムの最低DSE要件確認
- [ ] 各プログラムの過去3年間の入学者成績（メジアン）確認
- [ ] 推薦状が必要なプログラムの特定と教師への依頼

## よくある質問（FAQ）

**Q: JUPASに出願するにはHKEAAの受験番号が必要ですか？**
A: はい。JUPASに出願するには、まずHKEAAにDSE受験者として登録し、受験番号（Candidate Number）を取得する必要があります。

**Q: 成績発表前にJUPASプログラムの選択を変更できますか？**
A: 通常、出願後も一定期間内はプログラムの変更・追加が可能です。変更期限はJUPASウェブサイトで毎年告知されます。

**Q: 日本語で書かれた推薦状は受け付けてもらえますか？**
A: 原則として英語または中国語の推薦状が必要です。日本語の推薦状は公認翻訳を添付すれば受け入れる大学もありますが、事前に各大学の入学事務局に確認することを強く推奨します。

**Q: JUPASで希望が通らなかった場合、他に進学の選択肢はありますか？**
A: はい。主な代替選択肢として、(1) 翌年の再受験、(2) 非JUPAS経由での各大学への個別出願（同年または翌年）、(3) 副学位（Associate Degree）経由で大学3年次編入、があります。
