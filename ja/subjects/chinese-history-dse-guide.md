# DSE中国史（Chinese History）完全ガイド

## 日本人受験生のための試験構成・学習対策・進学への活用

---

## 1. DSE中国史とは

DSE中国史（Chinese History）は、先史時代から1997年の香港返還にいたる中国の通史をカバーする科目です。単なる歴史知識の暗記にとどまらず、**繁体字漢文による資料読解と論述解答**が求められる、DSEの中でもとりわけ特殊な科目です。

日本の「世界史」や「東洋史」とは異なり、中国史はあくまでも「中国の視点」から歴史を叙述します。歴史解釈の軸が日本の教科書と異なる場面があるため、日本人受験生は視点の切り替えに意識的に取り組む必要があります。

香港の大学進学（JUPAS）では、中国史は選択科目（Category B）として扱われ、成績優秀者は歴史学・中国語中文学・区域研究系学部で高く評価されます。

## 2. 試験構成

| 試験 | 内容 | 配点 | 試験時間 |
|------|------|------|----------|
| **Paper 1** | 中国史通史（資料問題・短答） | 45% | 1時間30分 |
| **Paper 2** | テーマ選択（論述問題・2問選択） | 55% | 2時間 |

**Paper 1** は資料（史料・地図・統計）を読み解く問題が中心です。繁体字漢文の一次資料が出題されることも多く、読解力が直接スコアに影響します。

**Paper 2** は複数のテーマ（例：中国近現代史/香港史/中国と周辺諸国の関係）から2問を選択し、論述形式で解答します。論点の明確さと史実の正確な引用が採点基準の核心です。

## 3. 主要学習範囲

### 古代〜清末

| 時代 | 主要トピック |
|------|-------------|
| 秦漢 | 中央集権体制の確立・郡県制・シルクロード |
| 隋唐 | 科挙制度・唐の対外関係（日本への影響含む）・安史の乱 |
| 宋元 | 南宋の経済発展・モンゴル帝国・海上貿易 |
| 明清 | 鎖国政策・清の拡大・アヘン戦争の背景 |

### 近現代（重点範囲）

| 時代 | 主要トピック |
|------|-------------|
| 1840–1912 | アヘン戦争・太平天国・洋務運動・辛亥革命 |
| 1912–1949 | 五四運動・国共内戦・日中戦争（1937–1945） |
| 1949–1978 | 中華人民共和国成立・大躍進・文化大革命 |
| 1978–1997 | 改革開放・天安門・香港返還 |

### 香港史（Paper 2 テーマとして頻出）

- 英国植民地統治下の制度と社会変化
- 香港経済の発展と中国との関係
- 1997年返還と「一国二制度」の背景

## 4. 日本人受験生の強み・弱み

| 項目 | 評価 | 詳細 |
|------|------|------|
| 隋唐文化・日中関係史 | 強み | 日本の歴史教育と重複が多く、遣唐使・漢字文化圏の知識が活用できる |
| 明清時代・貿易史 | 強み | 日本の「江戸時代の鎖国」との比較視点が論述に応用可能 |
| 日中戦争（1937–1945） | 強み（要注意） | 事実関係は把握しているが、中国側の叙述視点への切り替えが必要 |
| 繁体字漢文の読解 | 中立〜強み | 日本語の漢字知識が土台になるが、繁体字独自の字形に慣れる必要がある |
| 文化大革命の中国的解釈 | 弱み | 日本の教科書では批判的に描かれるが、DSEでは中立的叙述が求められる |
| 改革開放・経済史 | 弱み | 数値・政策名の暗記量が多く、日本人には馴染みが薄い |
| 論述の中国語表記 | 要注意 | 解答は繁体字中国語（書き言葉）が基本 — 日本語での解答は不可 |

## 5. 繁体字漢文読解の対策

日本人受験生にとって意外なアドバンテージが**繁体字漢文の読解力**です。日本語の漢字（特に旧字体）と繁体字は多くの字形を共有しており、文脈から意味を推測する力が活かせます。

### 繁体字と日本語の対応例

| 繁体字 | 日本の旧字 / 新字 | 意味 |
|--------|------------------|------|
| 國 | 國 / 国 | 国 |
| 體 | 體 / 体 | 体 |
| 實 | 實 / 実 | 実 |
| 發 | 發 / 発 | 発 |
| 長 | 長 / 长 | 長（読み方注意） |

### 対策のポイント

1. **歴史用語の繁体字表記を優先的に覚える** — 「鴉片戰爭」「辛亥革命」「文化大革命」など頻出語を繁体字で書けるように
2. **DSE過去問の一次資料問題を毎週1〜2問練習** — 資料の繁体字漢文を読み、設問に答える練習が最効率
3. **簡体字との変換練習** — 簡体字で知っている語を繁体字に変換する習慣をつけると誤字が減る

## 7. Level 4以上を目指す対策

### 優先順位

1. **繁体字漢文読解の集中練習（最優先）** — Paper 1の資料問題は読解なしに得点できない
2. **近現代史（1840–1997）の徹底理解** — Paper 2の論述テーマは近現代史から最も多く出題される
3. **年表暗記 → 論述への構造化** — 年号・事件・背景・影響の4項目をセットで暗記し、論述テンプレートに当てはめる

### 論述テンプレート（Paper 2）

```
【立論】〇〇の主な要因は△△にある。
【論拠1】史実として〜が挙げられる。（具体的年号・人物・政策）
【論拠2】さらに〜という側面も重要である。
【反論の検討】一方で〜という見方もあるが、〜の理由で主論が優位である。
【結論】以上より、〇〇は△△によって説明される。
```

### 学習時間の目安

| 学習内容 | 週あたり時間 |
|----------|------------|
| 通史の復習（年表・用語） | 2〜3時間 |
| 過去問・資料問題演習 | 2時間 |
| 論述練習（Paper 2） | 1〜2時間 |
| 繁体字書き取り練習 | 30分 |

## まとめ

DSE中国史は日本人受験生にとって**漢字知識というアドバンテージ**がある一方、繁体字漢文での解答と中国的歴史観への適応が大きな壁です。近現代史（1840–1997）を重点的に学習し、過去問の一次資料問題で繁体字読解力を鍛えることが、Level 4以上への最短ルートです。歴史・中文・区域研究系の大学学部を目指す日本人受験生には、戦略的に選択する価値の高い科目といえます。
