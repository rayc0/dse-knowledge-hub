# DSE試験戦略完全ガイド
## 日本人受験生のための科目選択・時間管理・Level 5★攻略

HKDSE（香港大学入学試験）は、4科目の必修と2〜3科目の選択で構成されます。どの科目を選ぶか、どう準備するかによって、合格できる大学・学部が大きく変わります。本ガイドでは、日本人受験生の視点から最も効果的な試験戦略を解説します。

---

## 1. 科目選択戦略

### 必修4科目（全員受験）

| 科目 | 備考 |
|------|------|
| 中国語（Chinese Language） | 日本人には最難関。早期対策必須 |
| 英語（English Language） | 日本人の得点源になりやすい科目 |
| 数学（Mathematics） | 計算力より概念理解が重要 |
| 通識教育（Citizenship & Social Development） | 旧Liberal Studies。社会問題の論述 |

### 選択科目の組み合わせパターン

**理系パターン（理工学部・医学部志望）**

```
推奨: Physics + Chemistry + Biology（または M1/M2）
特徴: HKU・CUHK理工学部はこの組み合わせを最も高く評価
注意: 3科目は負担が大きい。2科目（Phys+Chem）+M1/M2も有効
```

**文系パターン（法学・社会科学・人文系志望）**

```
推奨: BAFS + Economics + Geography または History
特徴: JUPAS出願で法学・商学系に有利
注意: 中国語の成績が特に重要視される
```

**混合パターン（ビジネス＋理系学部の両立）**

```
推奨: Physics + Economics + BAFS
特徴: HKU BBA（商学）+ 工学系への両面出願が可能
注意: 科目の性格が多様で勉強方法が分散しやすい
```

## 3. 6週間前〜当日タイムライン

### Year 12（HKDSE前年）のSBAデッドライン

**SBA（School-Based Assessment）**は学校内評価であり、試験本番より早く締め切られます。

| 時期 | タスク |
|------|--------|
| Year 12 9〜11月 | SBA課題の作成・提出（英語・中国語・BAFS等） |
| Year 12 1月〜3月 | SBA最終提出・内部評価 |
| Year 12 3〜4月 | 本番試験（DSE）開始 |

**重要**：SBAを後回しにすると、試験直前期と重なり負担が極大化します。Year 11から少しずつ取り組んでください。

### 試験6週前〜本番タイムライン

| 時期 | 優先タスク |
|------|------------|
| 6週前 | 全科目の過去問3年分を通し解き → 弱点科目を特定 |
| 5週前 | 弱点科目の集中対策（概念整理 + 過去問演習） |
| 4週前 | 論述科目（歴史・地理・経済）の答案テンプレート完成 |
| 3週前 | MCQ科目（理科・数学）の高速演習（1日1セット） |
| 2週前 | 全科目の時間計測模試（本番形式） |
| 1週前 | 体調管理最優先。新しい内容は入れない。要点ノートの復習のみ |
| 前日 | 試験会場の確認・持ち物チェック・早寝（23時までに就寝） |
| 当日 | 朝食は軽く・腸に優しいもの。会場には45分前到着 |

## 4. 共通解答テクニック

### Paper 1データ問題：グラフ読み取り3ステップ

経済・地理・BAFS等のデータ問題で使える汎用テクニックです。

```
ステップ1：軸と単位を確認（X軸・Y軸・凡例・単位）
ステップ2：トレンドを言語化（増加/減少/横ばい/変曲点の年代）
ステップ3：因果関係を追加（なぜそのトレンドが生じたか → 知識を接続）
```

**よくある失点パターン**：グラフを読んだだけで「なぜか」を書かない。
**対策**：必ず "This is because..." または "This can be attributed to..." で理由を添える。

### 論述問題：PEEL構造

詳細は各科目ガイドに記載していますが、共通原則は以下の通りです。

- **1段落 = 1論点**（複数の主張を1段落に詰め込まない）
- **証拠は具体的に**（年代・人名・統計・事件名を明示）
- **結論で問いに直接答える**（「したがって〜である」の形で締める）

### MCQ：消去法 vs 時間配分

| 科目 | 推奨戦略 |
|------|----------|
| 数学 | 計算で確実に解く。時間がかかる問題は後回し |
| Physics/Chemistry | 公式の記憶より「なぜその公式か」の理解で応用 |
| 歴史MCQ（Section A） | 消去法が有効。「最も適切な選択肢」を選ぶ問題が多い |

**時間配分の目安**：1問あたり最大90秒。最初の一周で確信できない問題はチェックして次へ進む。

## 5. 本番当日チェックリスト

### 持ち物

- [ ] 受験票（Admission Form）+ 顔写真付きID
- [ ] HBまたは2B鉛筆（MCQ用）・黒ボールペン（論述用）
- [ ] 消しゴム・定規
- [ ] 承認済み電卓（数学・BAFS）
- [ ] 飲み物・軽食（休憩時間用）
- [ ] ストップウォッチ（試験中の時間管理用）

### 試験会場での注意事項

- 試験開始15分前には着席。問題冊子を配られたら**まず全ページをめくって構成を確認**。
- MCQの答案はずれないよう、定期的に問題番号と答案用紙の行を照合する。
- 論述は**薄い字で見直せるようにする**（鉛筆の上に黒ペンで清書する方法も有効）。

### 答案記入ルール（よくあるミス）

- MCQ：HBまたは2B鉛筆のみ。ボールペン使用は無効になる可能性あり。
- 論述：黒または青ボールペン。鉛筆で論述答案を提出しないこと。
- 答案用紙の上部（受験番号・科目番号）の記入漏れはゼロ点になるリスクあり。

## 6. JUPAS出願との連動

JUPAS（Joint University Programmes Admissions System）は香港の大学への統一出願システムです。DSE結果が出る前に出願するため、スケジュール管理が重要です。

### 出願スケジュール

| フェーズ | 時期 | 内容 |
|----------|------|------|
| Early Round | 11月 | 第一次出願（プログラムリストの提出） |
| Main Round | 翌2〜3月 | 志望順位の確定・変更 |
| DSE試験 | 4月〜5月 | 本番試験 |
| 結果発表 | 7月10日前後 | HKDSE成績公開 |
| Offer発表 | 7月中旬 | JUPASのオファー通知 |

### 志望校選択のポイント

- **第1志望（Band A）**：届くか届かないかのチャレンジ校（Level 5以上を狙う科目で勝負）
- **第2〜4志望（Band B）**：現実的な目標校
- **第5〜6志望（Band C/D）**：安全校（保険として必ず入れる）

**日本人受験生へ**：中国語のLevel 3以上がほぼすべての学部で最低条件です。中国語対策を最優先にしながら、強みの英語・数学・理科でLevel 5★を狙う組み合わせが最も効果的です。

## 日本人受験生に多い失点パターンと対策

| 失点パターン | 原因 | 対策 |
|-------------|------|------|
| 中国語で足切りになる | 日常生活での中国語接触が少ない | Year 10から毎日1時間の中国語集中訓練 |
| 論述で「事実の羅列」になる | 英語での論理展開に不慣れ | PEEL構造を徹底。接続詞（Therefore/However/As a result）を意識 |
| MCQで時間切れ | 1問に時間をかけすぎる | 1問90秒ルールを徹底。迷ったらマークして次へ |
| SBAで低評価 | 締め切りを軽視・内容が薄い | 学校の指示をよく読み、フィードバックをもらいながら早期に完成 |
| 試験直前に体調を崩す | 過労・睡眠不足 | 試験2週前から規則正しい睡眠（7時間以上）を確保 |

## まとめ

DSEで高得点を取るには、「何を選ぶか」と「どう準備するか」の両方が重要です。日本人受験生の強みである英語・数学・理科を最大限に活かしつつ、中国語という最大の壁を早期に攻略する戦略が鍵です。

SBAの早期着手・6週間前からの本番形式練習・JUPAS出願スケジュールの把握——これらを計画的に進めることで、Level 5★獲得と志望大学合格の両立が現実になります。
