# DSE経済（Economics）完全ガイド：日本人受験生のための履修範囲・試験構成・ビジネス系進学への活用

DSE経済は、香港の大学進学において経済学・商学・会計・法律・政策系学部が評価する主要選択科目です。理系科目ほど数学力は要求されませんが、英語での経済分析論述が核心スキルとなります。このガイドでは、日本の高校「政治経済」との比較を交えながら試験構成・学習戦略を解説します。

## 1. 試験構成と配点

| ペーパー | 形式 | 配点比率 |
|---|---|---|
| Paper 1 | 多肢選択（MCQ）45問 | 45% |
| Paper 2 | データ解析・短答 | 25% |
| Paper 3 | 自由論述（Essay型） | 30% |

Paper 1は幅広い知識を素早く確認するスピード勝負です。Paper 2はグラフや統計データを読み取り、経済理論を応用する問題が出ます。Paper 3は特定の経済問題に対して、理論を使って論述する力が問われ、Level差が最も出る部分です。

## 2. ミクロ経済・マクロ経済の重要単元比較

| DSE経済の単元 | 日本の対応（政治経済/現代社会） | 深度比較 |
|---|---|---|
| 需要と供給・価格メカニズム | 市場経済の基礎 | DSEはグラフ分析が詳細 |
| 消費者行動・弾力性 | 日本には一部しかない | DSEはより体系的 |
| 市場構造（完全競争〜独占） | 日本では概論のみ | DSEは詳細な分析が必要 |
| 国民所得・GDP・インフレ | 国民経済計算 | ほぼ同等 |
| 財政政策・金融政策 | 日本でも取り扱う | DSEは実例分析が豊富 |
| 国際貿易・比較優位 | 日本では概説のみ | DSEは計算問題あり |
| 香港経済の特徴 | 日本には対応なし | DSE固有（重要） |

「香港経済の特徴」（自由貿易港・金融センター・土地供給問題など）はDSE独自トピックで、Paper 2・3で頻出します。香港の経済ニュースを日常的にチェックすることが有効な対策です。

## 3. Paper 3論述の構造テンプレート

Paper 3の論述問題では、以下の4段階構造で回答することが採点上有効です。

```
1. 主張（Claim）：自分の立場・結論を1文で明示
2. 理由と根拠（Reason）：経済理論・モデルを用いて説明（需給曲線・乗数効果など）
3. 反論への対処（Counter-argument）：反対意見を認め、それでも自説が優位な理由を述べる
4. 結論（Conclusion）：主張を再確認し、政策的含意や限界を言及
```

**例題：「最低賃金の引き上げは失業を増加させるか」**
- Claim: 最低賃金の引上げは一定条件下では失業を増加させうる
- Reason: 労働需要曲線上、賃金上昇は雇用量を減少させる（図示）
- Counter: ただし労働市場がモノプソニー的である場合は、雇用が増加する可能性がある
- Conclusion: 市場構造によって効果は異なり、一律の結論は出せない

## 4. 日本人受験生の強みと弱点

**強み：**
- 数学的な計算（弾力性の計算・乗数効果など）は高い正解率が期待できる
- 日本の社会科教育で国際経済・財政政策の基礎知識あり
- 統計・グラフの読取能力（Paper 2）は比較的得意

**弱み：**
- Paper 3の英語論述は「日本語で考え、英語に翻訳」するアプローチではレベルが上がりにくい
- 香港固有の経済問題（土地政策・通貨ペッグ制・自由貿易体制）は0からの学習が必要
- 「経済学的な主張の組み立て方」は日本の政治経済教育では訓練が少ない

## 6. 香港経済の実例を活用した問題傾向

DSE経済の過去問では、以下の香港固有の実例が繰り返し出題されます。

- **土地価格と住宅問題**：需給分析・外部性・政府介入
- **連動為替制度（通貨ペッグ）**：固定相場制のメリット・デメリット
- **自由貿易と保護主義**：比較優位・WTO・香港の貿易構造
- **最低賃金法（MWO）**：労働市場分析・弾力性
