# DSE生物（Biology）完全ガイド：日本人受験生のための履修範囲・試験対策・医療系進学への道

DSE生物は、医学・歯学・看護・薬学・生命科学への進学を目指す受験生にとって最も重要な選択科目の一つです。暗記量が多い一方で、論述力と図解読取力も問われる科目です。このガイドでは日本の「生物」との比較を交えながら、試験対策を体系的に解説します。

## 1. 試験構成と選択必修の関係（Core/Extension）

DSE生物は「必修（Compulsory）パート」と「選択（Elective）パート」の2段階構成です。

| ペーパー | 内容 | 配点比率 |
|---|---|---|
| Paper 1 | 必修パートのMCQ（30問） + 長問 | 50% |
| Paper 2 | 選択パートの長問（2択1） | 35% |
| SBA（学校内評価） | 実験・フィールドワーク | 15% |

選択パートは「生化学・生体技術・生態学」から2つ選択します。選択の戦略が得点に大きく影響するため、授業開始時に得意分野を見極めることが重要です。

## 2. 主要トピックと日本の生物との比較表

| DSE生物のトピック | 日本の対応単元（生物基礎/生物） | 深度比較 |
|---|---|---|
| 細胞構造と機能 | 細胞の構造・細胞膜 | ほぼ同等 |
| 光合成・細胞呼吸 | 同上 | DSEはやや定性的 |
| 遺伝・DNA・タンパク質合成 | 遺伝情報の発現 | 同等（計算問題あり） |
| 進化・自然選択 | 進化の仕組み | 同等 |
| 生態系・環境問題 | 生態系・生物と環境 | DSEは香港の実例が多い |
| 人体の機能（循環・消化・神経） | 動物の体の機能 | 同等（図解が多い） |
| 生化学（選択：Elective A） | 日本の「生物」にも一部あり | DSEはより詳細 |
| バイオテクノロジー（選択：Elective B） | 日本の「生物」にも一部あり | DSEは実用的応用が豊富 |

全体として、DSE生物は日本の高校「生物」とほぼ同等のカバレッジです。ただし、英語での図解問題と論述記述のスタイルに慣れる必要があります。

## 3. 医療系学部での生物の役割

### HKU医学部（MBBS）の要件

HKU（香港大学）医学部の一般的な入学要件は以下の通りです。

- 必須科目：**化学（Chemistry）** — 必須
- **生物（Biology）** — 必須ではないが、実質的に受験者のほぼ全員が履修
- 物理（Physics）— 推奨

生物が「公式必須」でない場合も、面接・入学試験（HKUSAT）では生物の基礎知識が前提とされます。化学と生物の両方を取得することが医学部合格への現実的な戦略です。

| 学部 | 大学 | 生物の位置付け |
|---|---|---|
| 医学部（MBBS） | HKU / CUHK | 事実上必須 |
| 歯学部（BDS） | HKU / CUHK | 推奨 |
| 看護学 | PolyU / CUHK | 推奨 |
| 薬学（B.Pharm） | CUHK / HKU | 化学 + 生物が理想 |
| 生命科学 | HKUST | 推奨 |

## 4. SBAの評価基準と効果的な実験レポートの書き方

SBAでは以下の4項目が評価されます。

| 評価項目 | 内容 |
|---|---|
| 計画（Planning） | 仮説・変数・実験設計の適切さ |
| 実施（Implementation） | 正確な観察・測定の記録 |
| 分析（Analysis） | データ処理・グラフ・傾向の考察 |
| 評価（Evaluation） | 誤差の分析・実験方法の改善提案 |

**効果的なレポートのポイント：**
- 図・グラフは必ずラベルと単位を記入
- 「改善提案」は「より精度の高い装置を使う」ではなく、具体的な変更点を述べる
- 結論は仮説と対応させて記述する

## 5. Level 5★を目指す学習法

### 5.1 暗記の効率化

生物は暗記量が多い科目ですが、ただ丸暗記するより「仕組みで理解する」方が長期記憶に残ります。例えば、DNAの複製メカニズムは「なぜそうなるか」を理解することで、問題のどんな切り口にも対応できます。

### 5.2 図解問題の対策

Paper 1・2の長問には、細胞の断面図・心臓の構造図・食物連鎖の図など図解を使った問題が頻出します。図を見て即座に部位名と機能を英語で答えられるよう、教科書の図を繰り返し書き写す練習が有効です。

### 5.3 論述問題の構造

Paper 2の論述問題は「●●を説明しなさい（Explain...）」の形式が中心です。以下の構造で回答すると採点者の評価が高くなります。

1. **定義または現象の説明**（1〜2文）
2. **仕組み・プロセスの詳述**（2〜4文）
3. **結果または影響の説明**（1文）

過去問で採点基準表（Mark Scheme）を確認しながら練習することが最善の対策です。
