# DSE複合理科（Combined Science）完全ガイド
## 日本人受験生のための試験構成・科目選択との比較

DSEの選択科目において、「理科系の科目をひとつ取りたいが、3科目は多すぎる」「文系志望だが理系学部も視野に入れたい」という受験生に注目されているのが、**Combined Science（複合理科）**です。1科目分の枠で2科目分の内容をカバーできるという独自の仕組みを持つこの科目について、日本人受験生向けに詳しく解説します。

---

## 1. Combined Scienceとは

Combined Scienceは、**1つの選択科目枠で2つの理科分野の基礎を学べる**科目です。単科理科（Physics・Chemistry・Biology）と比べて試験範囲は浅くなりますが、その分、時間と労力を他の科目に充てることができます。

### 3つのモジュール選択

受験生は以下の3種類のモジュールから**1つ**を選択します：

| モジュール | 内容 |
|-----------|------|
| **Module 1** | Physics + Chemistry |
| **Module 2** | Chemistry + Biology |
| **Module 3** | Physics + Biology |

各モジュールは、対応する2科目の**必須項目（Compulsory part）**のみを扱います。オプション科目や拡張内容（Extension Topics）は含まれません。

---

## 2. 試験構成

Combined Scienceの試験は2つのペーパーで構成されます。SBAは**ありません**（単科理科にはSBAがある場合がある）。

### Paper 1：多肢選択問題（MCQ）

| 項目 | 内容 |
|------|------|
| 形式 | 多肢選択（Multiple Choice） |
| 問題数 | 約36問（各モジュール18問ずつ） |
| 配点 | 全体の約33% |
| 時間 | 約1時間15分 |

### Paper 2：短答・構造問題

| 項目 | 内容 |
|------|------|
| 形式 | 短答・グラフ・計算・説明記述 |
| 問題数 | セクションAとBで分かれる（各モジュール対応） |
| 配点 | 全体の約67% |
| 時間 | 約1時間45分 |

**SBAなし**は大きなメリットです。単科理科（特にBiology・Chemistry）ではSBAの実験報告書作成が大きな負担になりますが、Combined Scienceでは試験のみで評価されます。

---

## 3. 単科理科との比較表

| 比較項目 | 単科理科 | Combined Science |
|----------|----------|------------------|
| 試験深度 | 深い（オプション含む） | 浅い（必須項目のみ） |
| SBA | あり（Chemistry/Biologyは実験報告） | なし |
| 入試評価 | HKU/CUHK理工学部で強く推奨 | PolyU/CityU工学系で認められる |
| Level 5★難易度 | 高い（競争が激しい） | 中程度（深さが不要なぶん取りやすい） |
| 時間効率 | 低い（1科目に多くの時間） | 高い（2科目分を1科目枠で） |
| 科目多様性 | 特定理科に特化 | 2分野の基礎をバランスよく習得 |

**結論**：時間と科目の多様性を優先するならCombined Science、特定理系学部への進学が明確ならば単科理科が有利です。

## 4. Combined Scienceが向く受験生

### タイプ1：文理混合で多様な出願を狙う受験生

```
例: BAFS + Economics + Combined Science（Module 1: Physics+Chemistry）
→ 商学系（HKU BBA・CUHK Business）への出願を主軸にしつつ、
  PolyU Engineering / CityU Applied Scienceへの出願も可能にする
```

### タイプ2：勉強時間に制約がある受験生

Year 12で課外活動・SBA・他科目の論述準備が重なる場合、単科理科2科目は非現実的です。Combined Scienceで理科を1科目枠に収めることで、全体のバランスを保てます。

### タイプ3：理系学部が第二志望の受験生

第一志望が法学・社会科学系の受験生が、理工学部を保険として出願する際に有効です。Combined Scienceでも、PolyU・CityU・HKUST（一部プログラム）への出願資格を満たすケースがあります。

## 5. 大学入学評価

Combined Scienceの大学側での評価は**学校によって異なります**。最新情報は各大学の公式アドミッション要件を確認してください（以下は一般的な傾向）。

| 大学 | 評価 |
|------|------|
| **HKU（香港大学）** | 理工学部・医学部は原則として単科理科（2科目）を要求。Combined Scienceは不可の場合が多い |
| **CUHK（香港中文大学）** | 理学部・工学部は単科理科を推奨。Combined Scienceは下位条件として認められる場合がある |
| **HKUST（科技大学）** | 工学部は単科Physics/Chemistry推奨。詳細は学部別要件で確認 |
| **PolyU（理工大学）** | 工学系・健康科学系でCombined Scienceを認めるプログラムが比較的多い |
| **CityU（城市大学）** | 一部の理工学部でCombined Scienceを認める。ただし単科理科より評価ポイントが低くなることがある |

**注意**：医学部・薬学部・看護学部への進学を目指す場合、Combined Scienceでは要件を満たさない可能性が高いです。必ず各大学の最新アドミッション要件を確認してください。

## 6. Level 5対策

Combined ScienceでLevel 5（以上）を目指すための戦略です。

### 「広く浅く」ではなく「概念の正確な理解」

Combined Scienceの出題は深くない分、**基礎概念の正確な理解と応用力**が問われます。計算問題は単科理科より複雑ではありませんが、説明記述（概念を英語で説明する問題）の割合が高いため、用語の正確な定義と英語での表現力が鍵です。

**よくある失点**：「大体わかる」のに英語で正確に説明できない。

**対策**：各概念のキーワード（例：activation energy、osmosis、Newton's second law）を英語で定義できるように、フラッシュカードで定着させる。

### 過去問はモジュール別に解く

Combined Scienceの過去問は、選択したモジュール（Module 1/2/3）に対応したセクションのみを解きます。他モジュールの問題に時間を使わないよう注意しましょう。

**演習計画の例（Module 1: Physics+Chemistry）**：

| 週 | 内容 |
|----|------|
| 1〜2週 | Physics必須項目の概念整理 + MCQ演習 |
| 3〜4週 | Chemistry必須項目の概念整理 + MCQ演習 |
| 5〜6週 | Paper 2（短答・計算）の過去問演習（時間計測） |
| 7〜8週 | 弱点項目の集中補強 + 模試（全ペーパー通し） |

### 時間管理のコツ

Paper 2では、計算問題に時間を使いすぎて記述問題が疎かになるパターンが多発します。

**目安**：
- MCQ（Paper 1）：1問2分以内
- Paper 2 短答・記述：1マーク≒1.5〜2分を目安に配分
- 確信できない問題は先に進み、最後に戻る

## 日本人受験生へのアドバイス

日本の理科教育（物理・化学・生物の独立した授業）に慣れた日本人受験生にとって、Combined Scienceの「2科目を1枠で」という概念は最初は違和感があるかもしれません。しかし、内容的には日本の中学理科〜高校理科の基礎に近いレベルであり、日本の理科教育を受けた受験生には十分取り組める難易度です。

特に**Module 1（Physics+Chemistry）**は、日本の高校物理・化学の基礎知識が大きく活きます。英語での解答形式に慣れることが最大の課題ですが、内容理解はスムーズに進むでしょう。

---

## まとめ

DSE Combined Scienceは、「効率的に理科をカバーしたい」受験生のための戦略的な選択肢です。単科理科ほど深くはありませんが、SBAなし・1科目枠という利点を活かし、他科目との組み合わせで最大の受験戦略を組み立てることができます。

目指す大学・学部の要件を事前に確認し、Combined Scienceが有効かどうかを判断した上で、計画的に取り組みましょう。
