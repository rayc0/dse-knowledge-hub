# 2026年DSE改革全面解析：核心科目變革、公民科取代通識、評分調整

> 摘要 — 2021年起，香港教育局啟動新高中課程改革，以「公民與社會發展科」（公民科）全面取代「通識教育科」，並優化中文、英文、數學等核心科目的課程及評核架構。首批接受新課程的考生將於2026年應考文憑試（DSE）。本文整合教育局公告、課程發展議會指引及JUPAS最新計分規則，為2026屆考生及家長提供完整改革路線圖。

---

## 一、改革背景：為何要改？

### 1.1 通識教育科的爭議

通識教育科自2012年成為DSE必修科目，原意是培養學生多角度思考、批判性分析社會議題的能力。然而，多年來該科備受批評：

- **課題政治化**：部分考題及評分標準被指存在政治傾向，引起社會爭議。
- **備考難度不均**：「開放式答案」令評分缺乏客觀標準，坊間補習機構趁機炒作「答題套路」。
- **課程深度不足**：涵蓋範疇廣而表淺，被批評流於「評論式」思考，缺乏系統知識基礎。
- **與香港實際情況脫節**：部分課題與香港社會現況及國家發展方向不符。

2021年2月，教育局宣布廢除通識教育科，由2021至2022學年起逐步以「公民與社會發展科」取代，並於2024年起正式落實至所有新高中學生。

### 1.2 教育局的改革目標

教育局在課程文件中列出以下目標：

1. **強化國家認同**：加深學生對國家歷史、憲制及香港在國家發展中角色的認識。
2. **提升國際視野**：透過「當代國際議題」模塊，培養學生全球公民意識。
3. **客觀可測量的評核**：以校本評核（SBA）及公開考試取代全開放式論述題，提高評分透明度。
4. **減輕考試壓力**：整體評核模式較通識科更為結構化，降低「意外失手」的風險。

## 二、公民與社會發展科（公民科）詳解

### 2.1 課程架構

公民科由三大主題組成：

| 主題 | 內容範疇 | 課時比例 |
|---|---|---|
| **主題一：香港** | 香港的法治與廉潔、公共財政、社會保障、文化身份認同 | 約30% |
| **主題二：國家** | 中國歷史發展（1900年至今）、「一國兩制」、改革開放、科技成就 | 約40% |
| **主題三：當代世界** | 全球化與區域融合、可持續發展、科技與社會 | 約30% |

### 2.2 評核模式

公民科採用「達標／不達標（A/U）」評級制度，而非昔日通識科的1至5**等級制：

- **公開考試（卷一）**：佔評分50%，考核主題一至三的知識應用，設選擇題及短答題。
- **校本評核（SBA）**：佔評分50%，以「探究報告」形式提交，學生須就指定主題進行資料搜集及分析。

> **重要**：公民科採用「達標（Attained）」或「不達標（Unattained）」評級，不計入JUPAS「最佳五科」加分。大學收生時僅要求「達標」即符合核心科目要求，不影響整體分數計算。

### 2.3 與通識教育科的核心對比

| 比較維度 | 通識教育科（舊） | 公民與社會發展科（新） |
|---|---|---|
| **評級制度** | 1–5**（7個等級） | 達標（A）／不達標（U） |
| **是否計入JUPAS分數** | 計入（最佳五科或六科之一） | **不計入**（僅需達標） |
| **考試形式** | 延伸式論述題為主 | 選擇題＋短答題＋校本評核 |
| **課程重點** | 香港本地及國際時事分析 | 香港、國家、世界三維架構 | 
| **校本評核比重** | 不設SBA | SBA佔50% |
| **備考策略** | 依賴「議題分析框架」 | 以知識記憶及探究技能為主 |
| **補習需求** | 極高（套路式答題盛行） | 相對較低（結構化課程） |

## 三、核心科目的其他優化

除公民科改革外，2026屆DSE的中文、英文及數學三個核心科目亦有相應調整。

### 3.1 中國語文（中文）

教育局對中文科作出以下優化：

- **取消校本評核（聆聽及說話）**：2021年起，中文科聆聽及說話考核改以公開試統一評核，避免校際評分標準不一的問題。
- **閱讀理解題型調整**：加強考核文言文理解能力，白話文與文言文比例調整為約5:5。
- **寫作卷精簡**：減少應用文種類的考核範疇，聚焦於說明文、議論文及描寫文。

### 3.2 英國語文（英文）

- **口語考試（SBA）保留**：英文科校本評核（口語）維持現行制度，佔整體成績20%。
- **聆聽及整合能力（聆聽卷）**：考核重點移向「實用英語應用」，加入更多職場及學術場景題材。
- **寫作題型微調**：加強「論說文」及「建議書」等應用文體的考核比重。

### 3.3 數學科（必修部分 ＋ 延伸部分）

- **必修部分**：課程範疇基本維持不變，但試題難度分佈作出優化，中等難度題目比例增加，令「精英vs.一般」之間的分數差距更具區分度。
- **延伸部分（單元一：微積分與統計 ／ 單元二：代數與微積分）**：評級由5**擴展計算方式不變，但課題深度略有調整，與高等教育銜接更緊密。

## 四、JUPAS計分制度的影響

2026年改革對JUPAS計分的最大影響，在於**公民科不再計入JUPAS分數**。

### 4.1 舊制 vs. 新制計分方式

| 計分項目 | 舊制（通識科年代） | 新制（2026年起） |
|---|---|---|
| **核心科目** | 中文、英文、數學、通識（必須達標） | 中文、英文、數學、公民科（必須達標） |
| **通識／公民科分數** | 計入最佳五科或六科 | **公民科不計入**JUPAS分數 |
| **最佳五科組合** | 四核心科目＋最佳一選修科 | 三核心科目（中英數）＋最佳兩選修科 |
| **最佳六科組合** | 四核心科目＋最佳兩選修科 | 三核心科目（中英數）＋最佳三選修科 |
| **延伸數學加成** | 視院校而定（部分院校加0.5等級） | 維持不變 |

### 4.2 對選科策略的實際影響

**對選修科目的重視度大幅提升**

由於公民科不再計分，考生在JUPAS計分中的「有效科目數量」實際由五科減至四科（中、英、數加最多三選修科）。這意味著：

- 選修科目的選擇變得更為關鍵——一科高分選修科的影響遠超通識科時代。
- 建議考生選擇與目標學科相關的選修科，例如：
  - 報讀理工科：優先選物理、化學、數學延伸（單元二）
  - 報讀商科：優先選經濟、企業、會計
  - 報讀人文社科：優先選歷史、地理、中國歷史、經濟

**公民科達標壓力相對降低**

由於公民科採用A/U評級，且不計入分數，考生只需確保「達標」即可，毋須花費大量精力追求高分。一般而言，只要定期完成校本評核並掌握基本知識架構，達標難度不高。

> **策略提示**：把原本備考通識科的時間，全數轉移至選修科目的深化複習，是2026屆考生的最佳資源配置策略。

### 4.3 各院校要求變化一覽

| 院校 | 最低核心科目要求 | 備注 |
|---|---|---|
| 香港大學（HKU） | 中英文3級或以上、數學2級或以上、公民科達標 | 部分學系另設英文或數學更高要求 |
| 香港中文大學（CUHK） | 中英文3級或以上、數學2級或以上、公民科達標 | 理學院要求數學達3級或以上 |
| 香港科技大學（HKUST） | 中英文3級或以上、數學2級或以上、公民科達標 | 工程學院要求物理或化學必修 |
| 香港理工大學（PolyU） | 中英文2級或以上、數學2級或以上、公民科達標 | 護理學院另有面試要求 |
| 香港城市大學（CityU） | 中英文2級或以上、數學2級或以上、公民科達標 | 法律學院要求英文4級或以上 |
| 香港浸會大學（HKBU） | 中英文2級或以上、數學1級或以上、公民科達標 | — |
| 香港嶺南大學（LU） | 中英文2級或以上、數學1級或以上、公民科達標 | — |
| 香港教育大學（EdUHK） | 中英文3級或以上、數學2級或以上、公民科達標 | 中文教育學系要求中文4級或以上 |

> 以上資料以各院校2025–2026年度官方收生資料為準，實際要求請參閱各大學網站或JUPAS官方公告。

## 五、改革時間線

| 年份 | 重要事件 |
|---|---|
| **2021年2月** | 教育局宣布廢除通識教育科，推行課程改革 |
| **2021–22學年** | 中四生開始修讀「公民與社會發展科」（新課程試行） |
| **2022–23學年** | 全港所有新高中學校正式採用新課程 |
| **2024年** | 首批修讀新課程的中六生備考DSE |
| **2024年4月** | 首批公民科DSE試卷公開應考 |
| **2026年4月** | **2026屆DSE正式考試**（全面實施新制核心科目評核） |
| **2026年7月** | DSE成績公佈，JUPAS新計分制度正式生效 |

## 六、考生備考建議

### 6.1 公民科備考重點

1. **熟悉三大主題知識框架**：系統整理香港、國家、當代世界三個模塊的核心知識點，切忌死記硬背，應理解脈絡。
2. **校本評核（SBA）早規劃**：SBA佔50%，選題與研究方向應盡早與老師確認，避免臨急抱佛腳。
3. **參考教育局樣本試卷**：教育局已於官網發布公民科公開試樣本，考生應作針對性練習。
4. **達標為主，勿過度投入**：由於公民科不計JUPAS分數，合理分配時間至選修科更為划算。

### 6.2 核心科目（中英數）備考建議

- **中文**：加強文言文閱讀理解練習，每週至少完成一篇作文並尋求批改。
- **英文**：重視聆聽及口語練習，善用真實生活情境（英語播客、英文新聞）提升語感。
- **數學**：確保必修部分基礎紮實，延伸部分選修者應定期做歷屆試題，熟悉出題規律。

### 6.3 選修科目策略

1. **選擇與興趣及能力相符的科目**：選修科現佔JUPAS計分的最大比重，分數的高低直接影響入學資格。
2. **與目標學系掛鉤**：提前了解目標院校及學系的入學要求，確保選修科符合「必選科目」條件。
3. **避免「湊合式」選科**：不要因某科目「課程較輕鬆」而選擇，高分難度往往與課程深度成正比。

## 七、常見問題（FAQ）

**Q：公民科不達標會有什麼後果？**
A：公民科屬DSE核心科目，若不達標（Unattained），考生將不符合絕大多數資助大學的最低入學要求，即使其他科目成績優異，亦難以透過JUPAS入讀本地大學。

**Q：通識科的DSE成績（2012–2023年）在新制下是否仍受承認？**
A：是的。現有持有通識科成績的考生（例如重考生），其成績在大學收生時仍受承認，院校視乎個別情況折算。惟2026年後新鮮考生將以公民科成績替代。

**Q：數學延伸部分（M1/M2）改革後是否更難？**
A：課程範疇作出微調，整體難度維持相若，但部分考題的情境設計更貼近高等教育需求。建議有志報讀理工科或商科（精算等）的考生繼續選修。

**Q：大灣區高校及海外大學承認公民科成績嗎？**
A：大部分以DSE成績收生的大學（包括內地985院校）均接納公民科達標作為核心科目要求。海外大學（如英國、澳洲院校）通常要求DSE整體成績符合入學標準，公民科的評級制度轉變對申請影響不大，但建議個別核實各院校要求。

## 重點總結

2026年DSE改革的核心要點如下：

1. **公民與社會發展科全面取代通識教育科**——採用「達標／不達標」評級，不計入JUPAS分數，考生只需達標即可。

2. **JUPAS計分結構調整**——核心科目由「四科」（含通識）變為「三科」（中英數），選修科目在計分中的比重大幅提升，選科策略需全面重新部署。

3. **中英數三科課程優化**——考核模式作出相應調整，整體難度維持，但評核重點有所轉移，須參照最新課程綱要備考。

4. **備考策略須因應新制調整**——減少對公民科的備考投入，將時間集中於核心科目及選修科；校本評核（SBA）是公民科的重要組成部分，切勿輕視。

5. **大學入學要求同步更新**——八大資助大學已更新最低核心科目要求，考生應定期查閱各院校JUPAS收生資料以掌握最新標準。
