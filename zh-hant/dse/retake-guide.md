# DSE重考完全攻略：自修生報名、備考策略與成績合併規則

> **一句摘要**：DSE重考（以自修生身份報考）並非洪水猛獸——了解成績合併規則、掌握科目取捨策略，配合有紀律的備考時間表，重考可以是扭轉命運的第二次機會。本文逐一拆解報名資格、成績計算、備考方法，助你作出最理智的決定。
>
> 本文由 DSE Knowledge Hub 編輯團隊獨立撰寫，資料來源：香港考試及評核局（HKEAA）官方規例、大學聯合招生辦法（JUPAS）收生指引。License: CC BY 4.0。

## 重點速覽（5條）

1. **自修生可保留舊成績**：DSE成績有效期為永久，重考時可選擇沿用上屆成績或以新成績取代，按科目獨立計算。
2. **成績合併有上限**：每份JUPAS申請計算分數時，考評局容許考生從不同年度的成績中揀選最佳組合，但須符合「最多三次應考同一科目」的限制。
3. **校本評核（SBA）處理方式特殊**：自修生一般無法完成在校SBA，部分科目設有「替代評核」安排，需提前了解清楚。
4. **報名時間緊迫**：自修生報名截止日期一般為每年10月至11月，比在校生報名時間早，錯過即等一整年。
5. **備考策略須因人而異**：重考不等於重做一次——應優先處理薄弱科目，而非全科重考，避免浪費時間和金錢。

## 一、誰是「自修生」？資格與定義

### 1.1 自修生的正式定義

根據HKEAA規定，**自修生（Private Candidate）**是指以個人身份（而非透過學校）報考DSE的考生。以下人士均須以自修生身份應考：

- 曾應考DSE並希望重考的畢業生
- 已離校但未曾應考DSE的人士
- 就讀於非參加DSE的學校的學生
- 年滿19歲（即超出在校生年齡限制）的人士

### 1.2 報名資格要求

自修生報名須滿足以下條件：

| 條件 | 說明 |
|---|---|
| 年齡 | 報考當年8月31日或之前年滿17歲 |
| 學歷 | 完成中四（S4）或以上課程，或持有同等學歷 |
| 科目限制 | 部分科目（如視覺藝術、音樂）自修生不可報考 |
| 校本評核科目 | 須事先查明相關科目的替代評核安排 |

> **注意**：即使你上屆曾以在校生身份應考DSE，重考時仍須以「自修生」身份重新報名，須繳交全額考試費用。

## 二、報名程序：截止日期、費用與注意事項

### 2.1 重要日期

自修生報名時間表一般如下（以2026年為參考，每年略有調整）：

| 階段 | 大約時間 |
|---|---|
| 自修生網上報名開放 | 10月初 |
| 報名截止日期 | 11月初至中旬 |
| 繳費截止 | 11月底 |
| 准考證發出 | 翌年2月至3月 |
| 考試期間 | 翌年4月至5月 |
| 成績公布 | 翌年7月中旬 |

錯過截止日期，即使僅遲一天，亦無法報名當年DSE，須等待下一個考試周期。建議在9月底已開始準備所需文件，切勿拖延。

### 2.2 考試費用

自修生須為每份卷別繳交考試費，費用每年由HKEAA公布。以下為費用結構示意（實際金額以HKEAA公告為準）：

- **每個科目**：分別計算，連同不同卷別的費用
- **校本評核（SBA）**：部分科目設替代費用
- **申請費**：另收行政費

一般而言，重考四至六科的總費用介乎數百至逾千港元不等，費用負擔相對輕，但須衡量時間機會成本。

### 2.3 報名所需文件

- 香港身份證（正本及副本）
- 上屆DSE成績通知書（如適用）
- 近照（符合HKEAA規格）
- 有效聯絡電郵及電話
- 繳費方式（信用卡或指定渠道）

## 三、成績合併規則：如何取得最佳組合

這是整個重考決策中最關鍵的部分，亦是最多考生誤解的地方。

### 3.1 核心原則：按科目獨立揀選

DSE成績是**永久有效**的。重考後，你可以按**每個科目**獨立選擇採用哪年的成績，而非整年度一刀切使用。例如：

- 中文：用2025年成績（4級）
- 英文：用2026年重考成績（5級，比上屆進步）
- 數學：用2025年成績（4級，重考不理想）
- 歷史：用2026年重考成績（5*，大幅進步）

這種「拼盤式」取最佳成績的做法，正是重考的最大優勢所在。

### 3.2 應考次數限制

根據HKEAA現行規例，同一科目最多可應考**三次**。第三次應考後，不論成績高低，均以最後一次成績為準（即第三次成績會**強制取代**前兩次）。

> **重要提醒**：第三次應考不再享有「揀選最佳成績」的權利——第三次成績為最終成績，無論是升是降。因此在決定是否第三次應考前，須格外謹慎評估。

### 3.3 JUPAS申請與成績合併

申請JUPAS時，考生須在系統內指明每科採用哪年的成績。大學收生部門會以考生所指定的成績組合作計算，不會自動採用最高分組合，考生須自行確保填報正確。

| 考生行動 | 說明 |
|---|---|
| 首次重考 | 可選用舊成績或新成績（按科目） |
| 第二次重考 | 同上，惟須注意已用去一次「第三次機會」 |
| 第三次應考 | 當科成績強制以最新一次為準，無法選擇 |

### 3.4 公民與社會發展科的特殊情況

公民與社會發展科以「達標（Attained）」或「不達標（Unattained）」作為評級，不設數字分級。由於此科只需「達標」即可滿足JUPAS最低入學要求（「332A」中的「A」），絕大多數考生無需重考此科——除非上屆不達標。

## 四、校本評核（SBA）：自修生的灰色地帶

### 4.1 SBA對自修生的挑戰

SBA（School-based Assessment，校本評核）是部分DSE科目的必要評核部分，由學校老師在課堂內評核，計入最終成績。自修生無法透過學校完成SBA，因此HKEAA為不同科目設有不同的替代安排：

| 科目 | 自修生SBA安排 |
|---|---|
| 中國語文 | 設替代評核（閱讀書目報告），須在指定日期前向HKEAA提交 |
| 英國語文 | 卷四（說話）由HKEAA安排集中考試，自修生可參加 |
| 數學 | 沒有SBA，無需額外安排 |
| 生物、化學、物理 | 設實驗室評核替代安排，自修生須自行安排 |
| 歷史、地理等人文科 | 部分設獨立探究替代安排，詳情須查閱HKEAA年度通告 |

### 4.2 實際行動建議

1. 在決定重考哪些科目**之前**，先逐一查閱HKEAA官網有關自修生SBA替代安排的最新指引
2. 若某科SBA難以安排（例如需要實驗室設施），需提前聯絡HKEAA或補習社了解可行方案
3. 中文科的閱讀書目報告是最常見的自修生挑戰之一，需提前閱讀指定書目並準備報告

## 五、科目選擇策略：重考哪些科目最值得？

### 5.1 重考 vs 不重考的判斷框架

| 考慮因素 | 應重考 | 不必重考 |
|---|---|---|
| 現有成績 | 3或以下，有明顯提升空間 | 已達4或以上，難以大幅進步 |
| 與目標課程的關係 | 是目標課程的必要或加分科目 | 目標課程對該科要求不高 |
| SBA安排 | 可妥善處理SBA替代安排 | SBA替代安排複雜，風險高 |
| 個人優勢 | 過去成績低於應有水準，有充分理由相信可進步 | 成績已反映真實水平 |
| 時間資源 | 可全力備考，毋須兼顧其他重大承諾 | 須同時處理工作、升學等事務 |

### 5.2 高回報科目分析

以下科目通常被視為「重考回報率較高」的科目，原因是評核方式較為規律，備考方向明確：

**數學（必修部分）**
- 沒有SBA，評核全靠卷一、卷二
- 過去卷目有跡可循，多做Past Papers效果顯著
- 從3級升至4級的分數差距通常較小

**英國語文**
- 卷一（閱讀）及卷二（寫作）各佔大比例
- 卷四（說話）自修生可參加集中考試
- 系統性操練寫作技巧可見明顯成效

**歷史 / 地理 / 經濟**
- 人文科目有大量過去試題可參考
- 重點在於掌握答題結構和分析框架
- 自修時間充裕，往往比在校時更有系統

### 5.3 謹慎重考的科目

- **中國語文**：SBA閱讀報告安排繁複；作文及閱讀均有相當難度，貿然重考若準備不足，風險較大
- **視覺藝術 / 音樂**：部分科目自修生不可報考，需先確認資格
- **體育**：自修生一般不可報考

## 六、備考時間規劃：12個月備考路線圖

重考考生通常有約12個月的備考時間（從上屆成績公布至翌年考試）。以下為建議時間規劃：

### 第一階段：決策期（7月至9月）

- 仔細分析上屆成績，逐科評估進步空間
- 研究目標課程的收生要求，確認哪些科目最需提升
- 查閱HKEAA自修生SBA安排，確定可重考科目
- 完成報名準備工作（文件、費用預算）

### 第二階段：基礎建立期（10月至12月）

- 完成報名（10月至11月截止）
- 重新梳理各科核心知識框架，不要急於做Past Papers
- 針對上屆失分點，逐項制定改善方案
- 建立穩定的每日學習習慣（建議每天至少4至6小時）

### 第三階段：強化操練期（1月至3月）

- 系統性完成近5至10年Past Papers
- 分析答題模式，針對失分原因調整策略
- 模擬考試環境，練習時間管理
- 中文科考生應在1月前提交SBA閱讀書目報告申請

### 第四階段：衝刺期（4月至5月）

- 鞏固已掌握的知識，避免臨時大量吸收新內容
- 每天模擬考試，保持狀態
- 注意作息、睡眠及飲食，避免過勞
- 考試期間保持冷靜，按計劃應考

## 七、心理準備：重考不是失敗的標誌

很多重考生面對的最大障礙，不是知識不足，而是心理壓力。以下幾點值得記住：

**重考是普遍現象，不是例外**。每年均有相當數量的考生以自修生身份重考，大學亦普遍接受重考成績，不會因此歧視考生。

**成績合併機制是你的保護網**。即使某科重考成績不理想，你仍可選用舊成績，風險相對可控（唯第三次應考例外）。

**一年的時間比你想像中更充裕**。在校生要上課、做功課、應付SBA，往往無法全心備考；重考生可全力投入，這是一個不可忽視的優勢。

## 重點總結

| 項目 | 關鍵要點 |
|---|---|
| 自修生資格 | 年滿17歲、完成中四或以上課程 |
| 報名截止 | 一般為10月至11月，切勿錯過 |
| 成績有效期 | 永久有效，可按科目揀選最佳年度成績 |
| 應考次數上限 | 同一科目最多三次；第三次成績強制取代前兩次 |
| SBA處理 | 各科安排不同，須提前查閱HKEAA通告 |
| 最值得重考的科目 | 數學、英文、人文科（有明顯進步空間者） |
| 備考時間 | 約12個月，分四階段有序推進 |
| JUPAS申請 | 須自行指定每科採用哪年成績，系統不自動選最優 |

---

重考DSE是一個重要決定，但若能充分掌握規則、制定清晰策略，重考不僅可行，更可能是改變升學命運的最佳機會。建議在作決定前仔細分析自身情況，必要時可向學校升學輔導老師或資深補習導師尋求意見。
