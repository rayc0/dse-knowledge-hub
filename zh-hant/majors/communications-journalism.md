# DSE考生選讀傳播與新聞：港大、中大、城大、浸大入學要求與就業出路

傳播與新聞學科向來吸引對社會事務、媒體創作和公共溝通有熱情的DSE考生，但同時也是「入讀容易，出路迷茫」這個刻板印象最常被提起的學科之一。本文不迴避現實，坦誠分析香港傳播業的就業前景，同時指出新媒體與科技傳播為這個領域帶來的新機遇，幫助考生做出知情的選擇。

---

## 1. 傳播系的4大方向

「傳播系」並非一個單一的職業路徑，而是涵蓋4個截然不同方向的傘形學科。選擇時需要明確自己最感興趣的是哪個方向。

### 方向一：新聞（Journalism）

新聞是傳播學科中最具公共責任感的方向，訓練學生以準確、公正、有深度的方式報道社會事件。

**學習內容**：新聞採訪與寫作、媒體法律與倫理、數據新聞、多媒體報道（文字、影片、播客）

**職業出路**：記者、編輯、新聞主播、文字記者、調查記者、新聞分析員

**現實考量**：香港傳統新聞業（紙媒、電視台）持續收縮，大量媒體裁員或停刊。然而，數字新聞和獨立媒體的崛起提供了新的可能。

### 方向二：公共關係與廣告（PR & Advertising）

公關和廣告是傳播系畢業生最廣泛的就業方向，薪酬也通常優於純新聞方向。

**學習內容**：品牌傳播策略、媒體關係、廣告文案、社交媒體管理、危機傳播

**職業出路**：公關顧問、品牌傳播經理、廣告文案、社交媒體策略師、市場推廣專員

**市場現況**：企業公關和數位營銷持續有需求，尤其是能操作社交媒體廣告、SEO及內容行銷的人才。

### 方向三：廣播（Broadcasting）

廣播方向涵蓋電視、廣播電台及網絡視頻製作，訓練學生掌握音視頻製作技術和內容創作。

**學習內容**：電視製作、電台廣播、紀錄片製作、視頻編輯、直播技術

**職業出路**：電視製作助理、電台主持、網絡視頻創作者、紀錄片導演、串流平台內容製作

**轉型現況**：傳統電視台（TVB、ViuTV）規模縮減，但YouTube、串流平台和企業視頻製作提供了新的就業市場。

### 方向四：數位媒體（Digital Media）

數位媒體是4個方向中增長最快、薪酬潛力最高的方向，結合傳播理論與數據分析、科技應用。

**學習內容**：社交媒體管理、數據分析、搜索引擎優化（SEO）、內容策略、媒體科技

**職業出路**：數字內容策略師、社交媒體經理、SEO專員、UX寫作（UX Writing）、電子商務傳播

**增長動力**：企業數字轉型和電子商務爆炸式增長，帶動對數字傳播人才的強勁需求。

### 英文的核心地位

**傳播系是香港學科中對英文要求最嚴格的非醫療類課程之一。**

原因很直接：傳播系的核心能力是語言表達，而香港的專業媒體環境（特別是商業新聞、金融媒體、國際媒體）以英語為主要工作語言。大部分院校的傳播和新聞課程亦以英語授課。

**英文5**（對應DSE Grade 5**）在頂尖院校（如港大）幾乎是不成文的標準**。即使是中大或浸大，英文5*以上也是競爭入讀的基本門檻。

**中文的重要性**：雖然官方要求中文只需Level 3，但有志於中文媒體（如中文報章、中文廣播）的考生，中文成績5*或以上能展示語文實力，並在選題和採訪方面更具競爭力。

## 4. 「傳播系出路差」的迷思與現實

「傳播系出路差」是香港社會廣為流傳的說法，但這個說法需要仔細拆解。

### 哪部分是真的？

**傳統新聞業確實在萎縮**，這是不爭的事實：

- 《蘋果日報》、《明報》等多份報章相繼停刊或大幅裁員
- 免費電視台（亞視停播）、收費電視（如有線新聞、now News）收縮
- 記者薪酬起點偏低（初級記者月薪約HK$15,000–22,000），且職業安全感下降
- 香港政治環境的變化影響部分媒體的發展方向和人才需求

**廣告業同樣面對數字化衝擊**：傳統廣告（電視廣告、印刷廣告）預算持續萎縮，廣告公司裁員頻繁。

### 哪部分是迷思？

**「傳播系出路差」混淆了「傳統媒體行業」和「傳播能力」本身的市場價值。**

事實上：

- **企業傳播和公關**需求穩定甚至增長：每間大型企業都需要傳播人才，負責媒體關係、危機公關、企業社會責任傳播
- **數字行銷人才極為搶手**：能操作社交媒體、SEO、內容行銷、數據分析的傳播系畢業生，薪酬和就業前景遠優於純新聞記者
- **內容創作和創作者經濟崛起**：YouTube、Instagram、TikTok、播客等平台創造了大量新的內容創作職位
- **跨界能力**：傳播系畢業生擅長寫作、溝通、研究和說故事，這些技能在科技、金融、政府等各行業均有需求

**坦誠結論**：如果你入讀傳播系是為了成為**傳統新聞記者**，前景確實充滿挑戰，需要有心理準備。但如果你願意擁抱數字媒體和多元出路，傳播系的就業前景比外界描述的要好得多。

## 5. 新媒體與高薪新方向

以下是傳播系畢業生中增長最快、薪酬最具吸引力的新興方向：

### 社交媒體管理（Social Media Management）

- **職責**：管理品牌的Facebook、Instagram、YouTube、TikTok等帳號，制定內容策略，執行推廣活動
- **薪酬範圍**：初級HK$16,000–22,000；資深HK$35,000–60,000
- **需求趨勢**：持續增長。幾乎所有企業都需要社交媒體管理人才，特別是能創作本地化中文內容的人才

### 內容策略（Content Strategy）

- **職責**：為企業制定整體內容方向，包括部落格、電子報、白皮書、視頻腳本，以及SEO內容規劃
- **薪酬範圍**：初級HK$18,000–25,000；資深HK$40,000–70,000
- **需求趨勢**：電商和企業網站對高質量SEO內容的需求持續增長

### UX寫作（UX Writing）

- **職責**：為軟件和App撰寫用戶界面文字（按鈕、錯誤提示、引導語），確保用戶在使用數字產品時的清晰體驗
- **薪酬範圍**：初級HK$22,000–30,000；資深HK$45,000–80,000
- **需求趨勢**：科技公司和金融科技公司是主要僱主。UX寫作是傳播系畢業生進入科技行業的最佳切入點之一

### 媒體科技（Media Technology）

- **職責**：結合傳播理論與技術應用，包括數據新聞、媒體分析、AI內容生成管理
- **薪酬範圍**：中高（因結合技術能力，薪酬通常高於純傳播職位）
- **需求趨勢**：AI工具（如ChatGPT、Claude等）改變媒體生產方式，能駕馭AI工具的傳播人才需求急增

### 企業傳播和內部傳播（Corporate Communication）

- **職責**：管理企業對內（員工通訊）和對外（媒體、投資者、公眾）的傳播策略
- **薪酬範圍**：初級HK$20,000–28,000；總監級HK$70,000–130,000+
- **需求趨勢**：上市公司、大型企業和政府機構持續需要企業傳播專才

## 6. 公關與廣告 vs 新聞記者：薪酬/工作性質/穩定性比較

許多傳播系畢業生在畢業後面對「走新聞還是走公關/廣告」的選擇。以下是客觀比較：

| 比較項目 | 新聞記者 | 公關顧問 | 廣告/數字行銷 |
|----------|----------|----------|---------------|
| 初級薪酬 | HK$15,000–22,000 | HK$18,000–26,000 | HK$16,000–25,000 |
| 中級薪酬 | HK$22,000–38,000 | HK$30,000–55,000 | HK$28,000–50,000 |
| 資深薪酬 | HK$35,000–65,000 | HK$55,000–120,000+ | HK$50,000–100,000+ |
| 工作時間 | 不規律，突發新聞需隨時待命 | 相對規律，但有危機時高壓 | 相對規律，project period較忙 |
| 工作穩定性 | 較低（行業萎縮） | 中至高（企業需求穩定） | 中至高（數字行銷需求增長） |
| 社會影響力 | 高（媒體監督功能） | 中（為客戶服務） | 中（品牌影響力） |
| 職業滿足感 | 高（有社會責任感） | 因人而異 | 因人而異 |
| 晉升路徑 | 記者→高級記者→編輯→主編 | 助理→顧問→經理→總監→合夥人 | 執行→策略師→經理→總監 |

**薪酬差距的現實**：公關和廣告行業的薪酬天花板遠高於新聞業，特別是在公關公司晉升至顧問或合夥人層級後。資深新聞工作者的薪酬通常難以匹敵同齡的資深公關從業員。

**選擇建議**：如果你對新聞工作有真誠的使命感，值得堅持。但如果主要考慮薪酬和職業穩定性，公關或數字行銷是更現實的選擇。

## 7. 實習和工作經驗的重要性

在傳播行業，有一個廣為人知的現實：**「Portfolio比學歷重要」**。

### 為什麼實習如此關鍵？

- **傳播行業靠人脈和作品**：僱主看的是你能做什麼，而不只是你持有什麼學位
- **媒體運作靠實際操作**：課堂無法完全模擬真實媒體環境，實習提供無法替代的實戰經驗
- **畢業即就業的捷徑**：部分實習生在實習期間表現出色，直接獲留用offer

### 如何建立傳播Portfolio？

1. **在校園媒體實踐**：校報、學生電台、YouTube頻道、校園Instagram，從大學一年級就開始建立作品
2. **主動申請暑期實習**：目標放在媒體機構（SCMP、明報、now新聞）、公關公司（Ogilvy、Edelman、本地PR公司）或企業傳播部門
3. **發展個人品牌**：建立個人部落格、LinkedIn、或相關社交媒體帳號，展示你的寫作和傳播能力
4. **自學數字工具**：Google Analytics、Hootsuite、Canva、Adobe Premiere Pro等工具，免費課程和YouTube教程已足夠入門

### 建議實習時間線

| 時間 | 行動 |
|------|------|
| 大學一年級 | 加入校內媒體、學生會傳訊部，開始建立基礎作品 |
| 大學一年級暑假 | 申請初級媒體實習（記者助理、社交媒體助理） |
| 大學二年級 | 繼續累積作品，嘗試自行Pitch文章給媒體發表 |
| 大學二年級暑假 | 申請較具規模的媒體機構或公關公司實習 |
| 大學三年級 | 確立方向（新聞/公關/數字媒體），深化相關作品集 |
| 大學三年級暑假 | 目標「準職位」實習，與業界建立實質人脈 |

**英國學位認可度**：英國傳播和新聞課程在香港業界廣受認可，特別是英文媒體僱主。但需注意英國課程費用（學費加生活費每年可達港幣40–60萬），性價比需仔細評估。

### 就香港DSE申請海外傳播課程

- **DSE英文要求**：英國大學一般要求英文5*–5**（相當於IELTS 6.5–7.0）
- **寫作樣本（Writing Sample）**：部分美國和英國院校要求提交新聞稿、文章或評論作為申請材料，應提前準備
- **IELTS/TOEFL**：部分美國院校不接受DSE英文成績，需另外考IELTS或TOEFL

## 總結

傳播與新聞學科是一個充滿活力的領域，但需要考生帶著清醒的眼光入讀。以下是幾個關鍵總結：

1. **英文能力是核心**：無論哪個方向，傳播系對英文能力的要求極高，英文5**幾乎是頂尖院校的不成文標準
2. **方向選擇影響出路**：「傳播系」是傘形學科——數字媒體和UX寫作的前景遠優於傳統新聞，入讀時應明確自己的方向
3. **Portfolio比Transcript重要**：在傳播行業，作品集和實習經驗往往比學業成績更能決定就業結果
4. **傳統媒體難，數字媒體旺**：傳統新聞和廣播行業持續收縮，但數字傳播、社交媒體、企業傳播的人才需求仍然強勁
5. **浸大和中大在本地傳媒最具優勢**：如目標是香港傳媒，浸大傳理和中大新傳是業界最認可的本地培訓基地
6. **公關和廣告薪酬天花板更高**：如考量薪酬和職業穩定性，公關顧問或數字行銷是比傳統新聞記者更實際的選擇
7. **AI改變行業版圖**：能掌握AI內容工具、懂數據分析的傳播系畢業生，在新媒體時代最具競爭力

對有語言天賦、對社會充滿好奇、且願意擁抱媒體轉型的考生而言，傳播系仍然是一個值得認真考慮的學科選擇。關鍵是從大學第一年就主動建立作品集，不要等到畢業才開始。
