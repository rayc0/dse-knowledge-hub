# DSE生畢業後移民加拿大升學：升讀大學及移民路徑全解析

加拿大是DSE畢業生升讀海外大學的**第二大熱門目的地**（僅次於英國），同時也是近年最多香港人選擇移民的國家。加拿大的優勢在於：大學費用低於英美、移民友好、英語環境、多元文化，以及近年針對香港人推出的特別移民計劃。

本文從DSE成績認可政策、主要大學申請策略、城市生活比較、到畢業後工作及移民路徑，全面解析香港學生赴加升學的選擇。

## 一、加拿大大學認可DSE的政策概覽

加拿大大多數大學**認可DSE成績**作為入學資格，但具體要求因校因科而異。

## 二、熱門學系的DSE要求

### 電腦科學（UBC/Waterloo）

加拿大電腦科學最強的是**滑鐵盧大學電腦科學**——全球排名極高，畢業生直接進入矽谷科技公司（Google、Amazon、Meta）的比例冠絕北美。但競爭極為激烈。

- **滑鐵盧CS**：要求英語5+、數學5+、其他強科2-3科達5
- **UBC CS**：要求4科達5，競爭同樣激烈
- **多倫多大學CS**：強調高中數學和英語成績，DSE 4科達5+

### 商科（Schulich / Rotman / Sauder）

- **多倫多大學Rotman**（全加最強商學院）：4科達5，要求英語4或以上
- **UBC Sauder**：4科達5，附上個人陳述
- **York大學Schulich**：4科達4，對DSE生相對友善

### 護士/醫療相關

- 加拿大醫學院幾乎不接受直接本科申請（需先讀學士再申請），但護理學（BScN）可以直接申請
- 一般要求生物科5級或以上

## 三、三大城市比較

| 比較項目 | 多倫多 | 溫哥華 | 蒙特利爾 |
|---|---|---|---|
| 氣候 | 四季分明，冬天冷 | 溫和多雨 | 很冷，多雪 |
| 香港人社群 | 極大（30萬+） | 極大（30萬+） | 較小 |
| 生活費（月） | CAD$1,800-2,500 | CAD$2,000-2,800 | CAD$1,200-1,800 |
| 就業機會 | 最多（金融、科技） | 科技、貿易 | 較少 |
| 語言環境 | 英語為主 | 英語為主 | 法語為主 |
| 主要大學 | 多倫多大學、York | UBC | 麥基爾 |

**對DSE生的建議**：
- 看重就業 → **多倫多**（金融科技中心）
- 看重生活質素和氣候 → **溫哥華**
- 預算有限 → **蒙特利爾**（費用最低，但需學法語）

## 五、畢業後工作及移民路徑

### 畢業後工作許可（PGWP）

加拿大提供**畢業後工作許可（Post-Graduate Work Permit）**：
- 課程長度1-2年 → PGWP最長2年
- 課程長度2年或以上 → PGWP最長3年

### Express Entry移民

工作1-3年後可透過**Express Entry**申請永久居留（PR）：
- Federal Skilled Worker（需要IELTS、學歷、工作經驗評分）
- 全球同等學歷認可，港人評分普遍較高

### 香港人特別移民計劃（BVIAHK / Stream C）

**針對香港人的特別安排**：
- **Stream A**：加拿大畢業生或具加拿大工作經驗
- **Stream B**：其他技術移民類別
- **Stream C**：香港人特別通道（無需工作經驗要求）

這些計劃是近年加拿大針對香港移民潮推出的政策，為香港學生提供了比一般國際學生更便捷的移民路徑。

## 六、總結：加拿大升學移民路徑

| 步驟 | 時間線 | 內容 |
|---|---|---|
| 1. 準備DSE成績 | 中六（現在） | 目標4科達5或以上 |
| 2. 申請大學 | 中六12月-翌年2月 | 直接向各校遞交申請 |
| 3. 入讀大學 | 9月 | 一般本科課程4年 |
| 4. 申請PGWP | 畢業後立即 | 最多3年工作許可 |
| 5. 累積工作經驗 | 工作1-3年 | 建立Express Entry分數 |
| 6. 申請PR | 工作1-3年後 | Express Entry或香港特別計劃 |
