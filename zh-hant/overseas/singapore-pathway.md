# DSE考生升讀新加坡大學

新加坡擁有全球頂尖大學NUS（國立大學）和NTU（南洋理工大學），兩者均在QS世界排名前15位。對DSE考生而言，新加坡是僅次於香港本地、英澳以外最受歡迎的升學目的地之一，主要因新加坡承認DSE成績、文化相近、使用英語教學。

---

## 1. 新加坡主要大學一覽

| 大學 | 簡稱 | QS世界排名（2025）| 強項科系 | 是否承認DSE |
|---|---|---|---|---|
| 新加坡國立大學 | NUS | 前8 | 商科、法律、理工、醫學 | ✅ 承認 |
| 南洋理工大學 | NTU | 前15 | 工程、傳播、商科 | ✅ 承認 |
| 新加坡管理大學 | SMU | 約500 | 商科、法律、資訊系統 | ✅ 承認 |
| 新加坡科技設計大學 | SUTD | 未排名 | 工程設計 | 視課程而定 |
| SIT（新加坡理工大學） | SIT | 應用型 | 護理、工程、酒店管理 | ✅ 承認 |

## 2. DSE入學基本要求

新加坡大學一般以DSE成績換算為「A-Level equivalent」來評估申請人。以下是一般指引（各課程要求以官網為準）：

| 要求項目 | NUS/NTU 一般要求 | SMU 一般要求 |
|---|---|---|
| 英文（English Language） | 5** 或 5* | 5* 或 5** |
| 最佳科目組合 | 4科（含英文）最少達3科5** | 4科合格（含英文5*） |
| 數學 | 科學/商科需5*或以上 | 商科需數學5* |
| 面試 | 法律、醫學、傳播等課程必須 | 商科可能需要 |
| 個人陳述（PS） | 必須，1,000字 | 必須 |

> **重要提示**: 新加坡大學的競爭極為激烈。DSE申請者通常需要比A-Level考生更高的分數才能獲得同等考慮，因為DSE申請池較小而篩選標準較嚴格。

## 3. 熱門科系DSE最低錄取分參考

以下為往年錄取的非官方參考數據（以DSE最佳5科成績計算，英文必算在內）：

| 科系 | 大學 | 參考最低DSE分 | 備註 |
|---|---|---|---|
| 商學院（BBA） | NUS | 英文5** + 3科5* | 競爭激烈，面試重要 |
| 電腦科學 | NUS | 英文5* + 數學5** + 2科5* | 數學加強必看 |
| 工程學院 | NTU | 英文5* + 3科5* | 物理需5*或以上 |
| 傳播學院（ADM） | NTU | 英文5** + 作品集 | 藝術類需portfolio |
| 法律 | NUS/SMU | 英文5** + 3科5** | 極難；LNAT考試額外要求 |
| 護理學 | NUS/SIT | 英文5* + 化學/生物5* | 面試必須 |

## 4. DSE vs A-Level vs IB 換算參考

新加坡大學通常以下列方式評估：

| DSE | A-Level（香港/英國） | IB |
|---|---|---|
| 5** | A* | 7 |
| 5* | A | 6–7 |
| 5 | B | 5–6 |
| 4 | C | 4–5 |
| 3 | D | 3–4 |

> **注意**: 這是非官方換算參考。各大學有自己的DSE評估系統，建議直接聯繫各大學招生辦查詢。

## 5. 新加坡留學費用

| 費用類別 | 每年估算（坡元 SGD）| 港元約合 |
|---|---|---|
| 學費（有政府補貼） | SGD 8,000–15,000 | HK$46,000–86,000 |
| 學費（無補貼/海外生） | SGD 22,000–45,000 | HK$126,000–258,000 |
| 住宿（宿舍） | SGD 5,000–8,000 | HK$29,000–46,000 |
| 餐飲 | SGD 4,000–6,000 | HK$23,000–35,000 |
| 交通 | SGD 1,200–2,000 | HK$7,000–11,500 |
| **總計（低估）** | **約SGD 18,000–25,000** | **約HK$103,000–143,000** |

> **政府補貼（Tuition Grant）**: 外國學生可申請新加坡政府學費補貼，條件是畢業後在新加坡工作3年。這大幅降低學費，但需簽署服務協議。

## 6. 學生簽證（Student's Pass）

| 項目 | 詳情 |
|---|---|
| 申請機構 | 新加坡移民與關卡局（ICA）透過SOLAR系統 |
| 申請時機 | 大學錄取通知後 |
| 費用 | SGD 30申請費 + SGD 60准證費 |
| 條件 | 全日制正規課程學生，有效大學錄取證明 |
| 工作許可 | 在學期間每週可合法工作最多16小時（兼職）|
| 延續 | 每學年需更新 |

## 7. 獎學金機會

| 獎學金 | 機構 | 金額 | 競爭程度 |
|---|---|---|---|
| NUS Global Merit Scholarship | NUS | 全額學費+生活津貼 | 極難 |
| NTU University Scholarship | NTU | 全額學費 | 極難 |
| ASEAN Undergraduate Scholarship | NUS/NTU | 學費50-100% | 非ASEAN不適用 |
| SINGA（研究生） | A*STAR | 研究生津貼 | 研究型碩博 |
| 個別院系獎學金 | 各學院 | 部分學費 | 視學院而定 |

> 港人DSE申請者可申請NUS和NTU的一般「Merit Scholarship」，但競爭與全球申請者同場。

## 8. 畢業後留新就業

| 項目 | 詳情 |
|---|---|
| 就業准證（EP） | 月薪需達SGD 5,000（畢業生最低，2025年） |
| 留用率 | NUS/NTU畢業生新加坡就業率約85-90% |
| 熱門行業 | 金融科技、科技、供應鏈、製造業研發 |
| 永久居民（PR）申請 | 就業2年後可申請，DSE+NUS/NTU背景有優勢 |
| Tuition Grant服務協議 | 接受補貼者需完成3年服務，否則退回差額 |

---

## 9. 申請注意事項

| 事項 | 說明 |
|---|---|
| 申請截止日期 | 一般每年2月底至3月底（NUS/NTU網站確認最新日期）|
| 面試語言 | 英語為主，部分面試官會用普通話或廣東話輔助 |
| 個人陳述（PS） | 強調為何選擇新加坡、未來職業規劃、DSE以外的活動經歷 |
| IELTS/TOEFL | DSE英文5*或以上通常豁免英語考試 |
| 文件認證 | DSE成績表需正式認證（考評局蓋章） |

## 總結

新加坡的NUS和NTU是DSE考生進入全球前20大學的最具競爭力途徑之一，學費比英美便宜、文化上更接近香港。英文5**加上3-4科5*是競爭熱門課程的基本門檻。接受政府補貼可大幅降低學費，但需要承擔畢業後3年留新服務協議。對於希望在東南亞金融科技或科技行業發展的香港學生而言，NUS/NTU學歷配合新加坡就業市場是值得認真考慮的選項。
