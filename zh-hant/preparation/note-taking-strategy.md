# DSE 高效筆記策略

很多同學花大量時間抄筆記，卻發現考試前不知道從何溫習。問題往往不是「寫多少」，而是「怎麼寫」。本指南介紹三種經研究驗證的筆記法，以及針對不同 DSE 科目的最佳應用方式。

---

## 1. 三種主要筆記法比較

| 筆記法 | 適合科目 | 優點 | 缺點 |
|---|---|---|---|
| **康乃爾筆記法** | 歷史、通識、BAFS | 左欄問題/右欄內容，強迫主動回憶 | 初學需要適應 |
| **思維導圖（Mind Map）** | 生物、地理、中史 | 視覺化概念關係，記憶點立體 | 不適合線性知識 |
| **線性筆記（傳統筆記）** | 數學、物理、化學 | 快速，適合公式推導 | 被動抄寫效果有限 |
| **流程圖/表格筆記** | 化學反應、歷史年表、英文文法 | 清晰對比，考試前快速翻閱 | 需要事先設計格式 |

---

## 2. 康乃爾筆記法詳解

最適合需要**理解與分析**的科目（通識、歷史、地理）：

**頁面格式**：
```
| 問題欄（左 1/3） | 筆記欄（右 2/3）          |
|                  | 課堂/書本內容抄寫         |
|                  |                           |
|__________________|___________________________|
| 總結欄（底部 1/5）                            |
| 用自己的話總結本頁核心概念                    |
```

**使用流程**：
1. **課堂/自習時**：在右欄記錄重點
2. **當天晚上**：在左欄寫出對應問題（不看右欄答案）
3. **隔天**：遮住右欄，用問題欄自測

> **研究顯示**：主動回憶（active recall）比單純重讀筆記有效 3–4 倍。

---

## 3. 各科目最佳筆記格式

| 科目 | 推薦格式 | 具體技巧 |
|---|---|---|
| 中文 | 表格（文體/主題/修辭手法/答題框架） | 按題型分類，考前快速翻閱 |
| 英文 | 線性筆記 + 例句收集 | 每學一個表達方式，附2–3個例句 |
| 數學 | 公式卡 + 錯題本 | 每個公式配一道例題；錯題必須重做 |
| 通識 | 康乃爾筆記 + 時事簿 | 左欄問題模擬考試提問方式 |
| 歷史/中史 | 時間線 + 思維導圖 | 先建立時間線，再用導圖理解因果 |
| 生物 | 思維導圖 + 圖解 | 系統/器官/功能三層次導圖 |
| 化學 | 反應方程式分類表 + 流程圖 | 按反應類型（氧化還原/酸鹼/有機）分冊 |
| 物理 | 公式推導線性筆記 + 概念圖 | 公式旁邊寫清楚各符號的物理意義 |

## 4. 數碼筆記 vs 手寫筆記

| 比較項目 | 數碼筆記（GoodNotes/Notion/OneNote） | 手寫筆記 |
|---|---|---|
| 搜尋功能 | ◎ 即時搜尋關鍵詞 | ✕ 需要翻閱 |
| 記憶效果 | △ 研究顯示稍遜 | ◎ 手寫有助加深記憶 |
| 整理方便 | ◎ 剪貼、重排容易 | △ 需要重抄 |
| 數學/圖解 | △ 觸控筆有限 | ◎ 任意繪圖 |
| 備份安全 | ◎ 雲端自動備份 | △ 遺失則永久消失 |

**建議**: 文字為主的科目（通識、歷史）可用數碼；計算/圖解科目（數學、物理）用手寫。

---

## 5. 考前筆記整理策略

考試前 4–6 週，從厚重的課堂筆記製作「精華筆記」：

| 步驟 | 做什麼 | 目標 |
|---|---|---|
| 第1步 | 瀏覽所有舊筆記，標記高頻考點 | 識別重要程度 |
| 第2步 | 每個主題濃縮到 1 頁紙 | 強迫篩選核心 |
| 第3步 | 加入歷屆試題的答題框架 | 連結考試應用 |
| 第4步 | 製作記憶口訣或縮寫 | 考場快速回憶 |
| 第5步 | 每天複習精華筆記（間距重複） | 鞏固長期記憶 |

---

## 6. 常見筆記錯誤與修正

| 錯誤行為 | 問題所在 | 正確做法 |
|---|---|---|
| 把課本全文抄下來 | 被動抄寫，記憶效果差 | 用自己的話提煉重點 |
| 筆記字體太工整，花大量時間 | 浪費時間在美化上 | 字跡清晰即可，不需要精美 |
| 從不複習舊筆記 | 短期記憶沒有鞏固 | 每週固定複習上週筆記 |
| 每科格式不一致 | 考前找不到重點 | 每科制定固定模板，沿用全年 |
| 筆記太多，找不到重點 | 資訊過載 | 配合「精華筆記」計劃定期濃縮 |

---

## 總結

DSE 備考的筆記策略核心是**主動整理而非被動抄寫**。康乃爾筆記法適合需要分析的文科，思維導圖適合概念多而複雜的理科，線性筆記適合數學/化學的公式推導。不論使用哪種方式，考前 4–6 週製作每主題一頁的「精華筆記」，配合間距重複法複習，才是最高效的備考策略。
