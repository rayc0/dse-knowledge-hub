# DSE數學科（必修）應試策略：M1/M2選擇、答題格式、常見失分原因

DSE數學科是升讀香港本地和海外大學的核心門檻科目。必修科（Compulsory Part）幾乎所有學生都需要應考，而延伸部分M1/M2則是攻讀理工科或商科考生的重要加分項。本文從結構認識到具體策略，涵蓋卷一答題節奏、卷二MC技巧、「Show all working」格式要求，以及5**/5*考生的備考方法。

---

## 1. 數學科完整結構

### 必修科（Compulsory Part）

| 卷別 | 形式 | 時間 | 分數 | 佔必修科總分 |
|------|------|------|------|-------------|
| 卷一 | 長答題（Section A1、A2、B） | 2.25小時 | 105分 | 65% |
| 卷二 | 多項選擇題（MC，45題） | 1.25小時 | 45分 | 35% (換算) |

**卷一三個部分細分**：
- Section A1（8題，22分）：基礎題，應在前30分鐘完成
- Section A2（6題，38分）：中級題，每題6-7分
- Section B（3題，45分）：進階題，每題14-15分，共答3題但可選4題中的3題

### 延伸部分（Extended Part）

| 模組 | 主要內容 | 時間 | 分數 |
|------|----------|------|------|
| M1（微積分與統計） | 統計學、微積分基礎、概率 | 2.5小時 | 100分 |
| M2（代數與微積分） | 數列、矩陣、微積分進階 | 2.5小時 | 100分 |

M1和M2是獨立科目，分開計算成績，可單獨選擇其中一個，亦可不選。

## 2. M1 vs M2選擇指引

選擇M1還是M2，應根據個人志向、學科配搭和數學強項綜合考慮。

### M1 vs M2比較表

| 比較項目 | M1（微積分與統計） | M2（代數與微積分） |
|----------|-------------------|-------------------|
| **難度** | 相對較易入門 | 較難，概念較抽象 |
| **主要內容** | 統計、概率、基礎微積分 | 矩陣、數列極限、進階微積分 |
| **適合科目配搭** | 商科、社會科學、生物、地理 | 物理、工程、純數學、計算機 |
| **就業/升學影響** | JUPAS入讀商學院、理學院部分課程 | 工程、物理、數學相關課程的要求或優勢 |
| **計算量** | 中等 | 高，需要大量代數運算 |
| **備考時間需求** | 相對較少（約4-6小時/週） | 較多（約6-8小時/週） |

### 選擇建議

- **商科、商學院目標**：選M1，統計學知識在大學商科課程中直接適用
- **工程、物理、電腦科學目標**：選M2，大學課程會直接延伸M2的代數和微積分內容
- **不確定方向**：先評估必修數學的強項——若對幾何和代數更強，選M2；若對統計和概率更有感，選M1
- **時間有限者**：M1的備考效率一般高於M2，若時間緊張可優先考慮M1

**重要提醒**：延伸部分的成績以「5**、5*、5、4、3、2、1、U」呈現，部分大學課程要求M1/M2達到特定等級（如理工大學工程學院要求M2達3級或以上）。報考前應查閱目標學科的具體要求。

## 3. 卷一答題節奏管理

卷一（2小時15分鐘，105分）的時間管理是高分的關鍵。

### 推薦時間分配

| 階段 | 時間 | 目標 |
|------|------|------|
| 前25-30分鐘 | Section A1（8題） | 每題3-4分鐘，全部完成，不留空白 |
| 接著40-45分鐘 | Section A2（6題） | 每題6-7分鐘，力求全對 |
| 剩餘55-60分鐘 | Section B（選3題） | 每題18-20分鐘，選自己最熟悉的3題 |
| 最後10分鐘 | 檢查 | 核對單位、負號、答案是否合理 |

### 節奏管理原則

**前20分鐘原則**：Section A1必須在前30分鐘完成。若某題卡超過5分鐘，跳過並做記號，稍後返回。

**Section B選題策略**：
- 進場前已確定自己最強的兩個範疇（統計、代數、微積分等）
- Section B通常包含概率/統計、代數/方程、幾何/三角，選擇自己最熟悉的3題
- 不要因為某題開頭容易就選，必須先瀏覽整題再決定

**遇到困難題的處理**：
1. 做完能做的部分，取部分分（follow-through marks）
2. 寫出相關公式即使未能完成計算，評卷員仍可給格式分
3. 不要為了一題放棄其他題

## 4. 卷二MC：消去法、估算法與倒推法

卷二45條MC題，1小時15分鐘，平均每題不到2分鐘。速度和準確並重。

### 三種核心MC策略

**策略一：消去法（Elimination）**
適用於計算繁瑣但能判斷答案範圍的題目。
- 先排除明顯不合理的選項（負數、超大數值等）
- 縮小選項後再進行精確計算

**策略二：估算法（Estimation）**
適用於代入計算費時的題目。
- 用整數或簡化數值代替複雜數值進行快速估算
- 適合多項式、指數等有明顯大小關係的題型

**策略三：倒推法（Back-solving）**
適用於方程或不等式題，直接把選項代入驗證。
- 從B或C選項開始代入（B/C通常是中間值，若錯可判斷方向）
- 驗算速度往往快於正面解題

### MC答題節奏

- 第1-20題（基礎題）：每題1-1.5分鐘
- 第21-35題（中級題）：每題1.5-2分鐘
- 第36-45題（難題）：每題2-3分鐘
- 最後5-10分鐘：檢查標記過的不確定題

**重要**：MC沒有扣分制（2012年後），所有題目必須填寫答案，絕不留空。

## 5. "Show All Working"的格式要求

DSE數學卷一評分時，即使最終答案錯誤，評卷員仍可按工作過程給予部分分數（Method Marks）。因此，「show all working」是最大化得分的關鍵策略。

### 格式要求細則

**必須展示的步驟**：
- 所用公式（即使是已知公式）
- 每一步代入和化簡過程
- 中間步驟的數值（不可跳步）
- 最終答案，並寫明單位（如適用）

**錯誤示例 vs 正確示例**

錯誤（跳步，即使答案正確也有扣分風險）：
```
x = (-b ± √(b²-4ac)) / 2a = 3 或 -1
```

正確（完整展示）：
```
x = (-b ± √(b²-4ac)) / 2a
  = (-(4) ± √((4)²-4(1)(-3))) / 2(1)
  = (-4 ± √(16+12)) / 2
  = (-4 ± √28) / 2
  = (-4 ± 2√7) / 2
  = -2 ± √7
∴ x = -2 + √7 ≈ 0.646 或 x = -2 - √7 ≈ -4.646
```

### 關於設「x為...」

解應用題時，必須在最開頭寫出「設x為...（單位）」，否則即使計算全對，也可能被扣去首步的設定分。

## 6. 常見失分原因彙整表

### 代數範疇

| 失分原因 | 具體例子 | 如何避免 |
|----------|----------|----------|
| **負號錯誤** | -(a-b) 展開為 -a-b（應為-a+b） | 展開前用括號標示，逐項處理 |
| **二次方漏解** | x²=9，只寫x=3（漏掉x=-3） | 開方必寫±，記「二次方程兩個根」 |
| **指數運算錯誤** | (a²)³=a⁵（應為a⁶） | 複習指數法則，考前快速溫習 |
| **分式化簡錯誤** | (a+b)/a = b（應保留分式） | 分式只可消去公因式，不可消去加項 |

### 幾何範疇

| 失分原因 | 具體例子 | 如何避免 |
|----------|----------|----------|
| **引理敘述不完整** | 只說「對頂角相等」，沒說明哪個角和哪個角 | 引理引用必包含「∵...∴...」格式 |
| **單位忘記** | 面積答案沒有寫cm²或m² | 最後步驟強制核對單位 |
| **圓心角混淆** | 混淆圓周角和圓心角的關係 | 記清楚「圓心角=2×圓周角」 |

### 概率與統計範疇

| 失分原因 | 具體例子 | 如何避免 |
|----------|----------|----------|
| **有放回/無放回混淆** | 題目說無放回，卻用有放回公式計算 | 審題時用螢光筆標出「有/無放回」 |
| **條件概率誤用** | P(A|B)和P(A∩B)混淆 | 複習貝葉斯公式和文氏圖 |
| **樣本空間遺漏** | 擲兩粒骰子，遺漏部分組合 | 系統列舉，用樹形圖或表格 |

### 三角函數範疇

| 失分原因 | 具體例子 | 如何避免 |
|----------|----------|----------|
| **角度範圍遺漏** | 0°≤θ≤360°，只找到兩個解中的一個 | 畫CAST圖，系統確認每個象限 |
| **弧度/角度混淆** | 計算機設在弧度模式但題目是角度 | 每次使用計算機前確認角度模式 |

## 7. CASIO fx-3650P計算機使用技巧

DSE指定計算機型號（fx-3650P或fx-50FH II），熟練使用可節省大量計算時間。

### 高效功能

**CALC功能（計算存儲公式）**：
- 輸入公式並儲存，考試中可重複代入不同數值
- 特別適合二次公式、統計計算
- 操作：輸入算式，按CALC，輸入變量值，按=

**STAT統計功能**：
- 輸入數據後可直接計算均值（x̄）、標準差（σ）、回歸直線
- 卷一Section A2的統計題和M1的統計題必用
- 操作：MODE→2（STAT）→選類型→輸入數據

**Fraction功能**：
- 可直接以分數運算，避免四捨五入誤差
- 在精確答案題目中必用（避免用小數導致誤差積累）

### 考試前計算機準備

- 考試前確認電池（帶備用電池）
- 清除存儲器（SHIFT→CLR→All→=）
- 確認角度模式（Degree for 三角，Radian for 弧度題）
- 熟悉CALC和STAT的操作流程（考前一週每日練習5分鐘）

## 8. 5**/5*考生的備考密度建議

### 時間表建議（最後3個月）

**考前3個月（12週）**

| 時期 | 週數 | 重點 |
|------|------|------|
| 鞏固期 | 第1-4週 | 每週2份卷二MC（歷屆真題），複習所有章節知識點 |
| 強化期 | 第5-8週 | 每週1份完整卷一（計時），針對弱項補強 |
| 衝刺期 | 第9-11週 | 每週2份完整卷一+1份卷二，分析錯題類型 |
| 收尾期 | 第12週 | 只做輕量複習（每日30分鐘），溫習公式表和常錯點 |

### 每日練習密度

**衝5**考生（目標5**）**：
- 平日：每日1.5-2小時數學（卷一練習+錯題複習）
- 週末：完整卷一或卷二模擬（計時）
- 每週整理「錯題本」，重做同類型題目直到完全掌握

**衝5*考生（目標5*）**：
- 平日：每日1小時數學（以卷一A1/A2為主）
- 週末：Section B練習（每次選2-3題）
- 以歷屆真題（2012-2024）為核心練習材料

### 錯題複習法

1. 做錯後不要立即看解答，先給自己5分鐘重試
2. 看解答後用自己的話寫出解題思路
3. 一週後重做同一題，確認已掌握
4. 整理「我常錯的題型」清單，考前重點溫習

## 總結

DSE數學科的高分策略可以總結為四個核心原則：

1. **格式先行**：Show all working，每步有據，最大化部分分數
2. **節奏管理**：卷一按時間分配作答，不死守難題
3. **MC善用策略**：消去法+估算法+倒推法靈活運用，所有題目必填答案
4. **計算機熟練**：CALC和STAT功能熟練操作，節省計算時間

M1/M2的選擇則取決於升學方向——商科選M1，工程/物理選M2。最重要的是在確定選擇後持之以恆地練習，延伸部分的積累效果遠高於臨時突擊。
