# DSE考試日策略：考前一週到放榜的完整心理及實戰指南

DSE考試是香港學生面對的最重大考試挑戰，多年溫習在這幾週內決出成果。然而很多考生在準備上投入大量時間，卻忽略了**應試策略本身**——考試日的心理狀態、時間分配、答題順序和突發事件應對，同樣決定最終成績。

本文從考前一週開始，全面覆蓋考試日的實戰策略。

---

## 一、考前一週：減少投入，鞏固記憶

### 最後一週的正確做法

很多人在最後一週「臨急抱佛腳」，試圖記憶新內容。這是反效果的：

| 最後一週應做 | 最後一週不應做 |
|---|---|
| 溫習已熟悉的重點筆記 | 開始全新課題 |
| 做1-2份過往試卷（計時） | 瘋狂做大量新卷 |
| 充足睡眠（7-8小時） | 開夜車溫習 |
| 準備文具和准考證 | 考前晚上大食 |
| 輕度運動放鬆 | 與人討論答案（增加焦慮） |

### 睡眠的重要性

腦神經科學研究表明，睡眠是記憶鞏固的關鍵過程。考試前一晚睡足7-8小時，比多溫習2小時更有效提升成績。**考前晚上零晨後才睡的考生，平均成績比正常睡眠的考生低半個等級**（HKEAA非正式統計）。

## 二、考試當日：出發前的準備

### 必備物品清單

| 物品 | 備注 |
|---|---|
| HK身份證（正本） | 主要身份證明文件 |
| 准考証 | 如遺失需提前聯絡HKEAA |
| 多支HB鉛筆 | 選擇題必須用鉛筆 |
| 黑色原子筆（3支） | 長題目及問答題 |
| 尺、圓規、計算機 | 視乎科目 |
| 透明水瓶 | 水，非含糖飲料 |
| 少量零食（健康） | 果仁、能量棒（長考適用） |

**絕對不可帶入考場**：電話（必須關機存入背包）、智能手錶、無線耳機。任何電子通訊設備被發現均屬違規，後果嚴重。

### 出發時間

預計**提早30-40分鐘**抵達考場。原因：
- 確認座位號碼及場地
- 熟悉環境，減少陌生感帶來的焦慮
- 遇到交通問題有緩衝時間

## 三、考試中：時間管理策略

### 開考前2分鐘

1. 快速瀏覽整份試卷結構
2. 確認題目數量與預期一致
3. 精算每個部分可用時間
4. 識別你最有把握的題目

### 時間分配原則

**先易後難**：先完成你有把握的題目，確保基本分數到手，再攻難題。

**按分值決定時間**：10分題給10-12分鐘，2分題給2-3分鐘。

**預留檢查時間**：無論哪一科，都預留最後5-10分鐘檢查。

### 答題不應有空白

即使不確定答案：
- 選擇題：猜一個答案（DSE大多不扣分）
- 問答題：寫出你所知道的相關概念，部分答對可得部分分
- 長題目：寫出計算思路，即使最終答案錯誤仍可能得步驟分

## 四、各科考試日策略

### 中文

- **閱讀理解**：先讀題目，帶著問題去讀文章，而非全讀完再答
- **寫作**：審題10分鐘，構思大綱再動筆，切勿開始後才改題目
- **口試**：放慢語速，不要急於填補沉默，思考後再發言

### 英文

- **Paper 1（閱讀）**：定位關鍵字再作答，不要「閱後感」式回答
- **Paper 2（寫作）**：第一段直接點題，避免無關開場白
- **Paper 4（口試）**：主動回應組員，用「Building on what [name] said...」引入

### 數學

- **Paper 1**：先做甲部（必答），再選丙部最熟悉的三條
- **Paper 2（MC）**：先答有把握的，難題做記號後續回
- 計算題每步驟都寫清楚，過程正確即使最終答案錯仍有步驟分

## 五、突發事件應對

| 突發情況 | 應對方法 |
|---|---|
| 腦海空白（Blank Out） | 深呼吸5次，跳過此題先做其他題 |
| 題目看不懂 | 再讀一遍，用常識推斷題目意圖 |
| 時間不夠 | 長題目只寫要點，確保有東西可得分 |
| 計算機沒電 | 向監考員要備用計算機 |
| 身體不適 | 立即舉手通知監考員，記錄於試卷 |
| 睇錯題號 | 冷靜，用連線標示清楚，繼續作答 |

## 六、放榜等待：心理調適

### 放榜前一個月

- 不要過度關注坊間「預測卷」或網絡上的分數討論
- 繼續維持規律生活，保持運動
- 思考備選方案（但不要過度焦慮）

### 放榜日

- 放榜結果出來前，準備好兩套方案：A計劃（理想成績）和B計劃（未達預期）
- 先自行查閱成績，在情緒穩定後才與家人討論
- 無論成績如何，允許自己有情緒反應，但不做即時重大決定

### 成績未如理想的後備選擇

| 選項 | 適合誰 |
|---|---|
| DSE重考 | 成績與目標差1-2個等級，有明確改善方向 |
| 副學士（AD） | 成績達入讀標準，願意多讀2年升本科 |
| 高級文憑（HD） | 傾向職業導向，希望更快就業 |
| 海外升學 | 成績達海外大學要求 |
| 先工作後再決定 | 需要時間思考方向 |
