# DSE考試週策略：考前一週、考試當天、考試後的完整計劃

DSE考試週是整個備考過程中最關鍵的7至10天。這段時間的策略不是「最後衝刺」，而是「保存狀態、穩定發揮」。本文提供從考前7天到最後一科完結的完整時間規劃，幫助考生以最佳狀態應考。

---

## 1. 考前7天倒數計劃

考前一週的核心原則：**不是吸收新知識，而是鞏固已有知識、保存體力**。

| 距考試天數 | 重點任務 | 禁忌事項 |
|---|---|---|
| 第7天（T-7） | 整理所有科目的考試日期時間表；確認准考證、文具、計算機等物品齊全 | 開始溫習全新篇章 |
| 第6天（T-6） | 最後一次完整複習考試當天第一科的所有重點；做1至2份舊試卷計時練習 | 通宵溫書 |
| 第5天（T-5） | 只複習高頻考點和弱項；縮減至半天溫習，下午開始休息 | 複習過多科目，分散注意力 |
| 第4天（T-4） | 輕度複習：瀏覽筆記、公式表、關鍵詞清單；不做完整試卷 | 因緊張而強迫自己繼續高強度溫習 |
| 第3天（T-3） | 早睡；確認考場路線、交通時間、備用交通方案 | 晚於凌晨12時睡覺 |
| 第2天（T-2） | 完全休息日：不溫習；進行輕鬆活動（散步、看電影）；早睡 | 「只看一下」地翻開參考書 |
| 第1天（考試前夕） | 整理考試用品；正常晚餐；不早於晚上10時半睡覺以避免過早醒來 | 與朋友互傳「最後溫書內容」；睡前刷社交媒體 |

### 重點提示

- **T-7至T-5**為主動複習期，目標是鞏固而非擴展。
- **T-4至T-2**為減壓期，降低認知負荷，讓大腦準備好在考場發揮。
- 每晚保證7至8小時睡眠。睡眠剝奪會直接損害記憶提取能力，對閱讀理解和寫作尤其致命。

## 2. 考試當天時間規劃表

以下以上午9時開考為標準計算（下午場相應後移）。

| 時間 | 行動 | 說明 |
|---|---|---|
| 06:30 | 起床 | 保留足夠時間清醒，避免頭腦昏沉入場 |
| 06:45 | 洗漱、更衣 | 穿著舒適、分層衣物（考場冷熱不定） |
| 07:00 | 早餐 | 選擇低GI食物（麥片、全麥麵包、雞蛋）；避免高糖早餐（會導致血糖驟降） |
| 07:30 | 出門 | 預留比平時多30分鐘的交通時間作緩衝 |
| 08:00 至08:20 | 抵達考場 | 建議在開考前40至50分鐘到達 |
| 08:20 至08:45 | 考場外等待 | 輕鬆瀏覽筆記重點（不超過10分鐘）；避免與人討論考試內容 |
| 08:45 至09:00 | 入場、準備 | 填寫考生編號；核對試題卷頁數；深呼吸放鬆 |
| 09:00 | 開考 | — |

### 交通備用方案

提前計劃兩條路線：主要路線（最快）和備用路線（若交通事故或地鐵延誤）。建議前一天以實際交通方式測試路線，確認步行時間。

## 3. 答題時間分配原則

以**英文卷一（Paper 1）**為例，演示時間分配邏輯：

| 題型 | 分數比重 | 建議時間 | 備注 |
|---|---|---|---|
| 閱讀理解（Part A/B） | 約50% | 45至50分鐘 | 先瀏覽問題，再針對性閱讀 |
| 延伸寫作（Integrated Skills） | 約50% | 40至45分鐘 | 留10分鐘計劃大綱 |
| 預留檢查時間 | — | 最後5分鐘 | 專注檢查漏答、格式 |

### 通用時間分配原則

1. **開考後先瀏覽整份試卷**（2至3分鐘）：了解題型、評估難度分布、決定作答順序。
2. **先易後難**：確保穩拿可得分數，不因難題浪費時間。
3. **設定內部截止時間**：每題或每部分事先定好完成時間，到時必須進入下一題，即使未完成。
4. **不因一題超時拖累整卷**：一題花費超過預定時間的150%，立即跳過，留空間做後面的題目。

## 4. 各科考試特定策略表

| 科目 | 最大失分點 | 防範技巧 |
|---|---|---|
| 中國語文（卷一：閱讀） | 引用原文不足、答案過於籠統、詞義解釋過淺 | 答案必須包含原文詞語；解釋詞義時分「字面義」+「文中義」兩層 |
| 中國語文（卷二：寫作） | 離題、篇幅不均（首段過長）、缺少文采 | 寫作前必花5分鐘擬提綱；首段控制在80至100字 |
| 英國語文（卷一：閱讀） | 答案未回應題目關鍵詞（key instruction words）、未達字數下限 | 每題答前先圈出關鍵詞（explain/describe/suggest）；答後數字數 |
| 英國語文（卷二：寫作） | 文體格式錯誤（letter/report/speech格式混淆）、論點重複 | 默背5種文體格式標準開頭結尾；每段只表達一個主要論點 |
| 數學（卷一） | 計算步驟不寫（被扣method marks）、代入數值時出錯 | 每步運算獨立成行；最後代入前重新核對代入值 |
| 數學（卷二：MC） | 計算失誤、時間不足（45題/65分鐘） | 平均每題不超過1分20秒；難題先跳過，做完一輪再回來 |
| 通識教育 | 論點單一（缺乏多角度）、只描述不分析 | 每個論點用PEEL結構（Point-Explain-Evidence-Link）；確保涵蓋至少兩個不同立場 |

## 5. 多科連考期間的睡眠與休息規劃

DSE考試期跨越約4週，科目之間往往只有1至2天間隔。長期的睡眠管理是成績穩定的基礎。

### 睡眠原則

- **目標睡眠時間**：每晚7至8小時，睡眠時間固定（例如每晚11時至早上7時）。
- **避免補眠失衡**：在考試週期間過度補眠（例如睡至中午）會打亂生理時鐘，令下一科考試時精神更差。
- **考後當晚**：完成一科後，當天可稍微放鬆（外出用餐、短暫娛樂），但仍須在午夜前入睡。

### 考試期間日程示例（兩科連考）

| 時段 | 活動 |
|---|---|
| 考完A科後（下午） | 休息2至3小時；輕食；不溫習B科 |
| A科考完當晚 | 輕度瀏覽B科重點（不超過1.5小時）；早睡 |
| B科考試前早上 | 正常起床；早餐；最後翻看B科提綱（20至30分鐘） |

## 6. 考後情緒管理

考後的心理狀態直接影響下一科的表現，尤其是連考或隔天便有另一科的情況。

### 核心原則：**考完即放下**

**不要和同學對答案**。這是最常見、最有害的習慣：
- 對答案無法改變已交出的卷
- 聽到同學的答案與自己不同，會製造不必要的焦慮
- 焦慮會擴散到下一科的備考心態

### 心理切換步驟

1. 走出考場後，先做5次深呼吸，刻意告訴自己「這科已完結」。
2. 進食、補充水分——認知表現與血糖直接相關。
3. 進行與考試完全無關的活動（散步30分鐘、聽音樂），讓前額葉皮層從高壓狀態恢復。
4. 下一科的複習**最早在考後3至4小時**才開始。

## 7. 突發情況處理

| 情況 | 即時應對 | 事後跟進 |
|---|---|---|
| 交通延誤/遲到 | 立即聯絡考評局熱線（2562 8638）；到場後向監考員說明情況 | 根據考評局規則，遲到30分鐘內通常仍可入場（但不獲延長時間） |
| 忘帶准考證 | 向監考員說明；出示其他身份證明文件（身份證）；部分情況可先入場後補交 | 告知家長或回家取準考證；不要擅自離開考場範圍 |
| 考試中腦袋空白 | 停筆，閉目10秒，做3次慢速深呼吸；先跳到下一題 | 深呼吸能降低皮質醇水平，有助恢復記憶提取 |
| 身體不適（頭暈/嘔心） | 舉手通知監考員；可申請到場外休息（時間照計）；嚴重時可申請特殊考慮（Special Consideration） |考後向校方提交醫生證明，申請「特殊情況」評分考慮 |

## 8. 考試當天禁忌清單

以下行為在考試當天可能降低表現，應完全避免：

- **禁忌1：與同學討論剛剛考完的題目**（製造焦慮，影響下午場或下一科）
- **禁忌2：考前大量攝取咖啡因**（加劇焦慮、手震，影響書寫）
- **禁忌3：早餐不吃或吃得過飽**（空腹專注力下降；過飽令人昏昏欲睡）
- **禁忌4：最後一刻才到達考場**（趕路的緊張狀態需15至20分鐘才能平復）
- **禁忌5：帶入考場的文具未事先測試**（原子筆沒油、計算機電池沒電）
- **禁忌6：在考場內查閱手機**（即使是等候期間，違規後果嚴重）
- **禁忌7：考試中途放棄作答**（空白答案必得0分；即使寫幾行也可能得到部分分數）
- **禁忌8：寫作題不擬草稿直接落筆**（結構混亂是寫作失分的主因）

---

## 總結

DSE考試週的成敗，往往不在最後幾天「溫了多少」，而在於能否以穩定的狀態、清醒的頭腦在考場上發揮平日水平。策略的核心是：**在對的時間做對的事**——知道何時複習、何時放下、何時休息、何時全力以赴。

按照本文的7天計劃執行，考試當天早到、吃好、不對答案，配合合理的時間分配，是任何科目在考場上穩定發揮的基礎。
