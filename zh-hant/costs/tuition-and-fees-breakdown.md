# DSE 備考及升學費用全面分析

DSE 備考及大學升讀的費用可從幾千至幾十萬港元不等，視乎你是否需要補習、選擇本地或海外升學，以及是否獲得資助。本指南列出各個費用環節的市場行情，幫助學生和家長做好財務規劃。

---

## 1. DSE 報考費用

| 費用項目 | 金額（2025–26年參考） | 備註 |
|---|---|---|
| 必修科報考費 (中、英、數、通識) | 每科約 HK$235 | 4科合共約 HK$940 |
| 選修科報考費 | 每科約 HK$235 | |
| M1/M2 數學延伸報考費 | 約 HK$235 | |
| 應用學習（ApL）報考費 | 約 HK$1,200–1,500 | 因課程而異 |
| 重考生附加費 | — | 部分情況適用 |
| 典型全套 DSE 報考費用 | **約 HK$1,400–2,500** | 視科目組合而定 |

> 查詢最新費用：香港考試及評核局官方網站 (hkeaa.edu.hk)

## 2. 補習費市場行情

補習是 DSE 備考最大的可變成本，差異懸殊：

| 補習類型 | 費用範圍（每月） | 優缺點 |
|---|---|---|
| 大型補習社 (張家明、呂Sir等) | HK$1,500–4,000 / 科 | 名師效應、班級大，互動少 |
| 小型補習中心 (10–15人班) | HK$800–2,000 / 科 | 較多互動，本地口碑 |
| 私人補習 (1對1) | HK$200–600 / 小時 | 最靈活，費用最高 |
| 網課/錄影課程 | HK$300–1,500 / 科 | 可重播，缺乏即時互動 |
| 不補習（自學） | 0 | 需高度自律及方向感 |

**典型補習開支試算（補 3 科、每科每月 HK$2,000，補 18 個月）**：
約 **HK$108,000**

## 5. 學生資助辦事處（SFAA）申請

| 資助計劃 | 最高金額 | 申請資格 |
|---|---|---|
| 免入息審查資助貸款 | 全額學費 | 所有本地全日制學生 |
| 學生財政援助計劃（入息審查） | 學費 + 生活費 | 家庭月入約 HK$40,000 以下 |
| 緊急貸款 | 視乎情況 | 突發財政困難學生 |

> 詳情及申請：studentfinance.gov.hk

## 6. 獎學金與助學金

| 獎學金 | 金額 | 申請方法 |
|---|---|---|
| 政府獎學基金（優異成績） | HK$40,000/年 | 自動考慮 |
| 各院校新生獎學金 | HK$20,000–80,000 | 各院校直接申請 |
| 香港賽馬會獎學金 | HK$50,000–100,000 | 校內推薦 |
| 利希慎基金、邵逸夫獎學金等 | 不等 | 校內推薦 |

## 7. 海外升學備考附加費用

如考慮同時申請海外大學：

| 項目 | 費用估計 |
|---|---|
| IELTS 考試 | HK$2,200–2,500 / 次 |
| SAT 考試 | USD$60–80 / 次（約HK$470–620）|
| A-Level 補習（如選擇） | 另計，可能每科 HK$3,000–6,000/月 |
| 大學申請費 | 英國 UCAS：HK$280（1–5校）；美國：USD$50–90/校 |
| 國際生顧問服務 | HK$10,000–80,000 |

## 總結

DSE 備考的核心費用是補習（差異最大）和報考費（固定）。不補習的學生可以節省6位數，但需要極高的自律能力。香港八大的本地生學費受資助，每年約 HK$42,100，並可透過學生資助辦事處申請貸款或資助。提早做好財務規劃——清楚知道自己的預算，才能在補習投資上做出最划算的決定。
