# DSE備考及大學費用全面拆解：補習、考試、學費與資助

每逢踏入中四，家長與學生最關心的問題之一，往往不是選科策略，而是：**DSE備考到底要花多少錢？讀大學又要花多少？有沒有資助可以申請？**

香港的DSE生態催生了龐大的補習產業。從名師補習社到一對一私補，從模擬試到坊間出版物，備考費用因人而異，差距可以極大。本文從DSE應試費用開始，逐層拆解補習、教材、大學學費、住宿生活費，以及各類政府資助計劃，協助家庭作出清醒的財務規劃。

---

## 一、DSE考試費用

DSE考試由香港考試及評核局（HKEAA）主辦，考試費用按科目數目計算。

### 1.1 報名費基本結構

| 費用項目 | 金額（HKD） |
|----------|-------------|
| 基本報名費 | 約 $340 |
| 每科必修科（中文、英文、數學、通識/公民）考試費 | 約 $310–$370 / 科 |
| 每科選修科考試費 | 約 $285–$370 / 科 |
| 重考個別科目（校外考生） | 同上，另加行政費 |

**典型在校考生（4科必修 + 2科選修）總報名費估算：約 $2,200–$2,600**

校內DSE考生（即中六在讀生）的報名費由學校統一處理，部分費用或由學校津貼。校外重考生則須自行全數繳交，且不享有在校生費用豁免。

### 1.2 特殊情況費用

- **音樂、視覺藝術**等涉及實物評核的科目，另設器材及評核場地費
- **體育科**校外考生須另繳運動設施使用費
- 遺失准考證、申請成績覆核均另收行政費（覆核費約 $70 / 科，如成績更改則退款）

## 二、補習費用：從名師到私人補習

補習是香港DSE備考文化的核心組成部分。根據補習形式不同，費用差異相當懸殊。

### 2.1 補習社課程

香港補習社分為兩大類：**連鎖大型補習社**（如英皇教育、Modern Education、導師學院）以及**獨立名師工作室**。

| 補習形式 | 每月費用估算（HKD） | 說明 |
|----------|----------------------|------|
| 大型補習社錄影班 | $400–$800 / 科 | 觀看預錄課堂，可反覆重看 |
| 大型補習社直播班 | $700–$1,200 / 科 | 即時課堂，互動性略高於錄影 |
| 名師小班（20人以下） | $1,200–$2,500 / 科 | 直接接觸名師，互動較多 |
| 名師精英小班（10人以下） | $2,500–$4,500 / 科 | 針對高分組，費用最高 |
| 獨立補習社小班 | $800–$1,800 / 科 | 水平參差，需自行評估導師質素 |

**以一個DSE考生補習3科、採用直播班計算，每月補習費約 $2,100–$3,600，全年（9個月）約 $18,900–$32,400。**

### 2.2 名師效應與費用溢價

香港DSE補習界存在「名師文化」。部分人氣名師（如補英文、數學或中文的知名導師）設有候補名單制度，收費較一般同行高出50%至100%。名師的品牌溢價建立在多年的口碑、高分學生見證及媒體曝光之上，實際教學效果因人而異。

家長在選擇名師補習前，建議：
- 查閱該名師學生的歷屆成績分佈（非只看頂尖個案）
- 了解小班規模——部分「小班」實為20至30人課堂
- 嘗試試堂（多數名師提供免費或低費試堂）

### 2.3 私人補習（一對一 / 一對二）

私人補習以鐘點計費，靈活性最高，亦最昂貴。

| 導師類別 | 每小時收費（HKD） |
|----------|-------------------|
| 在職大學生 / 副學士生 | $80–$150 |
| 大學畢業生（非教育學院） | $150–$250 |
| 現職或退休教師 | $200–$400 |
| 名師或資深專業導師 | $400–$800+ |

**每週補習2小時、每月上8節、採用大學畢業生導師，每月費用約 $1,200–$2,000 / 科。**

私人補習的優勢在於針對性強，可因應學生弱項調整進度。劣勢是費用較高，且導師質素較難保證。

### 2.4 網上課程與自學資源

近年網上補習平台（如Zoom教學、YouTube頻道、自學APP）大幅降低了入門門檻，部分資源甚至免費。

| 資源類型 | 費用 |
|----------|------|
| YouTube名師免費課堂 | 免費 |
| 網上補習訂閱制平台 | $200–$600 / 月 |
| 自學練習APP（如Past Paper APP） | 免費 / 約 $50–$100 一次性費用 |

## 三、教材與模擬試費用

### 3.1 教科書與參考書

| 項目 | 費用估算（HKD） |
|------|-----------------|
| 每科正版教科書（全套） | $400–$800 |
| 每科補充練習冊 | $80–$200 |
| 市售名師筆記 / 精讀本 | $60–$150 / 本 |
| 歷屆試題集（Past Papers） | $80–$150 / 冊 |

**6科教科書加基本參考書，整個中四至中六合計費用約 $8,000–$15,000。**購買二手教科書可節省30%至50%費用，DSE考生間的二手書交易平台（如Facebook群組、Carousell）選擇豐富。

### 3.2 模擬試（Mock Exam）

模擬試是DSE考前準備的重要環節，費用視乎主辦機構及科目數目而定。

| 模擬試類型 | 費用估算（HKD） |
|-----------|-----------------|
| 大型補習社主辦模擬試（每科） | $150–$350 |
| 學校統一安排模擬試 | 多數由學校承擔或象徵性收費 |
| 跨校聯合模擬試（全科） | $600–$1,200 |

部分補習社會將模擬試費用包含在課程套餐內，報名前宜確認。

## 四、DSE備考兩年總費用估算

以下為兩種典型備考策略的費用總覽（中五至中六合計）：

### 方案A：自學為主 + 少量補習

| 項目 | 費用（HKD） |
|------|-------------|
| DSE報名費 | $2,500 |
| 教科書及參考書 | $6,000 |
| 補習（2科錄影班，每月 $500/科，12個月） | $12,000 |
| 模擬試（3科） | $600 |
| **合計** | **約 $21,100** |

### 方案B：多科補習（直播班 + 私補）

| 項目 | 費用（HKD） |
|------|-------------|
| DSE報名費 | $2,500 |
| 教科書及參考書 | $10,000 |
| 補習社課程（3科直播班，每月 $1,000/科，18個月） | $54,000 |
| 私人補習（1科，每月 $1,500，18個月） | $27,000 |
| 模擬試（全科） | $1,000 |
| **合計** | **約 $94,500** |

兩個方案的費用差距超過四倍，說明補習策略是DSE備考最大的可控開支。

## 五、大學學費：資助學位 vs 自資課程

### 5.1 UGC資助學士學位（八大）

由大學教育資助委員會（UGC）資助的學士課程，學費受政府規管，每年劃一標準。

| 類別 | 年度學費（HKD） |
|------|-----------------|
| UGC資助學士學位（所有課程） | **$42,100 / 年**（2024/25學年） |
| 修業年期（一般課程） | 3至4年 |
| 四年制課程總學費 | **$168,400** |

醫科、牙科等專業學位年期較長（5至6年），總學費相應增加。醫科六年制總學費約 $252,600，惟部分年份可獲獎學金或政府資助抵減。

### 5.2 自資學士課程

自資課程（包括各院校自行開辦的學士、修讀銜接的副學士升本課程等）學費由院校自訂，差異較大。

| 課程類別 | 年度學費範圍（HKD） |
|----------|---------------------|
| 本地自資學士（一般） | $65,000–$120,000 |
| 護理、醫療相關自資學士 | $90,000–$130,000 |
| 商科自資學士 | $70,000–$100,000 |
| 副學士銜接學士（頂銜） | $55,000–$90,000 / 年（一般2年制） |

自資課程不受UGC學費規管，院校每年均可調整收費，報名前須確認最新學費。

### 5.3 副學士課程

副學士（Associate Degree）是部分DSE考生在取得學士學位前的過渡選擇。

| 副學士類型 | 年度學費估算（HKD） |
|-----------|---------------------|
| 社區學院（大學附屬） | $55,000–$80,000 |
| 獨立私立院校副學士 | $45,000–$70,000 |
| 修業年期 | 一般2年，總費用 $90,000–$160,000 |

副學士畢業後，升讀銜接學士一般需再讀2年，整體學費因此可能高於直接入讀UGC資助學位。

## 六、大學住宿費及生活費

### 6.1 學生宿舍費用

八大均設有學生宿舍，惟宿位有限，競爭激烈，未必每年均能成功申請。

| 宿舍類型 | 每年費用估算（HKD） |
|----------|---------------------|
| 大學宿舍（雙人房） | $15,000–$30,000 |
| 大學宿舍（單人房） | $25,000–$45,000 |
| 校外租房（市區） | $60,000–$120,000（視乎地區） |
| 校外租房（新界） | $36,000–$72,000 |

部分院校（如科技大學、中文大學）因校園地理位置較偏，住宿需求較高，宿位競爭尤為激烈。

### 6.2 日常生活費估算（每月）

| 項目 | 每月費用估算（HKD） |
|------|---------------------|
| 餐飲（校園飯堂為主） | $2,500–$4,000 |
| 交通（月票 / 八達通） | $500–$1,200 |
| 書本及學習用品 | $300–$600 |
| 個人護理及雜費 | $500–$1,000 |
| 娛樂及社交 | $500–$1,500 |
| **每月合計** | **$4,300–$8,300** |

**四年制大學生活費（不含學費及宿舍）估算：約 $206,400–$398,400。**

## 八、政府及院校資助計劃

香港設有多項針對本地學生的財務資助，協助減輕學費及生活費負擔。

### 8.1 學生資助辦事處（學資處）計劃

學資處（SFO，Student Finance Office）是本港學生財務援助的主要機關，旗下有兩大主要計劃：

#### 政府資助學位課程助學金計劃（TSFS）

- **全稱**：Tertiary Student Finance Scheme — Publicly-funded Programmes（TSFS）
- **適用對象**：就讀UGC資助及其他政府資助全日制課程的學生
- **資助形式**：助學金（免還款）及低息貸款混合
- **計算基準**：家庭經濟狀況審查（入息及資產），家庭收入愈低，助學金比例愈高
- **最高助學金金額**：可涵蓋全額學費及部分生活費，具體金額按年度及家庭情況而定
- **申請時間**：每年5至7月，於學資處官網提交申請

#### 非本地學生貸款計劃（NLSPS）

- **全稱**：Non-means-tested Loan Scheme for Post-secondary Students（NLSPS）
- **適用對象**：就讀本地認可院校的全日制學生，**不設入息審查**
- **資助形式**：純貸款，須連本帶息歸還
- **利率**：與政府發行的外匯基金票據掛鈎，歷年維持在低息水平（通常低於市場利率）
- **用途**：學費及生活費均可申請
- **優點**：免入息審查，中產家庭或父母不願提供財務資料者亦可申請

| 計劃 | 入息審查 | 資助形式 | 適用課程 |
|------|----------|----------|----------|
| TSFS | 需要 | 助學金 + 貸款 | UGC資助課程 |
| NLSPS | 不需要 | 純貸款 | 本地認可全日制課程 |

### 8.2 自資課程資助計劃

就讀自資學士或副學士課程的學生，主要可申請以下計劃：

- **免入息審查貸款計劃（NULS）**：針對自資院校學生，純貸款形式，不設審查
- **獎學金及助學金（個別院校）**：各院校設有院校級獎學金，競爭激烈，通常按成績或才藝頒發
- **學費減免計劃**：部分自資院校就家庭經濟有困難的學生提供學費減免，需直接向院校申請

### 8.3 院校獎學金與外部獎學金

除政府計劃外，學生亦可申請以下資助來源：

| 類別 | 例子 | 金額範圍（HKD） |
|------|------|-----------------|
| 院校入學獎學金 | 港大入學獎學金、中大成績優異獎 | $10,000–$80,000 / 年 |
| 政府獎學金 | 政府獎學金基金 | $36,000–$60,000 / 年 |
| 企業獎學金 | 滙豐、恒生、李錦記等 | $10,000–$50,000 / 年 |
| 慈善基金 | 東華三院、保良局、各宗教團體 | $5,000–$30,000 |
| 海外交流資助 | 大學訪問交流計劃（政府） | 視乎目的地，數千至數萬 |

**申請建議**：院校獎學金多於入學後第一學年方可申請，且需維持一定成績；部分獎學金亦設有課外活動或服務要求。應盡早向院校學生事務處索取獎學金目錄，建立申請時間表。

### 8.4 緊急資助及學生援助基金

部分院校設有**緊急援助基金**，針對突發財務困難（如父母失業、家庭意外）的學生，可提供一次性免還款援助或緊急貸款。申請通常毋需繁複審批，可直接向院校福利部門查詢。

## 九、財務規劃實用建議

### 9.1 DSE備考階段

1. **量力而為選擇補習**：補習並非成績的決定性因素。善用免費及低費資源（YouTube名師課堂、Past Paper APP），可在不大幅削減備考效果的前提下節省數萬元。
2. **二手書市場**：教科書及參考書透過Carousell、Facebook DSE二手書群組購買，通常可節省三至五成費用。
3. **善用學校資源**：學校圖書館、自修室及部分科目的免費補底課，是被忽略的低費資源。

### 9.2 大學學費規劃

1. **盡早了解資助申請流程**：TSFS申請通常在學年開始前數個月便需提交，錯過窗口則須等待下一學年。
2. **NLSPS作為備用**：即使家庭收入中等，NLSPS的低息貸款仍是可考慮的補充工具，特別適合不希望動用儲蓄的家庭。
3. **兼職工作**：大學生在學期間可透過合法兼職補貼生活費，部分院校亦提供校內工讀機會（Research Assistant、Teaching Assistant）。
4. **比較宿舍與校外租房的實際成本**：宿舍雖然便宜，但若需每年重新申請、不能保證宿位，應同時規劃校外租房的預算方案。

## 重點總結

**DSE備考及大學費用因選擇而差異極大，以下是本文的核心數字與要點：**

1. **DSE報名費**：約 $2,200–$2,600（按科目數目計）；重考考生須自行全額繳付。

2. **補習費用是最大可控開支**：兩年備考期，純自學僅需約 $2萬；多科名師補習可超過 $9萬，相差四倍以上。選擇補習形式前，建議先評估性價比。

3. **資助學位（八大）學費**：每年 $42,100，四年合計 $168,400，遠低於自資課程的每年 $65,000–$120,000。

4. **大學四年全費用**（學費 + 住宿 + 生活費）：資助學位住宿舍約 **$54萬**；自資課程校外租房可超過 **$100萬**。

5. **主要資助計劃**：**TSFS**（需入息審查，助學金 + 貸款混合）適合家庭收入有限的學生；**NLSPS**（免入息審查，純貸款）適合不符合TSFS資格但仍需財務支援的學生。兩者申請均透過學資處辦理。

6. **提前規劃是關鍵**：了解資助申請窗口、善用院校獎學金、比較課程費用，可在整個求學週期內節省數十萬港元。
