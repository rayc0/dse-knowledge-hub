# DSE物理科完全攻略：課程大綱、公式應用與實驗技巧

DSE物理科（HKDSE Physics）是香港中學文憑試中最具挑戰性的理科科目之一，同時也是報讀工程學、物理學、電腦科學、建築學等理工學科的必要科目。對於有志在大學修讀理工科的同學來說，物理科的成績往往直接影響JUPAS選科的競爭力。

本文以香港本地考生的視角出發，系統整理DSE物理科的課程架構、考試形式、核心公式應用、實驗技巧及高分答題策略，協助你建立清晰的應考框架。

---

## 一、課程架構：必修與選修單元

DSE物理課程分為兩大部分：**必修部分（Compulsory Part）**與**選修部分（Elective Part）**。

### 1.1 必修部分（五大主題）

所有考生均須掌握以下五個必修主題：

**（一）力學（Mechanics）**

力學是整個物理課程的基石，涵蓋運動學（勻速運動、勻加速運動、拋體運動）、牛頓三大定律、動量與衝量、功與能量、彈性碰撞與非彈性碰撞，以及萬有引力與衛星運動。力學在試卷中計算題佔比最重，必須熟練掌握各類方程式的推導與應用。

**（二）熱學（Heat and Gases）**

包括溫度與熱量的分別、熱容量（heat capacity）、潛熱（latent heat）、理想氣體定律（波義耳定律、查理定律、給呂薩克定律及綜合氣體定律），以及氣體分子動理論。熱學概念較為直觀，但考生容易混淆各條定律的適用條件，尤其是溫度單位須轉換為開爾文（K）。

**（三）波動與光學（Wave Motion and Optics）**

涵蓋波的基本性質（波長、頻率、振幅、波速）、縱波與橫波、干涉與衍射、光的折射與全內反射、透鏡成像原理，以及聲波特性。這一章節圖形題較多，需熟練繪製波形圖與光路圖，考生在解答時須清楚標示方向與角度。

**（四）電與磁（Electricity and Magnetism）**

包括靜電學、電路分析（歐姆定律、基爾霍夫定律、串聯與並聯電路）、電容、電磁感應、交流電及變壓器原理。電磁學是DSE物理概念密度最高的範疇，亦是歷年試卷的重心。電路分析題型多變，考生務必先判清串並聯結構，再進行計算。

**（五）放射性與核物理（Radioactivity and Nuclear Energy）**

內容涵蓋三種輻射的性質（α、β、γ射線）、半衰期計算、核反應方程式、核裂變與核聚變，以及輻射的危害與防護措施。這一章節在試卷中題目不多，但每年均有考核，且常以簡答或短計算的形式出現，相對容易得分。

---

### 1.2 選修部分（四選二）

DSE物理設有以下四個選修單元，考生須從中選取**兩個**作為應考範圍：

| 單元編號 | 英文名稱 | 中文名稱 |
|---------|---------|---------|
| Module 1 | Astronomy and Space Science | 天文學與太空科學 |
| Module 2 | Atomic World | 原子世界 |
| Module 3 | Energy and Use of Energy | 能源與能源應用 |
| Module 4 | Medical Physics | 醫學物理 |

**天文學與太空科學（Module 1）** 近年相當受歡迎，內容包括開普勒定律、天體運動、望遠鏡原理及宇宙學基礎，計算量適中，較重概念理解。

**原子世界（Module 2）** 難度稍高，涵蓋光電效應、德布羅意波、能級躍遷、激光原理及X射線，適合有意修讀物理或工程相關學科的考生深化理解。

**能源與能源應用（Module 3）** 結合時事熱點，包括各類發電方式、能源效益與熱力學基礎，論述性內容較多，適合文字表達能力強的考生。

**醫學物理（Module 4）** 涵蓋X射線成像、超聲波診斷及磁力共振（MRI）的物理原理，貼近現實應用，適合有意報讀醫科或生物醫學工程的考生。

## 二、考試形式詳解

DSE物理公開考試分為兩份試卷，合計佔總成績**80%**，其餘**20%**為校本評核（SBA）。

### 2.1 試卷一（Paper 1）：必修內容考核

- 考試時間：**2.5小時**
- 總分：**100分**
- 分三部分：
  - **Section A（多項選擇題）**：共33題，每題1分，共33分。考查基礎概念與快速判斷，選項陷阱多，須細心審題。
  - **Section B（短答題）**：約40分，要求考生在有限篇幅內列出計算步驟或作簡短解釋。
  - **Section C（論述題）**：約27分，要求考生對物理現象作較完整的分析，部分題目附有數據圖表，須讀圖解析。

### 2.2 試卷二（Paper 2）：選修內容考核

- 考試時間：**1.25小時**
- 總分：**60分**
- 內容涵蓋考生所選的兩個選修單元，每個單元約30分
- 題型以短答及論述為主，幾乎沒有純選擇題
- 題目常結合實驗情境或實際應用案例，考查考生能否靈活運用理論知識

### 2.3 校本評核（SBA）

SBA主要為實驗報告與探究活動，由任課老師評分後提交香港考試及評核局審核，佔總成績**20%**。對本地學校在讀考生而言，認真對待每次實驗課是穩定基礎分數的重要一環。私人候選人的SBA安排可能不同，建議向考評局查詢。

## 三、必備公式速查表

以下整理DSE物理各主題的核心公式，建議製作成自己的「公式手冊」隨時溫習：

### 力學

| 公式 | 說明 |
|------|------|
| v = u + at | 勻加速運動，v為末速，u為初速，a為加速度，t為時間 |
| s = ut + ½at² | 位移公式 |
| v² = u² + 2as | 不含時間的位移速度關係 |
| F = ma | 牛頓第二定律 |
| p = mv | 動量定義 |
| W = Fs cosθ | 功的定義，θ為力與位移夾角 |
| KE = ½mv² | 動能 |
| GPE = mgh | 重力勢能 |
| F = GMm/r² | 萬有引力定律 |

### 熱學

| 公式 | 說明 |
|------|------|
| Q = mcΔT | 比熱容公式，c為比熱容 |
| Q = mL | 潛熱公式，L為潛熱 |
| pV = nRT | 理想氣體狀態方程 |
| p₁V₁/T₁ = p₂V₂/T₂ | 綜合氣體定律 |
| T(K) = T(°C) + 273 | 溫度單位換算（切記！） |

### 波動與光學

| 公式 | 說明 |
|------|------|
| v = fλ | 波速、頻率與波長的關係 |
| n = sin i / sin r | 折射率（斯涅爾定律） |
| n = c / v | 折射率與光速的關係 |
| 1/f = 1/u + 1/v | 透鏡成像公式 |
| m = v/u | 放大率 |

### 電與磁

| 公式 | 說明 |
|------|------|
| V = IR | 歐姆定律 |
| P = IV = I²R = V²/R | 電功率 |
| Q = It | 電荷量 |
| E = V + Ir | 電動勢（含內阻） |
| F = BIL sinθ | 載流導體在磁場中的受力 |
| EMF = BLv | 感應電動勢 |
| Vs/Vp = Ns/Np | 變壓器電壓比 |

### 放射性

| 公式 | 說明 |
|------|------|
| A = A₀ × (½)^(t/t½) | 放射性活度隨時間衰減 |
| E = mc² | 質能方程（核反應能量計算） |

## 四、實驗技巧與高頻實驗題

### 4.1 高頻實驗題目

以下實驗在DSE試卷中出現頻率極高，必須深入掌握：

**（一）單擺實驗（Simple Pendulum）**

用於測量重力加速度 g。核心技巧：繪製 T²-L 圖，從斜率反推 g（斜率 = 4π²/g）。控制變量：保持擺幅小於10°，確保擺動在同一平面內。

**（二）弹簧常數測定（Hooke's Law）**

記錄不同質量下彈簧的伸長量，繪製 F-x 圖，斜率即為彈簧常數 k（F = kx）。需注意超出彈性限度後定律不再適用。

**（三）折射率測定**

利用大頭針或半圓形玻璃磚，測量入射角 i 與折射角 r，以 n = sin i / sin r 計算折射率。找出全內反射的臨界角時，須令光線由密介質射向疏介質。

**（四）歐姆定律與電阻測量**

改變電壓，記錄對應電流值，繪製 V-I 圖。斜率即為電阻 R。需說明電流表與電壓表的接法（安培表外接法 vs 內接法）對結果的影響。

**（五）玻義耳定律驗證（Boyle's Law）**

在恆溫條件下，記錄氣體在不同體積時的壓力值，繪製 P-1/V 圖以驗證線性關係，即 pV = constant。

**（六）放射性衰變分析**

利用半衰期圖表，從初始活度或核數推算特定時間後的剩餘量，或反推原始核素數量。

### 4.2 回答實驗題的標準框架

DSE實驗題的評分準則非常具體，每次回答須包含以下要素：

1. **控制變量**：明確說明哪些量需要保持不變，哪些是自變量與因變量
2. **誤差來源**：區分隨機誤差（random error）與系統誤差（systematic error），並說明其成因
3. **提高準確性的方法**：重複測量取平均值、使用精度更高的儀器、增加數據點數量
4. **作圖技巧**：選取合適的縱橫軸、標示單位、畫出最佳擬合直線（best-fit line）而非點與點連接

## 五、常見失分點分析

根據歷年DSE物理閱卷報告（Examiner's Report），以下是考生最常見的失分原因：

**1. 遺漏單位或單位錯誤**
計算電阻忘寫 Ω、頻率單位誤寫為 Hz/s，均會直接扣分，即使計算過程完全正確。

**2. 溫度單位未換算**
熱學題中忘記將攝氏溫度轉換為開爾文（K = °C + 273），是最普遍的計算錯誤之一。

**3. "Explain" 題回答過於簡略**
考生只寫出關鍵詞而未形成完整的因果邏輯鏈。考評局要求的是「因為……所以……」式的完整表述，缺乏物理語言連接會被扣分。

**4. 電路分析串並聯判斷錯誤**
部分考生未能正確辨別電路元件的串並聯關係，導致後續所有計算均出錯。建議先將複雜電路重新繪製為等效電路，再逐步計算。

**5. 混淆物理量定義**
例如將「速率（speed）」與「速度（velocity）」互換，或將「力（force）」與「衝量（impulse）」混用，是選擇題和短答題的常見失誤。

**6. 選修單元準備不足**
部分考生在選修單元的溫習時間不足，面對Paper 2的應用情境題時無從入手，白白失去30分。

**7. 圖形題方向與標示錯誤**
光路圖、波形圖及向量圖若方向畫反或遺漏箭頭標示，均會失分。繪圖時務必清楚標示所有物理量的方向。

## 六、成績等級參考

DSE物理成績以等級報告，從最低至最高為1至5，再細分5*與5**。以下為歷年大約的分數參考（Paper 1與Paper 2折算後滿分160分）：

| 等級 | 大致分數參考 |
|------|------------|
| 3級（合格） | 約55–70分 |
| 4級 | 約80–95分 |
| 5級 | 約100–115分 |
| 5*級 | 約116–132分 |
| 5**級 | 約133分以上 |

注意：以上分數為估算，每年因試卷難度調整而有所變動。DSE採用**準則參照評核（Criterion-referenced Assessment）**，成績反映考生的絕對表現而非相對排名，每年5**的比例約佔該科考生的2–3%。

## 七、備考策略與時間規劃

### 第一階段：系統建基（距考試12–18個月）

重點是深入理解必修五大主題，而非追求速度：
- 以教科書（Longman 或 Marshall Cavendish 版）為主，逐章掌握
- 每章完成後做10–15題DSE歷屆選擇題作自我測試
- 建立個人公式手冊，並為每條公式標注適用條件與常見錯誤

### 第二階段：選修單元攻讀（距考試8–12個月）

- 確定選修組合後，每個單元集中4–6週深入學習
- 整理每個單元的高頻概念與典型題型
- 結合生活實例加深理解，例如學習醫學物理時聯繫MRI的實際應用

### 第三階段：真題訓練（距考試4–8個月）

- 系統完成2012年至今所有DSE物理歷屆試題（Paper 1 + Paper 2）
- 每次模擬考試後詳細分析錯題，按概念錯誤、計算失誤、審題不清分類
- 重點練習「Explain」類文字題，對照官方評分準則（Mark Scheme）自行評分
- 計時練習，確保Paper 1可在2.5小時內穩定完成

### 第四階段：衝刺強化（距考試1–3個月）

- 回歸公式手冊，檢查所有公式是否熟記，單位書寫是否規範
- 針對歷年Examiner's Report中指出的高頻失分點專項練習
- 每週完成至少一套完整模擬試卷
- 鞏固作圖技巧，包括光路圖、電路圖及最佳擬合直線的繪製

### 考前最後一週

- 以溫習鞏固為主，避免學習全新內容
- 複習常用公式、圖形符號及實驗標準表述
- 保證充足睡眠，維持良好應考狀態

## 重點總結

- **課程架構**：必修五大主題（力學、熱學、波動與光學、電與磁、放射性）加上四選二選修單元（天文學、原子世界、能源、醫學物理）。

- **考試形式**：試卷一（必修，2.5小時，100分）加試卷二（選修，1.25小時，60分），另有校本評核佔總成績20%。

- **核心公式**：重點掌握力學運動方程、氣體定律（溫度必換算為K）、折射率、歐姆定律及電功率、半衰期衰減公式。製作個人公式手冊是最有效的溫習工具。

- **實驗技巧**：單擺測 g、歐姆定律、玻義耳定律、折射率測定為高頻實驗題，每次回答須涵蓋控制變量、誤差分析、提高準確性的方法及正確作圖。

- **最常見失分點**：遺漏單位、溫度未換算為開爾文、「Explain」題未形成完整因果表述、電路串並聯判斷錯誤。

- **備考核心**：系統建基 → 選修攻讀 → 歷屆真題訓練 → 針對失分點強化。取得5**的關鍵不只在於會計算，更在於能用準確的物理語言清晰表達分析過程，並在有限時間內規範作答。

物理是一門以理解為本的學科。掌握了核心概念的邏輯，許多題目的答案都可以從基本原理推導出來。願各位同學備考順利，在DSE物理科取得理想成績。
