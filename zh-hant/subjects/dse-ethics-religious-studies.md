# DSE 倫理與宗教科（E&RS）完全攻略 2026：課程結構、試卷分析、模組選擇與應試策略

> **核心 fact**：DSE 倫理與宗教科每年應考人數約 700–1,000 人，是全港報考人數最少的甲類選修科之一。卷二基督教模組為**開卷考試**（Open Book），是 DSE 四十多個科目中極罕見的開卷安排之一。2028 年課程改革後將取消校本評核（SBA）。

## 一、倫理與宗教科是什麼？為什麼選這科？

倫理與宗教（Ethics and Religious Studies，簡稱 **E&RS** 或 **ERS**）是香港中學文憑考試（HKDSE）甲類選修科，屬個人、社會及人文教育（PSHE）學習領域。

### 科目定位

- **倫理學（Ethics）部分**：以理性思考為基礎，研究規範倫理學理論（如功利主義、義務倫理、德性倫理），並將之應用於個人及社會議題的道德判斷。
- **宗教傳統（Religious Studies）部分**：深入了解一個宗教傳統（佛教或基督教），從宗教歷史、核心教義、宗教實踐等角度建立系統認識。

### 為何選擇 E&RS？

| 特點 | 說明 |
|------|------|
| 卷二開卷（基督教模組） | 基督教模組為開卷考試，可攜帶《聖經》及筆記進場，大幅降低記憶負擔 |
| 人文社科思維訓練 | 倫理學的推理框架（功利主義 vs 義務論）對法律、社工、哲學等大學學科有直接幫助 |
| 與通識/公民科銜接 | 部分社會議題（如安樂死、墮胎、死刑）在 E&RS 及原公民與社會發展科均有涉及 |
| 競爭人數少 | 全港約 700–1,000 名考生，相對於物理（逾萬人）、經濟（逾萬人），分級曲線樣本較小，高分比例穩定 |
| 開放作答空間 | 倫理論述題沒有唯一「正確立場」，重視論證質素而非背誦標準答案 |

> ⚠️ **選科前提醒**：E&RS 並非大部分理科或商科大學課程的指定選修科，若目標院校課程要求選修理科組合（如 HKUST Engineering），E&RS 可能**無法計入**加權成績或符合入學要求，選科前務必核查目標課程的選修科要求。

## 二、課程結構：三大範疇六個單元

E&RS 課程由課程發展議會與 HKEAA 聯合編製，涵蓋兩大學習範疇：

### 倫理部分（Ethics）

**必修範疇一：規範倫理學（Normative Ethics）**

| 單元 | 核心內容 |
|------|----------|
| 1. 規範倫理概論 | 什麼是道德？道德判斷的基礎 |
| 2. 功利主義（Utilitarianism） | 邊沁（Bentham）與彌爾（Mill）、行為功利主義 vs 規則功利主義、最大幸福原則 |
| 3. 義務倫理學（Deontology） | 康德（Kant）定言命令、完全義務 vs 不完全義務、人的尊嚴 |
| 4. 德性倫理學（Virtue Ethics） | 亞里士多德（Aristotle）、中道原則（Golden Mean）、習慣與品格培育 |
| 5. 關懷倫理學（Ethics of Care） | 吉利根（Gilligan）、關係責任、關懷的優先性 |

**必修範疇二：個人及社會議題（Personal and Social Issues）**

| 議題類別 | 常見考題範疇 |
|---------|------------|
| 生命倫理 | 安樂死（Euthanasia）、墮胎、死刑、人體器官捐贈 |
| 科技倫理 | 人工智能的道德責任、基因改造、核能 |
| 環境倫理 | 可持續發展、物種保護、氣候責任 |
| 社會公義 | 貧富懸殊、動物權利、全球化與勞工剝削 |

---

### 宗教部分（Religious Studies）

學生在 **模組一（佛教）** 或 **模組二（基督教）** 中**選擇一個**應考：

#### 模組一：佛教（Buddhism）

| 單元 | 主要內容 |
|------|----------|
| 佛教歷史 | 釋迦牟尼生平、佛教在亞洲的傳播（南傳 / 北傳 / 藏傳） |
| 基本教義 | 四聖諦（苦集滅道）、八正道、緣起論、無我論、業力與輪迴 |
| 宗教實踐 | 戒定慧三學、菩薩道、出家制度、佛教節慶與儀式 |
| 佛教在香港 | 香港佛教傳統、寺廟與修行團體、社會服務角色 |

> **考試語言**：佛教模組僅提供中文試卷（繁體中文）。

#### 模組二：基督教（Christianity）⭐ 開卷考試

| 單元 | 主要內容 |
|------|----------|
| 聖經與教會歷史 | 舊約 / 新約背景、早期教會、宗教改革 |
| 基督教核心教義 | 三位一體、原罪與救贖、道成肉身、復活與末世論 |
| 耶穌教導 | 登山寶訓、比喻（Parables）、十架神學 |
| 基督教實踐 | 禱告、禮拜、洗禮、聖餐、基督教倫理觀 |
| 基督教在香港 | 本地教會、基督教機構的社會服務 |

> **開卷安排**：基督教模組考生可攜帶**無注解或有注解的聖經（任何語言版本）**及**個人筆記**進場。惟筆記需為手寫或印刷，不可攜帶電子裝置。試題強調分析與論述，而非單純引用經文，開卷優勢在於核查原文引用，非「照抄筆記即可高分」。

> **考試語言**：基督教模組提供**英文及中文**兩種試卷，考生可選擇；如該年度沒有學校申請英文版，HKEAA 可能只提供中文版（2026 年已確認只有中文版本）。

根據 HKEAA 倫理與宗教科評核大綱：

| 評核組成部分 | 比重 | 時間 | 說明 |
|------------|------|------|------|
| **卷一（Paper 1）：倫理** | 50% | 約 2 小時 | 必答；涵蓋規範倫理學理論及應用倫理議題 |
| **卷二（Paper 2）：宗教傳統** | 35% | 約 1 小時 15 分 | 選擇佛教模組或基督教模組（開卷）其中一份作答 |

## 四、卷一（倫理）題型詳解

### 題型結構

**甲部（Section A）— 必答，共約 40 分**

- 短問題形式，測試考生對規範倫理理論的理解與辨析
- 常見題型：
  - 解釋功利主義如何回應「殺一救多」的道德困境
  - 康德的「定言命令」如何應用於某特定道德情境
  - 比較兩種倫理理論對同一議題的立場

**乙部（Section B）— 選答，共約 40 分**

- 長答題 / 論述題，需就特定個人或社會議題作出道德判斷
- 考生須清楚**表明立場**，並用**倫理理論支持論點**
- 常見議題：安樂死、死刑、人工智能道德責任、環境倫理

### 卷一必記「三理論框架」

**功利主義（Utilitarian approach）**
> "The greatest happiness for the greatest number"
- 判斷準則：**結果**（consequences）
- 關鍵概念：幸福（happiness / utility）、最大化（maximization）
- 優點：考慮社會整體利益；缺點：可能犧牲少數人的基本權利

**義務倫理學（Deontological approach）**
> 康德：人不應被作為手段，而應被視為目的本身
- 判斷準則：**行為本身的道德性**（intrinsic rightness/wrongness）
- 關鍵概念：定言命令（Categorical Imperative）、普遍性原則（Universalizability）
- 優點：尊重個人尊嚴；缺點：在複雜情境中可能過於僵化

**德性倫理學（Virtue Ethics approach）**
> 亞里士多德：美德透過習慣培養，幸福（Eudaimonia）是良好生命的目標
- 判斷準則：**行為者的品格**（character of the agent）
- 關鍵概念：德性（virtue）、中道（golden mean）、實踐智慧（phronesis）
- 優點：重視品格全面發展；缺點：難以提供具體行為指引

## 五、卷二（宗教傳統）策略分析

### 選擇模組的考量

| 考慮因素 | 佛教模組 | 基督教模組 |
|---------|---------|-----------|
| **開卷與否** | 否（閉卷） | ✅ 是（開卷） |
| **學校宗教背景** | 佛教或非宗教學校 | 基督教學校（有系統教授） |
| **語言選擇** | 中文（繁體）唯一選項 | 中文或英文（視乎當年安排） |
| **記憶負擔** | 較重（教義、歷史、術語全靠記憶） | 較輕（可帶聖經及筆記） |
| **準備難度** | 入門較難但概念有限 | 基督教學校生有優勢；非教徒需系統學習 |

**建議原則**：
- 就讀**基督教學校**且對聖經有基礎認識 → 強烈建議選**基督教模組**（開卷優勢顯著）
- 就讀**佛教或非宗教學校**且對佛教有一定認識 → 可考慮**佛教模組**，避免額外學習宗教背景
- 若**學校沒有系統教授任何一個模組** → 建議選基督教（開卷降低記憶壓力）

### 卷二答題關鍵

**佛教模組高分策略：**
1. 熟記核心術語中英對照（四聖諦 = Four Noble Truths；緣起 = Dependent Origination）
2. 分析題必須「先陳述教義，再解釋意義，後評價批判」三段結構
3. 引用具體佛教典故（如《法句經》、《心經》）增強論據說服力

**基督教模組開卷策略：**
1. 開卷**不等於不用準備**——試題考核分析能力，照抄聖經原文不得高分
2. 製作「主題索引」：將常考議題（如受苦問題、上帝存在、倫理立場）與相關聖經章節對應
3. 用**神學概念**（如 Grace、Redemption、Covenant）分析而非純粹故事復述
4. 時間管理：開卷考試考生常因翻查資料而超時，建議控制每題翻查不超過 5 分鐘

## 六、校本評核（SBA）要求

> 以下 SBA 要求適用於現行課程（2027 年 HKDSE 前）。2028 年起 SBA 將被取消。

### SBA 組成

E&RS 的 SBA 佔總評核 **15%**，由學生的本校 E&RS 老師評核，主要包括：

- **反思日誌（Reflective Journal）**：學生就課程中一個倫理議題或宗教主題進行個人反思，記錄思考過程
- **短篇報告 / 小組討論紀錄**：部分學校要求學生進行個案研究或小組倫理辯論，老師評核參與情況及深度

### SBA 評分重點

- **批判性反思能力**：能否識別議題中的道德衝突？能否從不同倫理理論角度分析？
- **宗教認識深度**：是否準確引用宗教教義？能否將教義聯繫日常生活？
- **個人立場清晰度**：表達個人道德立場時，是否有充分論證支持？

### 分級標準（Standards-referenced Reporting）

HKDSE E&RS 採用**標準參照評核**（SRR），共 7 級（Level 1 至 5**），各級有對應的表現標準描述。2024 年 HKDSE 各科普遍分佈：

| 等級 | 能力要求概述（E&RS 參考） |
|------|------------------------|
| **Level 5\*\*** | 倫理論述極具深度，能整合多種理論作精準分析；宗教知識廣博，批判性評價達學術水準 |
| **Level 5\*** | 論證結構嚴謹，能辨別不同倫理理論的優缺點；宗教分析能超越描述層次 |
| **Level 5** | 倫理分析全面，立場明確且有充分理論支持；宗教知識準確 |
| **Level 4** | 能正確應用主要倫理理論；宗教資料基本準確，具分析嘗試 |
| **Level 3** | 對倫理理論有基本認識；宗教知識尚算充足但分析較淺 |
| **Level 2** | 倫理及宗教知識部分正確，論述缺乏系統性 |
| **Level 1** | 基本知識理解不足，答題結構散亂 |

> **JUPAS 最低要求**：一般法律、社會工作、宗教研究等課程接受 E&RS 作為選修科；但大多數課程的 E&RS 計分與其他選修科無異。詳見各大學入學辦事處最新要求。

## 八、應試策略與備考計劃

### 八大高分原則

1. **論證結構先行**：每道論述題先用 30 秒確定立場，再列出兩至三個支持論點，最後作答。
2. **理論 → 應用 → 批評三步法**：陳述理論 → 應用至案例 → 指出理論局限，展示批判深度。
3. **避免「道德直覺」答案**：「這樣做是不對的，因為感覺不好」無法得高分；必須連繫倫理理論。
4. **宗教題目引用具體術語**：佛教題必用四聖諦、緣起等術語；基督教題必用恩典（Grace）、救贖（Redemption）等神學概念。
5. **開卷不等於低壓力**：基督教模組考生常因備考態度鬆懈而失分，開卷只是工具，分析能力才是核心。
6. **多做歷屆試題**：2014–2025 年的 Past Papers 全部公開（見 HKEAA 網站），練習答題節奏極為重要。
7. **注意時間分配**：卷一 2 小時需完成甲乙兩部，平均每分計 3 分鐘，切勿在單題超支。
8. **SBA 平均分全年拿**：SBA 佔 15%，反思日誌須認真對待，不可最後一刻草草了事。

### 兩年備考時間表

| 時期 | 重點任務 |
|------|---------|
| **中四上學期（Sep–Jan）** | 建立規範倫理學基礎：功利主義 → 義務倫理 → 德性倫理，每個理論至少能列舉兩個真實案例 |
| **中四下學期（Feb–Jul）** | 開始應用倫理議題（安樂死、死刑等），確定宗教模組選擇（佛教 vs 基督教） |
| **中五上學期（Sep–Jan）** | 深入學習宗教模組；開始撰寫 SBA 反思日誌初稿；做 2014–2019 年 Past Papers |
| **中五下學期（Feb–Jun）** | SBA 提交（通常 Mar–Apr）；完成模擬考試；檢視弱項議題 |
| **中六上學期（Sep–Jan）** | 系統複習：製作「理論 × 議題」矩陣表；精做 2020–2025 年 Past Papers |
| **中六下學期（Feb–Apr）** | 模擬考試計時練習；整理基督教開卷筆記索引（如選基督教模組）；調整答題節奏 |
| **考試前兩週** | 只做整卷計時練習，不再新學內容；確認開卷資料整理妥當 |

## 九、E&RS 選科常見誤解

### 誤解一：「E&RS 冇得讀，唔使溫書」

**事實**：E&RS 雖然沒有大量死記硬背的內容，但倫理論證的**深度**和**精準度**需要長期訓練。考生若以「最後衝刺」態度備考，難以在論述題取得 Level 4 以上成績。

### 誤解二：「基督教模組開卷，一定易過佛教模組」

**事實**：開卷考試的試題設計更側重**分析與評價**，而非知識複述。非基督徒學生若對聖經及基督教神學缺乏系統認識，即使可以翻書，在考試現場也難以有效引用及分析。

### 誤解三：「E&RS 只有宗教學校才讀」

**事實**：E&RS 是**所有中學**均可開辦的選修科，課程重心在**倫理學**（非宗教科）。部分世俗學校的學生選讀 E&RS 主要是因為倫理學的訓練對社會科學、法律或哲學學習有直接幫助。

### 誤解四：「E&RS 對升大學沒有幫助」

**事實**：以下大學課程可接受或受益於 E&RS 成績：

- **法律（LLB/JD）**：倫理推理訓練直接適用
- **社會工作（BSW）**：倫理決策在社工實踐中不可缺少
- **哲學（Philosophy）**：E&RS 是直接相關的前置訓練
- **護理（Nursing）**：醫療倫理是護理課程核心
- **宗教研究 / 神學（Religious Studies / Theology）**：直接對應

## 十、常見問題（FAQ）

**Q1：E&RS 是否只限基督教或佛教學校的學生報考？**
A：不是。E&RS 是向所有學校開放的甲類選修科。任何學生均可報考，宗教模組只是考試中的選擇題，與學校宗教背景無關。非宗教背景的學生一樣可以選讀。

**Q2：選了基督教模組，但自己不是基督徒，可以應考嗎？**
A：完全可以。HKEAA 的評核不涉及考生個人宗教信仰，只考核對基督教歷史、教義、實踐的理解及分析能力。許多高分考生本身並非基督徒。

**Q3：卷二基督教模組的「開卷」範圍是什麼？我能帶什麼入場？**
A：考生可攜帶聖經（任何語言版本，包括有注解版）及個人手寫或印刷的筆記。不可攜帶電子裝置。詳細規定每年以 HKEAA 考試程序手冊為準，建議考前核查最新版本。

**Q4：E&RS 的卷一和卷二分別需要多少時間應考？**
A：卷一（倫理）約 2 小時；卷二（宗教傳統）約 1 小時 15 分鐘。兩卷在**同一考試日**先後進行。

**Q5：SBA 如何評核？佔多少分？**
A：SBA 佔總評核 15%，由本校老師評核，主要形式為反思日誌或報告，學校間可能略有差異。2028 年課程改革後 SBA 將被取消，屆時公開試佔 100%。

**Q6：自修生（Private Candidate）可以報考 E&RS 嗎？SBA 怎辦？**
A：可以。自修生不需要參加 SBA，其 SBA 比例（15%）的標準將按 HKEAA 既定程序由公開試成績換算補回。詳情見 HKEAA 每年公布的「自修生報考須知」。

**Q7：E&RS 的歷屆試題（Past Papers）在哪裡取得？**
A：HKEAA 官方網站（www.hkeaa.edu.hk）→ HKDSE → 評核資訊 → 科目資料 → 倫理與宗教，可免費下載各年度試卷及考生表現示例。

**Q8：E&RS 能否計入 JUPAS 的「最佳六科」成績？**
A：可以。E&RS 屬甲類（Category A）選修科，與生物、化學、地理等科目同等地位，均可計入 JUPAS「最佳五科」或「最佳六科」計算（視乎大學計分方法）。但各院校對特定課程可能有指定選修科要求，須另行確認。

**Q9：2028 年後的新課程有什麼主要改變？**
A：教育局公布的修訂 E&RS 課程（2028 年 HKDSE 起適用）將取消 SBA（現佔 15%），評核由公開試（卷一 + 卷二）承擔 100%。課程宗旨及主要內容框架維持不變。詳情見 EDB 倫理與宗教科課程發展頁面。

**Q10：修讀 E&RS 對申請香港八大有幫助嗎？**
A：E&RS 對部分課程（法律、社工、護理、哲學、宗教研究）有直接相關性，申請時可在個人陳述（Personal Statement）中呈現 E&RS 訓練如何加強了批判思維及道德推理能力，有助展示學術準備度。
