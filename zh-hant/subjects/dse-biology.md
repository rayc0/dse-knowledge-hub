# DSE生物科完全攻略：課程大綱、溫習策略與答題技巧

DSE生物科是本港最受歡迎的選修理科之一，每年約有一萬名考生報考。有志修讀醫學、牙醫、藥劑、護理、生物醫學或生命科學的同學，幾乎都以生物科為必選。然而，生物科的考獲5級或以上的比率僅約15至18%，競爭相當激烈。

本文專為中四至中六的DSE生物科考生而寫，涵蓋課程架構、考試形式、校本評核（SBA）要求、常見失誤，以及實用的高分策略，助你有系統地備考。

---

## 課程架構概覽

DSE生物科分為**必修部分**及**選修部分**，必修部分佔考試內容的大部分比重。

### 必修部分（六大主題）

**1. 細胞與生命分子**
- 細胞結構與細胞器功能
- 細胞膜與物質運輸（擴散、滲透、主動運輸、胞吞胞吐）
- 酶的結構、功能及影響酶活性的因素
- 生物分子：碳水化合物、脂質、蛋白質、核酸
- 細胞分裂：有絲分裂與減數分裂
- DNA結構與複製

**2. 遺傳與進化**
- 單基因與雙基因雜交
- 伴性遺傳
- 基因表達（轉錄、翻譯）
- 基因突變及其影響
- 自然選擇與演化
- 物種形成與生物多樣性

**3. 生物與環境**
- 生態系統與能量流動
- 食物鏈、食物網、營養級
- 物質循環（碳循環、氮循環、水循環）
- 人類活動對環境的影響
- 生態保育
- 種群動態

**4. 人體生理：維持生命**
- 消化與營養
- 氣體交換與呼吸作用
- 血液與循環系統
- 排泄與腎臟功能
- 生殖（男女生殖系統、懷孕、避孕）
- 體內穩態（體溫調節、血糖調節、滲透調節）

**5. 健康與疾病**
- 營養與健康
- 傳染病（細菌、病毒、原蟲、真菌、蠕蟲）
- 免疫（先天免疫、適應性免疫、疫苗接種）
- 非傳染性疾病
- 藥物與酒精濫用

**6. 協調與反應（神經與激素）**
- 神經系統（中樞、外周、反射弧）
- 神經衝動與突觸
- 內分泌系統與主要激素
- 感覺器官（眼、耳、皮膚）

---

### 選修部分（四選二）

| 選項 | 主題 | 適合考生 |
|------|------|----------|
| 選項A | 人體生理：調節與控制 | 有意修讀醫科、護理的同學 |
| 選項B | 應用生態學 | 對環境科學有興趣的同學 |
| 選項C | 微生物與生物技術 | 有意修讀生物科技、藥劑的同學 |
| 選項D | 食物科學與技術 | 對食品工業或應用生物有興趣的同學 |

**選項A（人體生理：調節與控制）**最為普遍，與必修部分銜接緊密，亦是醫科及護理學系最常參考的內容。**選項C（微生物與生物技術）**涵蓋基因工程及發酵技術，近年出題比重上升，值得留意。

## 考試形式

| 考卷 | 形式 | 時間 | 佔分 |
|------|------|------|------|
| 卷一 | 多項選擇題（MC） | 1小時15分鐘 | 30% |
| 卷二（必修） | 結構題及論述題 | 1小時30分鐘 | 35% |
| 卷二（選修） | 結構題及論述題 | 1小時 | 20% |
| 校本評核（SBA） | 實驗報告與調查 | 全年進行 | 15% |

### 卷一：多項選擇題
- 共36題，每題均等計分
- 涵蓋必修部分全部範圍
- 部分題目要求閱讀實驗數據或圖表
- 常見陷阱：雙重否定句、「以下哪項正確/不正確」的混淆

### 卷二（必修）：結構題與論述題
- 分為甲部（所有人必答）和乙部（選答）
- 論述題要求完整解釋生理過程，必須用科學術語
- 圖表分析、實驗設計題比重逐年增加

### 卷二（選修）：選修單元考核
- 兩個選修單元各設相關題目
- 部分題目會將選修內容與必修主題結合考核

## 校本評核（SBA）

SBA佔總分15%，主要評核學生的**實驗技能**和**科學探究能力**。

**SBA常見實驗類型：**
- 酶活性實驗（如澱粉酶、過氧化氫酶）
- 光合作用與呼吸作用速率實驗
- 滲透作用實驗（如馬鈴薯實驗）
- 生態採樣（如樣方法、截線法）
- 食品中營養素的檢測

**SBA高分要訣：**
1. 實驗記錄須即場填寫，數據必須真實
2. 誤差分析要具體指出來源（如溫度波動、計時誤差），避免泛泛而談
3. 結論須與數據直接掛鉤，切忌「跳結論」
4. 改善建議要切實可行，例如「使用恆溫水浴」而非「做多次實驗」

## 歷屆出題趨勢分析

根據近五年的考題規律，以下主題的出題頻率特別高：

| 主題 | 近年出現頻率 | 備考重點 |
|------|------------|----------|
| 酶的作用機制 | 極高 | 誘導契合模型、底物濃度與速率關係 |
| DNA複製與蛋白質合成 | 極高 | 轉錄與翻譯步驟、密碼子對應 |
| 體內穩態（恆常性） | 高 | 血糖調節、體溫調節的負反饋機制 |
| 遺傳題（雜交分析） | 高 | 伴性遺傳、表現型與基因型比例計算 |
| 腎臟與排泄 | 高 | 各部位的滲透壓變化、重吸收機制 |
| 免疫系統 | 中高 | T細胞/B細胞角色、抗體產生過程 |
| 生態系統 | 中 | 能量流動效率計算、物種間關係 |
| 進化與自然選擇 | 中 | 工業黑化、抗藥性演化等應用例子 |

## 常見失誤與應對策略

### 失誤一：術語不精準

生物科要求使用**精確的科學術語**。例如：
- 不可說「細胞吸水」——應說「水分子經滲透作用進入細胞」
- 不可說「胰島素令血糖降低」——應說「胰島素促進細胞攝取葡萄糖，並促進肝細胞將葡萄糖轉化為糖原，從而降低血糖濃度」

**應對**：建立個人術語詞彙表，每個概念對應標準答案用語，反覆操練至自然運用。

### 失誤二：混淆有絲分裂與減數分裂

兩者的染色體行為、細胞數目、基因組成均不同，是歷年高頻考點。

| 比較項目 | 有絲分裂 | 減數分裂 |
|---------|---------|---------|
| 分裂次數 | 一次 | 兩次 |
| 子細胞數目 | 2個 | 4個 |
| 染色體數目 | 維持不變（2n→2n） | 減半（2n→n） |
| 基因組成 | 與母細胞相同 | 可能不同（交叉互換） |
| 發生位置 | 體細胞 | 生殖器官（精巢/卵巢） |

### 失誤三：論述題答案過於簡略

卷二論述題往往要求「解釋」或「描述」某生理過程，而非只是「列出」。考生常犯的錯誤是答案只有一兩句，未有完整描述因果關係。

**應對**：答論述題時，先在草稿紙寫出「步驟鏈」：觸發因素 → 感受器/訊號 → 效應器反應 → 結果，再整合成連貫段落。

### 失誤四：遺傳題計算錯誤

部分同學在「雙基因雜交」中忘記列出全部基因型組合，或在伴性遺傳題中混淆X染色體上基因的表示方式。

**應對**：所有遺傳題必須畫出龐雜特方格（Punnett Square），逐一填寫，切勿憑直覺估算。

### 失誤五：忽略SBA的備考

部分同學視SBA為「送分」，輕率應付，最終失去寶貴的15分。

**應對**：每次實驗前充分預習，了解實驗目的與預期結果；實驗後趁記憶清晰立即完成初步記錄。

## 分題溫習策略

### 第一階段：建立知識框架（距考試90天前）

- 以「主題」為單位，而非逐章閱讀
- 製作思維導圖：每個大主題對應一張A3紙
- 重點標記「機制類」知識（如滲透調節、血糖調節），因為此類知識在論述題中高頻出現
- 同時整理選修單元，與必修部分對應串連

### 第二階段：操練題型（距考試60至30天）

- 優先操練**歷屆卷二論述題**，對照評卷參考（Marking Scheme）找出自己的用語差距
- MC題目每周至少完成一份完整卷一，計時作答
- 針對薄弱環節：將同一主題的歷屆試題匯集成「專題練習」集中攻克
- 實驗數據題（圖表分析）需專項訓練：注意坐標軸單位、數據趨勢描述用語

### 第三階段：衝刺整合（距考試30天內）

- 每天以「1MC卷+1論述題」的迷你模擬形式維持手感
- 重溫術語詞彙表，確保每個術語在腦海中有完整的「標準答案句式」
- 模擬考試：完整作答歷屆試題，嚴格計時，檢查自己的答題節奏
- 最後一周：回顧錯題本，集中鞏固反覆出錯的知識點

## 選修單元選擇建議

| 志向 | 推薦選修組合 |
|------|------------|
| 醫科／牙醫／藥劑 | 選項A + 選項C |
| 護理／物理治療 | 選項A + 選項B 或 C |
| 生物科技／食品科學 | 選項C + 選項D |
| 環境科學／生態學 | 選項B + 選項C |
| 不確定志向 | 選項A + 選項C（最廣泛適用）|

選修選項A及C的考生人數最多，相關操練資源（補充題、模擬卷）亦最豐富，建議優先考慮。

## 重點總結

1. **必修部分為核心**：六大主題中，「細胞與生命分子」、「遺傳與進化」及「人體生理」歷來是出題重鎮，務必深入掌握。

2. **術語準確是關鍵**：論述題的評分標準極為嚴格，欠缺關鍵術語可能失去多個分數，建議建立個人詞彙表並定期溫習。

3. **SBA不可輕視**：15%的分數對整體成績影響顯著，認真對待每一次實驗。

4. **選修選項A+C最通用**：適合大多數志向理科或醫療相關學系的考生。

5. **答題要有層次**：論述題從「觸發→反應→結果」的邏輯鏈著手，避免答案零散或欠缺因果關係。

6. **及早操練歷屆卷**：生物科的高分不靠背誦，而靠準確理解後的準確表達，只有大量操練並對照評卷參考，才能找到評分員期望的答案模式。
