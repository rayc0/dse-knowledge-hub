# DSE視覺藝術科完全攻略：校本評核、試卷結構與備試策略

DSE視覺藝術科是香港會考選修科中最具個人特色的科目之一。與大多數科目不同，視覺藝術以**校本評核（SBA）為主**，公開考試比重較低，但兩者缺一不可。本文全面解析視覺藝術科的評核結構、備試策略和升學出路。

---

## 一、科目概覽

| 項目 | 詳情 |
|---|---|
| 考生人數（約） | 5,000-7,000人/年 |
| 科目性質 | 選修科（甲類） |
| 評核語言 | 中文或英文（考生自選） |
| JUPAS相關 | 視覺藝術、設計、建築、多媒體等系別主要參考 |
| 最高等級 | 5**（與其他科目相同） |

---

## 二、評核結構

DSE視覺藝術科分兩個評核部分：

### 組別一：藝術製作（60%）— 校本評核

- **個人研習**（Independent Study）：學生在中四至中六期間完成的藝術作品集
- 需製作**主題陳述（Artist Statement）**，說明創作意圖、媒介選擇和藝術語言
- 工作紀錄（Process Journal）：展示從概念、實驗到完成的創作過程
- 展覽形式：學校內部評審＋香港考試及評核局抽樣覆核
- **評分重點**：創意與原創性（35%）、技術運用（25%）、概念探索深度（40%）

### 組別二：藝術鑑賞與批評（40%）— 公開考試

- **卷一（短題目）**：分析提供的藝術作品，涵蓋描述、分析、詮釋、評價（DAIE框架）
- **卷二（延伸回應）**：就指定論題撰寫較長篇幅的藝術評論或比較分析
- 時間：約2.5小時

## 三、藝術製作（SBA）策略

### 主題選擇的重要性

主題（Theme）是整個個人研習的核心。一個好主題應具備：
- **個人相關性**：與考生個人經歷、文化背景或興趣連結
- **視覺可探索性**：能透過不同媒介、技法和概念角度深化
- **跨文化連結**：參照本地和國際藝術家的處理方式

**常見主題選擇方向**：身份認同、城市景觀、記憶與空間、生態環境、家庭關係、流行文化批判

### 媒介選擇建議

| 媒介 | 適合類型 | 備考難點 |
|---|---|---|
| 油畫/壓克力 | 技術成熟、渴望豐富色彩表現 | 需較長製作時間，存放空間 |
| 版畫（蝕刻/石版） | 注重過程和系列感 | 學校設備依賴度高 |
| 數碼媒體 | 有設計或攝影背景 | 需說明「藝術性」而非純設計 |
| 裝置藝術 | 概念強烈、喜愛實驗 | 展覽和文件記錄要求高 |
| 混合媒材 | 表達複雜概念 | 需主題一致性，避免雜亂 |
| 素描/水彩 | 傳統技法紮實 | 在高分等級需展示更高原創性 |

### 工作紀錄（Process Journal）技巧

- 紀錄每次創作決定的**原因**，而非只記「今日做了甚麼」
- 定期引用藝術家（至少2-3位），說明如何受其啟發及如何超越
- 用文字、草圖、照片、試驗記錄呈現思考過程
- 保持與主題的一致性：每件作品都應指向同一核心探問

## 四、藝術鑑賞（卷二）答題技巧

### DAIE分析框架

| 層次 | 內容 |
|---|---|
| 描述（Describe） | 客觀描述作品的視覺元素（顏色、線條、形狀、空間、質感） |
| 分析（Analyse） | 解讀各元素如何相互作用，形成整體視覺效果 |
| 詮釋（Interpret） | 推論藝術家的意圖、主題意涵和文化背景 |
| 評價（Evaluate） | 判斷作品的成效，提供個人觀點並作論証 |

**評卷重點**：必須使用正確的視覺藝術術語（如：構圖、肌理、色調對比、負空間），且論點需有具體視覺證據支持。

### 常見失分原因

| 問題 | 修正建議 |
|---|---|
| 只描述不分析（停留在D層） | 練習每個描述後加「這造成…效果」 |
| 術語混用或使用不當 | 建立個人術語詞彙表，考前複習 |
| 評價層空洞（「我覺得好靚」） | 以具體視覺元素支撐評價判斷 |
| 比較題中兩件作品各自獨立分析 | 明確建立比較框架，交替對比同一面向 |

## 五、升學出路

視覺藝術科的良好成績（Level 4以上）主要配合：

- **本地大學藝術及設計學系**：香港藝術學院、理工大學設計學院、城市大學創意媒體等
- **英美藝術院校**：Foundation Year申請多以個人作品集（Portfolio）為主，DSE成績為次
- **建築及室內設計**：需配合數學和英文成績

**作品集的重要性**：無論升讀本地還是海外設計/藝術院校，個人作品集幾乎都比DSE分數更重要。應在中六期間整理完整的Portfolio。

## 總結

DSE視覺藝術科的成功關鍵在於：**SBA要持續累積、概念要深入一致、鑑賞要掌握框架**。由於SBA佔60%，中四及中五的日常創作進度比臨考衝刺更重要。建議考生從中四起就有意識地以個人研習主題為中心進行創作，同步建立批判性鑑賞的語言能力。
