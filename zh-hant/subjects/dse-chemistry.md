# DSE化學科完全攻略：課程大綱、計算題技巧與實驗要點

> 摘要 — DSE化學科（HKDSE Chemistry）是香港中學文憑試其中一科最受理科生重視的選修科目，亦是入讀醫學、藥學、化工、環境科學等學系的必要條件。本文全面拆解化學科的課程架構、必修與選修單元、卷一卷二考試格式、高頻計算題解題步驟、實驗技巧及高效溫習策略，助考生備戰取得佳績。

---

## 一、課程架構概覽

DSE化學科由香港考試及評核局（HKEAA）管理，課程分為**必修部分**（Compulsory Part）及**選修部分**（Elective Part）。

### 必修部分（共12個單元）

| 單元編號 | 單元名稱 | 主要概念 |
|---|---|---|
| M1 | 基本化學概念 | 粒子、元素、化合物、混合物 |
| M2 | 原子結構及化學鍵 | 電子排布、離子鍵、共價鍵、金屬鍵 |
| M3 | 化學計量學 | 摩爾、相對原子質量、化學方程式計算 |
| M4 | 酸鹼化學 | 酸鹼定義、pH、中和反應、滴定 |
| M5 | 氧化還原反應 | 氧化數、半方程式、電化學電池 |
| M6 | 化學反應速率與化學平衡 | 碰撞理論、勒沙特列原理、平衡常數 |
| M7 | 化學熱力學 | 焓變、Hess定律、燃燒熱 |
| M8 | 週期表及元素週期律 | 電負性、原子半徑、電離能趨勢 |
| M9 | 金屬及其化合物 | 金屬活性、萃取方法、過渡金屬 |
| M10 | 有機化學一 | 碳氫化合物、官能團、異構體 |
| M11 | 有機化學二 | 酯化、加聚、縮聚、有機合成 |
| M12 | 環境化學 | 大氣污染、水質、溫室效應 |

### 選修部分（揀選一個模組，共三個）

| 選修模組 | 涵蓋內容 | 適合志向 |
|---|---|---|
| **模組一：工業化學** | 工業製氨（Haber法）、硫酸製造、電解工業 | 工程、化工 |
| **模組二：材料化學** | 聚合物、陶瓷、複合材料、奈米材料 | 材料科學、應用科技 |
| **模組三：醫藥化學** | 藥物分類、藥物作用機制、手性分子 | 醫科、藥學 |

**選模組三的考生特別注意**：模組三與醫藥學系入學相關性最高，但考試深度亦最大，需多花時間在手性（chirality）及藥物合成路線上。

## 二、考試格式詳解

### 試卷結構

| 試卷 | 性質 | 時限 | 佔分比例 |
|---|---|---|---|
| **卷一（Paper 1）** | 多項選擇題（36題） | 1小時15分鐘 | 36% |
| **卷二（Paper 2）** | 結構式短答題及延伸題 | 2小時15分鐘 | 64% |

### 卷二分題分佈

卷二分為甲部（必修）及乙部（選修）：
- **甲部**：共約100分，涵蓋全部必修單元，包括計算題、解釋題及實驗設計題
- **乙部**：約50分，考核所選模組的深入理解，常見跨單元整合題

> 實戰貼士：甲部計算題往往連環，先確認第一步正確再往下做；乙部解釋題需使用正確化學術語，避免口語化描述。

## 三、必考計算題類型及解題步驟

計算題是DSE化學科的重要失分位，以下列出五大必考題型的解題框架。

### 3.1 摩爾計算（Mole Calculation）

**常見題型**：給予質量或體積，計算反應物/生成物的量

**解題步驟**：
1. 列出平衡化學方程式
2. 確認已知量（質量 → 摩爾數：n = m/M；體積（氣體）→ 摩爾數：n = V/24dm³ at r.t.p.）
3. 按化學計量比找出目標物質的摩爾數
4. 換算成所需單位（質量、體積、濃度）

**例題**：4.8g 鎂與足量鹽酸反應，求生成氫氣的體積（r.t.p.）。
- n(Mg) = 4.8 ÷ 24 = 0.2 mol
- Mg + 2HCl → MgCl₂ + H₂，故 n(H₂) = 0.2 mol
- V(H₂) = 0.2 × 24 = **4.8 dm³**

---

### 3.2 酸鹼滴定計算（Acid-Base Titration）

**關鍵公式**：n = C × V（C為濃度mol dm⁻³，V為體積dm³）

**解題步驟**：
1. 確認酸鹼中和方程式及計量比
2. 計算已知溶液的摩爾數
3. 利用計量比求未知溶液的摩爾數
4. 用 C = n/V 求濃度

**常見陷阱**：
- 體積單位需從cm³換算至dm³（÷1000）
- 多元酸（如H₂SO₄）的計量比為1:2

---

### 3.3 化學平衡計算（Equilibrium Calculation）

**ICE表格法**（Initial-Change-Equilibrium）是標準解法：

```
         A   +   2B   ⇌   C
初始(I)   a       b        0
變化(C)  -x      -2x      +x
平衡(E)  a-x    b-2x       x
```

再將平衡數值代入 Kc = [C] / ([A][B]²) 求解。

---

### 3.4 Hess定律計算（Hess's Law）

**核心原則**：反應的焓變只與起始狀態和最終狀態有關，與途徑無關。

**解題技巧**：
- 將目標方程式拆解成已知反應的組合
- 反轉方程式時，ΔH取負值
- 乘以係數時，ΔH亦乘以相同倍數

---

### 3.5 電化學計算（Electrochemistry）

**氧化還原半方程式配平步驟**：
1. 確認氧化/還原物質
2. 配平原子數（先非O/H元素，再O用H₂O補，再H用H⁺補）
3. 配平電荷（加e⁻）
4. 合併兩個半方程式，消去e⁻

## 四、實驗技巧與要點

DSE化學科設有**學校本位評核（SBA）**，佔整體評分的20%。熟悉實驗操作及考試中的「實驗設計題」同樣關鍵。

### 4.1 基本實驗操作守則

| 操作 | 正確做法 | 常見錯誤 |
|---|---|---|
| 量取液體體積 | 使用量筒，視線與液面凹面齊平 | 視線偏高或偏低導致誤差 |
| 滴定操作 | 滴定前排走滴定管氣泡，記錄初始讀數 | 忘記記錄初始讀數 |
| 加熱試管 | 試管口朝向無人方向，不可直接對準底部 | 試管口朝向人，液體噴濺 |
| 氣體收集 | 根據氣體密度選擇向上/向下排氣法或排水法 | 用錯收集方法導致氣體外洩 |
| 過濾 | 濾紙貼緊漏斗，玻璃棒輕引液體 | 濾紙破損，沉澱穿過 |

### 4.2 高頻實驗考核主題

- **酸鹼滴定**：指示劑選擇（強酸強鹼用酚酞或甲基橙，弱酸強鹼用酚酞）
- **電解實驗**：陰極/陽極反應，電解產物預測
- **化學反應速率**：控制變因、收集氣體計時法
- **有機化學實驗**：酯化反應裝置、蒸餾分離、紙上層析

### 4.3 實驗設計題應對策略

考試中「設計實驗」題的標準作答框架：
1. **儀器及試劑**（列明所有所需物品）
2. **實驗步驟**（逐點列出，使用「加入」「混合」「加熱」等操作動詞）
3. **觀察及記錄**（預期現象）
4. **計算及結論**（如何利用數據得出結論）
5. **安全措施**（如有毒氣體需在通風櫥進行）

## 五、必修單元深度提示

### 有機化學（M10、M11）——高分重災區

有機化學佔卷二約30%分數，是拉開成績的關鍵單元。

**官能團與反應類型對應表**：

| 官能團 | 化合物類別 | 主要反應類型 |
|---|---|---|
| —OH（醇） | 醇類 | 氧化、酯化、消去 |
| —COOH（羧基） | 羧酸 | 酯化、中和 |
| —CHO（醛基） | 醛類 | 氧化（銀鏡反應、斐林試劑） |
| C=C（碳碳雙鍵） | 烯烴 | 加成、加聚 |
| —NH₂（胺基） | 胺類 | 縮聚（形成尼龍等） |
| —COO—（酯鍵） | 酯類 | 水解（酸性/鹼性） |

**常考有機合成路線**：由醇到酯的五步合成，考生需熟悉每步所用試劑及條件（如酸化催化劑、加熱溫度）。

---

### 化學反應速率與化學平衡（M6）——概念題重點

**勒沙特列原理（Le Chatelier's Principle）**是核心考點，題目常問「改變某條件後，平衡如何移動，各物質濃度如何變化」。

應對方法：
- 增加反應物濃度 → 平衡右移
- 增加壓力（對有氣體的反應）→ 平衡向氣體摩爾數較少的一方移動
- 升溫 → 平衡向吸熱方向移動
- 加入催化劑 → **平衡位置不變**，只加快達到平衡的速率

## 六、溫習策略與時間分配

### 各單元建議時間比重

| 單元類別 | 建議時間比重 | 原因 |
|---|---|---|
| 有機化學（M10、M11） | 25–30% | 題目多，失分空間大 |
| 計算題（M3、M4、M6、M7） | 25% | 計算題步驟分，需反覆練習 |
| 電化學（M5） | 15% | 半方程式配平是固定題型 |
| 選修模組 | 20% | 佔卷二約1/3分數 |
| 週期表及環境化學 | 10% | 記憶性內容，性價比高 |

### 複習三層次法

**第一層：建立概念圖**
每個單元畫出概念圖（Mind Map），把核心概念、公式、反應條件整理在一張紙上，便於快速回顧。

**第二層：計算題專項訓練**
每天做5至10條計算題，不同題型輪換。重點培養「閱題→辨別題型→套用框架→核對單位」的作答習慣。

**第三層：歷屆試卷模擬**
最後一個月每周完成至少一套完整試卷（卷一＋卷二），嚴格計時，訓練時間管理。完成後對答案時，重點分析失分原因（概念不清？計算失誤？表達不足？）。

### 答卷技巧

- **計算題**：必須展示清晰步驟，即使最終答案錯誤，中間步驟分仍可保留
- **解釋題**：使用化學術語，避免口語，如「分子間氫鍵」不要寫「分子之間有吸引力」
- **化學方程式**：必須配平，記得標明狀態符號（s/l/g/aq）及反應條件（△、催化劑等）
- **多項選擇題**：先排除明顯錯誤選項，再利用計算或推理確認答案

## 重點總結

DSE化學科的高分策略可歸納為以下六點：

1. **課程架構要清晰**：區分必修（12單元）與選修（三個模組），選對選修模組配合志向
2. **計算題要系統化**：摩爾計算、滴定、平衡、Hess定律、電化學五大題型各有固定框架，反覆練習至自動化
3. **有機化學不能放棄**：佔分最重，官能團反應、合成路線需熟記，是拉開成績的關鍵
4. **實驗技能要到位**：SBA佔20%，考試中的實驗設計題亦需用標準格式作答
5. **概念解釋要精準**：使用正確化學術語，勒沙特列原理、碰撞理論等需用專業語言表達
6. **歷屆試題是最佳教材**：HKEAA官網提供歷屆試卷及評卷參考，是備考最重要的資源
