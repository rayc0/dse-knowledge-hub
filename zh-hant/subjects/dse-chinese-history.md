# DSE中國歷史科完全攻略：卷一卷二結構、時代分析與高分答題策略

DSE中國歷史科（Chinese History，簡稱「中史」）是個人、社會及人文教育（PSHE）領域的選修科目，每年報考人數約 **15,000 至 18,000 人**，在所有選修科中屬報考規模較大的科目之一。中史科難度定位為「中等偏難」——涵蓋的時代跨度由先秦至當代中國，史實量龐大，但考試亦高度重視史料分析與論述能力，並非純粹的死記硬背。

---

## 一、科目概覽

### 中史科與歷史科的分別

許多同學混淆「中國歷史科」與「歷史科」，兩者是**完全獨立的選修科目**，課程內容、試卷結構和評分準則均不同。

| 比較項目 | 中國歷史科（Chinese History） | 歷史科（History） |
|----------|-------------------------------|-------------------|
| 課程範圍 | 中國文明史（先秦至近現代中國） | 近代世界史及香港史（19至20世紀） |
| 授課語言 | 主要以中文授課及應考 | 主要以英文或中文授課及應考 |
| 每年報考人數 | 約 15,000–18,000 人 | 約 8,000–10,000 人 |
| 選修時代/單元 | 三個時代（由考生在卷二選答） | 選修世界歷史單元（由學校決定） |
| 難度定位 | 中等偏難（時代跨度大、政治文化內容並重） | 中等偏難（分析及論文能力要求高） |
| 適合升學方向 | 歷史、中文、人文、教育、文化研究 | 歷史、法律、社會科學、國際研究 |

| 中史科基本資料 | 詳情 |
|----------------|------|
| 學習領域 | 個人、社會及人文教育（PSHE） |
| 校本評核（SBA） | 已取消，成績 100% 來自公開考試 |
| 5級門檻（歷年估算） | 約 60–63% 總分 |
| 5**門檻（歷年估算） | 約 70–73% 總分 |
| 相關升學課程 | 中國研究、歷史、中文、教育、人文社科 |

## 二、試卷結構

中史科分為**卷一**與**卷二**，兩卷均在同一天應考，合共考試時間約 **3.5 小時**。

### 卷一 — 必修部分：中國歷史發展（佔 50%）

卷一為**資料題卷**，時間 **2 小時**，佔總分 **50%**。

試卷圍繞三大必修時期命題，每個時期設一組試題，考生**須全部作答**：

| 必修時期 | 涵蓋朝代 / 課題 |
|----------|----------------|
| 上古至唐代 | 先秦諸子、秦漢統一、三國魏晉、隋唐盛世、對外交流與文化發展 |
| 宋代至清代中期 | 宋代政制、元代統治、明清政策、科舉制度、中外接觸 |
| 清末至現代中國 | 列強入侵、晚清改革、辛亥革命、五四運動、民國政局、中共建政、改革開放 |

**題型分布**：

- **甲部 — 史料分析題**：根據提供的文字或圖像史料，回答描述、解釋及評估問題
- **乙部 — 短論題**：就指定課題撰寫有組織的短答論述，需結合課程知識
- 每組題目通常包含 **4–6 個小題**，總分約 **25–30 分**

### 卷二 — 選修部分：三個時代論述題（佔 50%）

卷二為**論述題卷**，時間 **1.5 小時**，佔總分 **50%**。

試卷設**三個選修時代**，每個時代各有 **2 條題目**，考生從**三個時代中各選一條**，共**作答三條**題目。

每條論述題分值相近，考生須在限時內針對各時代課題提出論點、引用史實並加以分析評估。

## 三、選修時代策略

卷二的選修時代由**考生自行決定**（非由學校指定），是中史科備考的重要策略決定。考生須從多個可選時代中，選定三個在課內深入學習並在考試時作答的時代。

以下為常見選修時代的比較：

| 選修時代 | 主要課題 | 優點 | 缺點 | 適合學生類型 |
|----------|----------|------|------|-------------|
| **秦漢時代** | 秦始皇統一、漢代政制、絲路與對外關係、儒學確立 | 史料豐富，與必修內容呼應度高 | 政治制度術語較多，需深入理解 | 對制度史有興趣者 |
| **隋唐時代** | 科舉制度、唐太宗貞觀之治、武則天、安史之亂、中外文化交流 | 故事性強，事件脈絡清晰，論題較易組織 | 課題數量多，備考範圍較廣 | 喜歡敘事性強課題者 |
| **宋元時代** | 宋代政制改革、王安石變法、元代統治方式、宋元對外貿易 | 比較分析空間大（宋與元的對比） | 元代史料相對較少，答題難度偏高 | 擅長比較論述者 |
| **明清時代** | 明代政制、清朝滿漢關係、閉關鎖國、鴉片戰爭前夕 | 與必修清末課題形成連貫，知識協同效益高 | 內容多與其他必修時期重疊，需分清深度 | 已對清史有基礎者 |
| **近現代中國** | 晚清改革、辛亥革命、五四運動、國共內戰、中共建國、文化大革命 | 論題貼近現代，考生較有既有認識 | 政治敏感度高，評估要求客觀平衡 | 對近代史有興趣者 |

**三時代選擇建議**：

1. **避免三個時代集中同一段歷史**（如全選近現代）——若全部押注同一段歷史，一旦題目角度偏難即難以補救
2. **至少一個時代與卷一必修內容有知識重疊**——可減少備考總量
3. **優先選擇學校老師教學最詳盡的時代**——教師備課質量直接影響應試深度

## 四、核心答題技巧

### 卷一甲部：史料分析題答題要素

中史科史料題與歷史科（History）的 OPCVL 框架有所不同，以下是中史科答題的核心要素：

| 題型 | 答題重點 | 常見失分原因 |
|------|----------|-------------|
| **描述 / 概括題** | 提取史料關鍵信息，以自己語言重組，切勿逐字抄錄 | 全文照抄史料原文，欠缺整合組織 |
| **解釋 / 分析題** | 引用史料內容＋補充課程背景知識，說明成因或影響 | 只引史料，缺乏課程知識補充；或只靠課程知識，忽略史料 |
| **史料可信性評估** | 指出作者身份、立場、史料性質（一手/二手）；說明偏頗原因及史料限制 | 只說「可信」或「不可信」，未提供具體理由 |
| **史料比較題** | 指出兩份史料的相同點與不同點，分析成因（立場、時代、目的不同） | 只羅列比較點，未分析為何存在差異 |

### 卷二：論述題答題框架

中史科論述題建議採用以下結構：

```
【引言】
- 簡述時代背景（朝代、關鍵人物/事件、時間範圍）
- 提出清晰論點（Thesis）：直接回應題目關鍵詞
- 預告主要論據方向（2–3個）

【正文各段】
- 主題句：點明本段論點（須呼應論題）
- 史實引用：具體人物、政策、事件、年份
- 分析：說明史實如何支持論點
- 過渡句：連結下一段論點

【結論】
- 重申論點（勿與引言重複）
- 評估各因素的相對重要性
- 提出歷史意義或當代啟示（可選）
```

**論題關鍵詞識別**：
- 「試評述……」→ 須有正反兩面評估，不能只講優點或只講缺點
- 「你是否同意……」→ 須明確表態（同意/不同意/部分同意），並為立場提供論據
- 「分析……的原因」→ 須分類組織（政治、經濟、社會等面向），避免羅列式答法

## 五、常見失分原因及修正

| 失分原因 | 具體表現 | 修正方法 |
|----------|----------|----------|
| 抄錄史料原文 | 卷一甲部直接複製史料文字，未作整合 | 以自己語言重組，並說明信息的意思或重要性 |
| 論述缺乏論點 | 只逐點羅列史實，全文無貫穿性立場 | 引言必須有明確 Thesis，每段有主題句呼應論題 |
| 史實錯漏或混淆 | 朝代、年份、人物名稱錯誤（如混淆宋太祖/太宗） | 製作朝代時序表，定期核對關鍵人物事件 |
| 時間分配失當 | 卷一個別小題耗時過長，卷二論述題倉促 | 按配分估算時間，每 1 分鐘約 1–1.5 分鐘作答 |
| 評估欠平衡 | 「試評述」題只寫正面，未提反面或局限 | 設立「相反論點段」，提出反面論據後再鞏固立場 |
| 卷二時代知識不足 | 選了備考不充分的時代，臨場無法展開論述 | 每個選定時代至少練習 3–4 條歷屆真題論述 |

## 重點總結

DSE中國歷史科的高分關鍵在於三項核心能力：

**1. 史料分析與整合能力**：卷一佔 50%，考生須能有效提取史料信息、評估史料可信性，並結合課程背景知識補充分析，切忌單純抄錄。

**2. 論述組織與論點構建**：卷二要求在限時內針對三個時代各寫出具結構的論述，清晰論點配合具體史實是得分關鍵，純粹羅列史實難以達到 5 級以上。

**3. 選修時代的策略選擇**：善用三個時代的選擇空間，選擇知識最充分、老師最詳盡備課的時代，並確保三個時代的知識深度均衡，避免臨場知識不足。

掌握以上三項能力，配合系統操練歷屆試卷並仔細研讀評卷參考，取得 5 級或以上是可以實現的目標。
