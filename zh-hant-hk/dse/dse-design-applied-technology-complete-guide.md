# DSE 設計與應用科技（DAT）完全攻略 2026

設計與應用科技（Design and Applied Technology，DAT）係DSE選修科中最能結合創意思維與實用技能的一科。DAT涵蓋設計理論、實際製作及作品集呈現，對有志投身設計、建築、工業及創意產業的同學而言，係展示跨界能力的理想科目。

---

## DAT 考試結構總覽

| 評核部分 | 考核內容 | 佔分比例 |
|----------|----------|----------|
| 卷一（筆試） | 設計理論（Design Theory） | 35% |
| 校本評核一（SBA） | 設計作品集（Design Portfolio） | 30% |
| 校本評核二（SBA） | 實作項目（Practical Project） | 35% |

> **注意**：DAT的SBA佔分高達65%，意味著全年的設計工作比公開筆試更為重要。同學必須全年持續投入，不能只靠臨考衝刺。

---

## 卷一：設計理論（35%）

### 考試形式

卷一為2小時的公開筆試，分為兩部分：

| 部分 | 題型 | 佔卷一比例 |
|------|------|-----------|
| 甲部 | 結構性問題（選答） | 約50% |
| 乙部 | 論述題／設計案例分析 | 約50% |

### 設計理論核心考核內容

#### 設計過程（Design Process）

設計理論的核心，考核同學對完整設計流程的理解：

```
需求分析 → 設計簡介 → 調研分析 → 概念構思 → 設計發展 → 製作測試 → 評估改良
```

- **設計簡介（Design Brief）**：清晰界定設計目標、目標用戶、限制條件
- **調研方法**：問卷調查、訪問、市場分析、參考案例
- **概念草圖（Concept Sketches）**：多元創意方案的快速呈現
- **設計評估**：功能性、美觀性、可持續性、成本效益

#### 設計原則與元素

| 設計元素 | 設計原則 |
|----------|----------|
| 線條（Line） | 對比（Contrast） |
| 形狀（Shape） | 平衡（Balance） |
| 色彩（Colour） | 比例（Proportion） |
| 質感（Texture） | 節奏（Rhythm） |
| 空間（Space） | 統一（Unity） |
| 形態（Form） | 強調（Emphasis） |

#### 物料與製造工藝

- **物料類型**：金屬、木材、塑膠、紡織品、複合材料
- **物料特性**：硬度、韌性、延展性、導電性、可持續性
- **製造工藝**：切割、成型、接合、表面處理
- **製造工序**：批量生產、定制生產、自動化生產

#### 可持續設計（Sustainable Design）

- **3R原則**：Reduce（減少）、Reuse（重用）、Recycle（回收）
- **生命週期評估（LCA）**：從生產到棄置的環境影響
- **生態設計**：減少廢物、採用可再生物料、節能設計

#### 人體工學（Ergonomics）

- 設計如何配合人體尺寸、動作及感知需求
- 用戶友好設計（User-Friendly Design）
- 通用設計（Universal Design）：適合不同能力人士

#### 知識產權與設計倫理

- 版權、專利、商標的基本概念
- 抄襲與原創的界線
- 設計師的社會責任

## 四大選修範疇

同學需選擇其中一個選修範疇作為SBA的重點方向：

### 1. 產品設計（Product Design）

| 項目 | 內容 |
|------|------|
| 核心技能 | 三維設計、物料應用、功能性製作、原型製作 |
| 常見項目 | 家具設計、家用電器、兒童玩具、輔助工具 |
| 製作工具 | CAD軟件（SolidWorks/Fusion 360）、3D打印、木工/金工工具 |
| 評核重點 | 功能性、使用者體驗、物料選擇合理性 |

### 2. 傳意設計（Communications Design）

| 項目 | 內容 |
|------|------|
| 核心技能 | 平面設計、排版、品牌設計、視覺傳達 |
| 常見項目 | 海報設計、品牌識別系統、包裝設計、書籍設計 |
| 製作工具 | Adobe Illustrator、Photoshop、InDesign |
| 評核重點 | 視覺效果、信息傳達清晰度、設計一致性 |

### 3. 數碼媒體設計（Digital Media Design）

| 項目 | 內容 |
|------|------|
| 核心技能 | 動態圖像、互動設計、影片製作、網頁設計 |
| 常見項目 | 動畫短片、互動裝置、手機App介面、網站設計 |
| 製作工具 | Adobe After Effects、Premiere、Figma、Unity |
| 評核重點 | 用戶體驗、互動邏輯、技術執行力 |

### 4. 時裝及紡織品設計（Fashion and Textiles Design）

| 項目 | 內容 |
|------|------|
| 核心技能 | 服裝設計、紡織技術、縫製、時裝展示 |
| 常見項目 | 服裝系列設計、配件設計、可持續時裝 |
| 製作工具 | 縫紉機、染色技術、CAD時裝設計軟件 |
| 評核重點 | 創意概念、製作技術、穿著功能性 |

## 設計作品集（Design Portfolio）SBA：30%

### 作品集評分準則

設計作品集記錄整個設計過程，評核以下範疇：

| 評核範疇 | 具體要求 |
|----------|----------|
| 設計過程呈現 | 完整呈現從調研到完成品的全過程 |
| 創意思維 | 多元化的概念方案，展示創造力 |
| 技術能力 | 繪圖技巧、軟件應用、製作技術 |
| 設計溝通 | 清晰有效地傳達設計意念 |
| 反思與評估 | 對設計決策的反思，以及改良建議 |

### 作品集製作建議

1. **記錄每個設計決定**：為何選擇這個方案而非另一個？評分員希望看到你的思維過程
2. **視覺呈現多元化**：手繪草圖＋數碼效果圖＋實物照片並用
3. **清晰標注文字說明**：圖像旁要有足夠的文字解釋
4. **保持整潔有序**：版面設計本身就是評分的一部分
5. **定期備份**：數碼文件必須有備份，避免突發事故

## 實作項目（Practical Project）SBA：35%

### 實作項目要求

實作項目是DAT最核心的評核部分，同學需要完成一件實際的設計作品：

| 評核範疇 | 佔實作項目比例 |
|----------|--------------|
| 作品質量（完成度、技術執行） | 約40% |
| 設計創意（獨特性、解決問題能力） | 約30% |
| 物料與工藝 | 約20% |
| 可持續性考慮 | 約10% |

### 實作項目成功要素

1. **選題要實際可行**：不要選擇超出自己製作能力的項目
2. **提早開始製作**：留足夠時間修改和完善
3. **物料選擇要有理由**：能解釋為何選用這種物料
4. **原型測試**：在最終製作前先製作原型，測試並改良
5. **最終呈現要完整**：作品必須完成，不能只是半製成品

## DAT vs 視覺藝術科（VA）比較

| 比較項目 | 設計與應用科技（DAT） | 視覺藝術（VA） |
|----------|---------------------|--------------|
| 核心方向 | 解決問題、功能性設計、應用科技 | 藝術表達、創意、美學探索 |
| 製作成果 | 實用設計品、數碼作品 | 純藝術作品（繪畫、雕塑等） |
| 理論考核 | 設計理論、物料科技、設計過程 | 藝術史、藝術評賞 |
| SBA比重 | 65% | 60%+ |
| 技術要求 | 製作技術、CAD軟件、工具應用 | 藝術媒介（油彩、水彩、雕塑） |
| 選修人數 | 約3,000–4,000人/年 | 約5,000–6,000人/年 |
| Level 5** 比例 | 約3–5% | 約2–4% |
| 大學認可 | 設計、工程、建築相關課程 | 美術、藝術相關課程 |
| 適合人 | 有設計思維、喜歡解決問題 | 熱愛純藝術創作 |

## Level 5** 備考策略

### 各等級大概水平

| 等級 | 大概百分比 |
|------|-----------|
| Level 5** | 全港前約1–2% |
| Level 5* | 全港前約3–5% |
| Level 5 | 全港前約8–10% |
| Level 4 | 全港前約20% |

### 衝擊5**的具體行動

**筆試（卷一）35%**：
- 熟記設計理論詞彙（設計元素、設計原則、可持續設計）
- 練習設計案例分析，掌握評析格式
- 操練past paper，了解HKEAA出題風格

**設計作品集（SBA）30%**：
- 全年持續更新作品集，不要臨近截止日期才趕工
- 確保設計過程完整呈現（調研 → 概念 → 發展 → 評估）
- 版面設計要整潔，視覺呈現要專業

**實作項目（SBA）35%**：
- 選擇可展現個人最強技能的設計方向
- 確保作品完整完成，功能性強
- 材料選擇要有充分理由，並在作品集中解釋

### 常見失分陷阱

| 失分位置 | 具體問題 | 改善方法 |
|----------|----------|----------|
| 筆試設計理論 | 答題太籠統，缺乏專業術語 | 建立設計詞彙清單，答題時使用正確術語 |
| 作品集設計過程 | 只展示最終結果，缺乏過程記錄 | 每個設計階段都要有文字說明和視覺記錄 |
| 實作項目 | 作品未完成或製作粗糙 | 提早開始，留足夠時間精修 |
| 可持續性 | 忽略可持續設計考慮 | 設計每個環節都要加入可持續性分析 |

## 選修DAT的考慮因素

### 適合選DAT的同學

1. **有設計興趣**：對創意設計、產品造型、視覺傳達有天然興趣
2. **有動手能力**：喜歡製作實物，而非純粹文字考試
3. **有志投身設計業**：目標是設計、建築、時裝、數碼媒體等行業
4. **願意投入SBA**：DAT的65% SBA要求長期持續的工作

### 不適合選DAT的同學

1. **只想應付考試**：DAT的SBA要求大量非考試期間的工作投入
2. **缺乏設計基礎**：DAT需要一定的視覺表達和製作技能
3. **時間管理困難**：SBA的長期要求對時間管理有較高要求

---

## 總結

設計與應用科技係DSE中最能展現跨界能力的科目，備考重點在於：

1. **重視SBA（佔65%）**：全年持續投入，設計作品集和實作項目是勝負關鍵
2. **掌握設計理論**：設計原則、可持續設計、物料科技是筆試核心
3. **選擇合適的選修範疇**：根據自己的興趣和技能選擇最能發揮優勢的設計方向
4. **銜接大學Portfolio**：DAT的SBA作品集可直接用於大學申請
5. **全年規劃**：不能只靠考前衝刺，DAT需要長期系統性的設計工作

對熱愛設計的同學而言，DAT不只是一個DSE科目，更是進入設計行業的第一步。
