# DSE 中國文學完全攻略 2026

中國文學科（中文文學）係DSE選修科中最具文藝氣息的一科，考核同學對中國古典及現代文學的理解、賞析與論述能力。對熱愛文學的同學而言，中文文學既是興趣的延伸，亦是通往大學中文及人文學科的重要跳板。

---

## 中文文學科考試結構

| 評核部分 | 考核內容 | 佔分比例 | 時間 |
|----------|----------|----------|------|
| 卷一 | 指定篇章閱讀賞析 | 40% | 2小時 |
| 卷二 | 文學賞析與論述（包括非指定篇章） | 40% | 2小時 |
| 校本評核（SBA） | 朗誦及口語文學活動 | 20% | 全年持續 |

## 四大文類概覽

中文文學科涵蓋四大文類，各有獨特的分析框架：

### 詩歌（詩）

| 項目 | 說明 |
|------|------|
| 涵蓋範圍 | 古典詩詞（唐詩宋詞）＋現代新詩 |
| 分析重點 | 意象、聲韻、格律、情感、抒情技巧 |
| 古典詩重要技巧 | 對仗、用典、借景抒情、托物言志 |
| 現代詩重要技巧 | 自由詩形式、象徵、語言實驗 |

**常考古典詩詞範疇**：
- 唐詩：李白、杜甫、王維、白居易
- 宋詞：蘇軾、李清照、辛棄疾
- 元曲：關漢卿、馬致遠

**詩歌分析框架**：
1. **意象分析**：詩人用什麼具體事物（意象）表達情感？
2. **聲韻效果**：韻腳的安排如何強化情感？
3. **情感走向**：詩中情感如何發展？有否轉折？
4. **抒情方式**：直抒胸臆、借景抒情、托物言志？

### 散文（散）

| 項目 | 說明 |
|------|------|
| 涵蓋範圍 | 古典散文（議論/記敘/抒情）＋現代散文 |
| 分析重點 | 筆法、結構、語言風格、作者情懷 |
| 古典散文特點 | 駢散結合、典故引用、言志抒懷 |
| 現代散文特點 | 形散神聚、個人化風格、口語化傾向 |

**散文分析框架**：
1. **主旨**：作者想表達什麼核心思想？
2. **結構**：文章如何組織？開合起伏？
3. **語言**：用什麼修辭手法？語言風格（典雅/質樸/諷刺）？
4. **情感**：作者的情感態度是什麼？

### 小說（小）

| 項目 | 說明 |
|------|------|
| 涵蓋範圍 | 古典白話小說＋現代小說 |
| 分析重點 | 人物塑造、情節結構、主題、敘事技巧 |
| 古典小說特點 | 話本傳統、道德教化、章回結構 |
| 現代小說特點 | 心理描寫、社會批判、多元敘事角度 |

**小說分析框架**：
1. **人物塑造**：正面/側面描寫、對話、行動、心理
2. **情節結構**：起承轉合、衝突高潮、懸念設置
3. **主題思想**：小說揭示什麼社會問題或人性？
4. **敘事手法**：第一/第三人稱、倒敘插敘、視角轉換

### 劇本（劇）

| 項目 | 說明 |
|------|------|
| 涵蓋範圍 | 古典戲曲（崑曲/粵劇）＋現代話劇 |
| 分析重點 | 對白、衝突、人物性格、舞台指示 |
| 古典戲曲特點 | 唱詞（曲牌）、科介（動作指示）、虛擬表演 |
| 現代話劇特點 | 對白為主、場景說明、心理深度 |

**劇本分析框架**：
1. **衝突分析**：人物之間的核心矛盾是什麼？
2. **對白功能**：對白如何展現人物性格和推進情節？
3. **戲劇張力**：如何製造懸念和高潮？
4. **主題**：劇本探討什麼人生或社會命題？

## 卷一：指定篇章閱讀賞析（40%）

### 考試形式

卷一考核同學對指定篇章的深入理解與賞析能力，題型包括：

- **理解題**：解釋詞語、概括內容、辨別態度
- **賞析題**：分析寫作技巧及其效果
- **比較題**：比較兩段文字的異同
- **評價題**：評價作品的藝術成就或作者觀點

### 指定篇章備考方法

1. **精讀每篇作品**：不能只讀摘要，必須逐字精讀
2. **建立賞析筆記**：每篇作品整理主題、人物、技巧、名句
3. **熟悉作者背景**：作者的時代背景影響創作意圖
4. **比較不同作品**：找出不同作品在主題或技巧上的共同點和差異

---

## 卷二：文學賞析與論述（40%）

### 考試形式

卷二包含：
- **甲部**：非指定篇章賞析（閱讀未見過的文學作品作答）
- **乙部**：文學論述（就指定題目撰寫文學論述文章）

### 非指定篇章賞析技巧

非指定篇章考核同學能否把學過的文學分析技巧應用於未見過的作品：

1. **快速辨認文類**：詩/散文/小說/劇本？古典/現代？
2. **找出核心意象或主旨**：作品想表達什麼？
3. **分析具體技巧**：引用原文，指出技巧名稱，分析效果
4. **個人回應**：表達自己對作品的感受及評價（需有依據）

### 文學論述寫作

文學論述題通常要求同學就某一文學主題或技巧展開討論，例如：
- 「試論中國古典詩詞中『思鄉』主題的表現手法」
- 「現代小說如何反映社會現實——以所讀作品為例」

**文學論述高分要點**：
1. **論點明確**：開篇即提出清晰的文學判斷
2. **引用原文**：每個論點必須配合具體引文
3. **分析而非描述**：不是複述作品內容，而是分析藝術效果
4. **比較不同作品**：引用多於一部作品，展示廣度
5. **個人見解**：加入自己的文學評價，不只覆述老師觀點

## 古典文學 vs 現代文學比較

| 比較項目 | 古典文學 | 現代文學 |
|----------|----------|----------|
| 語言 | 文言文或白話章回體 | 現代白話文 |
| 典故 | 大量引用典故 | 較少典故，重個人創新 |
| 形式 | 嚴格格律（詩詞）、程式化（戲曲） | 形式自由，實驗性強 |
| 主題 | 山水田園、忠君愛國、個人情志 | 社會批判、個人認同、存在意義 |
| 分析難點 | 文言理解、典故識別 | 象徵解讀、現代技巧辨認 |
| DSE佔比 | 約40–50% | 約50–60% |

## 校本評核（SBA）：20%

### SBA組成部分

| 活動 | 形式 | 佔SBA比例 |
|------|------|-----------|
| 朗誦（個人） | 背誦並朗誦指定或自選詩文 | 約50% |
| 口語文學活動 | 小組朗讀劇本、詩歌合誦等 | 約50% |

### 朗誦評分準則

| 評分項目 | 具體要求 |
|----------|----------|
| 流暢度 | 不停頓、不漏字，語調自然 |
| 語音準確 | 字音正確，不讀錯字 |
| 情感表達 | 語調配合詩文情感，有抑揚頓挫 |
| 理解程度 | 朗誦速度和停頓反映對內容的理解 |

### 朗誦備考建議

1. **早期揀選篇章**：選擇自己有感情連結的詩文，容易投入
2. **理解後才背誦**：理解意思和情感後，背誦更自然
3. **錄音自我監察**：錄下自己的朗誦，找出需要改善的地方
4. **練習情感起伏**：不要用「背書」的語調，要有情感表達
5. **注意停頓節奏**：詩歌的句讀停頓需要符合意思

## 大學課程銜接

中文文學科對以下大學課程有直接加分效果：

| 大學 | 課程 | 與中文文學的關聯 |
|------|------|----------------|
| HKU | 中國語言及文學（Chinese）| 高度相關，部分課程認可 |
| CUHK | 中國語言及文學 | 高度相關，為理想準備 |
| HKU | 比較文學（Comparative Literature） | 相關，需配合英文科成績 |
| CUHK | 文化管理 | 相關 |
| HKBU | 中國語言文學 | 相關 |

### 重要說明

中文文學科並非大多數商科、理科或工科大學課程的必要條件，但對希望修讀人文學科、教育（中文教學方向）或傳媒的同學而言，選修中文文學係一個強烈的訊號，顯示其語文素養。

## Level 5 備考策略

> 注意：中文文學科的Level 5**比例在DSE選修科中相對較低，但Level 5已是相當優秀的成績。

### 各等級大概水平

| 等級 | 大概百分比 |
|------|-----------|
| Level 5** | 全港前約1–2% |
| Level 5* | 全港前約3–4% |
| Level 5 | 全港前約8–10% |
| Level 4 | 全港前約20% |

### 衝擊Level 5的具體行動

**卷一（指定篇章）**：
- 製作每篇指定作品的深度賞析筆記
- 熟記重要名句（引用準確是加分項）
- 練習各題型的答題格式

**卷二（論述）**：
- 閱讀指定篇章以外的中文文學作品，擴闊引用範圍
- 練習撰寫結構清晰的文學論述
- 培養個人的文學品味和觀點

**SBA（朗誦）**：
- 全年持續練習，不要臨近評核才準備
- 選擇適合自己聲線和情感的篇章

### 常見失分陷阱

| 失分位置 | 具體問題 | 改善方法 |
|----------|----------|----------|
| 賞析題 | 只說技巧名稱，不分析效果 | 固定格式：技巧＋原文＋效果分析 |
| 論述題 | 內容描述多於文學分析 | 時刻問自己「這個技巧的藝術效果是什麼？」 |
| 非指定篇章 | 不懂應用分析框架 | 平時練習分析各種未見過的文學作品 |
| SBA朗誦 | 語調平板，缺乏情感 | 理解作品後再背誦，注重情感投入 |

## 總結

中國文學科係DSE中最能展現個人人文素養的選修科。備考的核心是：

1. **深入理解文學作品**：不只背誦，要真正理解和感受
2. **掌握四大文類分析框架**：詩、散文、小說、劇本各有側重
3. **古典與現代均衡準備**：不能偏廢任何一方
4. **文學論述寫作訓練**：分析而非描述，有論點有依據
5. **SBA朗誦情感投入**：技巧與感情缺一不可

選修中文文學的同學，往往對語文有較深的熱情。把這份熱情轉化為系統的備考行動，Level 5是完全可以達到的目標。
