# DSE 組合科學完全攻略 2026

組合科學（Combined Science）係DSE一個獨特的選修科安排，讓同學在一個科目框架內同時修讀兩個科學範疇。然而，組合科學有一個極為重要但常被忽略的特性：**它只計算為一個選修科，而非兩個**。這個安排對JUPAS申請有深遠影響，同學必須在選科前充分了解。

---

## 組合科學三種組合

同學可以從以下三種組合中選擇其一：

| 組合 | 模組A | 模組B | 適合方向 |
|------|-------|-------|----------|
| 組合一 | 生物（Biology） | 化學（Chemistry） | 醫療、生命科學、藥學 |
| 組合二 | 生物（Biology） | 物理（Physics） | 生物物理、環境科學 |
| 組合三 | 化學（Chemistry） | 物理（Physics） | 工程、材料、能源科學 |

## 重要：組合科學只計一個選修科

這是同學選修組合科學前**最需要了解**的核心事實：

> **組合科學在DSE成績單上計為一個選修科，而非兩個獨立科學科。**

### 對JUPAS的影響

JUPAS計算最佳成績時，以「最佳六科」（4核心科＋2選修科）為基礎：

| 情況 | 可計科目數目 |
|------|------------|
| 選修2個獨立科學科（例如：化學＋生物） | 2個選修科位均可計入 |
| 選修組合科學（例如：生物＋化學合科） | 只佔1個選修科位 |

**實際例子**：

甲同學成績：中文5、英文5、數學5、通識/公民5、**化學5**、**生物5**（分開兩科）
→ 最佳6科 = 30分

乙同學成績：中文5、英文5、數學5、通識/公民5、**組合科學（生物+化學）5**、**另一選修5**
→ 最佳6科 = 30分（但組合科學只佔1個選修位）

若乙同學的第三選修成績較差，則相比甲同學少了一個高分選修科可計入。

## 組合科學 vs 獨立科學科全面比較

| 比較項目 | 組合科學 | 獨立科學科（例：生物＋化學） |
|----------|----------|--------------------------|
| DSE選修科位數 | **1個** | **2個** |
| 課程深度 | 較淺（各科只涵蓋部分範疇） | 較深（完整課程） |
| 學習時間 | 較少 | 較多 |
| 可釋出的選修空間 | 可再選1個其他選修科 | 已用2個選修位 |
| 大學認可度 | 部分課程不認可或有限制 | 大多數科學相關課程均認可 |
| Level 5** 難度 | 相對較容易（深度較淺） | 相對較難 |
| 適合入讀醫科/藥學 | 一般不足夠（需獨立科） | 符合大多數要求 |
| 工程科系認可 | 視乎大學，部分接受 | 大多數接受 |

## 組合科學的大學認可情況

## 誰適合選組合科學？

### 適合選組合科學的情況

1. **有意讀商科/社科/人文學科**：不需要深入科學知識，但希望保留科學背景
2. **想釋出選修科位**：用省下的一個選修位修讀其他感興趣的科目（例如：BAFS、地理）
3. **科學能力中等**：不打算修讀科學相關大學課程，但想保持理科選項
4. **時間管理困難**：組合科學比獨立兩科節省學習時間

### 不適合選組合科學的情況

1. **有志入讀醫科、牙醫、藥學**：必須選獨立科學科
2. **目標工程、理科課程**：獨立科學科更具競爭力
3. **科學科表現優秀**：應選獨立科以展現最強科目成績
4. **JUPAS計分策略**：若科學科是你的強項，分開兩科可以計入兩個高分選修位

## 組合科學 Level 5 備考策略

### 各等級大概水平

| 等級 | 說明 |
|------|------|
| Level 5** | 全港前約1–2% |
| Level 5* | 全港前約3–5% |
| Level 5 | 全港前約8–10% |

### 備考重點

**共同原則（適用於所有組合）**：
1. 組合科學的深度比獨立科淺，但考核範圍仍然廣泛
2. 多項選擇題（卷一）是拉分關鍵，爭取滿分
3. 結構性問題（卷二）要注意答題格式

**生物模組備考**：
- 重點掌握：遺傳學（基因型/表現型計算）、人體系統（功能及疾病）
- 實驗技能：掌握常見生物實驗的目的和程序

**化學模組備考**：
- 重點掌握：化學方程式配平、有機化學官能團、電化學
- 計算題：摩爾計算、濃度計算

**物理模組備考**：
- 重點掌握：力學計算（牛頓定律）、電路計算（歐姆定律）
- 圖形分析：速度-時間圖、電路圖

### 時間分配策略

組合科學考試中，同學需在有限時間內應付兩個科目：

| 部分 | 建議時間 |
|------|----------|
| 卷一甲（多選題A科） | 25–30分鐘 |
| 卷一乙（多選題B科） | 25–30分鐘 |
| 卷二甲（結構題A科） | 60–70分鐘 |
| 卷二乙（結構題B科） | 60–70分鐘 |

## 常見問題解答

**Q：組合科學的成績等級和獨立科學科一樣嗎？**
A：是的，組合科學同樣以Level 1–5**評級，與獨立科學科使用相同的評分標準。

**Q：組合科學可以計入「最佳六科」嗎？**
A：可以，但只計為一個選修科。

**Q：如果我選了組合科學，還可以再選一個獨立科學科嗎？**
A：一般而言，學校不會允許同學同時選修組合科學及其中一個組合內的獨立科學科（例如：不能同時選組合科學（生物+化學）和獨立化學科）。具體安排視乎學校政策。

**Q：組合科學的Past Paper在哪裡找？**
A：可在HKEAA官方網站（hkeaa.edu.hk）下載歷年試題及評分準則。

## 總結

組合科學係一個靈活但需要謹慎選擇的科目安排。核心要點：

1. **只計一個選修科位**——這是選科前必須清楚的事實
2. **深度比獨立科淺**——學習壓力相對較小，但大學認可度有限制
3. **適合非純理科方向的同學**——可用省出的選修位修讀其他科目
4. **有志醫科/藥學必須選獨立科**——組合科學通常不符合資格
5. **查閱最新大學要求**——各大學對組合科學的政策每年可能調整

選科前，務必充分考慮自己的升學方向，再決定是否選修組合科學。
