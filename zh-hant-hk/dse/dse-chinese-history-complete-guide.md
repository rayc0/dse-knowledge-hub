# DSE 中國歷史完全攻略 2026

中國歷史科（中史）係DSE選修科中極具特色的一科，涵蓋逾兩千年的中華文明發展。對香港本地生而言，中史科不單止考核記憶，更重視歷史分析與論述能力。本文全面拆解中史科考試結構、重點朝代、近代史大事、論述技巧及Level 5**備考策略。

---

## 中史科考試結構總覽

DSE中史科分為兩份試卷：

| 試卷 | 考核範圍 | 佔分比例 | 考試時間 |
|------|----------|----------|----------|
| 卷一 | 中國傳統時期（約前600年至1900年） | 50% | 2小時15分 |
| 卷二 | 中國近代及現代史（1840年至今） | 50% | 2小時15分 |

> **注意**：卷二設有必答部分及選答部分，同學需要按要求選擇回答。

## 卷一：中國傳統時期（約前600年–1900年）

### 必須掌握的主要朝代

| 朝代 | 時期 | 核心考核重點 |
|------|------|-------------|
| 春秋戰國 | 約前770–前221 | 百家爭鳴、儒家/道家/法家思想、七雄爭霸 |
| 秦朝 | 前221–前206 | 統一六國、中央集權、郡縣制、焚書坑儒 |
| 漢朝 | 前206–後220 | 文景之治、武帝擴疆、絲綢之路、儒學獨尊 |
| 魏晉南北朝 | 220–589 | 三國鼎立、民族融合、佛教傳播 |
| 隋朝 | 581–618 | 統一南北、開鑿大運河、科舉制創立 |
| 唐朝 | 618–907 | 貞觀之治、開元盛世、安史之亂、對外交流 |
| 宋朝 | 960–1279 | 重文輕武、商品經濟發展、四大發明、靖康之恥 |
| 元朝 | 1271–1368 | 蒙古統治、行省制度、中西交流 |
| 明朝 | 1368–1644 | 廢相集權、鄭和下西洋、閉關鎖國、宦官亂政 |
| 清朝（前期） | 1644–1840 | 康雍乾盛世、文字獄、閉關政策 |

### 卷一重點考核主題

#### 政治制度演變

- **中央集權的強化**：從分封制 → 郡縣制 → 行省制 → 廢除丞相（明太祖）
- **科舉制度**：隋代創立 → 唐代完善 → 宋代擴大 → 明清八股文
- **君主專制的發展**：各朝代如何削弱貴族/相權，強化皇權

#### 社會經濟發展

- **農業發展**：各朝代土地制度（均田制、租庸調製）
- **商業繁榮**：宋代商品經濟、紙幣（交子）、城市化
- **手工業**：絲織、陶瓷的發展及外銷

#### 民族關係與對外關係

- **和親政策**：漢唐時期與北方民族的關係
- **民族融合**：北魏孝文帝漢化政策
- **對外交流**：絲綢之路、鄭和下西洋、朝貢制度

## 論述題答題策略

中史科的論述題係失分的重災區，但亦係與其他考生拉開差距的關鍵。

### 論述題基本結構

**PEEL結構（每個論點）**：
- **P**oint（論點）：直接說明你的立場
- **E**vidence（史實）：引用具體史實、事件、人物、年份
- **E**xplanation（解釋）：解釋史實如何支持你的論點
- **L**ink（連結）：連結回題目要求

### 議論雙方的技巧

中史論述題常要求同學「評估」或「試論」某種說法，需要同時論及正反兩面：

**論述模板**：
```
[論點甲]：支持……的理由是……（以……為例）
[論點乙]：然而，亦有說法認為……（以……為例）
[比較/評估]：綜合以上分析，……更為重要/更能說明……，原因是……
```

### 常見論述題類型

| 題型 | 答題重點 |
|------|----------|
| 「評估某政策的成效」 | 正面成就＋局限/失敗，最後作總體評估 |
| 「比較兩個時期/人物」 | 找出相似點＋差異點，加以分析 |
| 「分析某事件的原因」 | 遠因/近因；內因/外因；政治/經濟/社會因素 |
| 「評價歷史人物」 | 功績＋過失，考慮歷史背景，避免以今非古 |
| 「某說法是否正確」 | 分析說法的正確之處＋不足之處，提出自己的判斷 |

### 引用史實的注意事項

1. **年份要準確**：例如「1895年《馬關條約》」，不能含糊說「甲午戰後」
2. **史實要具體**：例如指出條約的具體條款，不能只說「割地賠款」
3. **引用多於一個例子**：每個論點至少2個史實支持
4. **避免「堆砌史料」**：史實要服務論點，不是把所有知識都寫出來

## Level 5** 備考策略

### 各等級大概要求

| 等級 | 大概要求 |
|------|----------|
| Level 5** | 全港前約1–2% |
| Level 5* | 全港前約3–5% |
| Level 5 | 全港前約8–10% |
| Level 4 | 全港前約20% |

### 系統備考計劃

**Stage 1：建立知識框架（開考前6個月）**
1. 為每個朝代/時期製作一頁總結（政治/經濟/社會/文化）
2. 整理重要事件時序表
3. 分析各朝代的「興亡原因」（政治腐敗/外族入侵/社會矛盾等）

**Stage 2：操練論述技巧（開考前3個月）**
1. 每週至少完成2–3條論述題
2. 請老師或同學批改，針對論點是否清晰、史實是否準確
3. 分析HKEAA高分示範答案的論述結構

**Stage 3：Past Paper衝刺（開考前1個月）**
1. 完整模擬考試條件（計時、不看筆記）
2. 對照評分準則自我評估
3. 針對弱點的題目類型加強練習

### 常見失分原因

| 失分原因 | 改善方法 |
|----------|----------|
| 史實錯誤（年份/人名混淆） | 製作精確的時序表，反覆默寫 |
| 論述缺乏論點（只堆砌史實） | 每段必須有明確的「論點句」 |
| 答非所問 | 考前仔細審題，用題目關鍵詞貫穿答案 |
| 時間不足 | 練習計時，卷一/卷二各約2小時15分 |
| 字跡潦草 | 考試時放慢書寫速度，確保字體清晰 |

## 總結

中史科是一門兼顧記憶與分析的科目，備考要做到：

1. **系統整理知識**：按朝代/主題建立清晰的知識框架
2. **掌握論述格式**：每條論述題都有清晰論點及史實支持
3. **正反兩面分析**：避免偏頗，展示全面的歷史思維
4. **大量操練**：中史論述需要熟練，不能只靠記憶
5. **精準引用史實**：年份、條約名稱、人物要準確無誤

掌握以上要點，在中史科取得Level 5*甚至Level 5**是完全可以實現的目標。
