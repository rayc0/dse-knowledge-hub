# DSE 中國語文完全攻略 2026

中國語文（中文）係DSE核心科目之一，佔JUPAS計分的重要比重。對香港本地生而言，中文科既係母語考試，亦係最考核語文素養的科目。本文全面拆解中文科各卷結構、常見失分位置、高分文體技巧及Level 5**備考策略。

---

## 中文科考試結構總覽

DSE中文科共分五個評核部分：

| 評核部分 | 考核內容 | 佔分比例 | 時間 |
|----------|----------|----------|------|
| 卷一 閱讀能力 | 白話文＋文言文閱讀理解 | 22.5% | 2小時 |
| 卷二 寫作能力 | 指定文類寫作 | 22.5% | 2小時 |
| 卷三 聆聽及綜合能力 | 聆聽理解＋綜合訊息 | 22.5% | 約1.5小時 |
| 卷四 說話能力 | 小組討論 | 15% | 約25分鐘 |
| 校本評核（SBA） | 個人短講＋互動口語活動 | 17.5% | 全年持續 |

> **注意**：卷三聆聽部分及卷四說話能力均係在校考核，卷一及卷二為公開考試。

---

## 卷一：閱讀能力（22.5%）

### 題型結構

卷一包含白話文閱讀及文言文閱讀兩大部分：

| 部分 | 內容 | 大概配分 |
|------|------|----------|
| 甲部 白話文 | 2–3篇語段，附理解問題 | 約65–70分 |
| 乙部 文言文 | 1–2篇文言文，附翻譯及理解 | 約30–35分 |

### 白話文閱讀題型

- **詞語解釋**：根據語境解釋詞語意思（不能純粹查字典式回答）
- **內容理解**：從文章中找出具體資訊
- **寫作手法分析**：修辭手法（比喻、擬人、對比）、結構分析
- **主題思想**：作者的觀點、態度、意圖
- **讀者回應題**：表達個人感受及評價（需有文本支持）

### 文言文閱讀要點

- **翻譯題**：逐字對應，保留文言語氣，不能過度意譯
- **理解題**：結合文章內容及文言文法
- **常見文言虛詞**：之、乎、者、也、而、以、於、為

### 閱讀卷高分策略

1. **答題格式**：每題必須直接回應題目要求，不能只抄文章原文
2. **引用原文**：答理解題時，引用原文後加上自己的解釋
3. **修辭分析三步**：指出手法名稱 → 引用原文 → 分析效果（讀者感受/作者意圖）
4. **讀者回應題**：先表明立場，再引用文章具體內容支持，最後加個人延伸

## 卷二：寫作能力（22.5%）

### 題型結構

卷二分為兩部分，同學須完成兩題：

| 部分 | 要求 | 字數 |
|------|------|------|
| 甲部 指定文類寫作 | 按指定文類寫作（實用文類） | 約400–600字 |
| 乙部 命題／半命題寫作 | 自由命題或開放式題目 | 約600–800字 |

### 常見文類一覽

#### 實用文類（甲部常考）

| 文類 | 格式要求 | 常見題目 |
|------|----------|----------|
| 書信 | 稱謂、正文、署名、日期 | 投訴信、建議信、感謝信 |
| 演講稿 | 開場白、正文、結語 | 典禮講辭、辯論稿 |
| 報告 | 標題、引言、正文分點、建議 | 調查報告、工作報告 |
| 建議書 | 標題、背景、建議項目、結論 | 社區改善建議 |
| 通告 | 標題、對象、內容、落款 | 活動通知、校內通告 |
| 會議記錄 | 出席者、議程、討論、決議 | 學生會/機構會議 |
| 日記 | 日期、天氣、正文 | 個人心境、事件記錄 |
| 新聞稿 | 標題、副題、倒金字塔結構 | 活動新聞稿 |

#### 文學文類（乙部常考）

| 文類 | 寫作要求 |
|------|----------|
| 記敘文 | 事件發展清晰，人物刻畫生動，首尾呼應 |
| 描寫文 | 感官描寫豐富，情景交融，用詞精準 |
| 抒情文 | 情感真摯，借景抒情或托物言志 |
| 議論文 | 論點清晰，論據充分，論證嚴密 |
| 夾敘夾議 | 敘述與分析交替，層層深入 |

### 寫作高分技巧

#### 實用文類（格式分不能失）

1. **格式正確**：格式錯誤直接扣「表達」分數
2. **語氣適當**：書信要分清正式/非正式對象
3. **分點清晰**：報告、建議書必須用分點形式
4. **切題**：緊扣題目要求，不能答非所問

#### 文學寫作（內容分要高）

1. **開頭吸引人**：避免「光陰似箭」等老套開頭
2. **描寫具體**：用感官細節代替抽象描述（「汗水浸濕了衣領」而非「很熱」）
3. **首尾呼應**：結尾回應開頭的意象或主題
4. **用詞準確**：避免重複用詞，展示詞彙廣度

## 卷三：聆聽及綜合能力（22.5%）

### 考試形式

卷三分為聆聽部分及綜合能力部分：

| 部分 | 形式 | 佔卷三比例 |
|------|------|-----------|
| 甲部 聆聽 | 聽錄音回答問題 | 約50% |
| 乙部 綜合能力 | 閱讀資料＋聆聽資料，完成寫作任務 | 約50% |

### 聆聽題型

- **填表題**：從錄音中提取具體資訊填入表格
- **多項選擇題**：辨別錄音中的事實與意見
- **簡答題**：用短句回答錄音內容
- **推斷題**：根據說話語氣、用詞推斷說話者態度

### 聆聽技巧

1. **第一次播放前**：先瀏覽題目，了解要注意的重點
2. **關鍵詞標記**：在題目旁標記需要聆聽的關鍵詞
3. **辨別語氣**：廣東話語調中的肯定/懷疑/諷刺語氣
4. **數字日期**：特別注意數字、日期、地點等具體資訊

### 綜合能力寫作

綜合能力考核同學能否整合多種資料（文字＋圖表＋聆聽）完成指定任務，常見任務包括：
- 撰寫計劃書或建議書
- 撰寫演講稿或辯論稿
- 整理資料並呈現

**高分要點**：整合多個資料來源，不能只依賴單一資料；文類格式必須正確。

## 卷四：說話能力（15%）

### 考試形式

卷四為小組討論，4–6人一組，就指定議題進行約25分鐘的討論。

### 評分準則

| 評分範疇 | 具體要求 |
|----------|----------|
| 用語 | 普通話/廣東話的正確運用（視乎學校選擇） |
| 表達 | 表達清晰，組織有條理 |
| 態度 | 積極參與，尊重他人意見 |
| 互動 | 回應他人觀點，有效推進討論 |
| 內容 | 論點充分，能引用具體例子 |

> **注意**：DSE中文卷四以廣東話進行（普通話中文科另作安排）。

### 說話卷高分技巧

1. **主動帶動討論**：不要等別人先說，主動開展話題
2. **回應他人觀點**：「我同意某某的觀點，因為……但我想補充……」
3. **具體例子**：引用時事、統計數字支持論點
4. **適時總結**：擔任「整合者」角色，總結各方意見
5. **語言準確**：避免大量廣東話俚語，保持書面語風格

## 校本評核（SBA）：17.5%

### SBA組成部分

| 活動 | 形式 | 佔SBA比例 |
|------|------|-----------|
| 個人短講 | 就指定題目作約3–5分鐘短講 | 約50% |
| 互動口語活動 | 與同學進行討論/辯論/角色扮演 | 約50% |

### SBA備考建議

1. **個人短講**：選擇自己有深入了解的題目，準備充分的具體例子
2. **互動活動**：提前了解討論議題，準備正反兩方的論點
3. **全年持續**：SBA全年評核，不能只靠最後衝刺
4. **與老師確認格式**：不同學校的SBA安排略有差異

## 各卷備考資源對比

| 評核部分 | 最有效的備考方式 |
|----------|----------------|
| 卷一 閱讀 | 大量閱讀不同文體的文章，練習分析手法 |
| 卷二 寫作 | 每週至少寫一篇完整文章，請老師批改 |
| 卷三 聆聽 | 反覆練習past paper聆聽錄音，提升訊息提取能力 |
| 卷四 說話 | 組織討論小組，練習即席發言 |
| SBA | 全年持續準備，勿臨時抱佛腳 |

## Level 5** 備考策略

### 各等級大概水平

| 等級 | 大概百分比 |
|------|-----------|
| Level 5** | 全港前約1% |
| Level 5* | 全港前約3% |
| Level 5 | 全港前約8% |
| Level 4 | 全港前約20% |

### 衝擊5**的具體行動

**閱讀卷（卷一）**：
- 精讀HKEAA範文及高分示範答案，了解評分準則
- 練習在有限時間內完成完整答案（不能只寫要點）
- 文言文翻譯：確保語感準確，不生硬

**寫作卷（卷二）**：
- 建立自己的「詞庫」：每週學習10個精準詞語
- 練習不同文體，尤其不熟悉的實用文類
- 模仿高分範文的結構，再加入自己的創意

**聆聽綜合（卷三）**：
- 熟悉各類錄音格式（對話、訪問、廣播）
- 綜合寫作：練習整合多個資料來源的技巧

**說話（卷四）**：
- 平時多關注時事，備有充分論點
- 練習流暢的書面語口語表達

### 常見失分陷阱

| 失分位置 | 具體錯誤 | 改善方法 |
|----------|----------|----------|
| 卷一修辭分析 | 只寫出手法名稱，不分析效果 | 固定答題模式：手法＋原文＋效果 |
| 卷二書信格式 | 稱謂錯誤、署名格式不符 | 熟記各文類格式，考前複習 |
| 卷三聆聽 | 漏答關鍵資訊 | 提前瀏覽題目，做好筆記 |
| 卷四說話 | 全程沉默，只被動回應 | 主動帶動討論，展示領導力 |

## 總結

中文科雖係本地生的母語科，但Level 5**並不容易。關鍵在於：

1. **掌握各文類格式**：格式錯誤直接失分
2. **系統練習寫作**：每週操練，請人批改
3. **閱讀廣泛**：增加詞彙量及對不同文體的熟悉度
4. **聆聽訓練**：反覆練習past paper聆聽部分
5. **說話自信**：主動參與，不怕犯錯

中文5**係進入頂尖大學課程的重要砝碼，值得投入充足時間備考。
