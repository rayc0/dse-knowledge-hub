# DSE 全科備考策略完全指南

HKDSE 係香港每年最重要的公開考試，應考策略正確與否直接影響最終成績。本文從選科、時間管理、Mock 備考，到最後衝刺的心理準備，提供一套完整的備考路線圖，助你在有限時間內發揮最大效益。

## HKDSE 考試日曆與報名

### 重要日期（以 2026 年為例）

| 項目 | 時間 |
|------|------|
| JUPAS 第一輪選科截止 | 每年 1 月初 |
| DSE 報名截止 | 每年 8–9 月（中六開學前） |
| SBA 提交截止 | 每年 1–3 月（視乎科目） |
| Mock 考試（校內） | 每年 1–2 月 |
| DSE 正式考試 | 每年 3 月底至 5 月 |
| 成績公佈 | 每年 7 月 |
| JUPAS 放榜/院校取錄 | 每年 7–8 月 |

### 考試報名注意事項
- 報名費豁免：綜援家庭可申請
- 重考生（Private Candidate）可保留部分科目成績（成績有效期 5 年）
- 退出（Withdraw）某科須在指定日期前申請

## 選科策略：3 科還是 4 科？

### 必修科目（所有考生）
1. 中國語文（Chinese Language）
2. 英國語文（English Language）
3. 數學（Mathematics）
4. 公民與社會發展科（Citizenship & Social Development）

### 選修科選擇

| 策略 | 說明 | 適合對象 |
|------|------|---------|
| 選 3 科選修 | 精力集中，每科都做到最好 | 時間管理困難、目標科系要求明確 |
| 選 4 科選修 | 提高入學競爭力，保險選項 | 學習能力強、自律性高、時間充裕 |

### 選修科選科原則

**1. 對齊大學要求**
- 先查清目標學系的收生要求（Required Subjects）
- 醫科、法律、工程各有特定科目要求

**2. 揚長避短**
- 選自己有興趣及基礎的科目
- 唔要因為「聽說易拿 5**」而選不熟悉的科目

**3. 科目組合互補**
- 避免全選同類性質的科目（如 3 科理科同時選）
- 考慮 SBA 時間衝突：唔要同時選 4 科都有大型 SBA 項目

**4. 考慮 M1/M2**
- 有意讀工程、精算、物理的同學應同時報考 M2
- M1/M2 為額外計分，唔影響必修科

### 常見選科組合建議

| 方向 | 建議組合 |
|------|---------|
| 醫科/牙醫 | 化學 + 生物 + 物理（或數學延伸） |
| 工程 | 物理 + 化學/ICT + 數學 M2 |
| 商科/精算 | 經濟 + 數學 M1/M2 + BAFS |
| 法律/社科 | 歷史 + 經濟 + 地理/中史 |
| 藝術/設計 | 視覺藝術 + 中文文學 + 音樂 |

## Mock 考試備考策略

### Mock 的正確用法

1. **考前態度**：以正式考試的態度應考 Mock，唔要「反正唔計分」
2. **考後分析**：Mock 之後最重要係逐題分析失分原因
3. **唔要死記 Mock 答案**：Mock 試題唔等於 DSE 真題

### Mock 後分析框架

| 失分類型 | 應對方法 |
|---------|---------|
| 概念錯誤 | 重新溫習相關課題，理解而非死記 |
| 計算失誤 | 增加操練量，養成「Check 數」習慣 |
| 審題錯誤 | 養成圈重點詞（e.g. "apart from", "other than"）習慣 |
| 時間不夠 | 練習計時，學習跳過難題先做容易題 |
| 知識盲點 | 找出自己唔熟的課題，集中補強 |

## 如何善用 HKEAA 評卷準則

### 評卷準則（Marking Scheme）的重要性

HKEAA 每年公佈歷屆試題的 Marking Scheme，係備考最重要的資源之一。

**使用方法：**
1. 做完試卷後，先自己嘗試答題
2. 對照 Marking Scheme，找出答案差異
3. 分析「為何 Marking Scheme 咁答」，唔係死背答案
4. 留意「Accept"/"Any appropriate answer"——代表有彈性，唔係只有一個答案

### 理解評級描述（Level Descriptors）

每個 Level 都有具體描述，了解 Level 4 和 Level 5 的差異：

| Level | 一般描述 |
|-------|---------|
| Level 1 | 基本知識，碎片化理解 |
| Level 2 | 能識別主要概念，部分應用 |
| Level 3 | 理解核心概念，能基本應用 |
| Level 4 | 深入理解，能分析及評估 |
| Level 5 | 全面深入分析，獨立批判思考 |
| Level 5* | 優秀示範水平 |
| Level 5** | 頂尖，考生中最高 8–10% |

## SBA 準備時間線

### 通用 SBA 時間線（適用於大部分科目）

| 月份 | 工作 |
|------|------|
| 中五 9 月 | 確認 SBA 題目，與老師溝通 |
| 中五 10–11 月 | 完成初步研究/設計/創作 |
| 中五 12 月 | 提交初稿予老師批改 |
| 中六 1 月 | 根據老師意見修改 |
| 中六 2–3 月 | 完成最終版本，提交 |

### SBA 高分原則
- **格式完整**：按照 HKEAA 要求的格式提交，缺少任何部分直接扣分
- **字數/長度適當**：唔係越長越好，重質不重量
- **正確引用**：如有引用資料，必須正確標注來源
- **提早完成**：預留時間修改，唔要最後一刻才交

## 最後一個月衝刺策略（3 月）

### 最後 4 週計劃

**第 1 週：盤點弱點**
- 列出每科最弱的 3 個課題
- 重新溫習這些課題的教科書章節

**第 2 週：密集操練**
- 每天至少完成一份過往試卷（計時完成）
- 重點科目的長題答題練習

**第 3 週：重點複習**
- 製作每科的重點筆記（1–2 頁總結）
- 練習「快速複習」，確保核心概念清晰

**第 4 週（考試週前）：維持狀態**
- 唔要加入新內容，維持複習節奏
- 確保充足睡眠，每天最少 7–8 小時
- 輕度運動保持專注力

## 考試日當天策略

### 考試前一晚
- 準備好所有文具（鉛筆、原子筆、計算機、尺等）
- 確認考場地址及交通路線
- 早睡，唔要通宵溫習

### 考試當天
- 提早 30 分鐘到達考場
- 帶水及輕食（如果允許），保持血糖穩定
- 入場前唔要看筆記，影響心理狀態

### 答題策略
1. **先瀏覽全卷**：了解題目分佈，規劃時間
2. **先易後難**：先做有把握的題目，累積分數
3. **時間控制**：每題根據分數分配時間，唔要在一題上花太多時間
4. **留時間檢查**：最後 10–15 分鐘用作審題及修改

## 考試期間精神健康

### 常見壓力信號
- 睡眠困難、食慾不振
- 無法集中注意力
- 持續焦慮、情緒波動

### 應對方法

| 方法 | 說明 |
|------|------|
| 規律運動 | 每天 20–30 分鐘步行或輕度運動，有效減壓 |
| 社交連繫 | 唔要孤立自己，與同學互相支持 |
| 分享感受 | 跟父母、老師或輔導員說出壓力感 |
| 正念練習 | 每天 5–10 分鐘冥想或深呼吸 |
| 休息計劃 | 每溫習 45–50 分鐘休息 10 分鐘 |

### 重要提醒
- DSE 成績唔係人生的唯一出路
- JUPAS 外有副學士、毅進、海外升學等多種選擇
- 如有嚴重焦慮，及早諮詢學校輔導老師或撒瑪利亞防止自殺會（2382 0000）

## Level 5** 整體戰術

### 頂尖考生的共同特徵
1. **理解而非死記**：能夠靈活應用概念到新題型
2. **大量操練**：每科至少做 5–7 份歷屆試卷
3. **分析評卷準則**：了解評分員想要什麼答案
4. **時間管理精準**：考試中每分每秒都有計劃
5. **保持一致狀態**：全年穩定學習，而非臨時死衝

### 各核心科目 Level 5** 重點

| 科目 | Level 5** 關鍵 |
|------|--------------|
| 中文 | 卷一閱讀分析深度，卷二文章有個人觀點 |
| 英文 | 卷四口語自然流暢，卷二論說清晰有力 |
| 數學 | 步驟完整，不跳步，高難題概念清晰 |
| 選修科（理科） | 實驗報告完整，理論與應用結合 |
| 選修科（文科） | 多角度分析，引用史實/數據支撐論點 |
