# DSE 數學 M1 vs M2 選修決策指南 2025：唔同大學 Program 對 Module 的要求、分數計算與選科策略

DSE 數學延伸部分（Mathematics Extended Part）分為 M1（微積分與統計）和 M2（代數與微積分）兩個選修模組。對於許多中四選科的同學來說，這個決定往往在資訊不足的情況下草草作出，結果到了中六才發現選錯了 Module，影響大學申請。

本指南將從課程內容、大學要求、JUPAS 計分方式、學習負擔，以及選科策略等多個角度，幫助你做出最適合自己的決定。

---

## 一、M1 和 M2 各自涵蓋什麼內容？

### M1：微積分與統計（Calculus and Statistics）

M1 的課程重心分為兩大部分：

**微積分部分（約 50% 內容）：**
- 極限與微分基礎（Differentiation）
- 積分基礎（Integration）
- 指數函數與對數函數的微積分應用

**統計部分（約 50% 內容）：**
- 概率論基礎（Probability）
- 離散及連續隨機變量（Discrete and Continuous Random Variables）
- 二項分佈（Binomial Distribution）
- 泊松分佈（Poisson Distribution）
- 正態分佈（Normal Distribution）
- 統計推斷基礎（Statistical Inference）：點估計、信賴區間、假設檢定

M1 的特色是將微積分與統計並重，統計部分較多數學中的其他科目深入，是修讀商科、社會科學、生物、心理學的同學較理想的選擇。

### M2：代數與微積分（Algebra and Calculus）

M2 的課程深度和廣度均比 M1 更高，主要涵蓋：

**代數部分：**
- 數學歸納法（Mathematical Induction）
- 二項式定理（Binomial Theorem）
- 三角函數（Trigonometry）：加法公式、積化和差、和差化積
- 數列與級數（Sequences and Series）
- 向量（Vectors）：二維、三維，點積與向量積
- 矩陣（Matrices）：行列式、逆矩陣、線性方程組

**微積分部分：**
- 極限與連續性
- 微分技巧：連鎖法則（Chain Rule）、隱函數微分、參數方程微分
- 積分技巧：代換積分、分部積分
- 定積分的應用

M2 本質上是大學工程、物理、數學、計算機科學等理科課程的預備課程，內容比 M1 更接近大學頭一年的課程要求。

## 二、各大學 Program 對 M1/M2 的要求

這是選科決定中最關鍵的因素。不同大學、不同課程對 Module 的要求差異很大，以下按主要院校列出。

### 香港大學（HKU）

| 課程 | M1/M2 要求 |
|------|-----------|
| 工程學（BEng） | M2 必修（大部分課程） |
| 計算機科學（BCS） | M2 必修或強烈建議 |
| 精算學（BSc Actuarial Science） | M1 或 M2 均接受，M2 更具優勢 |
| 理學（BSc，物理、數學方向） | M2 必修 |
| 商學院（BBA/BBE） | M1 或 M2 均接受 |
| 醫學（MBBS） | 無強制要求，但 M2 有優勢 |
| 護理 | 無 M1/M2 要求 |

### 香港科技大學（HKUST）

HKUST 理工科課程幾乎全面要求 M2：

| 課程 | M1/M2 要求 |
|------|-----------|
| 工程（所有範疇） | M2 必修 |
| 理學（物理、數學、化學） | M2 必修 |
| 計算機科學 | M2 必修 |
| 商學院（BSBA） | M1 或 M2 均接受 |
| 環境管理及科技 | M1 或 M2 均接受 |

**重要注意**：HKUST 部分課程在計算入學分數時，M2 可計入「最佳 5 科」，M1 則不被計入某些理科課程的最佳分計算，這影響極大（見第三節）。

| 理學（物理、數學） | M2 必修 |

| 商學院 | M1 或 M2 均接受 |

## 三、JUPAS 計分方式：Module 如何影響最佳五科？

這是很多同學最容易忽略的技術細節，直接影響 JUPAS 的分數計算和最終錄取。

### M1/M2 在 JUPAS 的地位

M1 和 M2 屬於數學科的「延伸部分」，在 JUPAS 計分時有以下特性：

1. **獨立科目**：M1/M2 的成績獨立列出，不算在必修數學（Compulsory Mathematics）內
2. **可計入最佳科目**：在計算「最佳五科」或「最佳六科」時，M1/M2 的成績可以作為一個獨立科目計入
3. **額外加分**：部分課程計算入學成績時，M1/M2 可以加入最佳科目的計算，相當於多一個得分機會

### 實際計分例子

假設某同學成績如下：
- 中文：5
- 英文：5*
- 數學（必修）：5
- 物理：5*
- 化學：5
- M2：5

若該課程計算「最佳 5 科」且 M2 可計入：

最佳 5 科可以選：英文(5*) + 數學(5) + 物理(5*) + 化學(5) + M2(5) = 5+5+5+5+5 = 25（4星 = 5分，3星 = 4分，以此類推）

在這個例子中，M2 的分數幫助這位同學在計算最佳五科時有更多「最強科目」可選，提升了整體入學分。

### 重要提醒：不是所有課程都計 M1/M2

部分課程在計算最佳科目時，**不計或不優先計算** M1/M2，例如某些文科或人文課程。在選擇課程志願時，必須查閱各院校官方的計分規則，避免錯誤估算分數。

## 四、M1 vs M2：哪類學生適合哪個？

### 適合選 M1 的學生

以下特質的同學，M1 通常是更合適的選擇：

**1. 目標課程方向：商科、社會科學、心理學、生物醫學**
- 這些範疇的大學課程均要求統計學知識
- M1 的統計部分（概率、隨機變量、假設檢定）與大學課程高度相關
- M2 的矩陣、向量等代數內容在商科/社科課程中較少用到

**2. 對微積分有一定興趣，但不想深入純數學**
- M1 的微積分部分深度中等，難度低於 M2
- 適合希望接觸高等數學但不想投入大量時間的同學

**3. 數學必修科成績中等（3-4 級）**
- M1 整體難度較 M2 低，對數學能力要求稍低
- 若必修數學已感吃力，M2 的難度可能會顯著影響整體成績

**4. 計劃主修精算學（Actuarial Science）或統計學**
- 精算學的大學課程包含大量統計內容，M1 的統計部分是良好基礎
- 但需注意：部分大學（如 HKU）的精算課程要求 M2，需具體查閱要求

### 適合選 M2 的學生

以下特質的同學，M2 通常是更合適的選擇：

**1. 目標課程方向：工程、物理、數學、計算機科學**
- 這些課程的大學入學幾乎全面要求 M2
- M2 的矩陣、向量、積分技巧是大學工程數學的直接預備
- 選 M1 可能直接令你失去申請資格

**2. 數學必修科成績優秀（5 或以上）**
- M2 對代數操控能力要求較高，必修科成績優秀的同學更容易適應
- M2 5 級以上的成績在 JUPAS 計算中有更高加分效果

**3. 對純數學有濃厚興趣**
- M2 的數學歸納法、二項式定理、矩陣等內容對純數學愛好者更有趣
- 若有興趣修讀大學數學系，M2 是必需的基礎

**4. 希望保持最多升學選擇**
- M2 接受的課程涵蓋範圍比 M1 廣（工程、理科、部分商科）
- M1 不被大部分工程/理科課程承認或計分
- 若升學方向仍未確定，M2 提供更多彈性（尤其是理工科方向）

## 五、學習負擔比較

### 課時與作業量

M2 的課時通常比 M1 多約 10-20%，因為課程內容更廣、難度更高。以中六學年為例：

| 項目 | M1 | M2 |
|------|----|----|
| 每週課時（參考） | 2-3 小時 | 3-4 小時 |
| 考試卷數 | 1 份（3 小時） | 1 份（2.5 小時） |
| 概念難度 | 中等 | 中至高 |
| 運算量 | 中等 | 較高 |
| 常見失分陷阱 | 統計概念混淆 | 積分技巧出錯 |

### 備試策略差異

**M1 備試重點：**
- 概率與統計定義要背熟
- 假設檢定步驟要標準化
- 微積分部分相對直接，重在練習

**M2 備試重點：**
- 積分技巧需大量練習（代換、分部）
- 矩陣運算要準確快速
- 向量題目要建立幾何直覺

## 六、常見選科錯誤與如何避免

### 錯誤一：「朋友選 M2，我也選 M2」

選 Module 應根據自己的目標課程方向，而非朋輩影響。M2 難度較高，若你的目標是商科或社科，選 M2 可能既增加了備試壓力，又無助於提升整體分數競爭力（因為商科課程通常同等對待 M1/M2）。

### 錯誤二：不查清楚目標課程要求就選科

這是最常見、後果最嚴重的錯誤。選 M1 的同學若中途決定申請工程課程，往往發現無法達到入學要求。解決方法：在選科前，查閱所有你可能有興趣的課程的官方要求，預留一定彈性。

### 錯誤三：以「容易得高分」為唯一考量

部分同學選 M1 是因為覺得比 M2 容易取得高分。這個考量本身無問題，但需要同時確認目標課程是否接受 M1。若 M1 的高分在目標課程中根本不計入最佳科目，就失去了意義。

### 錯誤四：忽略 M1/M2 在 JUPAS 計分中的效益

選修 Module 的邏輯之一，是讓 JUPAS 計算最佳科目時多一個高分選項。若你的 Module 成績預計只有 2-3 級，它可能拖低而非提升你的最佳科目分數。在這種情況下，有些同學考慮是否值得修讀 Module。

### 錯誤五：中途放棄 Module

部分同學在中五或中六初期決定放棄 Module，原因通常是壓力太大或成績不理想。放棄 Module 前需考慮：目標課程是否要求它？若是必修項，放棄便直接喪失申請資格；若只是可選項，且成績持續落後，放棄以集中精力其他科目有時是合理選擇。此決定應在諮詢老師和升學輔導後做出。

## 七、若你現在才考慮：可以在中五或中六時轉換嗎？

理論上，部分學校允許學生在中四升中五時轉換 Module，但中五升中六時轉換難度極高，因為課程進度差距太大。

**若你正在中四/中五初期**：仍有機會轉換，但需立即告知學校和任課老師，並評估能否補追課程內容。

**若你已在中六**：轉換已基本不可行。應把精力放在如何在現有 Module 下取得最佳成績，並調整 JUPAS 志願以匹配你的 Module。

## 八、總結：一個快速決策框架

回答以下三個問題，幫助你快速判斷：

**問題 1：你的主要目標課程是工程、物理、數學或計算機科學嗎？**
- 是 → 選 M2（幾乎無選擇）
- 否 → 繼續問題 2

**問題 2：你的主要目標課程是商科、社科、心理或生物醫學嗎？**
- 是 → 選 M1（統計部分高度相關）
- 否 → 繼續問題 3

**問題 3：你對數學的興趣和能力如何？**
- 必修數學 5 級或以上，對數學有濃厚興趣 → 考慮 M2
- 必修數學 3-4 級，或對數學興趣一般 → M1 更穩妥
- 升學方向完全未定 → M2 保留更多理工科選擇
