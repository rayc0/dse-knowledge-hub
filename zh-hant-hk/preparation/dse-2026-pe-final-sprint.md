## 二、九大課程主題快速掌握

DSE 體育科理論涵蓋九個主題（Parts I–IX），卷二長答題必從其中三個主題各出一條。以下按出題頻率由高至低排列：

### ⭐⭐⭐ 高頻主題（必掌握）

#### 第二部分：人體結構與功能

| 系統 | 必考核心知識點 |
|---|---|
| 骨骼系統 | 206 塊骨；關節類型（球窩關節、鉸鏈關節、平面關節）；滑液關節的結構（軟骨、滑液、關節囊） |
| 肌肉系統 | 三類肌肉（骨骼肌/平滑肌/心肌）；主動肌 vs 拮抗肌 vs 協同肌；肌肉收縮類型（向心/離心/等長） |
| 心血管系統 | 心臟四腔室；體循環 vs 肺循環路徑；心率/每搏輸出量/心輸出量的關係（CO = HR × SV） |
| 呼吸系統 | 氣體交換位置（肺泡 ↔ 毛細血管）；潮氣量/肺活量/最大攝氧量（VO₂max） |
| 神經系統 | 中樞神經（腦 + 脊髓）vs 周邊神經；本體感覺（proprioception）；反射弧結構 |

#### 第三部分：運動分析（生物力學）

| 概念 | 考試重點 |
|---|---|
| 牛頓運動三定律 | 慣性定律（持球靜止）；加速度定律（F = ma）；作用力與反作用力（起跑蹬地） |
| 槓桿原理 | 一類槓桿（頸部點頭）；二類槓桿（提踵）；三類槓桿（肘部彎曲，最常見於人體） |
| 線性運動 | 速度/加速度計算；拋體運動中的水平與垂直分量 |
| 重心與穩定性 | 重心高低、支撐面積大小、重心在支撐面內的位置——三項因素決定穩定性 |
| 旋轉運動 | 角速度與慣性矩（跳水收腹加快旋轉）；力矩（torque） |

#### 第五部分：訓練生理學基礎

| 訓練原則 | 說明 |
|---|---|
| 超負荷原則（Overload） | 訓練強度須超過日常水平方能進步 |
| 特異性原則（Specificity） | 訓練方式須配合目標運動（跑步不等於改善游泳） |
| 漸進原則（Progression） | 逐步增加訓練量，避免過度訓練 |
| 可逆性原則（Reversibility） | 停止訓練後，體能改善會逐漸消退（detraining） |
| 個別差異（Individual Differences） | 訓練反應因人而異 |

**能量系統三大重點：**

| 系統 | 燃料 | 持續時間 | 運動例子 |
|---|---|---|---|
| ATP-PC 磷酸肌酸系統 | 磷酸肌酸（PC） | 0–10 秒 | 100m 衝刺、舉重 |
| 乳酸系統（無氧醣解） | 糖原 → 乳酸 | 10 秒–2 分鐘 | 400m、800m |
| 有氧系統 | 糖原/脂肪/蛋白質 | 2 分鐘+ | 馬拉松、長泳 |

### ⭐⭐ 中頻主題（重點補充）

#### 第四部分：體適能與營養

- **體適能五要素**：心肺耐力、肌力、肌耐力、柔韌性、體成分
- **BMI 計算**：BMI = 體重(kg) ÷ 身高²(m²)；正常範圍 18.5–22.9（亞洲標準）
- **營養六大類**：碳水化合物（主要燃料）、蛋白質（肌肉修復）、脂肪（長時間運動）、維生素、礦物質、水
- **運動前中後補水**：運動中每 15–20 分鐘飲 150–250 ml；電解質補充在 60 分鐘以上運動中尤為重要

#### 第六部分：運動創傷

| 創傷類型 | 定義 | 處理 |
|---|---|---|
| 急性創傷 | 突然受力導致（骨折、扭傷、脫臼） | PRICE（保護→休息→冰敷→加壓→抬高） |
| 慢性創傷 | 反覆微創傷積累（肌腱炎、應力性骨折） | 降低訓練量、物理治療 |
| PRICE 原則 | **P**rotect → **R**est → **I**ce → **C**ompression → **E**levation | 急性受傷後首 48 小時適用 |

> **常考問題**：扭傷（Sprain）損傷韌帶；拉傷（Strain）損傷肌肉或肌腱——兩者須分清。

#### 第七部分：運動心理學

| 概念 | 考試重點 |
|---|---|
| 動機（Motivation） | 內在動機（享受）vs 外在動機（獎品）；自我效能（self-efficacy）對表現的影響 |
| 焦慮（Anxiety） | 特質焦慮（長期性格）vs 狀態焦慮（比賽前臨時）；最適喚醒水平（Optimal Arousal） |
| 目標設定（Goal Setting） | SMART 原則（具體/可量度/可達成/相關/有時限） |
| 意象訓練（Mental Imagery） | 運動前在腦海中預演動作，提升技術表現 |
| 歸因（Attribution） | Weiner 四歸因：能力/努力（內部）；任務難度/運氣（外部） |

### ⭐ 低頻主題（選擇性溫習）

#### 第一部分：歷史與發展
- 奧運會歷史：現代奧運 1896 年雅典；殘奧 1960 年羅馬
- 香港體育歷史：殖民地時期到 2020 東京奧運（張家朗劍擊金牌）

#### 第八部分：社會層面
- 商業化對體育運動的影響（贊助、電視轉播）
- 性別平等與殘疾人士參與運動的政策

#### 第九部分：運動管理
- 大型賽事籌辦步驟
- 義工管理與場館管理基本概念

## 三、卷二長答題框架（三條必答）

卷二每題按三個**不同**主題出題，每題分值相同。答題格式須清晰、有結構。

### 萬用長答框架（以生物力學題為例）

**題目例子**：「試分析打乒乓球正手扣殺時涉及的槓桿類型及其力學優勢。」

**答題框架**：

```
1. 指出概念（1–2 分）
   → 人體肘部彎曲運動屬於三類槓桿（Third-class lever）

2. 定義支點、力點、阻力點（3–4 分）
   → 支點：肘關節
   → 力點：肱二頭肌附著點（橈骨近端）
   → 阻力點：手持球拍的手掌（最遠端）

3. 分析特點（3–4 分）
   → 三類槓桿的力點在支點與阻力點之間
   → 機械優勢 < 1（力距小、速度增益大）
   → 適合高速揮拍動作，犧牲力量換取速度

4. 聯繫實際（1–2 分）
   → 乒乓球正手扣殺追求高速旋轉，三類槓桿設計恰好符合此需要
```

### 運動生理題答題要點

**題目例子**：「解釋為何長跑運動員訓練後靜止心率會下降。」

```
1. 點出心輸出量公式：CO = HR × SV
2. 解釋：有氧訓練後心肌肥大（cardiac hypertrophy），每搏輸出量增加
3. 因果：為維持相同心輸出量，靜止心率可下降（心臟更有效率）
4. 數據支持：精英耐力運動員靜止心率可低至 40 bpm
```

### 運動心理題答題要點

**題目例子**：「分析一名運動員在比賽前出現過度焦慮時，教練可採取的心理介入策略。」

```
1. 識別問題（過度喚醒 → 表現下降，倒 U 理論）
2. 策略一：放鬆技巧（深呼吸、漸進式肌肉鬆弛）
3. 策略二：重新框架（認知重組，將焦慮解讀為興奮）
4. 策略三：意象訓練（回憶過往成功比賽經歷，建立自信）
5. 結論：結合認知與身體放鬆策略，令喚醒水平回歸最適水平
```

## 四、卷一 MC 高頻考點速記

| 主題 | 高頻選擇題考點 |
|---|---|
| 關節 | 球窩關節（肩、髖）＝最大活動範圍；鉸鏈關節（肘、膝）＝單軸運動 |
| 肌肉收縮 | 離心收縮（eccentric）＝肌肉在拉長時用力（下樓梯股四頭肌） |
| 心輸出量 | CO = HR × SV；運動時 CO 增加主要靠 HR 增加（短時間）及 SV 增加（訓練後） |
| VO₂max | 有氧能力最重要指標；訓練可提高 5–15% |
| ATP-PC | 最快速但最短暫的能量系統；磷酸肌酸儲量約 8–10 秒 |
| PRICE | 急性創傷首選處理；**不要**在急性期熱敷（加劇腫脹） |
| 槓桿類型 | 二類槓桿（提踵）是人體唯一力學優勢 > 1 的槓桿（阻力點在中間） |
| 穩定性 | 支撐面越大、重心越低、重心越靠近支撐面中心 → 越穩定 |
| SMART | Specific, Measurable, Achievable, Relevant, Time-bound |
| 歸因 | 「努力」= 內部可控；「運氣」= 外部不可控——比賽前正面歸因更有利 |

## 五、常見失分錯誤（7 大陷阱）

| # | 錯誤 | 正確做法 |
|---|---|---|
| 1 | 混淆扭傷（韌帶）與拉傷（肌肉/肌腱） | 扭傷 = Sprain = Ligament；拉傷 = Strain = Muscle/Tendon |
| 2 | 熱敷急性創傷 | 急性期（48 小時內）用冰敷（Ice），慢性期才用熱敷 |
| 3 | 心輸出量公式錯誤 | CO = **HR × SV**，不是 HR + SV |
| 4 | 三類槓桿誤認為一類 | 人體最常見是三類（力點在中間）；一類（支點在中間）最少見 |
| 5 | 卷二長答缺乏「聯繫運動實例」 | 每條長答必須舉具體運動例子（籃球、游泳等），不能只寫理論 |
| 6 | 能量系統混淆（乳酸系統 vs 有氧系統邊界） | 記住：>2 分鐘持續運動 → 進入有氧系統 |
| 7 | 心理題只列概念不解釋機制 | 必須說明「為什麼這個策略有效」，而非只列策略名稱 |

## 常見問題 (FAQ)

**Q: 卷二三條長題是否可以選答？**
A: 不可以。根據 HKEAA 2026 評核大綱，卷二三條長題全部必答，每條來自不同主題，考生無選擇權。

**Q: 體育科是否可以只靠實習試（Paper 3）拿 3 級？**
A: 不太可能。理論試佔 60%，實習試佔 40%。若理論試極差（如 10%），即使實習試滿分（40%），總分仍不足 50%，難以達到 3 級（約需 40–50% 以上）。建議兩者並重。

**Q: 卷一 MC 有負分嗎？**
A: 沒有。DSE 體育科 MC 不設負分，不確定的題目應憑感覺作答，絕不留空。

**Q: 長答題要寫多少字才夠？**
A: 一般建議每條長答寫 300–400 字（約 1.5–2 頁答卷），並輔以圖表（如槓桿圖、心臟循環圖）可加強說服力。不少考生因答案太短（只列出重點名詞）而失分。

**Q: 記不住人體骨骼和肌肉的英文名詞怎麼辦？**
A: DSE 體育科接受中英文作答。如記不住 "rectus femoris"，寫「股直肌」同樣有效。重要的是解釋其功能和位置，不是死背名稱。

**Q: 能量系統的題目常常如何出？**
A: 常見題型是「分析某運動項目（如 400 米跑）主要使用哪個能量系統並說明理由」。答題時必須指出時間（10 秒–2 分鐘→乳酸系統），說明是否有氧，並聯繫實際配速。

**Q: 心理學部分的 Inverted-U Theory 如何答？**
A: 說明過低喚醒（too low arousal）→ 注意力不足，動作遲緩；最適水平（optimal arousal）→ 最佳表現；過高喚醒（too high arousal）→ 焦慮、協調受損。需畫出倒 U 形圖，並舉具體運動例子（如射擊需低喚醒，足球需中高喚醒）。
