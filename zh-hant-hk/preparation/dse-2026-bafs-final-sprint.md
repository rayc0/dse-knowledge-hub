## 一、2026 BAFS 試卷結構一覽

DSE BAFS 2026 全面採用「必修 + 選修」雙軌制，考生選擇**一個選修單元**（會計或商業管理），並於卷二應考該單元的延伸題。

| 試卷 | 範疇 | 時間 | 比重 | 題型 |
|------|------|------|------|------|
| **卷一** | 必修部分（全體考生） | 1 小時 | **25%** | 甲部：24 題選擇題 / 乙部：3 題短答題 |
| **卷二 2A** | 選修：會計 | 2.5 小時 | **75%** | 結構式長題目（4–5 題） |
| **卷二 2B** | 選修：商業管理 | 2.5 小時 | **75%** | 結構式長題目 + 個案分析（4–5 題） |

> **重要：** 卷二選修必須在中四時選定，考生只應考其中一份卷二（2A 或 2B），不可同時應考兩份。

### 2026 重大改動

- **財務報表格式：** 由 2026 起，所有財務報表（損益表、資產負債表、股東權益表）**一律採用垂直式（Vertical Format）**，水平式格式不獲接受，否則扣分。
- **卷一比重大減：** 由舊制約 40% 降至新制 25%，考生必須把更多備試時間投放於卷二。
- **短答題加入：** 舊制卷一純選擇題，新制新增乙部 3 題短答，考核概念應用能力。

## 二、卷一必修部分：三大範疇精華

卷一甲部 24 題選擇題共 24 分，乙部 3 題短答題共 18 分，共 42 分（折合 25%）。考試時間僅 1 小時，必須快、準、穩。

### 2.1 商業環境（Business Environment）

**高頻考點：**

| 主題 | 常見考法 |
|------|---------|
| 企業擁有權類型 | 獨資 / 合夥 / 有限公司三者比較；有限與無限責任 |
| 企業社會責任（CSR）/ ESG | 定義 + 具體措施 + 對持份者的影響 |
| 持份者分析 | 辨認持份者 + 分析利益衝突 |
| 商業道德 | 判斷行為是否道德 + 建議改善措施 |
| 本港營商環境 | 金融中心優勢、GBA 機遇、監管制度 |

**答題貼士：** CSR/ESG 題必須聯繫**具體個案情境**，切勿只背書面定義。「對環境的影響 → 採取了什麼措施 → 對企業聲譽的效益」是標準作答結構。

---

### 2.2 基礎管理（Basic Management）

**高頻考點：**

| 主題 | 常見考法 |
|------|---------|
| 管理功能（POLC） | 計劃、組織、領導、控制各功能的具體應用 |
| 主要商業功能 | 生產、市場推廣、人力資源、財務會計之間的協作 |
| 中小企特點 | 靈活性、資源限制、政府支援措施 |
| 管理風格 | 獨裁式 vs 民主式 vs 放任式的適用情況 |

**答題貼士：** 管理功能題考「應用」多於「背誦」。題目通常給出個案，要求考生指出**哪個管理功能出現問題**，並建議改善。記緊緊扣個案細節。

---

### 2.3 基礎個人理財（Basic Personal Finance）

**高頻考點：**

| 主題 | 常見考法 |
|------|---------|
| 金錢的時間值 | 現值 / 終值計算；複利計算 |
| 消費者信貸 | 信用卡 vs 私人貸款 vs 按揭；年利率（APR）比較 |
| 個人理財規劃 | 收支預算編制；儲蓄 vs 投資取捨 |
| 證券市場投資 | 股票、債券、基金基本概念；風險與回報關係 |

**計算題必記：**
- 複利終值：`FV = PV × (1 + r)ⁿ`
- 現值：`PV = FV ÷ (1 + r)ⁿ`
- 每月還款額：先確認年利率轉月利率（÷12），再代入年金公式

## 三、卷二 2A 選修：會計核心考點

卷二 2A 是「高難度 + 高回報」的選修。2.5 小時需完成 4–5 條結構式長題，每題通常分為多個小問。以下是歷屆出現頻率最高的考點：

### 3.1 財務報表編製（最重要）

**2026 格式必須記住：**

```
損益表（Income Statement）— 垂直式
──────────────────────────────────
收入（Revenue）                  XXX
減：銷售成本（Cost of Sales）    (XXX)
                               ─────
毛利（Gross Profit）             XXX
加：其他收入（Other Income）      XXX
減：各項費用
  行政費用                      (XXX)
  分銷費用                      (XXX)
  財務費用                      (XXX)
                               ─────
除稅前溢利（Profit Before Tax）  XXX
減：所得稅                      (XXX)
                               ─────
本期溢利（Profit for the Period） XXX
```

**常見扣分點：**
- 使用舊式水平格式（T 字形）→ **直接失分**
- 帳目名稱不準確（如用「Debtors」而非「Trade Receivables」）
- 結轉錯誤：毛利計算錯誤影響整條題目

---

### 3.2 試算表、調整分錄（高頻）

**調整項目清單（必須全部掌握）：**

| 調整項目 | 方向 | 常見錯誤 |
|---------|------|---------|
| 預付費用（Prepaid Expenses） | 費用↓，資產↑ | 方向搞錯 |
| 應計費用（Accrued Expenses） | 費用↑，負債↑ | 忘記加入報表 |
| 預收收入（Deferred Income） | 收入↓，負債↑ | 混淆預付與預收 |
| 應收收入（Accrued Income） | 收入↑，資產↑ | 忘記確認 |
| 折舊（Depreciation） | 資產↓，費用↑ | 直線 vs 遞減餘額法混淆 |
| 壞賬（Bad Debts）| 費用↑，應收賬款↓ | 與呆賬準備混淆 |
| 呆賬準備（Allowance for Doubtful Debts） | 注意淨額變動 | 計算增減額 vs 總額之差 |

**折舊計算必記：**
- 直線法：`年折舊 = (成本 - 殘值) ÷ 使用年期`
- 遞減餘額法：`年折舊 = 賬面值 × 折舊率`

---

### 3.3 銀行往來賬核對（Bank Reconciliation）

歷屆必考，2025 及 2026 樣本試卷均有出現。

**核對方向要清晰：**

```
銀行結單結餘（Per Bank Statement）         XXX
加：未兌現入數（Uncredited Deposits）      XXX
減：未兌現支票（Unpresented Cheques）     (XXX)
                                        ─────
調整後銀行結單結餘                         XXX
                                        =====

現金賬結餘（Per Cash Book）               XXX
加：銀行直接入數（Bank Credits not yet in CB）  XXX
減：銀行直接扣數（Bank Debits not yet in CB）  (XXX)
加/減：賬目錯誤                          XXX
                                        ─────
調整後現金賬結餘                          XXX
                                        =====
```

**兩邊調整後結餘必須相等。**

---

### 3.4 合夥人賬目（Partnership Accounts）

**利潤分配次序（固定格式）：**
1. 利息支出（Interest on Drawings）→ 加至淨利潤
2. 資本利息（Interest on Capital）→ 從淨利潤扣除
3. 薪金津貼（Salary Allowance）→ 從淨利潤扣除
4. 剩餘利潤按約定比例分配

> 記住：**先做損益撥充賬（Profit and Loss Appropriation Account），再做合夥人資本賬 / 往來賬。**

**入夥 / 退夥：**
- 重估資產：借資產，貸「商譽」或「重估盈餘」
- 商譽（Goodwill）分攤：按**新比例**貸給新合夥人，按**舊比例**借回

---

### 3.5 股份公司賬目（Company Accounts）

| 考點 | 關鍵公式/格式 |
|------|------------|
| 股本發行（Issue of Shares） | 申請→分配→繳款 各期的分錄 |
| 紅股發行（Bonus Issue） | 借：保留溢利 / 股份溢價，貸：股本 |
| 供股（Rights Issue） | 按現有股東比例，折讓價發行 |
| 財務比率分析 | 流動比率、速動比率、存貨週轉期、應收賬款週轉期、股本回報率 |

**比率分析答題三步：**
1. 計算比率（列出公式 + 代入數值）
2. 比較基準（與行業標準或上年比較）
3. 評論含義（對流動性 / 盈利能力的影響）

## 四、卷二 2B 選修：商業管理核心考點

商管選修側重「個案應用」，考生必須將理論扣緊題目情境，用「3D答題法」——**Define（定義）→ Describe（結合個案）→ Deliver（建議方案）**。

### 4.1 財務管理（Financial Management）— 最高頻

| 考點 | 重要概念 |
|------|---------|
| 融資來源 | 內部（保留溢利、出售資產）vs 外部（銀行貸款、發行股票/債券）；短 / 中 / 長期區分 |
| 預算編制 | 銷售預算 → 生產預算 → 現金預算（Cash Budget）的計算流程 |
| 資本投資評估 | 回收期法（Payback Period）、淨現值法（NPV）、內部回報率（IRR） |
| 營運資金管理 | 現金轉換週期 = 存貨週轉期 + 應收賬款週轉期 - 應付賬款週轉期 |

**現金預算必考格式：**
```
期初現金結餘               XXX
加：現金收入               XXX
                          ───
可用現金                   XXX
減：現金支出              (XXX)
                          ───
期末現金結餘               XXX
                          ===
```

**NPV 計算：**
- `NPV = Σ [現金流入 ÷ (1+折現率)ⁿ] - 初始投資`
- NPV > 0 → 接受；NPV < 0 → 拒絕

---

### 4.2 市場推廣管理（Marketing Management）

**高頻考點：**

| 主題 | 常見考法 |
|------|---------|
| 市場分割（Market Segmentation） | 列出 4 種分割基礎 + 解釋適用情況 |
| 7P 服務市場推廣組合 | 服務業額外 3P：People / Process / Physical Evidence |
| 產品生命週期（PLC） | 各階段的市場推廣策略（如成長期加強促銷、成熟期差異化） |
| 定價策略 | 滲透定價 vs 撇脂定價 vs 競爭定價 vs 心理定價 |
| 消費者行為 | 影響購買決策的因素（文化、社會、個人、心理） |

**個案題答題框架：** 指出目標市場 → 選擇推廣策略 → 解釋選擇原因（結合個案） → 預期效果。

---

### 4.3 人力資源管理（HRM）

**高頻考點：**

| 主題 | 常見考法 |
|------|---------|
| 人力資源規劃 | 人力資源審核（Human Resource Audit）+ 影響需求的因素 |
| 招聘與甄選 | 內部 vs 外部招聘的優缺點；甄選方法比較（面試/測試/評估中心） |
| 培訓與發展 | 在職培訓 vs 離職培訓；培訓對員工及企業的效益 |
| 績效評估 | 目的 + 方法（360度、評分量表、關鍵事件法） |
| 獎酬管理 | 內在報酬（成就感、晉升）vs 外在報酬（薪酬、福利）；激勵理論 |

**Maslow 需求層次論常被考核：** 答題時須由低層次至高層次解釋，並聯繫企業現況。

---

### 4.4 營運管理（Operations Management）

| 考點 | 重要概念 |
|------|---------|
| 生產方式 | 單件生產 / 批量生產 / 流水線生產 / 大量客製化各自的成本、靈活性比較 |
| 質量管理 | TQM 全面質量管理的原則；質量控制 vs 質量保證 |
| 存貨管理 | JIT（零庫存）的優缺點；EOQ 最低訂購量公式 |
| 科技在營運中的應用 | AI、自動化、電子商務對效率與人力的影響 |

## 六、七大常見失分原因

| # | 錯誤 | 後果 | 預防方法 |
|---|------|------|---------|
| 1 | 財務報表用水平（T字形）格式 | 2026年直接扣分 | 只練習垂直式，把T字形格式從記憶中刪除 |
| 2 | 帳目名稱不準確（如 Debtors ≠ Trade Receivables） | 每次扣 0.5–1 分，積少成多 | 製作「正確帳目名稱對照表」，考前重溫 |
| 3 | 折舊方法混淆（直線 vs 遞減） | 計算全錯 | 題目明確指出用哪種，必須細讀題目 |
| 4 | 商管題純背定義，不結合個案 | 0–1 分（滿分通常 4–6 分） | 練習「定義 + 個案細節 + 建議」三段式 |
| 5 | 銀行核對方向錯誤 | 整條核對題失分 | 畫清「銀行結單 vs 現金賬」兩邊框架再填 |
| 6 | 現金預算期初/期末混淆 | 計算連鎖出錯 | 先寫清楚月份標題再填數字 |
| 7 | 時間分配失當（卷一花太多時間） | 卷二來不及完成 | 硬性規定自己 35 分鐘完成甲部選擇題 |

## 七、2026 最後衝刺重點預測

根據歷屆試卷（2012–2025）出現頻率分析：

### 會計（2A）極高概率出現：
- 含多項調整的財務報表（壞賬 + 折舊 + 預付 + 應計，四合一）
- 銀行往來賬核對 + 更新現金賬
- 合夥人利潤分配 + 往來賬 / 資本賬
- 財務比率計算 + 評論（流動性 + 盈利能力各一組）

### 商管（2B）極高概率出現：
- 現金預算編制（含融資需要評估）
- 市場推廣個案（新產品上市 / 進入新市場）
- HRM 個案（員工士氣低落 / 高流失率的改善建議）
- 資本投資評估（Payback Period + NPV 二選一）

## 常見問題（FAQ）

**Q：我選的是會計選修，卷一必修有沒有會計題？**
A：卷一必修部分（Paper 1）主要考商業環境、基礎管理和個人理財，**不包括雙式記賬和財務報表**。這些內容屬於卷二 2A 選修範疇。

**Q：2026 起財務報表只接受垂直式，T 字形格式究竟有幾嚴重？**
A：HKEAA 在官方科目簡介中明確列明 2026 起只接受垂直式（Statement of Financial Position / Income Statement 格式），使用水平式可能導致格式分全失，影響巨大。務必只練習垂直式。

**Q：商管選修的個案題，答幾多點才算足夠？**
A：以 4 分題為例，通常需要 2 個完整論點（各含定義 + 個案應用）。6 分題需 3 個論點。每論點約 3–4 句：定義理論 → 指出個案中的具體情況 → 解釋效果。

**Q：計算機可以用什麼？**
A：可使用 HKEAA 認可的計算機（不可有程式設計功能）。建議用有「財務計算」功能的計算機處理 NPV / IRR 計算，但必須確保已在認可名單內。

**Q：卷二 2A 的比率分析，用哪些比率最安全？**
A：流動比率（Current Ratio）、速動比率（Quick Ratio）、存貨週轉期（Inventory Turnover Days）、應收賬款週轉期（Receivables Days）、淨利率（Net Profit Margin）、股本回報率（Return on Equity）是歷屆最常考的 6 個，確保每個公式都能背出。

**Q：個人理財的計算題公式複雜，考場忘記了怎辦？**
A：試卷不提供公式，必須背熟。建議在試卷開始前先在草稿紙寫下所有公式（FV、PV、月供計算），作為「個人公式表」，再開始作答。

**Q：BAFS 合格率怎樣？Level 3 以上難嗎？**
A：根據 HKEAA 歷年報告，BAFS 的 Level 3+ 比率約 50–55%，Level 4+ 約 30%。只要財務報表格式正確、比率計算無誤，Level 3 是可達到的目標。
