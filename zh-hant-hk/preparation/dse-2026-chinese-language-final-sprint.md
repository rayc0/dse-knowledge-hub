## 二、卷一：閱讀能力卷攻略

### 2.1 甲部：指定文言範文

2026 年 DSE 指定文言範文共 **12 篇**，考生必須熟讀全文，掌握字詞釋義、句子翻譯及主旨思想。

| 篇目 | 作者 | 重點考核範疇 |
|---|---|---|
| 魚我所欲也 | 孟子 | 義利之辨、捨生取義；論說手法 |
| 過秦論（節錄） | 賈誼 | 秦亡原因分析；排比、對比修辭 |
| 出師表 | 諸葛亮 | 忠君愛國、請求北伐；書信體 |
| 師說 | 韓愈 | 尊師重道；駁論手法 |
| 廉頗藺相如列傳（節錄） | 司馬遷 | 人物性格、史傳敘事技巧 |
| 蘭亭集序 | 王羲之 | 生死觀；序文體裁 |
| 始得西山宴遊記 | 柳宗元 | 寄情山水；遊記寫法 |
| 岳陽樓記 | 范仲淹 | 憂國憂民精神；先天下之憂而憂 |
| 逍遙遊（節錄） | 莊子 | 寓言說理；絕對自由觀 |
| 勸學（節錄） | 荀子 | 學習的重要性；比喻論證 |
| 山居秋暝 | 王維 | 空靈意境；動靜結合 |
| 登樓 | 杜甫 | 憂國感時；借景抒情 |

**文言範文複習策略：**
1. **字詞必背**：每篇挑選 10–15 個高頻虛詞（「而」「之」「乎」「焉」「也」）及實詞，背誦其在文中的具體用法。
2. **翻譯練習**：逐句翻譯，特別注意文言特殊句式（倒裝、省略、判斷句）。
3. **主旨與手法**：每篇須掌握：主旨（作者意圖）、論說/寫作手法（排比、對比、比喻、引用）及藝術特色。
4. **甲部配題模式**：通常設字詞解釋（2–4 分）、句子翻譯（4–6 分）、理解及欣賞（8–12 分）三類題型。

### 2.2 乙部：白話文及文言閱讀理解

乙部設 **兩篇文章**（一般為白話文配文言文，或兩篇白話文），涵蓋不同文體：記敘文、議論文、說明文、散文等。

#### 答題技巧

**1. 字詞解釋題（2–4 分）**
- 先看字詞在句中的語境，再推斷詞義。
- 白話文字詞需用「書面語」作解釋，避免口語解釋。
- 答案需具體：「指……的意思」或「形容……的情況」。

**2. 句子理解題（4–8 分）**
- 先找出句子的「字面意思」，再解釋其「引申意思」或在文中的「作用」。
- 使用「作者藉此表達……」的句式作結。

**3. 段落功能／結構分析題（4–8 分）**
- 常見問法：「第X段有何作用？」「以下哪段最能支持作者觀點？」
- 答案框架：「此段（內容）……，從結構上而言，（承接上文／引出下文／點題／作結）……，令文章（效果）……。」

**4. 比較題（6–10 分）**
- 比較兩篇文章的觀點、寫法或主旨。
- 必須按題目要求逐點對比，不可只分別描述兩篇。

**5. 延伸評論題（8–12 分）**
- 考評賞析及個人觀點的表達能力。
- 需引用文章具體內容支持觀點，切忌泛泛而談。

#### 閱讀策略：精讀 vs 略讀

| 策略 | 適用場景 | 操作方法 |
|---|---|---|
| **略讀**（Skimming） | 初次瀏覽文章，掌握整體結構及主旨 | 讀首尾段、各段首句，建立文章「骨架」 |
| **精讀**（Scanning） | 針對具體題目，找答案段落 | 根據題目關鍵詞，定位相關段落再細讀 |
| **標記法** | 閱讀時主動整理文章信息 | 圈出段落主題句、標記重要論點、標示修辭手法 |

**時間分配（卷一，1.5 小時）：**
- 甲部（指定範文）：35–40 分鐘
- 乙部（閱讀理解）：45–50 分鐘
- 留 5 分鐘覆核答案

## 三、卷二：寫作能力卷攻略

### 3.1 甲部：實用文寫作

甲部要求考生在**規定文體及情境**下寫作，常見文體包括：

| 文體 | 常見考核情境 | 格式要點 |
|---|---|---|
| 書信 | 投訴信、建議信、求職信、感謝信 | 稱謂、正文、署名、日期；語氣視收信人而定 |
| 演講稿 | 學校活動、社區論壇、典禮致辭 | 開場白（稱呼聽眾）、主體論點、結語呼籲 |
| 報告 | 調查報告、建議書、計劃書 | 標題、引言、正文分點、結論建議 |
| 新聞稿 | 活動宣傳、機構公告 | 標題、日期地點、正文（5W1H）、聯絡方法 |
| 告示／通告 | 校方通知、社區公告 | 標題（通告）、收件人、正文、發出機構及日期 |
| 電郵 | 業務往來、學術聯絡 | 主旨行、收件人稱謂、正文段落、署名 |

**實用文高分策略：**
1. **格式分（Format Marks）**：實用文首要確保格式正確，即使內容欠佳，格式分仍可拿穩（一般佔 2–4 分）。
2. **語氣配合目的**：投訴信需措辭有禮但堅定；建議書需客觀陳述利弊。
3. **字數控制**：甲部一般要求 400–600 字，切勿過短或過長。
4. **避免口語**：全程使用書面語，「係」換成「是」、「咁」換成「如此/這樣」。

### 3.2 乙部：文學/論說/創意寫作

乙部提供多個題目，考生選**一題**作答，字數一般為 600–800 字。

#### 常見題型及應對策略

**論說文（議論文）**
- 結構：引言（提出論點）→ 正文（正反論證，至少兩個正面論點）→ 反駁（承認反面觀點但指出其局限）→ 結論（重申立場，呼籲或展望）
- 高分要素：論點清晰、論據充分具體、論證邏輯嚴密、語言精準
- 常見失分：論點模糊、例子不具體、結論重複引言而無深化

**記敘文／散文**
- 需有真實感及細節描寫（人物、環境、心理）
- 情節需有轉折，感情需有層次變化
- 語言風格需一致（不可前部分寫散文後部分突然變議論）

**描寫文**
- 善用感官描寫（視覺、聽覺、觸覺、嗅覺）
- 適當運用比喻、擬人等修辭手法
- 注意空間層次感（遠近、高低）

#### 審題四步法

1. **確定文體**：題目是否有明確指定文體？還是開放選擇？
2. **確定主旨**：題目要求表達甚麼核心思想或感受？
3. **確定立場**（論說文）：正面立場或反面立場，選擇自己最有把握論述的一方。
4. **構思材料**：列出 3–4 個具體例子或論據，確認足夠支撐論點才開始寫。

**卷二時間分配（2 小時）：**
- 甲部（實用文）：45 分鐘（含 5 分鐘構思）
- 乙部（創意/論說）：65 分鐘（含 10 分鐘構思）
- 覆核：10 分鐘

## 四、卷三：聆聽及綜合能力卷攻略

### 4.1 甲部：聆聽理解

卷三甲部播放 **兩至三段錄音**，考核考生從口語材料中提取信息、辨別觀點及判斷語氣的能力。

#### 聆聽理解答題策略

**播放前（讀題時間）：**
- 快速瀏覽題目，標記關鍵問題詞（「原因」「態度」「建議」「比較」）
- 預測可能出現的信息類別（數字、地點、人名、觀點）

**播放中：**
- 邊聽邊在試卷空白處速記要點（符號縮寫：↑漲、↓跌、✓同意、✗反對）
- 特別留意錄音中的「但是」「然而」「相反」等轉折詞，這是考點所在
- 注意說話人語氣（肯定/懷疑/讚賞/批評）

**播放後：**
- 整理速記，對照題目選答
- 未確定的題目暫時跳過，待第二次播放時補回

#### 常見考核重點

| 題型 | 考核能力 | 答題要點 |
|---|---|---|
| 辨別正誤（T/F） | 細節提取 | 嚴格按照錄音內容，不可加入自己的推斷 |
| 選擇題 | 主旨理解 | 選項往往相近，需找最能概括錄音要點者 |
| 填充題 | 具體信息提取 | 保持錄音中的措辭，避免自行改寫 |
| 短答題 | 理解及分析 | 用完整句子作答，引用錄音內容加以解釋 |
| 記錄表格 | 整理信息 | 按表格欄目分類填寫，注意格式要求 |

### 4.2 乙部：綜合能力

綜合能力部分要求考生閱讀 **多份文字資料**（一般 3–4 份），從中整合信息，以特定身份完成寫作任務（如：撰寫建議書、演講稿、報告、討論摘要等）。

#### 綜合能力解題框架

**步驟一：明確任務（2 分鐘）**
- 確認寫作身份（你是誰）、寫作對象（寫給誰看）、寫作目的（要達成甚麼效果）

**步驟二：瀏覽資料（10–15 分鐘）**
- 每份資料標記「支持」或「反對」特定立場的信息
- 記錄每份資料的核心觀點（一兩句）

**步驟三：規劃結構（5 分鐘）**
- 決定引用哪些資料作為支持論點的依據
- 規劃段落結構（引言＋論點一＋論點二＋論點三＋結論）

**步驟四：寫作（35–45 分鐘）**
- 引用資料時需加以解釋分析，不可只是照抄資料原文
- 整合不同資料的觀點，而非逐一描述每份資料
- 保持統一的寫作角度（不可前後矛盾）

**卷三時間分配（約 2 小時）：**
- 甲部聆聽（錄音及作答）：45–50 分鐘
- 乙部綜合能力（閱讀及寫作）：65–70 分鐘
- 覆核：5 分鐘

## 七、常見失分陷阱

### 卷一常見錯誤
1. **答案不在文章中**：考生加入自己的想法而非引用文本，閱讀理解須嚴格按文章作答。
2. **翻譯不夠準確**：文言翻譯要字字對應，切忌「意譯過度」。
3. **未回答「效果」**：手法題只說「用了排比」卻未說明「有甚麼效果」，必然失分。
4. **字數不足**：較大分值的評論題（8–12 分）需要充分發揮，不可草草了事。

### 卷二常見錯誤
1. **文體混淆**：議論文出現大量情節描述；記敘文夾雜大篇議論，應保持文體一致。
2. **例子太抽象**：「很多人認為……」「有調查顯示……」卻無具體數字或事例，說服力不足。
3. **結構鬆散**：無明確段落分題，一大段文字到底，評分者難以追蹤論點。
4. **口語入文**：「其實」「不過」「所以」等白話連接詞在書面文章中需換成「事實上」「然而」「因此」。

### 卷三常見錯誤
1. **照抄資料原文**：綜合卷若只是搬錄資料，屬「未能整合」，無法得高分。
2. **忽略寫作身份**：未能維持題目要求的身份及語氣（如應以學生身份寫信卻用官方語調）。
3. **遺漏聆聽細節**：聆聽題答題粗疏，只寫大意而遺漏具體要求的細節信息。

## 九、臨場策略

### 卷一考場策略
- **先甲後乙**：先完成指定範文（甲部），因為範文熟悉，心理壓力較小，有助建立信心。
- **標記返回法**：不確定的題目在題目旁打「？」，先答確定題目，時間許可再回頭處理。
- **字數估算**：短答題 4–6 分對應約 60–80 字；8–12 分對應約 150–200 字。

### 卷二考場策略
- **選題看三遍**：乙部選題前，需仔細閱讀所有選項，評估每題自己能提供多少具體內容，再選最有把握的題目。
- **先列提綱再寫作**：用 5–10 分鐘在草稿紙列出段落提綱，確保結構完整，避免寫到一半不知如何繼續。
- **保留修改時間**：最後 10 分鐘必須用於修改，重點查看：文體格式、口語詞彙、標點符號。

### 卷三考場策略
- **聆聽前讀清題目**：每次播放前善用讀題時間，圈出關鍵問題詞，做好心理準備。
- **綜合卷整合導向**：下筆前用 3–5 分鐘製作「資料地圖」，將各份資料的要點歸類為「論點A支持者」「論點B支持者」，寫作時橫向整合而非逐份描述。
- **身份代入**：始終以題目要求的身份（如：學生會主席、社區義工）落筆，語氣措辭需符合身份設定。

## 重點公式與框架速查

```
【論說文結構框架】
引言：點明議題 + 表明立場（約 100 字）
論點一：主題句 + 解釋 + 具體例子 + 小結（約 150–200 字）
論點二：主題句 + 解釋 + 具體例子 + 小結（約 150–200 字）
反駁：承認對立面觀點 + 指出其局限性（約 100–150 字）
結論：重申立場 + 展望/呼籲（約 100 字）

【實用文格式：書信】
[日期]
[收信人姓名/職銜]
[稱謂]：（如：敬啟者、XXX校長鈞鑒）
[正文—分段論述]
此致（或：謹此，祈請考慮）
[署名]
[身份]

【閱讀手法題答題框架】
「作者用了[手法]，寫出了[內容/對象]的[特點]，
令讀者感受到[效果]，強調了[主旨]。」

【綜合能力整合框架】
任務確認（身份+對象+目的）
→ 整合資料（橫向歸類，非逐份描述）
→ 寫作（格式+立場+引用+分析）
→ 覆核（文體語氣是否一致）
```

## 常見問題（FAQ）

**Q1：卷二乙部論說文與議論文有甚麼分別？**
A：兩者基本相同。議論文強調邏輯論證；論說文則可帶有較強的說服或感召目的。DSE 中通常統稱「論說文」，結構相近，均需清晰立場及充分論據。

**Q2：指定範文會整篇考還是節錄？**
A：DSE 甲部指定文言題通常以**節錄段落**形式出現，但問題可能涉及全篇主旨及人物形象，故需熟讀全文，不可只讀節錄部分。

**Q3：作文有沒有字數要求？**
A：DSE 官方不設硬性字數下限，但評卷員會視乎字數是否足夠表達完整論點。一般建議甲部（實用文）400–600 字，乙部（論說/記敘文）600–800 字，少於 400 字通常難以取得 4 級或以上。

**Q4：卷三綜合能力可以照抄資料嗎？**
A：不可以。評分準則明確要求考生「整合」而非「抄錄」資料。引用資料時需用自己的語言概括及解釋，並說明與論點的關係。

**Q5：口試（卷四）現在可以做甚麼準備？**
A：口試已於校本進行，成績已記錄。考生無需為卷四做額外準備，集中精力於卷一至卷三即可。

**Q6：中文科評分是「全卷」還是「逐題」？**
A：逐題評分。每一題都有獨立分值，即使某題表現較差，也不影響其他題目的得分，考生應盡力完成每道題目。
