# DSE考試完全指南（香港本地學生版）

香港中學文憑考試（HKDSE，下稱DSE）自2012年起取代舊制的香港中學會考（HKCEE）及香港高級程度會考（HKAL），成為香港唯一的高中公開考試。無論你是正在備考的中六生，還是家長、補習老師，了解DSE的評分機制都是制定應試策略的第一步。

## DSE的等級制度

DSE採用**等級制**（Level-based grading）而非百分制，每科成績分為以下等級：

| 等級 | 說明 |
|------|------|
| 5** | 最高成績，卓越表現 |
| 5* | 優異表現 |
| 5 | 優良 |
| 4 | 良好 |
| 3 | 達標 |
| 2 | 基本達標 |
| 1 | 未達標 |
| U | 不予評級（Unclassified） |

值得注意的是，5**並非「100分」的概念。考評局（HKEAA）每年根據考生整體表現劃定各等級的分數界線（cut score），即使同樣是5**，每年的實際得分下限可能略有不同。一般而言，取得5**的考生屬該科前約1至2%的表現。

## 核心科目與選修科目

DSE科目分為**核心科目**（Compulsory subjects）和**選修科目**（Elective subjects）：

**四大核心科目：**
- 中國語文（Chinese Language）
- 英國語文（English Language）
- 數學（Mathematics）——分必修部分（Compulsory Part）及延伸部分（M1/M2）
- 通識教育（Liberal Studies，2023年起改革為「公民與社會發展科」）

**選修科目：** 考生通常修讀2至3科，涵蓋人文、理科、商科及應用學習（ApL）等範疇。

## 分數如何計算？

DSE的分數計算主要用於JUPAS（大學聯合招生辦法）的評核。各大學會指定計分方式，常見的有：

**「422」最低要求：**
大部分本地大學要求考生在四大核心科目中達到「Level 3, 3, 2, 2」，即中文及英文各達Level 3，其餘兩科達Level 2，以取得基本入學資格。

**大學計分公式（以五科為例）：**
大多數學院取最佳五科（Best 5）成績計算入學分數。各等級的換算分如下：

| 等級 | 計算分數 |
|------|----------|
| 5** | 7分 |
| 5* | 6分 |
| 5 | 5分 |
| 4 | 4分 |
| 3 | 3分 |
| 2 | 2分 |
| 1 | 1分 |

以此計算，最高可能分數為35分（五科全部5**）。競爭激烈的醫學科或法律科，入學要求往往達30分或以上。

## 5**的實際意義

「5**」是DSE最高等級，代表考生的成績處於全港考生的頂尖水平。以數學科為例，一般要答對卷二（MC部分）約90%以上題目，並在卷一的長題（Long Questions）取得幾乎滿分，才有機會達到5**。

每年取得5**的人數因科目而異：
- **英文科**：5**人數相對較多，約佔全港考生2至3%
- **中文科**：5**競爭激烈，約佔1至1.5%
- **數學必修部分**：約佔2至3%
- **延伸部分M2**：5**比例相對寬鬆，約佔5至8%（因修讀人數較少）

取得多科5**的考生有機會獲得「狀元」（Top Scholar）媒體報道，但實際上DSE並無官方「狀元」制度，僅屬坊間稱謂。

## 成績公布與覆核機制

DSE成績通常於每年7月中旬公布。考生若對成績有異議，可在指定時間內申請**覆核（Rechecking）**或**覆卷（Re-marking）**：

- **覆核**：由考評局重新核對分數計算有否錯誤，費用較低。
- **覆卷**：由另一名閱卷員重新批改，有機會改變等級，但費用較高。

覆卷後成績可上調或下調，考生在申請前應審慎考慮。

## JUPAS Band A至E的含義

JUPAS以「Band」制度反映某個課程對考生的取錄意向：

- **Band A**：成績達到或超過該課程往年的中位數，錄取機會最高
- **Band B**：成績略低於中位數，仍有合理錄取機會
- **Band C**：成績低於中位數一個等級左右，機會較低
- **Band D**：成績明顯偏低，建議作備用考慮
- **Band E**：成績遠低於往年收生要求，錄取機會極低

建議考生在JUPAS填報時，確保至少有2至3個課程屬於Band A，以穩固保底選擇。

## 常見誤解澄清

1. **誤解：所有科目成績都會計入JUPAS分數**
   事實：大學通常取最佳五科（部分課程取最佳六科），其餘科目不計入主分，但可能作額外要求。

2. **誤解：通識/公民科不重要**
   事實：核心科目必須達到最低等級要求，否則即使選修科目成績再高，亦可能無緣入讀。

3. **誤解：5**代表滿分**
   事實：5**只代表達到頂尖等級標準，實際得分並非必然是100%。
